细胞信号传导

Cell Signal Transduction - 考研专业课复习笔记

一、细胞信号传导概述

1.1 基本概念

细胞信号传导（Cell Signal Transduction）是指细胞感受、传递和响应环境信息的过程，是细胞与细胞之间、细胞与环境之间进行信息交流的基本方式。这一过程使得多细胞生物能够协调各种生理活动，维持内环境稳态。

1.2 细胞通信的类型

  • 内分泌信号传递（Endocrine Signaling）：信号分子通过血液循环传递到远距离的靶细胞
  • 旁分泌信号传递（Paracrine Signaling）：信号分子作用于邻近细胞
  • 自分泌信号传递（Autocrine Signaling）：细胞产生的信号分子作用于自身
  • 突触信号传递（Synaptic Signaling）：神经元之间的特殊信号传递方式
  • 接触依赖性信号传递（Contact-dependent Signaling）：细胞表面分子直接接触传递信号

1.3 信号转导的基本过程

  1. 信号识别：受体蛋白识别并结合特定的信号分子
  2. 信号转换：受体构象改变，激活胞内信号转导级联反应
  3. 信号放大：通过酶级联反应实现信号的放大效应
  4. 细胞响应：激活转录因子，调节基因表达或酶活性
  5. 信号终止：通过负反馈机制终止信号传递

二、信号分子与受体

2.1 信号分子的分类

按化学性质分类：

  • 脂溶性信号分子：类固醇激素、甲状腺激素等，可穿透细胞膜
  • 水溶性信号分子：蛋白质激素、神经递质等，需要膜受体介导
  • 气体信号分子：NO、CO、H₂S等，可自由穿透膜结构

2.2 受体的类型和功能

按定位分类：

  • 细胞表面受体：
    • G蛋白偶联受体（GPCR）
    • 受体酪氨酸激酶（RTK）
    • 离子通道型受体
    • 细胞因子受体
  • 胞内受体：
    • 核受体超家族
    • 胞质受体

三、第二信使系统

3.1 cAMP信号系统

环腺苷酸（cAMP）是最重要的第二信使之一。当G蛋白偶联受体激活Gₛ蛋白时，激活腺苷酸环化酶，催化ATP生成cAMP。

cAMP-PKA信号通路：

  1. cAMP浓度升高
  2. cAMP结合PKA调节亚基
  3. PKA催化亚基释放并激活
  4. 磷酸化下游靶蛋白
  5. 调节酶活性和基因转录

3.2 cGMP信号系统

环鸟苷酸（cGMP）主要通过激活蛋白激酶G（PKG）发挥作用，在血管舒张、视觉传导等过程中起重要作用。

3.3 钙离子信号系统

Ca²⁺是最重要的第二信使之一，胞内Ca²⁺浓度的变化可以激活多种Ca²⁺结合蛋白。

钙信号的特点：

  • 钙火花：局部Ca²⁺浓度的瞬时升高
  • 钙波：Ca²⁺信号在细胞内的传播
  • 钙振荡：Ca²⁺浓度的周期性变化

钙调蛋白（Calmodulin, CaM）：

钙调蛋白是细胞内最重要的Ca²⁺结合蛋白，可调节多种酶的活性：

  • 钙调蛋白依赖性蛋白激酶（CaMK）
  • 钙调蛋白依赖性磷酸酶（Calcineurin）
  • 腺苷酸环化酶
  • 磷酸二酯酶

四、G蛋白偶联受体信号转导

4.1 GPCR的结构特征

G蛋白偶联受体具有七次跨膜结构域，是细胞表面受体中数量最多的一类。

4.2 G蛋白的分类和功能

  • Gₛ蛋白：激活腺苷酸环化酶，升高cAMP水平
  • Gᵢ/Gₒ蛋白：抑制腺苷酸环化酶，降低cAMP水平
  • Gq/G₁₁蛋白：激活磷脂酶C，产生IP₃和DAG
  • G₁₂/G₁₃蛋白：激活Rho信号通路

4.3 磷脂酶C介导的信号通路

当Gq蛋白激活磷脂酶C（PLC）时，PLC催化PIP₂分解产生两个第二信使：

  • IP₃（肌醇1,4,5-三磷酸）：激活内质网Ca²⁺释放
  • DAG（二酰甘油）：激活蛋白激酶C（PKC）

PKC信号通路的生物学效应：

  • 调节基因转录
  • 影响细胞增殖和分化
  • 调节离子通道活性
  • 参与学习记忆过程

五、酶联受体信号转导

5.1 受体酪氨酸激酶（RTK）

RTK是一类重要的细胞表面受体，配体结合后发生二聚化和自磷酸化。

RTK的激活机制：

  1. 配体结合导致受体二聚化
  2. 激酶结构域相互磷酸化
  3. 磷酸化酪氨酸作为停靠位点
  4. 招募含SH2结构域的信号蛋白

5.2 Ras-MAPK信号通路

这是RTK下游最重要的信号通路之一，参与细胞增殖、分化和凋亡的调节。

信号传递过程：

  1. RTK激活 → Grb2-SOS复合物招募
  2. SOS激活Ras → Ras-GTP形成
  3. Ras激活Raf → MAPK级联启动
  4. Raf → MEK → ERK → 磷酸化级联
  5. ERK入核 → 磷酸化转录因子

5.3 PI3K-Akt信号通路

PI3K-Akt通路是调节细胞存活、代谢和增殖的关键信号通路。

信号转导机制：

  1. RTK激活招募PI3K
  2. PI3K催化PIP₂生成PIP₃
  3. PIP₃招募PDK1和Akt
  4. PDK1磷酸化激活Akt
  5. Akt磷酸化多个下游靶点

Akt的主要靶蛋白：

  • mTOR：调节蛋白质合成和细胞生长
  • Bad：抑制细胞凋亡
  • GSK3β：调节糖原合成
  • FoxO：调节基因转录

六、细胞因子受体信号转导

6.1 JAK-STAT信号通路

JAK-STAT通路是细胞因子受体介导的主要信号转导途径，参与免疫反应、细胞分化等过程。

信号转导过程：

  1. 细胞因子结合 → 受体二聚化
  2. JAK激活 → 相互磷酸化
  3. 受体磷酸化 → STAT招募
  4. STAT磷酸化 → 二聚化
  5. STAT入核 → 调节基因转录

6.2 TGF-β/Smad信号通路

TGF-β超家族通过Smad蛋白介导信号转导，调节细胞增殖、分化、凋亡和胚胎发育。

Smad蛋白的分类：

  • R-Smad（Smad1,2,3,5,8）：受体调节的Smad
  • Co-Smad（Smad4）：共同介导的Smad
  • I-Smad（Smad6,7）：抑制性Smad

七、其他重要信号通路

7.1 Wnt信号通路

Wnt信号通路在胚胎发育、干细胞维持和肿瘤发生中起重要作用。

经典Wnt/β-catenin通路：

  • 无Wnt信号时：β-catenin被降解复合物磷酸化降解
  • 有Wnt信号时：降解复合物失活，β-catenin积累入核

7.2 Hedgehog信号通路

Hedgehog通路在胚胎发育模式形成和成体组织稳态维持中发挥关键作用。

7.3 Notch信号通路

Notch通路是细胞间直接接触的信号传递方式，调节细胞命运决定和分化。

7.4 NF-κB信号通路

NF-κB是重要的转录因子，在炎症反应、免疫应答和细胞存活中起关键作用。

十、深度思考题

1. 试分析G蛋白偶联受体信号转导的分子机制，并阐述其在疾病发生中的作用。

参考答案：
  1. GPCR信号转导的分子机制：GPCR具有七次跨膜结构域，配体结合后发生构象改变，激活与其偶联的G蛋白。G蛋白由α、β、γ三个亚基组成，激活后α亚基与GTP结合并解离，进而激活下游效应器如腺苷酸环化酶、磷脂酶C等，产生第二信使cAMP、IP₃、DAG等，最终调节细胞功能。
  2. 信号放大效应：一个激活的GPCR可以激活多个G蛋白分子，每个G蛋白又可激活多个效应器分子，通过级联放大实现信号的显著放大，使细胞能够对微弱的外界刺激产生强烈响应。
  3. 信号特异性：不同的GPCR偶联不同类型的G蛋白（Gs、Gi/o、Gq/11等），激活不同的信号通路，产生特异性的生物学效应，体现了信号转导的精确性和多样性。
  4. 疾病相关性：GPCR功能异常与多种疾病相关，如视紫红质突变导致视网膜色素变性、甲状腺刺激激素受体突变引起甲状腺功能异常、多巴胺受体异常与帕金森病和精神分裂症相关。
  5. 药物靶点价值：约40%的现代药物以GPCR为靶点，如β受体阻滞剂治疗高血压、抗组胺药治疗过敏反应等，体现了GPCR在药物开发中的重要地位。

2. 比较分析RTK-Ras-MAPK通路和PI3K-Akt通路的异同点，并说明它们在肿瘤发生中的作用。

参考答案：
  1. 相同点：两条通路都由RTK激活，都参与细胞增殖、分化和存活的调节，都是重要的原癌基因通路，在多种肿瘤中异常激活。两条通路都涉及蛋白质磷酸化级联反应，都能调节转录因子活性。
  2. 不同点：Ras-MAPK通路主要调节细胞增殖和分化，通过ERK磷酸化转录因子如c-Fos、c-Jun等；而PI3K-Akt通路主要调节细胞存活和代谢，通过Akt磷酸化Bad、FoxO、mTOR等蛋白。前者更多影响基因转录，后者更多影响蛋白质功能。
  3. 信号传递机制差异：Ras-MAPK通路通过小G蛋白Ras激活，经历三级激酶级联（Raf-MEK-ERK）；PI3K-Akt通路通过脂质激酶PI3K产生PIP₃，招募并激活Akt。
  4. 肿瘤发生中的作用：Ras-MAPK通路异常激活促进细胞过度增殖，Ras基因突变见于30%的人类肿瘤；PI3K-Akt通路异常激活抑制细胞凋亡，促进肿瘤细胞存活，PI3K和PTEN突变在多种肿瘤中常见。
  5. 治疗靶点：两条通路的关键分子都是肿瘤治疗的重要靶点，如MEK抑制剂、PI3K抑制剂、mTOR抑制剂等已用于临床肿瘤治疗。

3. 阐述钙离子作为第二信使的特点，并分析钙信号在细胞生理功能中的重要作用。

参考答案：
  1. 钙离子的独特性质：Ca²⁺是最重要的第二信使之一，具有浓度梯度大（胞外~1mM，胞内~100nM）、结合蛋白多样、信号快速可逆等特点。Ca²⁺可以产生钙火花、钙波、钙振荡等复杂的时空信号模式。
  2. 钙信号的产生机制：主要通过两个途径：一是IP₃激活内质网IP₃受体释放储存的Ca²⁺；二是膜去极化或配体门控激活质膜Ca²⁺通道。钙诱导钙释放（CICR）机制可以放大钙信号。
  3. 钙结合蛋白的作用：钙调蛋白是最重要的Ca²⁺结合蛋白，结合Ca²⁺后构象改变，激活CaMK、钙调神经磷酸酶等多种酶。其他Ca²⁺结合蛋白如troponin C、parvalbumin等在特定组织中发挥作用。
  4. 生理功能的多样性：肌肉收缩（激活肌钙蛋白）、神经传递（触发囊泡释放）、基因转录调节（激活CREB等转录因子）、细胞分裂（调节纺锤体形成）、细胞凋亡（激活钙依赖性蛋白酶）等。
  5. 钙稳态的维持：细胞通过Ca²⁺-ATPase、Na⁺/Ca²⁺交换体等机制维持胞内Ca²⁺稳态。钙稳态失调与多种疾病相关，如心律失常、神经退行性疾病、肌肉疾病等。

4. 分析JAK-STAT信号通路的调节机制，并讨论其在免疫反应中的重要性。

参考答案：
  1. JAK-STAT通路的基本机制：细胞因子结合受体导致受体二聚化，招募并激活JAK激酶，JAK相互磷酸化激活，进而磷酸化受体酪氨酸残基，为STAT蛋白提供停靠位点。STAT被磷酸化后二聚化并转位入核，调节靶基因转录。
  2. 多层次调节机制：正调节包括细胞因子刺激、JAK激活、STAT磷酸化等；负调节包括SOCS蛋白抑制JAK活性、蛋白酪氨酸磷酸酶去磷酸化STAT、PIAS蛋白抑制STAT转录活性等，形成精密的调控网络。
  3. 信号特异性：不同的JAK（JAK1-3、TYK2）和STAT（STAT1-6）组合产生特异性信号，如IFN-γ激活JAK1/JAK2-STAT1，IL-4激活JAK1/JAK3-STAT6，实现对不同细胞因子的特异性响应。
  4. 免疫反应中的关键作用：调节T细胞分化（Th1、Th2、Th17、Treg）、B细胞活化和抗体产生、巨噬细胞极化、NK细胞功能、干扰素抗病毒反应等，是适应性和固有免疫的核心调节通路。
  5. 疾病相关性和治疗意义：JAK-STAT通路异常与自身免疫病、免疫缺陷、血液肿瘤等密切相关。JAK抑制剂如托法替尼已用于类风湿关节炎治疗，体现了该通路的重要临床价值。

5. 论述信号转导网络的整合特性，并分析信号通路间的相互作用对细胞命运决定的影响。

参考答案：
  1. 信号网络的复杂性：细胞内存在数百条信号通路，它们不是独立运行的，而是形成复杂的相互作用网络。同一信号分子可激活多条通路，同一通路可被多种信号调节，形成信号的汇聚和分散。
  2. 信号整合的分子基础：共同的信号分子（如Ca²⁺、cAMP）、蛋白质间相互作用（如SH2、SH3结构域介导）、转录因子的协同作用、microRNA的调节等，使不同通路在分子水平上相互影响。
  3. 时空动态调节：不同信号通路具有不同的激活动力学和空间分布，通过时间和空间的精确调控实现信号整合。如ERK的瞬时激活促进增殖，持续激活促进分化。
  4. 细胞命运决定的机制：细胞增殖、分化、凋亡等命运决定需要多个信号通路的协调作用。如神经干细胞分化需要Notch（维持干性）、Wnt（促进增殖）、BMP（促进分化）等通路的平衡。
  5. 病理意义和治疗启示：信号网络失衡是疾病发生的重要机制，如肿瘤中多条促增殖通路激活而促凋亡通路失活。理解信号网络有助于开发多靶点治疗策略，提高治疗效果并减少耐药性。
