细胞核与染色质

考研专业课复习笔记 | 基于《细胞生物学》第五版（丁明孝主编）

核心知识点

一、核被膜（Nuclear Envelope）

1. 核被膜的结构组成

核被膜是包围细胞核的双分子膜结构，由外核膜和内核膜组成。外核膜与内质网相连，表面附着核糖体；内核膜光滑，与核纤层紧密结合。两膜之间的空隙称为核周隙（perinuclear space），宽度约20-40nm。

2. 核孔复合体（Nuclear Pore Complex, NPC）

核孔复合体是核被膜上的大型蛋白质复合物，直径约120nm，由约30种不同的核孔蛋白（nucleoporins, Nups）组成。核孔复合体具有八重对称结构，包括：

  • 胞质纤维：延伸至胞质侧
  • 核篮：延伸至核质侧
  • 中央通道：允许物质通过的孔道
  • 中央栓：调节通道开闭
重要考点：核孔复合体的主动运输过程需要消耗GTP，能够识别携带核定位信号（NLS）的蛋白质，实现选择性运输。

3. 核纤层（Nuclear Lamina）

核纤层位于内核膜下方，由核纤层蛋白（lamins）构成的纤维网络结构。核纤层蛋白属于中间丝蛋白家族，主要功能包括：

  • 维持核形态和结构稳定性
  • 参与DNA复制和转录调控
  • 在细胞分裂时解聚和重组
  • 与染色质相互作用

二、染色质（Chromatin）

1. 染色质的基本组成

染色质是细胞核内DNA与蛋白质的复合物，主要成分包括：

  • DNA：携带遗传信息的双螺旋分子
  • 组蛋白：富含碱性氨基酸的结构蛋白
  • 非组蛋白：参与基因调控的各种蛋白质
  • 少量RNA：主要是转录产物

2. 基因组DNA的分类

根据序列特征，真核生物基因组DNA可分为：

  • 单一序列（Unique Sequence）：单拷贝或少数拷贝，多为编码基因
  • 中度重复序列：重复次数10¹-10⁵，包括rRNA、tRNA、组蛋白基因等
  • 高度重复序列：重复次数>10⁵，主要分布在端粒、着丝粒区域

3. 组蛋白的类型与特点

组蛋白是富含精氨酸和赖氨酸的碱性蛋白质，等电点在pH 10.0以上。主要包括五种类型：

  • H1：连接组蛋白，分子量约21kDa
  • H2A：核小体组蛋白，分子量约14kDa
  • H2B：核小体组蛋白，分子量约13.8kDa
  • H3：核小体组蛋白，分子量约15.4kDa
  • H4：核小体组蛋白，分子量约11.3kDa

4. 核小体（Nucleosome）结构

核小体是染色质的基本结构单位，包含：

  • 核心颗粒：由H2A、H2B、H3、H4各2分子组成的八聚体
  • DNA缠绕：146bp DNA在八聚体上缠绕1.65圈
  • 连接DNA：相邻核小体间的DNA片段（约20-80bp）
  • H1组蛋白：结合在连接DNA上，稳定高级结构

三、染色质的组装与类型

1. 染色质的多级组装

染色质具有多级组装结构：

  1. 一级结构：DNA双螺旋（2nm）
  2. 二级结构：核小体串珠状结构（11nm）
  3. 三级结构：螺线管结构（30nm纤维）
  4. 四级结构：高度压缩的分裂期染色体（700nm）

2. 染色质的类型

根据压缩程度和转录活性，染色质分为：

  • 常染色质（Euchromatin）：
    • 结构疏松，转录活跃
    • 富含转录活性基因
    • 组蛋白修饰多样
  • 异染色质（Heterochromatin）：
    • 结构致密，转录沉默
    • 分为组成性异染色质和兼性异染色质
    • 富含重复序列

四、染色体的功能元件

1. 复制起点（Origin of Replication）

真核细胞染色体含有多个复制起点，特征序列为富含A-T的保守序列，长度约11-14bp。复制起点的激活受到严格调控，确保DNA只复制一次。

2. 着丝粒（Centromere）

着丝粒是姐妹染色单体连接的区域，包含两个核心序列：

  • 80-90bp的A-T富集区
  • 11bp高度保守序列：TGATTTCCGAA

着丝粒结合着丝粒蛋白形成动粒，是纺锤体微管附着的位点。

3. 端粒（Telomere）

端粒位于染色体末端，由富含TG的重复序列组成。人类端粒序列为(TTAGGG)n，具有以下功能：

  • 保护染色体末端免受降解
  • 防止染色体融合
  • 解决DNA复制的末端问题
  • 与细胞衰老密切相关

五、核仁与核体

1. 核仁（Nucleolus）结构与功能

核仁是细胞核内最显著的结构，主要功能是rRNA的转录、加工和核糖体亚基的组装。核仁包含三个区域：

  • 纤维中心（FC）：含有rDNA和RNA聚合酶I
  • 致密纤维组分（DFC）：rRNA转录和早期加工场所
  • 颗粒组分（GC）：核糖体亚基组装的后期阶段

2. 核体（Nuclear Body）

核体是细胞核内的非膜结构域，包括：

  • Cajal体：参与snRNA和snoRNA的修饰
  • PML体：参与转录调控和DNA修复
  • 剪接体：mRNA前体的剪接场所

六、核基质与核骨架

1. 核基质（Nuclear Matrix）

核基质是细胞核内除核被膜、核纤层、染色质和核仁外的网架结构体系，主要成分包括：

  • 纤维蛋白网络
  • 各种酶类
  • 转录因子
  • RNA加工因子

2. 核骨架的功能

  • 维持核形态和内部结构
  • 提供DNA复制和转录的支架
  • 参与基因表达调控
  • 组织核内各种生化反应

七、核质运输

1. 亲核蛋白的核输入

亲核蛋白通过以下步骤进入细胞核：

  1. 识别核定位信号（NLS）
  2. 与输入蛋白（importin）结合
  3. 通过核孔复合体
  4. Ran-GTP水解提供能量
  5. 在核内释放货物蛋白

2. RNA的核输出

成熟的mRNA、tRNA、rRNA通过输出蛋白（exportin）介导的机制输出到胞质，过程同样需要Ran-GTP参与。

名词解释库

细胞核（Cell Nucleus）
真核细胞内最大、最重要的细胞器，被核被膜包围，含有细胞的遗传物质DNA，是细胞遗传和代谢的控制中心。
染色质（Chromatin）
间期细胞核中由DNA和组蛋白构成的能被碱性染料着色的物质，是遗传信息的载体。
染色体（Chromosome）
细胞分裂期染色质高度螺旋、折叠、缩短变粗后形成的条状结构，确保遗传物质准确分配。
核被膜（Nuclear Envelope）
包围细胞核的双分子膜结构，由内外两层膜组成，具有选择透过性，控制核质间物质交换。
核孔复合体（Nuclear Pore Complex, NPC）
核被膜上的大型蛋白质复合物，直径约120nm，介导核质间大分子物质的选择性运输。
核纤层（Nuclear Lamina）
位于内核膜下方由核纤层蛋白构成的纤维网络结构，维持核形态和结构稳定性。
组蛋白（Histone）
富含精氨酸和赖氨酸的碱性蛋白质，是染色质的主要结构蛋白，包括H1、H2A、H2B、H3、H4五种。
非组蛋白（Non-histone Protein）
细胞核中除组蛋白外的所有蛋白质，种类多样，主要参与基因表达调控和DNA代谢。
核小体（Nucleosome）
染色质的基本结构单位，由组蛋白八聚体和缠绕其上的146bp DNA组成。
常染色质（Euchromatin）
结构疏松、转录活跃的染色质区域，富含活性基因，在间期细胞核中呈浅染状态。
异染色质（Heterochromatin）
结构致密、转录沉默的染色质区域，分为组成性和兼性两类，在间期细胞核中呈深染状态。
着丝粒（Centromere）
姐妹染色单体连接的区域，含有特殊DNA序列，是纺锤体微管附着的位点。
端粒（Telomere）
位于染色体末端的特殊DNA-蛋白质结构，由富含TG的重复序列组成，保护染色体稳定性。
核仁（Nucleolus）
细胞核内最显著的结构，是rRNA转录、加工和核糖体亚基组装的场所。
核基质（Nuclear Matrix）
细胞核内除核被膜、核纤层、染色质和核仁外的网架结构体系，参与基因表达调控。
核定位信号（Nuclear Localization Signal, NLS）
蛋白质中的特定氨基酸序列，通常富含碱性氨基酸，指导蛋白质进入细胞核。
输入蛋白（Importin）
介导蛋白质从胞质向核内运输的载体蛋白，识别核定位信号并通过核孔复合体。
Ran-GTP系统
调控核质运输方向性的GTP酶系统，在核内浓度高，在胞质中浓度低，提供运输动力。

选择题测试

以下是从30道题库中随机抽取的15道选择题，请选择最佳答案：

深度思考题

1. 试述核孔复合体的结构特点及其在核质运输中的作用机制。
  1. 结构特点：核孔复合体是核被膜上的大型蛋白质复合物，直径约120nm，由约30种核孔蛋白组成，具有八重对称结构。包括胞质纤维、核篮、中央通道和中央栓等组分，形成选择性通透的分子筛。
  2. 选择性识别：核孔复合体能够识别携带核定位信号（NLS）的蛋白质，通过与输入蛋白（importin）结合形成运输复合物，实现选择性运输。小分子（<40kDa）可自由扩散，大分子需要载体蛋白介导。
  3. 能量依赖性：大分子的主动运输需要消耗GTP，Ran-GTP系统提供方向性和动力。核内Ran-GTP浓度高，胞质中Ran-GDP浓度高，形成浓度梯度驱动运输过程。
  4. 双向运输：核孔复合体既介导蛋白质的核输入，也介导RNA的核输出。输入和输出使用不同的载体蛋白系统，但都依赖Ran-GTP梯度调控运输方向。
  5. 调控机制：核孔复合体的通透性受到细胞周期、信号转导等多种因素调控，在细胞分裂时核孔复合体解体，分裂完成后重新组装，确保核质运输的精确调控。
2. 比较常染色质与异染色质的结构特征、功能差异及其生物学意义。
  1. 结构特征差异：常染色质结构疏松，核小体间距较大，组蛋白修饰多样化（如H3K4me3、H3K27ac等激活性修饰）；异染色质结构致密，核小体紧密排列，富含抑制性组蛋白修饰（如H3K9me3、H3K27me3）。
  2. 转录活性差异：常染色质转录活跃，富含活性基因和转录因子结合位点，RNA聚合酶II易于接近；异染色质转录沉默，基因表达受到抑制，转录因子难以结合。
  3. DNA可及性：常染色质中DNA更容易被DNase I消化，表明结构开放；异染色质对DNase I抗性强，DNA可及性低，反映其紧密的包装状态。
  4. 复制时序：常染色质在S期早期复制，异染色质在S期晚期复制。这种时序差异与基因表达调控和染色体稳定性维持密切相关。
  5. 生物学意义：这种结构功能分化实现了基因表达的精确调控，常染色质确保管家基因和组织特异性基因的适时表达，异染色质维持基因组稳定性并抑制转座子活性，两者协调作用维持细胞身份和基因组完整性。
3. 阐述核小体的组装过程及其在基因表达调控中的作用。
  1. 核小体组装过程：首先H3-H4二聚体结合形成四聚体，作为核小体组装的核心；随后H2A-H2B二聚体分别结合到H3-H4四聚体两侧，形成组蛋白八聚体；146bp DNA以左手螺旋方式在八聚体上缠绕1.65圈，形成核心颗粒；最后H1组蛋白结合到连接DNA上，稳定核小体间的高级结构。
  2. 动态调控机制：核小体具有动态性，可通过ATP依赖的染色质重塑复合物（如SWI/SNF、ISWI）进行滑动、弹出或重新定位。组蛋白伴侣蛋白（如CAF-1、HIRA）参与核小体的组装和拆卸过程。
  3. 转录调控作用：核小体包装的DNA转录活性降低，形成转录屏障。转录激活需要染色质重塑，使转录因子和RNA聚合酶能够接近DNA。核小体定位影响转录起始位点的选择和转录效率。
  4. 表观遗传调控：组蛋白尾部的化学修饰（甲基化、乙酰化、磷酸化等）调控染色质结构和基因表达。不同修饰组合形成"组蛋白密码"，招募特定的效应蛋白，实现基因表达的精确调控。
  5. 遗传稳定性：核小体结构保护DNA免受物理和化学损伤，但也可能影响DNA修复过程。DNA损伤修复时需要暂时拆卸核小体，修复完成后重新组装，维持染色质结构的完整性。
4. 分析端粒结构与功能的关系，并讨论端粒与细胞衰老的分子机制。
  1. 端粒结构特征：端粒由富含TG的重复序列组成（人类为TTAGGG重复），3'端形成单链突出，与端粒结合蛋白（TRF1、TRF2、POT1等）结合形成T-loop结构。这种特殊结构掩盖了染色体末端，防止被识别为DNA断裂。
  2. 保护功能机制：端粒帽子结构（telomere cap）保护染色体末端免受核酸外切酶降解，防止染色体末端融合和重排。端粒结合蛋白复合物（shelterin complex）维持端粒结构稳定性，抑制DNA损伤反应的激活。
  3. 复制问题解决：由于DNA聚合酶无法完全复制线性DNA的末端（末端复制问题），端粒酶通过其RNA模板合成端粒DNA，补偿复制过程中的端粒缩短。端粒酶由TERT（催化亚基）和TERC（RNA组分）组成。
  4. 细胞衰老机制：大多数体细胞中端粒酶活性低或缺失，导致端粒随细胞分裂逐渐缩短。当端粒长度降至临界值时，激活DNA损伤检查点，诱导细胞周期阻滞和细胞衰老。这是细胞内在的"分子时钟"机制。
  5. 疾病相关性：端粒功能异常与多种疾病相关：端粒过度缩短导致早衰综合征，端粒酶异常激活与肿瘤发生相关。端粒长度可作为细胞衰老和疾病风险的生物标志物，为抗衰老和肿瘤治疗提供新的靶点。
5. 论述核仁的结构组织与功能，以及核仁在细胞生理中的重要作用。
  1. 核仁结构组织：核仁由三个功能区域组成：纤维中心（FC）含有rDNA基因和RNA聚合酶I；致密纤维组分（DFC）是rRNA转录和早期加工的场所；颗粒组分（GC）进行核糖体亚基组装的后期阶段。这种结构反映了rRNA加工的时空顺序。
  2. rRNA转录与加工：核仁是rRNA基因转录的专门场所，RNA聚合酶I转录45S前体rRNA，随后通过一系列剪切、修饰（甲基化、假尿苷化）过程，产生成熟的18S、5.8S和28S rRNA。这个过程需要大量snoRNA和相关蛋白的参与。
  3. 核糖体亚基组装：核仁中进行核糖体大小亚基的组装，包括rRNA与核糖体蛋白的结合、rRNA的折叠和修饰、亚基的成熟等步骤。组装完成的亚基通过核孔输出到胞质，参与蛋白质合成。
  4. 细胞周期调控：核仁大小和活性与细胞代谢状态密切相关，在细胞分裂时核仁解体，分裂完成后重新组装。核仁应激（nucleolar stress）可激活p53通路，调控细胞周期和凋亡，是细胞应对不良环境的重要机制。
  5. 疾病相关性：核仁功能异常与多种疾病相关：核仁应激参与肿瘤抑制，某些遗传病（如Werner综合征）表现为核仁结构异常，病毒感染常影响核仁功能。核仁也是多种药物的作用靶点，为疾病治疗提供新的策略。
