细胞生物学研究方法

考研专业课复习笔记 | 基于《细胞生物学》第五版

一、细胞生物学研究方法概述

细胞生物学研究方法是揭示细胞结构与功能奥秘的重要工具。随着科学技术的不断发展，从最初的光学显微镜到现代的超分辨率显微镜，从传统的组织化学染色到分子生物学技术，研究方法的进步极大地推动了细胞生物学的发展。

重要提示：掌握各种研究方法的原理、特点和应用范围是理解细胞生物学的基础，也是考研的重点内容。

二、光学显微镜技术

2.1 普通光学显微镜

光学显微镜是细胞生物学研究的基础工具，其工作原理基于光的折射和放大。现代光学显微镜的分辨率约为0.2μm，放大倍数可达1500-2000倍。

基本结构与原理：

  • 照明系统：包括光源、聚光镜和光阑，提供均匀的照明
  • 成像系统：物镜和目镜组成，实现两次放大
  • 调焦系统：粗调和细调螺旋，实现精确对焦
  • 机械系统：载物台、镜臂等支撑结构

2.2 相差显微镜

相差显微镜利用光的相位差原理，将透明的细胞结构转换为可见的明暗对比，特别适合观察活细胞。

技术特点：

  • 无需染色即可观察活细胞
  • 能够显示细胞内部结构的细微差别
  • 适合观察细胞分裂、细胞运动等动态过程

2.3 微分干涉相差显微镜（DIC）

DIC显微镜通过偏振光和棱镜系统，产生具有立体感的高对比度图像，分辨率和对比度都优于普通相差显微镜。

2.4 荧光显微镜

荧光显微镜利用荧光物质在特定波长激发下发出荧光的原理，实现对特定分子的定位和定量分析。

基本组成：

  • 激发光源：汞灯、氙灯或LED光源
  • 激发滤光片：选择特定波长的激发光
  • 二向色镜：分离激发光和荧光
  • 发射滤光片：选择特定波长的荧光

常用荧光染料：

  • DAPI：蓝色荧光，特异性结合DNA
  • FITC：绿色荧光，常用于蛋白质标记
  • 罗丹明：红色荧光，用于多重标记
  • GFP及其衍生物：绿色荧光蛋白，活细胞标记

2.5 激光扫描共焦显微镜（LSCM）

共焦显微镜通过点光源照明和针孔光阑，消除焦平面外的光线干扰，获得高分辨率的光学切片图像。

技术优势：

  • 提高轴向分辨率，获得清晰的光学切片
  • 可进行三维重建
  • 适合厚标本的观察
  • 可进行活细胞动态观察

三、电子显微镜技术

3.1 透射电子显微镜（TEM）

TEM利用电子束穿透超薄切片，通过电磁透镜成像，分辨率可达0.1-0.2nm，是观察细胞超微结构的主要工具。

样品制备流程：

  1. 固定：戊二醛初固定，锇酸后固定
  2. 脱水：系列酒精脱水
  3. 包埋：环氧树脂包埋
  4. 切片：超薄切片机制备60-80nm切片
  5. 染色：醋酸铀和柠檬酸铅双重染色

3.2 扫描电子显微镜（SEM）

SEM利用电子束扫描样品表面，通过二次电子成像，获得具有立体感的表面形貌图像。

应用特点：

  • 观察细胞表面形貌
  • 分辨率约为3-5nm
  • 景深大，立体感强
  • 适合观察细胞表面结构

3.3 冷冻电子显微镜技术

冷冻电镜技术通过快速冷冻保持样品的天然状态，避免了化学固定和脱水过程中的人工假象。

四、超分辨率显微镜技术

4.1 PALM/STORM技术

光激活定位显微镜（PALM）和随机光学重建显微镜（STORM）通过单分子定位实现超分辨率成像，分辨率可达20-50nm。

4.2 STED显微镜

受激发射损耗显微镜（STED）通过受激发射原理压缩有效激发体积，实现超分辨率成像。

4.3 结构化照明显微镜（SIM）

SIM通过结构化照明模式和数学重建算法，将分辨率提高约2倍。

五、其他重要研究方法

5.1 扫描隧道显微镜（STM）和原子力显微镜（AFM）

STM和AFM属于扫描探针显微镜，可以观察原子级别的表面结构，分辨率达到原子水平。

5.2 细胞分离技术

超速离心技术：

  • 差速离心：根据沉降系数分离不同大小的细胞器
  • 密度梯度离心：根据密度差异分离细胞组分

5.3 免疫技术

免疫荧光技术：

  • 直接免疫荧光：荧光标记的一抗直接结合抗原
  • 间接免疫荧光：一抗结合抗原，荧光标记的二抗结合一抗

5.4 分子生物学技术

  • 原位杂交：检测特定核酸序列的细胞内定位
  • FRET技术：检测蛋白质间相互作用
  • 酵母双杂交：体内检测蛋白质相互作用

5.5 细胞培养技术

动物细胞培养：

  • 原代培养：直接从组织分离的细胞培养
  • 传代培养：细胞系的连续培养
  • 细胞株：具有无限增殖能力的细胞系

六、重要名词解释

分辨率(Resolution)
显微镜能够分辨两个相邻点的最小距离，是衡量显微镜性能的重要指标。光学显微镜的理论分辨率约为0.2μm。
数值孔径(Numerical Aperture, NA)
物镜的聚光能力指标，NA = n·sinα，其中n为介质折射率，α为孔径角的一半。NA越大，分辨率越高。
荧光(Fluorescence)
物质吸收特定波长的光后，发射出波长更长的光的现象。荧光显微镜正是利用这一原理进行成像。
共焦(Confocal)
激发光焦点与检测光焦点重合的光学系统，能够消除焦平面外的光线干扰，提高图像质量。
超分辨率(Super-resolution)
突破光学衍射极限的显微技术，分辨率可达几十纳米，包括PALM、STORM、STED等技术。
电子束(Electron Beam)
电子显微镜中用于成像的电子流，波长远小于可见光，因此具有更高的分辨率。
超薄切片(Ultrathin Section)
用于透射电镜观察的极薄样品切片，厚度通常为60-80nm，以保证电子束能够穿透。
免疫荧光(Immunofluorescence)
利用荧光标记的抗体特异性结合抗原，在荧光显微镜下观察特定蛋白质分布的技术。
原位杂交(In Situ Hybridization, ISH)
在细胞或组织原位检测特定核酸序列的分子生物学技术，可用于基因表达定位分析。
FRET(Förster Resonance Energy Transfer)
荧光共振能量转移，当两个荧光分子距离很近时发生的能量转移现象，用于检测蛋白质相互作用。

八、深度思考题

1. 比较光学显微镜、电子显微镜和超分辨率显微镜的原理、特点及应用范围，分析它们在细胞生物学研究中的互补作用。

参考答案：

  1. 光学显微镜：基于光的折射和放大原理，分辨率约0.2μm，适合观察活细胞和动态过程。优点是操作简便、成本低、可观察活体样品；缺点是分辨率有限，无法观察超微结构。主要应用于细胞形态观察、细胞计数、活细胞动态研究等。
  2. 电子显微镜：利用电子束成像，分辨率可达0.1-0.2nm。TEM适合观察细胞内部超微结构，SEM适合观察表面形貌。优点是分辨率极高，能观察分子水平结构；缺点是样品需要特殊处理，无法观察活体，制样过程可能产生人工假象。
  3. 超分辨率显微镜：突破光学衍射极限，分辨率达20-50nm。包括PALM/STORM、STED、SIM等技术。优点是兼具高分辨率和活细胞观察能力；缺点是设备昂贵、技术复杂、对荧光标记要求高。
  4. 互补作用：三种显微镜在细胞生物学研究中形成完整的观察体系。光学显微镜提供整体概况和动态信息，电子显微镜揭示超微结构细节，超分辨率显微镜填补了两者之间的分辨率空白，实现了从宏观到微观的全方位观察。
2. 详细阐述免疫荧光技术的原理、实验流程及在细胞生物学研究中的应用，并分析直接免疫荧光和间接免疫荧光的优缺点。

参考答案：

  1. 基本原理：免疫荧光技术基于抗原-抗体特异性结合和荧光显微镜成像原理。荧光标记的抗体与细胞内特定抗原结合，在荧光显微镜下可观察到荧光信号，从而确定目标蛋白的细胞内分布。
  2. 实验流程：①细胞固定（4%多聚甲醛）；②透化处理（Triton X-100）；③封闭非特异性结合位点（BSA或血清）；④一抗孵育；⑤洗涤；⑥二抗孵育（间接法）；⑦洗涤；⑧封片观察。
  3. 直接免疫荧光：优点是步骤简单、特异性高、无交叉反应；缺点是信号较弱、成本高、每种抗原需要不同荧光标记的抗体。
  4. 间接免疫荧光：优点是信号放大、成本低、一种二抗可检测多种一抗；缺点是步骤复杂、可能有交叉反应、背景较高。
  5. 应用领域：蛋白质亚细胞定位、蛋白表达水平检测、细胞周期分析、病理诊断、药物筛选等。在细胞生物学研究中是不可或缺的重要技术。
3. 分析电子显微镜样品制备过程中各个步骤的作用机制，讨论制样过程可能产生的人工假象及其避免方法。

参考答案：

  1. 固定步骤：戊二醛初固定保持蛋白质结构，锇酸后固定保持脂质结构并增强电子密度对比。固定不充分会导致结构模糊，过度固定会产生沉淀假象。
  2. 脱水步骤：系列酒精脱水去除组织中的水分，为包埋做准备。脱水不彻底会影响包埋质量，脱水过快会导致组织收缩变形。
  3. 包埋步骤：环氧树脂渗透并固化，为超薄切片提供支撑。包埋不完全会导致切片时组织脱落，树脂聚合不充分会影响切片质量。
  4. 切片步骤：制备60-80nm超薄切片，保证电子束穿透。切片过厚会影响分辨率，过薄会导致结构不完整。
  5. 染色步骤：醋酸铀和柠檬酸铅提供电子密度对比。染色不均会导致对比度差异，过度染色会掩盖细微结构。
  6. 人工假象及避免：①收缩假象：控制固定和脱水速度；②沉淀假象：优化固定液配比和pH；③切片假象：保持刀具锋利，控制切片速度；④染色假象：严格控制染色时间和浓度。
4. 论述超分辨率显微镜技术的发展意义，比较PALM/STORM、STED和SIM三种主要技术的原理差异及各自优势。

参考答案：

  1. 发展意义：超分辨率显微镜突破了光学衍射极限（~200nm），实现了纳米级分辨率的活细胞成像，填补了光学显微镜和电子显微镜之间的技术空白，为细胞生物学研究提供了革命性工具。
  2. PALM/STORM技术：基于单分子定位原理，通过光激活或光转换荧光蛋白的随机激活，精确定位单个分子位置。优势是分辨率极高（20-50nm），适合蛋白质分布研究；劣势是成像速度慢，需要特殊荧光标记。
  3. STED技术：利用受激发射损耗原理，通过环形激光束压缩有效激发体积。优势是成像速度快，可用常规荧光染料；劣势是激光功率高，可能对样品造成损伤。
  4. SIM技术：通过结构化照明和数学重建算法提高分辨率约2倍。优势是设备相对简单，成像速度快，对样品损伤小；劣势是分辨率提升有限，需要复杂的图像处理。
  5. 应用前景：这些技术在蛋白质相互作用、细胞器结构、病毒感染机制等研究中发挥重要作用，推动了细胞生物学向分子水平的深入发展。
5. 结合具体实例，分析细胞培养技术在细胞生物学研究中的重要作用，讨论原代培养和细胞系培养的特点及应用选择。

参考答案：

  1. 重要作用：细胞培养技术为细胞生物学研究提供了可控的实验系统，使研究者能够在体外条件下研究细胞的生长、分化、代谢、信号转导等基本生命过程，是现代细胞生物学不可或缺的基础技术。
  2. 原代培养特点：直接从组织分离获得，保持细胞原有特性，遗传稳定性好，更接近体内状态。但培养难度大，传代次数有限，细胞异质性高。适用于研究细胞的生理功能、药物筛选的初步评估等。
  3. 细胞系培养特点：经过转化或永生化处理，具有无限增殖能力，培养简便，重现性好，成本低。但可能偏离原有特性，遗传不稳定。适用于大规模实验、机制研究、蛋白表达等。
  4. 应用实例：①HeLa细胞系用于细胞周期研究；②原代神经元培养用于神经发育研究；③干细胞培养用于再生医学研究；④癌细胞系用于抗癌药物筛选。
  5. 选择策略：根据研究目的选择合适的培养系统。研究生理功能优选原代培养，机制研究可选择细胞系，药物筛选需要考虑两者结合使用，以确保结果的可靠性和临床相关性。
