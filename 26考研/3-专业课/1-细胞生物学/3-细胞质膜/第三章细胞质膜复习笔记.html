第三章 细胞质膜

考研生物化学与细胞生物学复习笔记

一、细胞质膜的结构模型

1. 流动镶嵌模型（Fluid Mosaic Model）

Singer(辛格)和Nicolson(尼科尔森)于1972年提出的流动镶嵌模型是目前被广泛接受的生物膜结构模型。该模型的核心观点包括：

  • 膜脂双分子层：构成膜的基本骨架，具有流动性
  • 膜蛋白镶嵌：蛋白质以不同方式镶嵌在脂双分子层中
  • 动态结构：膜结构具有流动性和可塑性
  • 不对称性：膜的内外两侧组成和结构不同
重要提示：流动镶嵌模型强调生物膜是一个动态的、流动的结构，而不是静态的固定结构。

2. 脂筏模型（Lipid Raft Model）

脂筏模型是对流动镶嵌模型的重要补充，描述了膜中存在的特殊微区域：

  • 组成特点：富含胆固醇、鞘脂和特定蛋白质
  • 物理性质：相对于周围膜区域更加有序和致密
  • 功能意义：参与信号转导、膜融合等重要生物学过程

二、细胞质膜的化学组成

1. 膜脂（Membrane Lipids）

（1）甘油磷脂（Glycerophospholipids）

甘油磷脂是细胞膜中最主要的脂类成分，约占膜脂总量的50-60%：

  • 磷脂酰胆碱（PC）：最丰富的甘油磷脂，主要分布在膜的外叶
  • 磷脂酰乙醇胺（PE）：主要分布在膜的内叶
  • 磷脂酰丝氨酸（PS）：几乎全部分布在膜的内叶
  • 磷脂酰肌醇（PI）：参与信号转导过程

（2）鞘脂（Sphingolipids）

鞘脂主要包括：

  • 鞘磷脂（Sphingomyelin）：主要分布在膜的外叶
  • 糖脂（Glycolipids）：包括脑苷脂和神经节苷脂

（3）固醇（Sterols）

胆固醇是动物细胞膜中的主要固醇：

  • 含量：约占膜脂总量的25-30%
  • 分布：在膜的内外两叶中分布相对均匀
  • 功能：调节膜的流动性和通透性

2. 膜蛋白（Membrane Proteins）

（1）整合膜蛋白（Integral Membrane Proteins）

整合膜蛋白深度嵌入或完全穿越脂双分子层：

  • 跨膜蛋白：完全穿越膜的蛋白质，具有跨膜结构域
  • 单侧膜蛋白：仅嵌入膜的一侧
  • 特点：与膜结合紧密，需要去垢剂才能提取

（2）周边膜蛋白（Peripheral Membrane Proteins）

周边膜蛋白通过非共价键与膜表面结合：

  • 结合方式：通过静电作用、氢键等与膜结合
  • 提取方式：可用高盐溶液或极端pH提取
  • 功能：参与信号转导、细胞骨架连接等

3. 膜糖类（Membrane Carbohydrates）

膜糖类主要以糖蛋白和糖脂的形式存在：

  • 分布：几乎全部位于膜的外表面
  • 功能：细胞识别、细胞黏附、免疫反应等
  • 组成：包括葡萄糖、半乳糖、甘露糖、岩藻糖、唾液酸等

三、细胞质膜的基本特性

1. 膜的流动性（Membrane Fluidity）

膜的流动性是生物膜的重要物理性质：

影响因素：

  • 温度：温度升高，流动性增加
  • 脂肪酸组成：不饱和脂肪酸含量高，流动性增加
  • 胆固醇含量：在生理温度下，胆固醇降低流动性
  • 脂肪酸链长：链长增加，流动性降低

生物学意义：

  • 维持膜蛋白的正常功能
  • 调节膜的通透性
  • 影响膜融合过程
  • 参与信号转导

2. 膜的不对称性（Membrane Asymmetry）

细胞质膜的内外两侧在组成和结构上存在显著差异：

脂质不对称性：

  • 外叶：富含PC、鞘磷脂和糖脂
  • 内叶：富含PE、PS和PI

蛋白质不对称性：

  • 膜蛋白具有特定的取向
  • 糖基化修饰主要在膜的外侧

维持机制：

  • 翻转酶（Flippase）：将脂质从外叶转移到内叶
  • 翻出酶（Floppase）：将脂质从内叶转移到外叶
  • 扰乱酶（Scramblase）：促进脂质的双向转移

四、细胞质膜的主要功能

1. 屏障功能

  • 选择性通透：控制物质的进出
  • 维持细胞内环境：保持离子浓度梯度
  • 防止有害物质进入：保护细胞内部结构

2. 转运功能

  • 被动转运：简单扩散、促进扩散
  • 主动转运：原发性主动转运、继发性主动转运
  • 胞吞胞吐：大分子物质的转运

3. 信号转导功能

  • 受体蛋白：识别和结合信号分子
  • 信号传递：将胞外信号传递到胞内
  • 信号放大：放大微弱的信号

4. 细胞识别与黏附

  • 糖蛋白和糖脂：参与细胞表面识别
  • 黏附分子：介导细胞间的黏附
  • 免疫识别：MHC分子等参与免疫反应

5. 酶活性

  • 膜结合酶：如Na⁺-K⁺-ATPase
  • 代谢调节：参与多种代谢途径
  • 信号转导酶：如腺苷酸环化酶

五、重要名词解释库

流动镶嵌模型(Fluid Mosaic Model)
由Singer(辛格)和Nicolson(尼科尔森)于1972年提出的生物膜结构模型，认为生物膜是由流动的脂双分子层和镶嵌其中的蛋白质组成的动态结构。
整合膜蛋白(Integral Membrane Proteins)
深度嵌入或完全穿越脂双分子层的膜蛋白，与膜结合紧密，需要去垢剂才能从膜中提取出来。
周边膜蛋白(Peripheral Membrane Proteins)
通过非共价键与膜表面结合的蛋白质，结合相对松散，可用高盐溶液或极端pH条件提取。
甘油磷脂(Glycerophospholipids)
以甘油为骨架的磷脂类化合物，是细胞膜中最主要的脂类成分，包括磷脂酰胆碱、磷脂酰乙醇胺等。
鞘脂(Sphingolipids)
以鞘氨醇为骨架的脂类化合物，包括鞘磷脂和糖脂，主要分布在膜的外叶。
脂筏(Lipid Rafts)
膜中富含胆固醇、鞘脂和特定蛋白质的微区域，相对于周围膜区域更加有序和致密，参与信号转导等重要功能。
膜不对称性(Membrane Asymmetry)
细胞膜内外两侧在脂质组成、蛋白质分布和糖基化修饰等方面存在的显著差异。
翻转酶(Flippase)
一类ATP依赖性的膜蛋白，能够将磷脂从膜的外叶转移到内叶，维持膜的不对称性。
磷脂酰丝氨酸(Phosphatidylserine, PS)
一种重要的甘油磷脂，几乎全部分布在细胞膜的内叶，在细胞凋亡时会翻转到外叶作为凋亡信号。
胆固醇(Cholesterol)
动物细胞膜中的主要固醇类化合物，约占膜脂总量的25-30%，在生理温度下能够降低膜的流动性。

七、深度思考题

1. 试分析流动镶嵌模型的提出对细胞生物学发展的重要意义，并结合现代研究成果讨论该模型的局限性和发展。

  1. 历史意义：流动镶嵌模型的提出标志着对生物膜结构认识的重大突破。在此之前，人们对膜结构的认识主要基于静态的"三明治"模型，认为膜是由蛋白质-脂质-蛋白质三层结构组成的静态结构。Singer(辛格)和Nicolson(尼科尔森)的模型首次强调了膜的动态性和流动性，为理解膜的功能奠定了理论基础。
  2. 理论贡献：该模型成功解释了膜的选择性通透性、蛋白质的功能多样性以及膜融合等现象。它强调膜蛋白可以在脂双分子层中自由移动，这为理解受体聚集、信号转导等过程提供了理论依据。同时，模型中提出的膜不对称性概念，为后来发现的膜脂不对称分布和膜蛋白取向性提供了理论预测。
  3. 现代发展：随着研究技术的进步，科学家们发现了脂筏(lipid rafts)的存在，这些富含胆固醇和鞘脂的微区域具有不同于周围膜区域的物理化学性质。这一发现丰富了流动镶嵌模型，表明膜并非完全均一的流体，而是存在功能性的微区域分化。
  4. 局限性分析：原始的流动镶嵌模型过分强调了膜的均一性和流动性，忽略了膜中存在的有序结构域。现代研究表明，膜中存在多种类型的微区域，如脂筏、胞质膜-细胞骨架连接点等，这些区域的流动性受到限制。此外，模型对膜蛋白与脂质之间的特异性相互作用描述不够充分。
  5. 未来发展方向：现代膜生物学正朝着更加精细化的方向发展，包括单分子技术的应用、膜蛋白结构的高分辨率解析、膜脂-蛋白相互作用的分子机制研究等。这些研究将进一步完善我们对生物膜结构和功能的认识。

2. 细胞膜的不对称性是如何建立和维持的？这种不对称性对细胞功能有何重要意义？

  1. 不对称性的建立机制：膜不对称性的建立始于内质网中新合成脂质的不对称插入。新合成的磷脂主要插入到内质网膜的胞质侧，然后通过囊泡转运系统传递到其他膜系统。在这个过程中，膜的拓扑结构得以保持，从而建立了基本的不对称性模式。糖基化修饰也在内质网和高尔基体中进行，确保糖蛋白和糖脂只出现在膜的非胞质侧。
  2. 不对称性的维持机制：膜不对称性的维持主要依赖于三类转移酶的协调作用。翻转酶(flippase)是ATP依赖性的，主要将氨基磷脂(如PS和PE)从外叶转移到内叶；翻出酶(floppase)将脂质从内叶转移到外叶；扰乱酶(scramblase)在特定条件下(如细胞凋亡)促进脂质的双向快速转移。这些酶的活性受到严格调节，确保膜不对称性的稳定维持。
  3. 生理功能意义：膜不对称性对细胞功能具有多重重要意义。首先，它为膜蛋白提供了特定的脂质环境，影响蛋白质的构象和活性。例如，Na⁺-K⁺-ATPase需要特定的脂质环境才能正常发挥功能。其次，不对称性参与信号转导过程，如PIP₂主要分布在膜的内叶，作为重要的信号分子参与多种信号通路。
  4. 病理生理意义：膜不对称性的破坏与多种疾病相关。在细胞凋亡过程中，PS从内叶翻转到外叶，作为"eat-me"信号被巨噬细胞识别。在某些病理条件下，如镰状细胞贫血，红细胞膜的不对称性受损，导致PS异常暴露，引起细胞被过早清除。此外，一些病原体利用膜不对称性的改变来逃避宿主免疫系统。
  5. 研究意义和应用前景：深入理解膜不对称性的机制对于药物开发具有重要意义。例如，靶向翻转酶的药物可能用于治疗某些血液疾病；利用膜不对称性的改变作为疾病诊断标志物也是一个有前景的研究方向。

3. 比较分析整合膜蛋白和周边膜蛋白的结构特点、功能差异及其在细胞生命活动中的作用。

  1. 结构特点对比：整合膜蛋白具有疏水性的跨膜结构域，通常由α螺旋或β桶状结构组成，这些结构域与膜脂的疏水尾部相互作用，使蛋白质稳定地嵌入膜中。跨膜蛋白通常具有多个跨膜片段，形成复杂的三维结构。相比之下，周边膜蛋白主要由亲水性氨基酸组成，通过静电作用、氢键或范德华力与膜表面的极性基团结合，不具有跨膜结构域。
  2. 膜结合方式差异：整合膜蛋白与膜的结合是不可逆的，需要使用去垢剂或有机溶剂才能从膜中提取，这种强结合保证了蛋白质功能的稳定性。周边膜蛋白的结合相对可逆，可以通过改变离子强度、pH值或使用螯合剂等温和条件从膜上解离，这种可逆结合使得蛋白质能够在膜结合和游离状态之间转换。
  3. 功能多样性分析：整合膜蛋白主要承担跨膜转运功能，如离子通道、载体蛋白和ATP酶等，它们能够介导物质在膜两侧的转运。此外，许多受体蛋白也是整合膜蛋白，负责识别和结合胞外信号分子。周边膜蛋白则主要参与膜表面的生化反应，如酶催化、信号转导的调节、膜与细胞骨架的连接等。
  4. 调节机制不同：整合膜蛋白的活性调节主要通过构象变化、磷酸化修饰或与调节亚基结合等方式实现。由于其稳定的膜结合状态，调节相对缓慢但持久。周边膜蛋白的调节更加灵活，可以通过膜结合/解离的动态平衡快速响应细胞内信号变化，如许多信号转导蛋白在激活后会从胞质转移到膜表面。
  5. 生物学意义和协同作用：两类膜蛋白在细胞生命活动中发挥协同作用。整合膜蛋白提供了稳定的膜功能基础，如维持细胞内外环境的分隔和物质转运；周边膜蛋白则提供了灵活的调节机制，能够快速响应环境变化。例如，在信号转导过程中，整合膜蛋白受体接收信号后，会招募周边膜蛋白到膜表面，形成信号转导复合体，实现信号的放大和传递。

4. 胆固醇在细胞膜中的作用机制是什么？它如何影响膜的物理性质和生物学功能？

  1. 分子结构与膜中定位：胆固醇是一个两亲性分子，具有刚性的固醇环结构和一个短的疏水尾部。在膜中，胆固醇的羟基与磷脂的极性头基相互作用，而其疏水的固醇环和尾部则插入到脂肪酸链之间。这种独特的结构使胆固醇能够调节膜的有序性和流动性。胆固醇在膜中的分布相对均匀，但在脂筏区域浓度更高。
  2. 对膜流动性的双重调节作用：胆固醇对膜流动性的影响具有温度依赖性。在生理温度下，胆固醇通过其刚性的固醇环结构限制磷脂分子的运动，从而降低膜的流动性，这种作用被称为"凝聚效应"。然而，在较低温度下，胆固醇能够阻止磷脂分子的紧密堆积，防止膜进入凝胶态，从而维持膜的流动性。这种双重调节作用使细胞膜能够在不同温度条件下保持适宜的流动性。
  3. 膜通透性的调节：胆固醇通过影响膜的有序性来调节膜的通透性。适量的胆固醇能够减少膜的通透性，特别是对小分子物质如水、离子和气体的通透性。这种作用对于维持细胞内外环境的稳定性至关重要。胆固醇含量的改变会显著影响膜的屏障功能，过高或过低的胆固醇含量都可能导致膜通透性异常。
  4. 脂筏形成与信号转导：胆固醇是脂筏形成的关键组分。脂筏是膜中富含胆固醇和鞘脂的微区域，具有独特的物理化学性质。这些区域为特定的膜蛋白提供了适宜的环境，参与信号转导、膜融合等重要生物学过程。许多重要的信号转导蛋白，如GPI锚定蛋白、某些受体蛋白等，都优先定位在脂筏中。胆固醇的存在对于维持这些蛋白质的正常功能至关重要。
  5. 病理生理意义：胆固醇代谢异常与多种疾病相关。胆固醇含量过高可能导致动脉粥样硬化，而某些遗传性疾病如Smith-Lemli-Opitz综合征则是由于胆固醇合成缺陷引起的。此外，一些病原体如某些细菌毒素需要胆固醇才能发挥作用，而抗真菌药物如两性霉素B则通过与胆固醇结合来破坏真菌细胞膜。理解胆固醇的作用机制对于疾病的预防和治疗具有重要意义。
