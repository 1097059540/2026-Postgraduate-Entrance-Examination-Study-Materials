细胞周期与细胞分裂

考研专业课复习笔记 | 基于《细胞生物学》第五版

核心知识点

一、细胞周期概述

细胞周期是指细胞从一次分裂完成开始到下一次分裂完成为止的整个过程，是细胞生命活动的重要特征之一。

1. 细胞周期的意义

  • 生物繁育基础：单细胞生物通过细胞增殖直接增加个体数量
  • 生长发育基础：多细胞生物从受精卵发育需要无数次细胞增殖
  • 组织修复：损伤组织通过细胞增殖进行修复和再生

2. 细胞周期的时相划分

完整的细胞周期分为两个主要阶段：

  • 分裂间期（Interphase）：占细胞周期的90%以上
    • G₁期（Gap 1）：DNA合成前期，细胞生长，合成蛋白质、酶类等
    • S期（Synthesis）：DNA合成期，DNA复制和组蛋白合成
    • G₂期（Gap 2）：DNA合成后期，为分裂做准备
  • 分裂期（M期，Mitosis）：包括核分裂和胞质分裂
重要提示：细胞周期时间长短主要取决于G₁期，而S期、G₂期和M期的时间相对恒定。M期通常仅持续半小时左右。

3. 细胞群体分类

  • 周期中细胞：持续进行细胞周期，不断分裂
  • G₀期细胞：暂时脱离细胞周期的静止期细胞，可重新进入周期
  • 终末分化细胞：高度分化，终生不再分裂的细胞

二、细胞周期调控

1. 细胞周期检查点

  • G₁/S检查点（限制点R点）：位于G₁期末，决定细胞是否进入S期
  • G₂/M检查点：检查DNA复制是否完成
  • 纺锤体检查点：确保所有染色体正确附着于纺锤体

2. 细胞周期同步化

  • 天然同步化：某些组织细胞自然同步分裂
  • 人工选择同步化：通过物理方法选择特定时相细胞
  • 人工诱导同步化：使用化学试剂阻断细胞周期

三、有丝分裂过程

1. 前期（Prophase）

  • 染色体凝缩：染色质螺旋化、折叠，形成可见染色体
  • 中心体复制分离：建立两极纺锤体，确保分裂对称性
  • 纺锤体装配开始：微管开始组装

2. 前中期（Prometaphase）

  • 核膜崩解：与核纤层蛋白磷酸化相关
  • 纺锤体装配完成：形成完整的有丝分裂器
  • 染色体整列：Mad和Bub蛋白参与动粒敏化

3. 中期（Metaphase）

  • 染色体排列：所有染色体排列在赤道面上
  • 纺锤体成熟：呈现典型纺锤形结构
  • 检查点激活：确保所有染色体正确附着

4. 后期（Anaphase）

  • 后期A：姐妹染色单体分离，向两极运动
  • 后期B：纺锤体极间距离增加
  • 微管动力学：动粒微管缩短，极微管延长

5. 末期（Telophase）

  • 染色体去凝缩：染色体逐渐展开
  • 核膜重组：核纤层蛋白去磷酸化，核膜重新形成
  • 核仁重建：RNA合成功能恢复

6. 胞质分裂（Cytokinesis）

  • 分裂沟形成：细胞表面环形缢缩
  • 收缩环作用：肌动蛋白-肌球蛋白收缩
  • 细胞分离：最终形成两个子细胞

四、减数分裂过程

1. 减数分裂的特点

  • 染色体数目减半：从二倍体变为单倍体
  • 基因重组：增加遗传多样性
  • 两次连续分裂：减数分裂Ⅰ和减数分裂Ⅱ

2. 减数分裂前期Ⅰ的五个阶段

  • 细线期：染色质开始凝缩
  • 偶线期：同源染色体配对（联会）
  • 粗线期：交叉重组发生，合成P-DNA
  • 双线期：同源染色体分离，四分体结构清晰
  • 终变期：染色体进一步凝缩，交叉端化

3. 联会复合体

联会复合体是减数分裂特有的蛋白质结构，由侧元件、中央元件和横丝组成，介导同源染色体的精确配对和基因重组。

五、有丝分裂与减数分裂的比较

比较项目 有丝分裂 减数分裂
分裂次数 1次 2次连续分裂
染色体数目 保持不变 减半
同源染色体配对 不配对 配对（联会）
基因重组 不发生 发生交叉重组
生物学意义 生长发育、组织修复 产生配子、增加遗传多样性

名词解释库

细胞周期(Cell Cycle)
指细胞从一次分裂完成开始到下一次分裂完成为止的整个过程，包括分裂间期和分裂期两个阶段。
G₁期(Gap 1 Phase)
DNA合成前期，细胞主要进行生长，合成细胞生长所需的各种蛋白质、糖类、脂质等，但不合成DNA。
S期(Synthesis Phase)
DNA合成期，细胞进行DNA复制，同时合成新的组蛋白，是细胞周期中DNA含量发生变化的时期。
G₂期(Gap 2 Phase)
DNA合成后期，细胞继续生长并为即将到来的细胞分裂做准备，合成分裂所需的蛋白质。
限制点(Restriction Point, R Point)
位于G₁期末的细胞周期检查点，决定细胞是否继续进入S期进行DNA复制，是细胞分化与增殖的关键控制点。
有丝分裂器(Mitotic Apparatus)
在细胞有丝分裂过程中，由纺锤体、中心体、染色体组成的复合细胞器，专门执行有丝分裂功能。
纺锤体(Spindle Apparatus)
细胞分裂过程中与染色体分离直接相关的细胞器，由微管组成，负责牵引染色体向两极移动。
动粒(Kinetochore)
位于染色体着丝粒处的蛋白质复合体，是纺锤体微管附着的部位，在染色体分离中起关键作用。
联会(Synapsis)
减数分裂前期Ⅰ偶线期发生的同源染色体配对过程，通过联会复合体介导，是基因重组的前提。
联会复合体(Synaptonemal Complex, SC)
减数分裂特有的蛋白质结构，由侧元件、中央元件和横丝组成，介导同源染色体的精确配对。
二价体(Bivalent)
减数分裂前期Ⅰ中配对的两条同源染色体形成的复合结构，含有4条染色单体，又称四分体。
交叉(Crossing Over)
减数分裂前期Ⅰ粗线期发生的同源染色体间DNA片段交换过程，是基因重组的分子基础。
重组节(Recombination Nodule)
联会复合体上进行基因重组的特殊结构，含有重组所需的酶类，是交叉发生的部位。
黏连蛋白(Cohesin)
维持姐妹染色单体结合的蛋白质复合体，在细胞分裂后期被分离酶切割，使染色单体分离。
凝缩蛋白(Condensin)
参与染色体凝缩的SMC蛋白家族成员，通过ATP水解驱动染色体的螺旋化和压缩。

深度思考题

1. 试述细胞周期检查点的分子机制及其在肿瘤发生中的作用

  1. G₁/S检查点机制：该检查点主要由p53蛋白介导，当检测到DNA损伤时，p53被激活并诱导p21蛋白表达，p21结合并抑制CDK2-cyclin E复合体，阻止细胞进入S期。同时，Rb蛋白保持低磷酸化状态，继续结合E2F转录因子，阻止S期基因的转录。
  2. G₂/M检查点机制：主要检查DNA复制是否完整和DNA损伤修复是否完成。ATM和ATR激酶检测DNA损伤，激活Chk1和Chk2激酶，进而抑制CDC25磷酸酶活性，维持CDK1的磷酸化抑制状态，阻止细胞进入M期。
  3. 纺锤体检查点机制：由Mad2、BubR1、Bub3等蛋白组成的纺锤体检查点复合体监控染色体与纺锤体的正确结合。未正确附着的动粒会产生"等待"信号，抑制APC/C的活性，阻止分离酶的激活，确保所有染色体正确分离。
  4. 在肿瘤发生中的作用：检查点功能缺陷是肿瘤发生的重要机制。p53基因突变导致G₁/S检查点失效，使细胞在DNA损伤情况下仍能继续分裂；纺锤体检查点缺陷导致染色体不稳定性，促进肿瘤进展；这些缺陷使细胞逃脱正常的生长控制，获得无限增殖能力。

2. 分析减数分裂中基因重组的分子机制及其生物学意义

  1. 重组的分子机制：基因重组始于SPO11蛋白介导的DNA双链断裂，形成3'单链突出端。RAD51和DMC1蛋白介导同源搜索和链侵入，形成D-loop结构。随后发生DNA合成和第二端捕获，形成双Holliday连接点。最终通过分辨酶的切割，产生交叉型或非交叉型重组产物。
  2. 联会复合体的作用：联会复合体为重组提供结构支架，其中央元件含有重组节，集中了重组所需的酶类。侧元件结合染色体轴心，维持同源染色体的正确配对。SYCP1、SYCP2、SYCP3等蛋白是联会复合体的主要组成成分。
  3. 重组的调控：重组频率和分布受到严格调控。交叉干扰确保每对同源染色体至少有一个交叉，保证正确分离。重组热点的存在使重组在特定区域更容易发生。组蛋白修饰和染色质结构也影响重组的发生。
  4. 生物学意义：基因重组增加了配子的遗传多样性，为自然选择提供原材料；促进有利基因组合的形成，提高物种适应性；打破基因间的连锁关系，增加进化的灵活性；在医学上，重组异常与某些遗传疾病和不育症相关。

3. 比较分析植物细胞和动物细胞在细胞分裂过程中的异同点

  1. 纺锤体形成的差异：动物细胞具有中心体，形成有星纺锤体，中心体复制和分离确立分裂极；植物细胞缺乏中心体，形成无星纺锤体，通过分散的微管组织中心(MTOC)组装纺锤体。尽管结构不同，但都能有效介导染色体分离。
  2. 胞质分裂机制的差异：动物细胞通过肌动蛋白-肌球蛋白收缩环从外向内收缩，形成分裂沟；植物细胞则通过细胞板的形成从内向外分裂，涉及高尔基体衍生的囊泡融合和纤维素沉积，最终形成新的细胞壁。
  3. 细胞壁的影响：植物细胞的细胞壁限制了细胞形状的改变，使其无法像动物细胞那样通过收缩进行分裂。细胞板的形成需要精确的囊泡运输和膜融合过程，涉及胼胝质的暂时沉积和纤维素的最终形成。
  4. 共同的分子机制：两者在DNA复制、染色体凝缩、纺锤体检查点、细胞周期调控等方面具有高度保守的分子机制。CDK-cyclin复合体、检查点蛋白、DNA修复机制等在植物和动物中都发挥重要作用，体现了细胞分裂的基本原理的普遍性。

4. 论述细胞周期调控异常与疾病发生的关系

  1. 肿瘤与细胞周期失控：肿瘤细胞的核心特征是细胞周期调控的丧失。原癌基因(如cyclin、CDK)的过度表达促进细胞增殖；抑癌基因(如p53、Rb)的失活使检查点失效；DNA修复基因的缺陷导致基因组不稳定性。这些异常使细胞获得无限增殖能力，逃脱凋亡和衰老。
  2. 遗传性疾病与细胞分裂缺陷：某些遗传性疾病源于细胞分裂相关基因的突变。如着色性干皮病由DNA修复基因缺陷引起；早衰症与DNA损伤检查点异常相关；某些智力发育障碍与纺锤体蛋白突变有关，导致神经元分裂异常。
  3. 不育症与减数分裂异常：减数分裂过程的异常是导致不育的重要原因。联会复合体蛋白缺陷导致配对失败；重组异常影响配子形成；纺锤体检查点缺陷导致非整倍体配子产生。这些异常不仅影响生育能力，还可能导致后代染色体异常。
  4. 衰老与细胞周期调控：细胞衰老与细胞周期的永久性阻滞密切相关。端粒缩短、DNA损伤积累、氧化应激等因素激活p53-p21通路，导致细胞周期阻滞。衰老细胞的积累与多种年龄相关疾病的发生有关，如心血管疾病、神经退行性疾病等。

5. 分析现代细胞分裂研究技术及其在基础研究和临床应用中的价值

  1. 活细胞成像技术：荧光蛋白标记技术结合高分辨率显微镜，使研究者能够实时观察细胞分裂过程。GFP标记的组蛋白可追踪染色体动态；荧光标记的微管蛋白揭示纺锤体装配过程。这些技术为理解细胞分裂的动态机制提供了直观证据，并有助于发现新的调控因子。
  2. 单细胞测序技术：单细胞RNA测序和单细胞ATAC测序技术揭示了细胞周期不同阶段的基因表达谱和染色质可及性变化。这些技术帮助识别细胞周期特异性基因，理解细胞分裂的转录调控网络，并为肿瘤异质性研究提供新视角。
  3. 基因编辑技术：CRISPR-Cas9系统使精确的基因编辑成为可能，研究者可以特异性敲除或修饰细胞周期相关基因，研究其功能。在临床上，基因编辑技术为治疗遗传性疾病提供了新途径，如修复DNA修复基因缺陷。
  4. 临床应用价值：这些技术在肿瘤诊断和治疗中具有重要价值。细胞周期分析可用于评估肿瘤细胞的增殖活性；靶向细胞周期的药物(如CDK抑制剂)为肿瘤治疗提供新策略；生殖医学中，这些技术有助于评估配子质量和改善辅助生殖技术的成功率。
