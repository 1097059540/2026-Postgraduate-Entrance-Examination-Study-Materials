细胞增殖调控与癌细胞

考研专业课复习笔记 | 细胞生物学第十三章

第一部分：核心知识内容

一、细胞增殖调控的发现历程

1. MPF（成熟促进因子）的发现

1960年代，Yoshio Masui发现成熟蛙卵的提取物能促进未成熟卵的胚胞破裂（GVBD）。后来Sunkara将不同时期HeLa细胞的提取液注射到蛙卵母细胞中，发现G1和S期的抽取物不能诱导GVBD，而G2和M期的则具有促进胚胞破裂的功能，这种诱导物质被称为有丝分裂因子（MF），后统称为MPF（成熟促进因子）。

重要发现：Rao和Johnson通过细胞融合实验发现，M期细胞能诱导间期细胞产生早熟凝集染色体（PCC），证明了M期细胞具有某种促进分裂的因子。

2. p34cdc2激酶的发现

1970年代，Leland Hartwell以芽殖酵母为实验材料，利用温度敏感突变株分离出几十个与细胞分裂有关的基因（CDC基因）。Paul Nurse等人以裂殖酵母为材料，发现了cdc2、cdc25、wee1等关键基因。

  • cdc2基因：编码34KD的蛋白激酶，促进细胞周期进行
  • wee1基因：抑制CDC2活性，突变型提早分裂
  • cdc25基因：促进CDC2活性，突变型无法分裂

3. 周期蛋白（Cyclin）的发现

1983年Timothy Hunt在海胆卵受精后的卵裂过程中发现两种蛋白质的含量随细胞周期剧烈振荡：在间期开始合成，G2/M时达到高峰，M结束后突然消失，故命名为周期蛋白（Cyclin）。

MPF的组成：
MPF = p34cdc2（CDK1）+ Cyclin B

二、细胞周期调控的分子机制

1. CDK（周期蛋白依赖性激酶）

CDC2与细胞周期蛋白结合才具有激酶活性，称为细胞周期蛋白依赖性激酶（CDK）。目前发现的CDK在动物中有7种，各种CDK分子均含有相似的激酶结构域，包含保守序列PSTAIRE。

  • CDK1（CDC2）：调控G2/M转换
  • CDK2：调控G1/S和S期进程
  • CDK4/CDK6：调控G1期进程

2. 周期蛋白（Cyclin）的分类与功能

  • G1/S周期蛋白（Cyclin E）：与CDK2结合，启动S期
  • S期周期蛋白（Cyclin A）：与CDK2结合，维持S期进程
  • M期周期蛋白（Cyclin B）：与CDK1结合，启动M期
  • G1期周期蛋白（Cyclin D）：与CDK4/6结合，推进G1期

3. CDK抑制因子（CKI）

细胞中存在CDK抑制因子（CKI），能够抑制CDK活性，主要分为两大家族：

  • CIP/KIP家族：p21、p27、p57，广谱抑制多种CDK
  • INK4家族：p15、p16、p18、p19，特异抑制CDK4/6

三、细胞周期检验点

1. G1/S检验点（限制点）

检查DNA损伤、营养状况、生长信号等。p53蛋白在此发挥关键作用，DNA损伤时激活p53，诱导p21表达，抑制CDK2活性，阻止细胞进入S期。

2. S期检验点

监控DNA复制过程，确保DNA复制的准确性和完整性。

3. G2/M检验点

检查DNA复制是否完成、DNA是否有损伤。只有通过检验，细胞才能进入M期。

4. 纺锤体检验点

确保所有染色体正确附着在纺锤体上，防止染色体分离错误。APC/C（后期促进复合体）在此发挥重要作用。

四、癌细胞的特征与发生机制

1. 癌细胞的基本特征

  • 无限增殖能力：突破Hayflick界限
  • 失去接触抑制：细胞密度增高时仍继续分裂
  • 失去锚定依赖性：可在悬浮状态下生长
  • 侵袭和转移能力：突破基底膜，远处转移
  • 诱导血管生成：分泌血管生成因子
  • 逃避免疫监视：逃避机体免疫系统清除

2. 癌基因与抑癌基因

癌基因（Oncogene）：正常情况下调控细胞增殖的基因，突变或过度表达时促进癌变。

主要癌基因类型：

  • 生长因子：如PDGF
  • 生长因子受体：如EGFR、HER2
  • 信号转导蛋白：如Ras、Myc
  • 转录因子：如Jun、Fos
  • 细胞周期调控蛋白：如Cyclin D、CDK4
抑癌基因（Tumor Suppressor Gene）：正常情况下抑制细胞增殖，失活时导致癌变。

主要抑癌基因：

  • p53基因："基因组守护者"，调控细胞周期检验点
  • Rb基因：调控G1/S转换
  • APC基因：调控细胞粘附和Wnt信号通路
  • BRCA1/BRCA2：参与DNA损伤修复

3. 肿瘤发生的多步骤模型

肿瘤的发生是基因突变逐渐积累的结果，通常需要3-7个关键基因的突变。典型的结肠癌发生模型：

正常上皮 → APC突变 → 早期腺瘤 → K-ras突变 → 中期腺瘤 → p53突变 → 晚期腺瘤 → 其他突变 → 癌变

第二部分：重要名词解释

成熟促进因子
(Maturation Promoting Factor, MPF)
存在于M期细胞中的一种促进细胞分裂的因子，由CDK1（p34cdc2）和Cyclin B组成，能够磷酸化多种靶蛋白，如核纤层蛋白、组蛋白H1等，导致核膜解体、染色体凝缩等M期特征性变化。
周期蛋白依赖性激酶
(Cyclin-Dependent Kinase, CDK)
一类丝氨酸/苏氨酸蛋白激酶，必须与相应的周期蛋白结合才具有激酶活性。不同的CDK-Cyclin复合体调控细胞周期的不同阶段，是细胞周期调控的核心分子机器。
周期蛋白
(Cyclin)
一类在细胞周期中含量呈周期性变化的蛋白质，在特定时期合成，在特定时期被降解。不同类型的周期蛋白与相应的CDK结合，调控细胞周期的不同阶段。
CDK抑制因子
(CDK Inhibitor, CKI)
能够结合并抑制CDK活性的蛋白质，主要包括CIP/KIP家族（p21、p27、p57）和INK4家族（p15、p16、p18、p19），在细胞周期负调控中发挥重要作用。
细胞周期检验点
(Cell Cycle Checkpoint)
细胞周期中的质量控制机制，监控细胞内外环境，确保细胞周期有序进行。主要包括G1/S检验点、S期检验点、G2/M检验点和纺锤体检验点。
癌基因
(Oncogene)
正常情况下参与调控细胞增殖、分化的基因，当发生突变、扩增或异常表达时，能够促进细胞恶性转化。包括生长因子、生长因子受体、信号转导蛋白、转录因子等。
抑癌基因
(Tumor Suppressor Gene)
正常情况下抑制细胞过度增殖的基因，当发生突变或缺失时，失去抑制功能，导致细胞恶性转化。典型代表有p53、Rb、APC等。
早熟凝集染色体
(Prematurely Condensed Chromosome, PCC)
当间期细胞与M期细胞融合时，在MPF作用下，间期细胞的染色体提前凝缩形成的染色体结构。不同时期的PCC形态不同：G1期为单线状，S期为粉末状，G2期为双线状。
接触抑制
(Contact Inhibition)
正常细胞在培养皿中生长到相互接触形成单层时停止分裂的现象。癌细胞失去这种特性，能够堆积生长形成多层。
锚定依赖性
(Anchorage Dependence)
正常细胞需要附着在固体表面才能生长增殖的特性。癌细胞失去这种依赖性，能够在悬浮状态下生长，这与其侵袭转移能力相关。

第三部分：选择题测试

第四部分：深度思考题

1. 试述细胞周期检验点的分子机制及其在维持基因组稳定性中的作用。

参考答案：
  1. G1/S检验点（限制点）机制：p53蛋白作为核心调控因子，当检测到DNA损伤时，p53被激活并稳定化，转录激活p21基因。p21蛋白结合并抑制CDK2-Cyclin E复合体，阻止Rb蛋白磷酸化，维持Rb与E2F转录因子的结合，抑制S期基因表达，使细胞周期停滞在G1期，为DNA修复争取时间。
  2. S期检验点机制：ATR激酶检测复制叉停滞，激活Chk1激酶，抑制CDC25A磷酸酶活性，维持CDK2的抑制性磷酸化状态，减缓S期进程。同时激活同源重组修复机制，确保DNA复制的准确性。
  3. G2/M检验点机制：ATM/ATR激酶检测DNA双链断裂，激活Chk1/Chk2激酶，抑制CDC25C磷酸酶，阻止CDK1去磷酸化激活，防止细胞过早进入M期。
  4. 纺锤体检验点机制：Mad2、BubR1等蛋白监控染色体-纺锤体附着状态，未正确附着的染色体产生"等待"信号，抑制APC/C活性，阻止分离酶激活，确保染色体正确分离。
  5. 基因组稳定性维持：这些检验点协同工作，防止DNA损伤细胞继续分裂，避免突变传递给子代细胞，维持基因组完整性，是防止癌变的重要屏障。

2. 分析p53基因被称为"基因组守护者"的分子基础，并阐述其失活在肿瘤发生中的作用机制。

参考答案：
  1. p53的分子结构与功能：p53是转录因子，包含DNA结合域、转录激活域和寡聚化域。正常情况下与MDM2结合被快速降解，DNA损伤时通过磷酸化修饰稳定化，激活下游靶基因转录。
  2. DNA损伤应答机制：ATM/ATR激酶磷酸化p53的Ser15位点，阻断与MDM2结合；p300/CBP乙酰化p53，增强其转录活性。稳定的p53结合靶基因启动子，激活转录。
  3. 细胞命运决定：轻度损伤时，p53激活p21转录，诱导细胞周期停滞，启动DNA修复；严重损伤时，p53激活PUMA、Bax等促凋亡基因，诱导细胞凋亡，清除潜在危险细胞。
  4. p53失活的后果：超过50%的人类肿瘤存在p53突变，失活后细胞失去DNA损伤检验点控制，损伤细胞继续分裂，突变累积加速，基因组不稳定性增加。
  5. 肿瘤发生机制：p53失活使细胞获得抗凋亡能力，突破增殖限制，配合其他癌基因激活或抑癌基因失活，最终导致恶性转化。这解释了p53在肿瘤抑制中的核心地位。

3. 比较分析癌基因和抑癌基因在肿瘤发生中的不同作用机制，并举例说明其在临床诊断和治疗中的应用。

参考答案：
  1. 作用机制差异：癌基因是"油门"，正常时促进细胞增殖分化，异常激活时过度促进增殖；抑癌基因是"刹车"，正常时抑制过度增殖，失活时失去抑制功能。癌基因突变通常是显性的，一个等位基因突变即可发挥作用；抑癌基因失活通常是隐性的，需要两个等位基因都失活（Knudson二次打击假说）。
  2. 典型例子分析：Ras癌基因正常时调控细胞增殖信号，突变后持续激活增殖通路；p53抑癌基因正常时监控DNA完整性，突变后失去检验点功能。HER2在乳腺癌中扩增，导致生长信号过强；Rb在视网膜母细胞瘤中缺失，失去G1/S控制。
  3. 临床诊断应用：HER2检测用于乳腺癌分型和预后评估；p53突变检测用于多种肿瘤的风险评估；BRCA1/2突变检测用于遗传性乳腺癌筛查；K-ras突变检测用于结肠癌靶向治疗选择。
  4. 治疗应用策略：针对癌基因的抑制剂：如HER2单抗（曲妥珠单抗）、EGFR抑制剂（吉非替尼）、Ras下游通路抑制剂；针对抑癌基因缺陷的合成致死疗法：如BRCA缺陷肿瘤使用PARP抑制剂、p53缺陷肿瘤的MDM2抑制剂。
  5. 个体化医疗前景：基于肿瘤基因谱的精准治疗，多靶点联合治疗克服耐药，免疫治疗结合分子靶向治疗，为肿瘤患者提供更有效的个体化治疗方案。

4. 阐述细胞周期调控异常与肿瘤发生的关系，并分析现代肿瘤治疗中针对细胞周期的治疗策略。

参考答案：
  1. 细胞周期调控异常类型：CDK过度激活（如Cyclin D扩增、CDK4扩增）导致G1/S转换失控；检验点缺陷（如p53、ATM突变）使损伤细胞逃避监控；CKI失活（如p16缺失、p21下调）失去周期抑制；DNA修复缺陷（如BRCA1/2突变）导致基因组不稳定。
  2. 肿瘤发生机制：正常细胞周期调控确保DNA准确复制和分离，异常时导致基因组不稳定，突变累积加速。失去增殖控制的细胞获得生长优势，逐步获得恶性表型，最终形成肿瘤。
  3. CDK抑制剂治疗：CDK4/6抑制剂（如palbociclib）用于ER阳性乳腺癌，阻断G1/S转换；CDK1抑制剂用于p53缺陷肿瘤，利用合成致死原理；CDK2抑制剂正在开发中，针对特定肿瘤类型。
  4. 检验点抑制剂：Chk1抑制剂增强化疗敏感性，特别是在DNA修复缺陷肿瘤中；ATR抑制剂针对复制应激高的肿瘤；WEE1抑制剂与DNA损伤药物联用，增强疗效。
  5. 联合治疗策略：细胞周期抑制剂与传统化疗联合，增强杀伤效果；与免疫治疗联合，调节肿瘤微环境；与靶向治疗联合，克服耐药机制。个体化选择基于肿瘤分子特征，实现精准治疗。
