细胞质基质与内膜系统

考研专业课复习笔记 | 基于《细胞生物学》第五版

一、细胞质基质的涵义与功能

1.1 细胞质基质的基本概念

细胞质基质是指真核细胞的细胞质中，除去可分辨的细胞器以外的胶状物质。它是细胞质的连续相，填充在各种细胞器之间，为细胞内的各种生化反应提供场所。

主要成分：

  • 水：占细胞质基质的70-85%
  • 蛋白质：包括酶类、结构蛋白、调节蛋白等
  • 各种RNA：mRNA、tRNA、rRNA等
  • 有机分子：糖类、脂类、氨基酸、核苷酸等
  • 无机离子：K⁺、Na⁺、Mg²⁺、Ca²⁺、Cl⁻、PO₄³⁻等

1.2 细胞质基质的主要功能

（1）中间代谢过程的主要场所

细胞质基质是糖酵解、磷酸戊糖途径、脂肪酸合成等重要代谢途径的主要场所。这些代谢过程为细胞提供能量和合成原料。

（2）蛋白质合成与修饰

游离核糖体在细胞质基质中进行蛋白质合成，主要合成胞质蛋白、核蛋白、过氧化物酶体蛋白和线粒体蛋白。同时，细胞质基质也是蛋白质翻译后修饰的重要场所。

（3）蛋白质的选择性降解

通过泛素-蛋白酶体系统，细胞质基质能够选择性地降解错误折叠、损伤或不需要的蛋白质，维持细胞内蛋白质的稳态。

（4）帮助蛋白质正确折叠

细胞质基质中含有多种分子伴侣（如热激蛋白），协助新合成的蛋白质正确折叠，防止蛋白质聚集和错误折叠。

二、细胞内膜系统概述

2.1 内膜系统的定义

内膜系统是指细胞质内在形态结构、功能和发生上具有相互联系的膜相结构的总称。它包括核膜、内质网、高尔基体、溶酶体、内体和分泌泡等。

2.2 内膜系统的组成与特点

  • 结构连续性：各组分在结构上相互连接或通过膜泡运输相互联系
  • 功能协调性：各组分在功能上相互配合，形成完整的物质转运和加工体系
  • 发生相关性：各组分在发生上具有共同的起源或相互转化关系

三、内质网的结构与功能

3.1 内质网的形态结构

内质网是由单层膜围成的管状和囊状结构组成的膜系统，在细胞质中形成复杂的网状结构。根据膜表面是否附着核糖体，分为两种基本类型：

（1）糙面内质网（RER）

  • 膜表面附着大量核糖体，呈颗粒状外观
  • 主要由扁平囊状结构组成
  • 在分泌细胞中特别发达

（2）光面内质网（SER）

  • 膜表面无核糖体附着，表面光滑
  • 主要由管状结构组成
  • 在类固醇激素分泌细胞和肝细胞中发达

3.2 内质网的主要功能

（1）蛋白质合成（糙面内质网）

糙面内质网是膜蛋白和分泌蛋白合成的主要场所，包括：

  • 分泌性蛋白质（抗体、激素、酶等）
  • 膜蛋白（质膜、内膜系统的膜蛋白）
  • 可溶性驻留蛋白（内质网、高尔基体的驻留蛋白）

（2）脂质合成（光面内质网）

  • 磷脂合成：主要在内质网膜的胞质面进行
  • 胆固醇合成：所有酶都定位于内质网
  • 类固醇激素合成：在特定细胞的光面内质网中进行

（3）蛋白质修饰与加工

  • N-连接糖基化：在蛋白质的天冬酰胺残基上添加寡糖链
  • 蛋白质折叠：在分子伴侣协助下进行正确折叠
  • 二硫键形成：在蛋白质二硫键异构酶催化下形成
  • 信号肽切除：信号肽酶切除信号序列

3.3 内质网应激与细胞应答

当内质网中积累大量未折叠或错误折叠的蛋白质时，会引起内质网应激，细胞启动以下应答机制：

（1）未折叠蛋白质应答反应（UPR）

  • 增加分子伴侣的表达
  • 扩大内质网容量
  • 降低蛋白质合成速率
  • 激活内质网相关的蛋白质降解途径

（2）内质网超负荷反应（EOR）

当UPR无法缓解内质网应激时，细胞启动凋亡程序，清除受损细胞。

四、高尔基体的结构与功能

4.1 高尔基体的形态结构

高尔基体是由扁平膜囊、小管和小泡组成的膜系统，具有明显的极性结构：

（1）顺面（形成面，cis面）

  • 靠近内质网的一面
  • 由管状囊泡形成网络结构（顺面高尔基网络，CGN）
  • 功能：初级分选站，对来自内质网的蛋白质进行鉴别

（2）中间膜囊

  • 由扁平囊和管道组成
  • 功能：蛋白质糖基修饰、糖脂形成、多糖合成

（3）反面（成熟面，trans面）

  • 远离内质网的一面
  • 反面高尔基网络（TGN）呈管网状结构
  • 功能：蛋白质分类包装、"晚期"蛋白质修饰

4.2 高尔基体的主要功能

（1）蛋白质糖基化修饰

  • N-连接糖基化的修饰：去除甘露糖，添加其他糖基
  • O-连接糖基化：在丝氨酸或苏氨酸残基上添加糖基
  • 糖蛋白的成熟：形成复杂型糖链结构

（2）蛋白质分选与包装

高尔基体根据蛋白质的分选信号，将蛋白质包装到不同的载体中：

  • 分泌蛋白 → 分泌囊泡 → 细胞外
  • 膜蛋白 → 载体囊泡 → 质膜
  • 溶酶体酶 → 载体囊泡 → 溶酶体

（3）脂质代谢

  • 鞘磷脂和糖脂的合成
  • 胆固醇的酯化
  • 膜脂的修饰和转运

五、溶酶体的结构与功能

5.1 溶酶体的基本特征

溶酶体是单层膜围成的囊状细胞器，内含多种酸性水解酶，是细胞的"消化系统"。

主要特点：

  • 酸性环境：内部pH约为5.0-5.5
  • 膜结构：富含糖蛋白，具有保护作用
  • 标志酶：酸性磷酸酶、β-葡萄糖醛酸酶等
  • 异质性：大小、形态、酶含量存在差异

5.2 溶酶体的分类

（1）初级溶酶体

刚从高尔基体出芽形成的溶酶体，含有新合成的酸性水解酶，尚未参与消化活动。

（2）次级溶酶体

初级溶酶体与内吞泡、自噬泡等融合形成的溶酶体，正在进行消化活动。

（3）残质体

消化活动结束后，含有不能被消化的残留物质的溶酶体。

5.3 溶酶体的主要功能

（1）消化胞吞内容物

  • 消化通过胞饮、吞噬作用进入细胞的物质
  • 分解病原微生物
  • 处理细胞外基质成分

（2）细胞自噬

  • 大自噬：降解细胞器和大分子复合物
  • 微自噬：直接吞噬胞质成分
  • 分子伴侣介导的自噬：特异性降解某些蛋白质

（3）细胞防御

  • 清除入侵的病原体
  • 降解有害物质
  • 维持细胞内环境稳定

5.4 溶酶体的发生过程

溶酶体的形成主要通过M6P（甘露糖-6-磷酸）分选途径：

  1. 溶酶体酶在内质网合成，获得M6P标记
  2. 在高尔基体中，M6P受体识别并结合溶酶体酶
  3. 受体-酶复合物被包装到载体囊泡中
  4. 载体囊泡与早期内体融合
  5. 在酸性环境中，酶与受体分离
  6. 受体回收，酶留在溶酶体中

六、过氧化物酶体

6.1 过氧化物酶体的基本特征

过氧化物酶体是单层膜围成的细胞器，含有多种氧化酶，主要功能是进行氧化反应和解毒。

主要特点：

  • 标志酶：过氧化氢酶（catalase）
  • 基质：含有结晶状的尿酸氧化酶
  • 分布：肝细胞、肾细胞中含量丰富

6.2 过氧化物酶体的主要功能

（1）脂肪酸β氧化

特别是长链脂肪酸和支链脂肪酸的氧化分解。

（2）解毒作用

  • 分解过氧化氢（H₂O₂）
  • 氧化有毒物质（如酒精、甲醛等）
  • 参与药物代谢

（3）胆汁酸合成

参与胆固醇向胆汁酸的转化过程。

（4）嘌呤代谢

参与嘌呤碱基的分解代谢。

6.3 过氧化物酶体的发生

过氧化物酶体主要通过分裂增殖的方式产生：

  • 蛋白质在胞质核糖体合成
  • 通过PTS（过氧化物酶体靶向信号）导入
  • 过氧化物酶体生长并分裂
  • 形成新的过氧化物酶体

名词解释库

细胞质基质(Cytoplasmic Matrix)
真核细胞的细胞质中，除去可分辨的细胞器以外的胶状物质，是细胞质的连续相，主要由水、蛋白质、RNA、有机分子和无机离子组成。
内膜系统(Endomembrane System)
细胞质内在形态结构、功能和发生上具有相互联系的膜相结构的总称，包括核膜、内质网、高尔基体、溶酶体、内体和分泌泡等。
糙面内质网(Rough Endoplasmic Reticulum, RER)
膜表面附着核糖体的内质网，主要功能是合成膜蛋白和分泌蛋白，在分泌细胞中特别发达。
光面内质网(Smooth Endoplasmic Reticulum, SER)
膜表面无核糖体附着的内质网，主要功能是脂质合成、药物解毒和钙离子储存，在类固醇激素分泌细胞中发达。
信号肽(Signal Peptide)
位于新生蛋白质N端的短肽序列，指导蛋白质转运到特定的细胞器或膜系统，通常在转运完成后被切除。
分子伴侣(Molecular Chaperone)
协助蛋白质正确折叠、组装和转运的蛋白质，不参与最终产物的组成，包括热激蛋白、伴侣素等。
泛素(Ubiquitin, Ub)
由76个氨基酸组成的小蛋白质，通过共价结合标记需要降解的蛋白质，是蛋白质降解的重要调节因子。
蛋白酶体(Proteasome)
细胞质中负责降解泛素标记蛋白质的大型蛋白质复合物，是细胞内蛋白质降解的主要途径。
内质网应激(ER Stress)
当内质网中积累大量未折叠或错误折叠蛋白质时引起的细胞应激状态，激活未折叠蛋白质应答反应。
未折叠蛋白质应答(Unfolded Protein Response, UPR)
细胞对内质网应激的保护性应答机制，通过增加分子伴侣表达、扩大内质网容量等方式缓解应激。
高尔基体(Golgi Apparatus)
由扁平膜囊、小管和小泡组成的极性膜系统，主要功能是蛋白质修饰、分选和包装。
顺面高尔基网络(Cis-Golgi Network, CGN)
高尔基体靠近内质网一面的网络结构，作为初级分选站，对来自内质网的蛋白质进行鉴别。
反面高尔基网络(Trans-Golgi Network, TGN)
高尔基体远离内质网一面的网络结构，负责蛋白质的最终分类、包装和运输。
N-连接糖基化(N-linked Glycosylation)
在蛋白质天冬酰胺残基上共价连接寡糖链的翻译后修饰过程，主要在内质网和高尔基体中进行。
O-连接糖基化(O-linked Glycosylation)
在蛋白质丝氨酸或苏氨酸残基上共价连接糖基的翻译后修饰过程，主要在高尔基体中进行。
溶酶体(Lysosome)
单层膜围成的酸性细胞器，含有多种水解酶，负责细胞内物质的消化和降解。
甘露糖-6-磷酸(Mannose-6-Phosphate, M6P)
溶酶体酶的特异性标记，通过M6P受体介导溶酶体酶的分选和转运。
自噬(Autophagy)
细胞降解自身组分的过程，包括大自噬、微自噬和分子伴侣介导的自噬，是细胞维持稳态的重要机制。
过氧化物酶体(Peroxisome)
单层膜围成的细胞器，含有过氧化氢酶等氧化酶，主要功能是脂肪酸氧化和解毒。
过氧化氢酶(Catalase)
过氧化物酶体的标志酶，催化过氧化氢分解为水和氧气，具有重要的解毒功能。
膜泡运输(Vesicular Transport)
通过膜泡在内膜系统各组分间转运蛋白质和脂质的过程，是内膜系统功能协调的重要机制。
包被蛋白(Coat Protein)
覆盖在转运膜泡表面的蛋白质，如COPII、COPI、网格蛋白等，参与膜泡的形成和分选。
热激蛋白(Heat Shock Protein, HSP)
在应激条件下大量表达的分子伴侣蛋白，协助蛋白质折叠，防止蛋白质聚集和变性。
信号识别颗粒(Signal Recognition Particle, SRP)
识别信号肽并引导核糖体结合到内质网膜的核糖核蛋白复合物，是蛋白质转运的关键因子。
转位子(Translocon)
内质网膜上的蛋白质转运通道，允许新生蛋白质跨膜转运或插入膜中。

选择题测试

深度思考题

1. 试述内质网应激的分子机制及其生物学意义，并分析其与疾病发生的关系。

参考答案：
  1. 内质网应激的触发机制：当内质网中积累大量未折叠或错误折叠的蛋白质时，超过了内质网的折叠能力，导致内质网稳态失衡。这种情况可由多种因素引起，包括营养缺乏、氧化应激、钙稳态失衡、病毒感染等。内质网应激的核心是蛋白质折叠负荷与折叠能力之间的不平衡。
  2. 未折叠蛋白质应答反应（UPR）的分子机制：UPR主要通过三个信号传导途径实现：PERK途径（磷酸化eIF2α，抑制蛋白质翻译）、IRE1途径（剪切XBP1 mRNA，激活转录因子）、ATF6途径（激活转录因子，上调分子伴侣表达）。这些途径协同作用，减少新蛋白质合成，增加分子伴侣表达，扩大内质网容量，激活内质网相关降解途径。
  3. 生物学意义：UPR是细胞的保护性机制，帮助细胞适应应激条件，维持蛋白质稳态。它不仅调节蛋白质折叠，还参与脂质代谢、氧化应激应答、炎症反应等多个生物学过程。UPR的适度激活有利于细胞存活，但持续的内质网应激会导致细胞凋亡。
  4. 与疾病的关系：内质网应激与多种疾病密切相关，包括糖尿病（胰岛β细胞功能障碍）、神经退行性疾病（蛋白质聚集）、癌症（肿瘤细胞适应性）、心血管疾病（动脉粥样硬化）等。理解内质网应激机制为疾病治疗提供了新的靶点和策略。

2. 比较分析溶酶体和过氧化物酶体在结构、功能和发生机制方面的异同点。

参考答案：
  1. 结构特征比较：相同点：都是单层膜围成的细胞器，都含有特异性酶类。不同点：溶酶体内部呈酸性（pH 5.0-5.5），膜富含糖蛋白，形态多样；过氧化物酶体内部接近中性，基质常含结晶状尿酸氧化酶，形态相对规则。溶酶体的标志酶是酸性磷酸酶，过氧化物酶体的标志酶是过氧化氢酶。
  2. 功能特征比较：溶酶体主要进行水解反应，功能包括消化胞吞物质、细胞自噬、防御病原体等，是细胞的"消化系统"；过氧化物酶体主要进行氧化反应，功能包括脂肪酸β氧化、解毒、胆汁酸合成等，是细胞的"解毒工厂"。两者都参与细胞稳态维持，但作用机制不同。
  3. 发生机制比较：溶酶体通过内膜系统途径发生，酶在内质网合成，经高尔基体修饰和分选，通过M6P受体介导的途径形成；过氧化物酶体主要通过分裂增殖方式产生，蛋白质在胞质核糖体合成，通过PTS信号导入，过氧化物酶体生长后分裂形成新的过氧化物酶体。
  4. 进化和疾病意义：溶酶体的出现使真核细胞获得了强大的消化能力，过氧化物酶体的出现帮助细胞适应有氧环境。两者功能缺陷都会导致严重疾病：溶酶体储积病（如Tay-Sachs病）和过氧化物酶体病（如Zellweger综合征），说明它们在细胞生命活动中的重要性。

3. 阐述高尔基体极性结构的形成机制及其在蛋白质分选中的作用。

参考答案：
  1. 极性结构的形成机制：高尔基体的极性结构是由其功能需求决定的。顺面（cis面）靠近内质网，主要接收来自内质网的转运囊泡；反面（trans面）远离内质网，主要负责蛋白质的最终分选和输出。这种极性结构通过膜泡运输的方向性、酶的区域化分布、膜脂组成的梯度变化等机制维持。膜囊成熟模型认为，膜囊从顺面向反面移动过程中逐渐成熟，获得不同的酶活性。
  2. 蛋白质修饰的区域化：不同区域具有特异的酶活性：顺面主要进行N-连接糖基化的初步修饰，去除部分甘露糖；中间膜囊进行复杂的糖基修饰，包括添加半乳糖、岩藻糖等，同时进行O-连接糖基化；反面进行最终的糖基修饰和蛋白质分选标记的添加。这种区域化确保了蛋白质修饰的有序进行。
  3. 分选机制的精确性：反面高尔基网络（TGN）是蛋白质分选的关键部位，通过识别不同的分选信号将蛋白质包装到相应的载体中。分泌蛋白通过默认途径进入分泌囊泡，膜蛋白通过特定信号定位到质膜，溶酶体酶通过M6P标记被分选到溶酶体。这种精确的分选机制确保了蛋白质的正确定位。
  4. 调控机制和病理意义：高尔基体的极性结构受到多种因素调控，包括细胞骨架、小GTP酶、膜脂组成等。极性结构的破坏会导致蛋白质分选错误，引起细胞功能异常。在某些疾病中，如阿尔茨海默病，高尔基体结构异常与疾病发生密切相关，说明其结构完整性对细胞功能的重要性。

4. 分析细胞质基质在细胞生命活动中的多重作用及其调控机制。

参考答案：
  1. 代谢中心的作用：细胞质基质是多种重要代谢途径的场所，包括糖酵解、磷酸戊糖途径、脂肪酸合成、氨基酸代谢等。这些代谢过程为细胞提供能量（ATP）、还原力（NADPH）和合成原料。细胞质基质的胶体性质为酶-底物相互作用提供了适宜的微环境，其离子组成和pH调节影响酶活性和代谢流向。
  2. 蛋白质质量控制系统：细胞质基质含有完整的蛋白质质量控制系统，包括分子伴侣系统（协助蛋白质折叠）、泛素-蛋白酶体系统（降解错误蛋白质）、应激应答系统（应对蛋白质应激）。这些系统协同作用，确保蛋白质的正确折叠和及时清除有害蛋白质，维持细胞蛋白质稳态。
  3. 信号转导的整合平台：细胞质基质是多种信号转导途径的汇聚点，包括第二信使系统、蛋白激酶级联、转录因子激活等。胞质中的信号分子浓度、蛋白质修饰状态、分子间相互作用等都影响信号传递的效率和特异性。细胞质基质的动态性质使其能够快速响应外界刺激。
  4. 调控机制的复杂性：细胞质基质的功能受到多层次调控：转录水平（调节酶的表达量）、翻译水平（调节蛋白质合成）、翻译后修饰（调节酶活性）、代谢物反馈（调节代谢流向）、细胞骨架组织（影响空间分布）等。这种多层次调控确保了细胞质基质功能与细胞需求的精确匹配，体现了细胞调控的精密性和适应性。

5. 论述内膜系统各组分间的协调配合机制及其在细胞功能中的重要性。

参考答案：
  1. 结构连续性与功能协调：内膜系统各组分在结构上相互连接或通过膜泡运输相互联系，形成了一个功能协调的整体。内质网与核膜相连，通过COPII囊泡向高尔基体运输蛋白质；高尔基体通过不同类型的载体囊泡向溶酶体、质膜等运输蛋白质；这种结构连续性确保了物质转运的高效性和准确性。
  2. 蛋白质加工的流水线机制：从内质网到高尔基体再到最终目的地，蛋白质经历了一个完整的加工流水线：内质网负责初步折叠和N-糖基化，高尔基体负责糖基修饰和分选标记，溶酶体等负责最终功能执行。每个步骤都有特定的酶系统和质量控制机制，确保蛋白质的正确加工和定位。
  3. 膜脂代谢的协调机制：内膜系统不仅转运蛋白质，还参与膜脂的合成、修饰和分布。内质网是磷脂合成的主要场所，高尔基体参与鞘脂和糖脂的合成，不同膜结构具有特异的脂质组成。膜脂的不对称分布和动态调节影响膜蛋白功能和膜泡运输效率。
  4. 质量控制和应激应答的整合：内膜系统具有完整的质量控制网络，包括内质网的UPR系统、高尔基体的分选检查点、溶酶体的降解系统等。当系统某一环节出现问题时，会激活相应的应激应答机制，调节其他组分的功能以维持整体稳态。这种整合性质量控制确保了细胞功能的稳定性和适应性。
  5. 疾病机制和治疗靶点：内膜系统功能异常与多种疾病相关，如蛋白质折叠病、溶酶体储积病、分泌功能障碍等。理解内膜系统的协调机制为疾病诊断和治疗提供了重要线索，如分子伴侣增强剂、蛋白酶体抑制剂、基因治疗等都是基于内膜系统功能的治疗策略。
