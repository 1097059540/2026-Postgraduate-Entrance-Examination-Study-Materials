分子生物学研究法（上）——DNA、RNA及蛋白质操作技术

基于《现代分子生物学》第五版 朱玉贤主编 · 考研复习笔记

第一部分：核心知识要点

一、重组DNA技术基础

重组DNA技术是现代分子生物学的核心技术，通过体外重组不同来源的DNA分子，构建重组载体，实现基因克隆和表达。

1. 基因克隆的基本步骤

  1. 目的基因的获取：通过PCR扩增、cDNA合成或基因组DNA提取获得
  2. 载体的选择和处理：质粒载体、噬菌体载体、人工染色体载体等
  3. DNA连接：使用DNA连接酶将目的基因与载体连接
  4. 转化：将重组载体导入宿主细胞
  5. 筛选和鉴定：通过抗性筛选和分子鉴定确认重组子

2. 载体系统分类

  • 质粒载体：pBR322、pUC系列，适合小片段DNA克隆（<10kb）
  • 噬菌体载体：λ噬菌体载体，适合中等大小DNA片段（10-20kb）
  • 人工染色体载体：YAC、BAC、PAC，适合大片段DNA克隆（>100kb）

二、PCR技术原理与应用

聚合酶链式反应（PCR）是体外扩增特定DNA序列的技术，具有高特异性、高敏感性和快速便捷的特点。

1. PCR反应原理

PCR反应包含三个基本步骤，在热循环仪中重复进行：

  1. 变性（Denaturation）：94-96°C，DNA双链解开
  2. 退火（Annealing）：50-65°C，引物与模板结合
  3. 延伸（Extension）：72°C，DNA聚合酶合成新链

2. PCR反应体系组成

  • 模板DNA：含有目标序列的DNA
  • 引物：正向和反向引物，长度通常为18-25个核苷酸
  • DNA聚合酶：耐热DNA聚合酶，如Taq酶
  • dNTPs：四种脱氧核苷三磷酸
  • 缓冲液：维持适宜的pH和离子强度
  • Mg²⁺：DNA聚合酶的辅助因子

3. PCR技术的改进与应用

  • 实时荧光PCR（qPCR）：定量检测DNA/RNA，使用SYBR Green I或TaqMan探针
  • 逆转录PCR（RT-PCR）：检测RNA，先逆转录成cDNA再PCR扩增
  • 巢式PCR：提高特异性，使用两轮PCR扩增
  • 多重PCR：同时扩增多个目标序列

三、核酸电泳技术

核酸电泳是分离和分析DNA、RNA的基本技术，基于核酸分子在电场中的迁移速率差异。

1. 琼脂糖凝胶电泳

  • 原理：DNA在琼脂糖凝胶中按分子量大小分离
  • 应用：DNA片段大小分析、纯化回收
  • 检测方法：溴化乙锭（EB）染色、SYBR Safe等安全染料

2. 聚丙烯酰胺凝胶电泳（PAGE）

  • 变性PAGE：分离单链DNA/RNA，使用尿素变性剂
  • 非变性PAGE：保持核酸二级结构，分析蛋白质-核酸相互作用

四、RNA操作技术

RNA操作技术包括RNA提取、逆转录、RNA干扰等，是研究基因表达和功能的重要手段。

1. RNA提取与纯化

  • 总RNA提取：TRIzol法、柱纯化法
  • mRNA纯化：Oligo(dT)磁珠法，利用poly(A)尾巴
  • 非编码RNA分离：差异离心、凝胶电泳分离

2. cDNA合成技术

互补DNA（cDNA）合成是以mRNA为模板，通过逆转录酶合成的DNA拷贝。

  • 引物选择：Oligo(dT)、随机引物、基因特异性引物
  • 逆转录酶：M-MLV、SuperScript等
  • 方向性克隆：定向cDNA文库构建

3. RACE技术

快速扩增cDNA末端（RACE）技术用于获得完整长度的cDNA序列。

  • 5' RACE：获得mRNA的5'端序列
  • 3' RACE：获得mRNA的3'端序列
  • RAMPAGE技术：改进的RACE技术，提高效率和准确性

五、蛋白质操作技术

蛋白质操作技术包括蛋白质分离纯化、结构分析和功能研究等方面。

1. 蛋白质电泳技术

  • SDS-PAGE：变性聚丙烯酰胺凝胶电泳，按分子量分离
  • Native-PAGE：非变性电泳，保持蛋白质天然结构
  • 等电聚焦电泳：按等电点分离蛋白质

2. 二维电泳技术

  • 2-DE：等电聚焦+SDS-PAGE，高分辨率蛋白质分离
  • 2D-DIGE：差异凝胶电泳，比较不同样品间蛋白质表达差异

3. 质谱技术

  • MALDI-TOF MS：基质辅助激光解吸电离飞行时间质谱
  • ESI-MS：电喷雾电离质谱
  • 蛋白质组学：大规模蛋白质鉴定和定量分析

六、基因文库构建技术

基因文库是克隆载体的集合，包含某一生物体的全部或部分基因信息。

1. 基因组DNA文库

  • 构建原理：将基因组DNA随机切割后克隆到载体中
  • 特点：包含内含子、外显子和调控序列
  • 应用：基因定位、启动子分析

2. cDNA文库

  • 构建原理：以mRNA为模板合成cDNA后克隆
  • 特点：只含编码序列，无内含子
  • 应用：基因表达、蛋白质生产

七、现代分子克隆技术

1. Gateway克隆技术

Gateway克隆基于λ噬菌体的位点特异性重组系统，实现高效的基因克隆。

  • attB×attP重组：BP反应，产生Entry克隆
  • attL×attR重组：LR反应，产生Expression克隆

2. 等温一步法载体构建

  • Gibson Assembly：基于同源重组的无缝克隆
  • In-Fusion克隆：利用15bp同源序列进行重组

第二部分：重要名词解释库

重组DNA技术(Recombinant DNA Technology)

将不同来源的DNA分子在体外进行切割和重新组合，构建具有新遗传特性的重组DNA分子的技术。是现代分子生物学和基因工程的核心技术。

聚合酶链式反应(Polymerase Chain Reaction, PCR)

一种体外扩增特定DNA序列的技术，通过变性、退火、延伸三步循环，实现DNA的指数级扩增。具有高特异性、高敏感性和快速便捷的特点。

载体(Vector)

能够自主复制并携带外源DNA片段的DNA分子，包括质粒载体、噬菌体载体、人工染色体载体等。载体是基因克隆的重要工具。

互补DNA(Complementary DNA, cDNA)

以mRNA为模板，通过逆转录酶合成的DNA拷贝。cDNA只包含编码序列，不含内含子，常用于基因表达和蛋白质生产。

基因文库(Gene Library)

包含某一生物体全部或部分基因信息的克隆载体集合。分为基因组DNA文库和cDNA文库两大类。

RACE技术(Rapid Amplification of cDNA Ends)

快速扩增cDNA末端技术，用于获得完整长度的cDNA序列。包括5' RACE和3' RACE两种方法。

实时荧光PCR(Real-time PCR, qPCR)

在PCR扩增过程中实时检测产物的技术，可实现DNA/RNA的定量分析。常用SYBR Green I或TaqMan探针进行检测。

二维电泳(Two-Dimensional Electrophoresis, 2-DE)

结合等电聚焦和SDS-PAGE的蛋白质分离技术，可实现高分辨率的蛋白质分离，广泛应用于蛋白质组学研究。

Gateway克隆(Gateway Cloning)

基于λ噬菌体位点特异性重组系统的克隆技术，通过attB×attP和attL×attR重组反应实现高效基因克隆。

TAIL-PCR(Thermal Asymmetric Interlaced PCR)

热不对称交错PCR技术，用于扩增已知序列侧翼的未知DNA序列，常用于转基因植物的插入位点分析。

第四部分：深度思考题

1. 比较分析PCR技术与传统基因克隆技术的优缺点，并论述PCR技术在现代分子生物学研究中的重要地位。

  1. PCR技术的优点：
    • 快速高效：数小时内完成DNA扩增，而传统克隆需要数天至数周
    • 高特异性：通过引物设计实现特定序列的精确扩增
    • 高敏感性：可从极少量模板（单个细胞）中扩增目标序列
    • 操作简便：自动化程度高，减少人工操作误差
    • 成本较低：不需要复杂的载体构建和细胞培养
  2. PCR技术的缺点：
    • 扩增长度限制：常规PCR难以扩增超过10kb的长片段
    • 引入突变：DNA聚合酶缺乏3'-5'外切酶活性可能引入错误
    • 需要已知序列信息：引物设计需要目标序列的部分信息
    • 污染敏感：高敏感性导致容易受到外源DNA污染
  3. 传统基因克隆的优势：
    • 可克隆大片段DNA：载体系统支持大容量基因克隆
    • 保真度高：宿主细胞的修复系统保证序列准确性
    • 稳定保存：克隆子可长期保存和传代
    • 功能研究：可进行基因表达和功能分析
  4. PCR在现代分子生物学中的重要地位：
    • 基因诊断：疾病基因检测、病原体检测
    • 法医学应用：DNA指纹分析、亲子鉴定
    • 进化研究：古DNA分析、系统发育研究
    • 基因工程：基因克隆前的预扩增、定点突变
    • 定量分析：qPCR技术实现基因表达定量检测

2. 详细阐述基因组DNA文库与cDNA文库的构建原理、特点差异及其在分子生物学研究中的不同应用。

  1. 基因组DNA文库构建原理：
    • DNA提取：从细胞或组织中提取完整的基因组DNA
    • 随机切割：使用限制性内切酶或物理方法将基因组DNA切割成适当大小的片段
    • 载体连接：将DNA片段与载体连接形成重组分子
    • 转化筛选：将重组载体导入宿主细胞并筛选阳性克隆
  2. cDNA文库构建原理：
    • mRNA提取：从特定组织或细胞中提取总RNA或mRNA
    • 逆转录：使用逆转录酶以mRNA为模板合成第一链cDNA
    • 第二链合成：合成互补的第二链形成双链cDNA
    • 载体克隆：将cDNA插入载体并转化宿主细胞
  3. 两种文库的主要差异：
    • 序列完整性：基因组文库包含完整基因结构（外显子+内含子+调控序列），cDNA文库仅含编码序列
    • 大小差异：基因组文库片段较大，cDNA文库片段相对较小
    • 表达特异性：cDNA文库反映特定时期、组织的基因表达模式
    • 克隆难度：基因组文库构建相对简单，cDNA文库需要高质量RNA
  4. 应用领域差异：
    • 基因组文库应用：基因定位与作图、启动子和调控序列分析、基因组结构研究、转基因动物构建
    • cDNA文库应用：蛋白质表达与纯化、基因功能研究、药物靶点筛选、进化分析
    • 互补使用：基因组文库提供结构信息，cDNA文库提供功能信息

3. 分析蛋白质组学研究中二维电泳技术的原理、优势及局限性，并讨论质谱技术如何弥补这些不足。

  1. 二维电泳技术原理：
    • 第一维等电聚焦：蛋白质在pH梯度中迁移至其等电点位置
    • 第二维SDS-PAGE：按分子量大小进一步分离蛋白质
    • 检测显色：银染、考马斯亮蓝染色或荧光标记检测
    • 图像分析：计算机辅助分析蛋白质斑点的位置、强度和体积
  2. 二维电泳的优势：
    • 高分辨率：可同时分离数千个蛋白质
    • 直观性：蛋白质表达模式一目了然
    • 定量分析：可比较不同条件下蛋白质表达差异
    • 重现性：标准化操作保证结果可重复
    • 成本效益：相对经济的蛋白质组学研究方法
  3. 二维电泳的局限性：
    • 动态范围有限：难以检测低丰度蛋白质
    • 疏水蛋白问题：膜蛋白等疏水蛋白难以溶解和分离
    • 极端pI蛋白：极酸性或极碱性蛋白质分离困难
    • 分子量限制：极大或极小分子量蛋白质分离效果差
    • 操作复杂：需要熟练技术和标准化流程
  4. 质谱技术的补充作用：
    • 蛋白质鉴定：通过肽质量指纹图谱或串联质谱确定蛋白质身份
    • 定量蛋白质组学：SILAC、iTRAQ、TMT等标记技术实现精确定量
    • 翻译后修饰分析：磷酸化、糖基化、泛素化等修饰的检测
    • 蛋白质相互作用：免疫共沉淀结合质谱分析蛋白质复合物
    • 无凝胶方法：液相色谱-质谱联用技术避免凝胶电泳的局限性

4. 论述现代基因克隆技术（如Gateway克隆、Gibson Assembly）相比传统限制酶克隆的技术革新及其在功能基因组学研究中的应用前景。

  1. 传统限制酶克隆的局限性：
    • 酶切位点依赖：需要合适的限制酶切位点，限制了克隆灵活性
    • 多步骤操作：酶切、去磷酸化、连接等多个步骤，效率较低
    • 方向性问题：需要特殊设计保证插入方向正确
    • 载体改造困难：更换载体需要重新设计克隆策略
  2. Gateway克隆技术革新：
    • 位点特异性重组：基于λ噬菌体重组系统，无需限制酶
    • 方向性保证：attB和attP位点确保正确方向插入
    • 载体转换便利：一次克隆可转换到多种表达载体
    • 高通量兼容：适合大规模基因克隆项目
    • 减少操作步骤：BP和LR反应简化克隆流程
  3. Gibson Assembly技术优势：
    • 无缝克隆：基于同源重组，无需引入额外序列
    • 多片段组装：可同时组装多个DNA片段
    • 灵活性高：不依赖特定酶切位点
    • 一步反应：等温条件下完成组装
    • 高效率：组装成功率高，背景低
  4. 在功能基因组学中的应用前景：
    • 高通量基因功能研究：快速构建基因过表达或敲除载体
    • 合成生物学：设计和构建人工生物系统
    • 代谢工程：重构代谢途径，优化产物合成
    • 蛋白质工程：快速构建突变体文库进行定向进化
    • 基因治疗载体：高效构建治疗性基因载体
    • 农业生物技术：作物改良和新品种培育

5. 结合具体实例，分析RNA干扰技术与CRISPR基因编辑技术在基因功能研究中的应用原理、优缺点比较及发展趋势。

  1. RNA干扰技术原理与应用：
    • 作用机制：小干扰RNA（siRNA）引导RISC复合物特异性降解目标mRNA
    • 应用实例：线虫基因功能研究、植物抗病毒、哺乳动物基因敲减
    • 技术优势：操作简便、成本低廉、可逆性、适合高通量筛选
    • 局限性：敲减效率不稳定、脱靶效应、无法完全敲除基因
  2. CRISPR基因编辑技术原理：
    • 作用机制：Cas9蛋白在gRNA引导下特异性切割目标DNA序列
    • 编辑类型：基因敲除、敲入、点突变、表观遗传修饰
    • 应用实例：疾病模型构建、基因治疗、作物改良、药物筛选
  3. 两种技术的比较分析：
    • 作用层面：RNAi作用于转录后水平，CRISPR作用于DNA水平
    • 效果持久性：RNAi效果可逆且短暂，CRISPR产生永久性改变
    • 特异性：CRISPR特异性更高，脱靶效应相对较少
    • 技术难度：RNAi操作更简便，CRISPR需要更精确的设计
    • 应用范围：CRISPR应用更广泛，可进行多种类型的基因编辑
  4. 发展趋势与前景：
    • 技术融合：RNAi与CRISPR结合使用，实现时空特异性调控
    • 精确度提升：碱基编辑、引导编辑等技术提高编辑精确度
    • 递送系统优化：改进载体和递送方法，提高体内应用效率
    • 安全性增强：减少脱靶效应，提高临床应用安全性
    • 应用拓展：从基础研究扩展到临床治疗和农业应用
    • 伦理规范：建立完善的伦理审查和监管体系
