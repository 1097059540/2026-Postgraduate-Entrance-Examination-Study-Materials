疾病与人类健康 - 考研复习笔记

第一部分：核心知识体系

一、肿瘤与癌症

1. 基本概念

肿瘤是机体在各种致瘤因素作用下，局部组织的细胞在基因水平上失去对其生长的正常调控，导致异常增生而形成的新生物。

癌症是恶性肿瘤的统称，具有细胞分化和增殖异常、生长失去控制、浸润性和转移性等生物学特征。

2. 癌基因与抑癌基因

  • 原癌基因：细胞内与细胞增殖相关的基因，对维持机体正常生命活动必需
  • 癌基因：能够潜在地诱导细胞发生癌变的基因，分为病毒癌基因和细胞转化基因
  • 抑癌基因：又称肿瘤抑制基因，正常时起抑制细胞增殖和肿瘤发生的作用

原癌基因激活机制

  1. 点突变：单个碱基替换，改变蛋白质结构
  2. LTR插入：长末端重复序列插入，影响基因表达
  3. 基因重排：原癌基因间或与非原癌基因间的重排
  4. 缺失：负调控序列缺失或突变
  5. 基因扩增：基因拷贝数增加，导致过量表达

二、病毒感染与疾病

1. HIV（人类免疫缺陷病毒）

结构特征：属反转录病毒科，球状、20面体对称，直径约100nm，基因组由两条单链正链RNA组成

主要基因：gag（结构蛋白）、pol（酶蛋白）、env（外膜蛋白）

HIV致病机理

  • 主要攻击CD4⁺T淋巴细胞
  • 病毒在细胞内增殖，干扰正常代谢
  • 形成多核巨细胞导致细胞死亡
  • 诱导自身免疫应答
  • 编码超级抗原过度激活T细胞

2. HBV（乙型肝炎病毒）

基因组特征：环状双链DNA分子，有部分单链区，两条单链长度不同

cccDNA：共价闭合环状DNA，是乙肝病毒前基因组RNA复制的原始模板

3. 禽流感病毒（AIV）

感染人类的机制

  • HA受体结合位点突变：使病毒易于感染人类细胞
  • PB2蛋白627位氨基酸突变：增强病毒复制能力
  • NS1第92位氨基酸突变：显著增强致病能力

4. SARS-CoV（严重急性呼吸综合征冠状病毒）

主要通过近距离飞沫传播和直接接触传播，相比人禽流感更易控制，因为可通过隔离有效切断传播途径。

三、基因治疗

基因治疗：通过一定方式将人正常基因或有治疗作用的DNA片段导入人体靶细胞以矫正或置换致病基因的治疗方法。

1. 基因治疗策略

  1. 基因置换：用正常基因替换致病基因
  2. 基因修复：对致病基因进行精确修复
  3. 基因增补：插入正常基因而不删除突变基因
  4. 基因沉默：抑制致病基因的异常表达
  5. 自杀基因治疗：诱导肿瘤细胞"自杀"

2. 基因治疗途径

  • ex vivo途径：体外导入基因后输回人体
  • in vivo途径：直接将基因导入人体内

3. 载体系统

病毒载体 vs 非病毒载体

载体类型 优点 缺点
病毒载体 转化效率高、宿主范围广、易于制备 随机整合风险、免疫原性高、安全性问题
非病毒载体 毒性低、免疫反应小、使用简单、无大小限制 转导效率低、表达短暂、易被降解

四、肿瘤免疫治疗

CAR-T细胞疗法

CAR-T细胞疗法：通过基因修饰获得携带识别肿瘤抗原特异性受体T细胞的个性化治疗方法。

优点

  • 治疗更精准，克服肿瘤免疫逃脱
  • 多靶向，不受MHC限制
  • 杀瘤范围广，可广泛使用
  • 效果持久，具有免疫记忆功能

缺点与风险

  • 细胞因子风暴：可导致高烧、低压、休克甚至死亡
  • 毒副作用：感染、发热、乏力等
  • 疾病复发问题
  • 对实体瘤效果有限

第二部分：重要名词解释库

肿瘤
Tumor/Neoplasm
机体在各种致瘤因素作用下，局部组织的细胞在基因水平上失去对其生长的正常调控，导致异常增生而形成的新生物。
原癌基因
Proto-oncogene
细胞内与细胞增殖相关的基因，对于维持机体正常生命活动来说是必需的，是具有转变成癌基因潜力的正常基因，在进化过程中高度保守。
癌基因
Oncogene
一种能够潜在地诱导它所存在的细胞发生癌变的基因，分为病毒癌基因(V-onc)和细胞转化基因(C-onc)。
抑癌基因
Tumor Suppressor Gene
又称抗癌基因，抑癌基因的存在使其在被激活情况下具有抑制细胞增殖作用，但在一定情况下被抑制或丢失后可减弱甚至消除抑癌作用。
基因治疗
Gene Therapy
通过一定方式将人正常基因或有治疗作用的DNA片段导入人体靶细胞以矫正或置换致病基因的治疗方法，是一种以改变人遗传物质为基础的生物医学治疗手段。
基因诊断
Genetic Diagnosis
又称DNA诊断或分子诊断，是指利用分子生物学技术和方法直接检测基因结构及其表达水平是否正常，从而对疾病作出诊断的方法。
逆转录酶
Reverse Transcriptase
又称RNA指导的DNA聚合酶，是存在于一部分RNA病毒中具有逆转录活性的一种特殊的DNA聚合酶。
cccDNA
Covalently Closed Circular DNA
共价闭合环状DNA，是在乙肝病毒的复制过程中，病毒DNA进入宿主细胞核，在DNA聚合酶的作用下，两条链的缺口均被补齐，形成超螺旋的共价、闭合、环状DNA分子。
基因领域效应
Gene Territory Effect
同一DNA链上两个具有相同转录方向的基因间隔小于一定长度时，影响有效转录所必需的染色质结构的形成，从而使这两个基因中的一个或两个不能转录或转录活性显著降低的现象。
CAR-T细胞疗法
Chimeric Antigen Receptor T-cell Therapy
通过基因修饰获得携带识别肿瘤抗原特异性受体T细胞的个性化治疗方法。提取患者外周血T细胞，通过基因重组赋予T细胞特异性识别肿瘤抗原的能力。
细胞因子风暴
Cytokine Storm
当CAR-T细胞快速杀死癌细胞时，产生的大量细胞因子会对宿主其他组织器官发起攻击，导致高烧、低压、休克甚至死亡的严重免疫反应。
病毒载体
Viral Vector
经过改造的病毒，删除其致病基因，保留感染和基因表达能力，用于将治疗基因导入靶细胞的载体系统。

1. 试分析原癌基因激活成为癌基因的分子机制，并讨论这些机制在肿瘤发生发展中的作用。

参考答案：

  1. 点突变机制：原癌基因编码序列发生点突变，导致蛋白质结构改变，功能异常激活。如ras基因的点突变导致其编码的GTP酶活性丧失，持续激活细胞增殖信号。这种机制在多种肿瘤中都有发现，是肿瘤发生的重要分子基础。
  2. 基因重排机制：染色体易位、倒位等重排事件使原癌基因与强启动子或增强子结合，导致基因过度表达。典型例子是Burkitt淋巴瘤中c-myc基因与免疫球蛋白重链基因重排，导致c-myc过度表达，促进细胞无限增殖。
  3. 基因扩增机制：原癌基因拷贝数增加，导致基因产物过量表达。如乳腺癌中HER2基因扩增，导致生长因子受体过度表达，细胞对生长信号过度敏感，失去正常的生长控制。
  4. 插入激活机制：病毒DNA或转座子插入原癌基因附近，激活其表达。这种机制在病毒致癌过程中尤为重要，病毒整合位点的选择性决定了肿瘤发生的类型和特征。
  5. 表观遗传修饰：DNA甲基化、组蛋白修饰等表观遗传变化影响原癌基因表达调控，导致异常激活。这种机制的可逆性为肿瘤治疗提供了新的靶点和策略。

2. 基因治疗作为新兴的医疗技术，面临哪些主要挑战？请从技术、安全性和伦理角度进行全面分析。

参考答案：

  1. 技术挑战：载体系统的转导效率有限，难以实现精确的组织特异性靶向。病毒载体存在包装容量限制，非病毒载体转导效率低。基因表达的时空调控困难，难以模拟内源基因的精确表达模式。这些技术瓶颈限制了基因治疗的临床应用效果。
  2. 安全性问题：载体的免疫原性可能引起严重免疫反应，甚至危及生命。插入突变风险可能激活癌基因或失活抑癌基因，导致肿瘤发生。基因表达的不可控性可能产生意想不到的生物学效应。长期安全性评估困难，需要长期随访观察。
  3. 伦理考量：生殖细胞基因编辑涉及后代遗传，存在伦理争议。基因增强与基因治疗的界限模糊，可能导致社会不公。知情同意的复杂性，患者难以完全理解潜在风险。基因信息的隐私保护和歧视防范问题。
  4. 监管挑战：缺乏统一的国际监管标准，不同国家法规差异较大。临床试验设计复杂，评价标准不统一。成本效益分析困难，医疗资源分配的公平性问题。
  5. 社会接受度：公众对基因治疗的认知不足，存在恐惧和误解。媒体报道的影响，可能放大风险或过度宣传效果。医患沟通的挑战，需要专业的遗传咨询服务。

3. HIV作为获得性免疫缺陷病毒，其致病机理复杂多样。请详细分析HIV感染、复制及致病的分子机制。

参考答案：

  1. 病毒结构与感染机制：HIV为逆转录病毒，具有包膜结构。gp120糖蛋白识别并结合CD4受体和CCR5/CXCR4共受体，gp41介导膜融合过程。病毒进入细胞后，逆转录酶将RNA基因组逆转录为DNA，整合酶将病毒DNA整合到宿主染色体中，形成前病毒。
  2. 复制周期特点：HIV复制依赖宿主细胞的转录机制，但逆转录酶缺乏校正功能，导致高突变率。病毒可在潜伏感染和活跃复制间转换，潜伏感染使病毒逃避免疫监视和药物作用。复制过程中产生大量病毒颗粒，持续破坏免疫细胞。
  3. 免疫系统破坏机制：HIV主要感染CD4+ T细胞、巨噬细胞和树突状细胞，直接杀伤效应导致免疫细胞数量减少。病毒蛋白干扰细胞正常功能，影响免疫应答的启动和维持。慢性免疫激活导致免疫系统功能紊乱和衰竭。
  4. 变异与耐药机制：逆转录酶的高错误率和病毒的快速复制导致基因组高度变异。选择压力下产生耐药突变株，单一药物治疗容易失败。病毒准种的存在使得完全清除病毒极其困难。
  5. 临床表现的分子基础：免疫缺陷导致机会性感染和肿瘤发生。病毒直接作用和免疫反应共同导致器官损伤。神经系统感染引起AIDS痴呆综合征。慢性炎症状态导致心血管疾病等并发症增加。

4. 肿瘤免疫治疗代表了癌症治疗的新方向。请分析当前主要的肿瘤免疫治疗策略及其作用机制。

参考答案：

  1. 免疫检查点抑制剂：PD-1/PD-L1和CTLA-4抑制剂通过阻断免疫抑制信号，重新激活T细胞的抗肿瘤活性。这些检查点分子正常情况下防止免疫过度激活，但肿瘤细胞利用这一机制逃避免疫监视。抑制剂的应用解除了免疫"刹车"，使T细胞能够有效识别和杀伤肿瘤细胞。
  2. CAR-T细胞治疗：通过基因工程技术改造患者T细胞，使其表达嵌合抗原受体，特异性识别肿瘤抗原。CAR结构包括胞外抗原识别域、跨膜域和胞内信号域，能够绕过MHC限制直接识别肿瘤细胞。治疗过程包括T细胞采集、体外改造扩增和回输，在血液肿瘤治疗中显示出显著疗效。
  3. 肿瘤疫苗：通过呈递肿瘤特异性抗原激活免疫系统产生特异性免疫应答。包括多肽疫苗、DNA疫苗、病毒载体疫苗等多种形式。个体化新抗原疫苗基于患者肿瘤的基因突变谱设计，理论上具有更好的特异性和安全性。
  4. 过继性细胞治疗：包括肿瘤浸润淋巴细胞(TIL)治疗、TCR-T细胞治疗等。TIL治疗利用肿瘤组织中已经识别肿瘤抗原的T细胞，体外扩增后回输。TCR-T细胞治疗通过导入高亲和力TCR增强T细胞的肿瘤识别能力。
  5. 免疫调节剂：细胞因子如白介素-2、干扰素等能够增强免疫细胞功能。免疫佐剂如TLR激动剂能够激活先天免疫，促进适应性免疫应答的产生。单克隆抗体如抗CD20、抗HER2等通过抗体依赖的细胞毒作用杀伤肿瘤细胞。

5. 病毒在人类疾病中扮演重要角色。请比较分析DNA病毒和RNA病毒的致病特点，并讨论病毒感染的预防控制策略。

参考答案：

  1. DNA病毒致病特点：基因组相对稳定，突变率较低，感染后容易建立持续感染。如HBV能够形成cccDNA，在肝细胞核内长期存在，导致慢性感染。HPV感染可导致宫颈癌，病毒DNA整合到宿主基因组中，干扰细胞周期调控。疱疹病毒具有潜伏感染特性，在免疫抑制时重新激活。
  2. RNA病毒致病特点：基因组不稳定，突变率高，容易产生新的变异株。流感病毒的抗原漂移和抗原转换导致季节性流行和大流行。HIV的高变异率使其能够逃避免疫监视和产生耐药性。SARS-CoV-2的变异导致传播力和致病力的改变，影响疫苗效果。
  3. 疫苗预防策略：预防性疫苗是最有效的预防手段，通过激活免疫记忆提供长期保护。活疫苗免疫效果好但安全性相对较低，灭活疫苗安全性高但免疫效果相对较弱。mRNA疫苗代表了疫苗技术的新发展，能够快速应对新发病毒。
  4. 抗病毒药物治疗：核苷类似物如阿昔洛韦、利巴韦林等干扰病毒DNA/RNA合成。蛋白酶抑制剂如洛匹那韦阻断病毒蛋白质加工。神经氨酸酶抑制剂如奥司他韦阻止流感病毒释放。广谱抗病毒药物的研发是当前重点方向。
  5. 公共卫生措施：监测系统能够及时发现和报告病毒感染病例，为防控决策提供依据。隔离和检疫措施能够阻断传播链，控制疫情扩散。个人防护措施如戴口罩、手卫生等简单有效。健康教育提高公众的防病意识和能力。国际合作对于控制跨境传播的病毒感染至关重要。
