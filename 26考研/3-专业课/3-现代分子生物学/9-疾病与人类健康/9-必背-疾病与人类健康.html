国科大生命科学学院 细胞生物学疾病与人类健康必背考点

一、核心名词解释（10个）

原癌基因（Proto-oncogene）：正常细胞中参与调节细胞生长、分化和凋亡的基因，当被激活或过度表达时可转化为癌基因，促进细胞恶性转化。主要产物包括生长因子、生长因子受体、信号转导蛋白和转录因子。

抑癌基因（Tumor Suppressor Gene）：正常细胞中抑制细胞生长和增殖的基因，具有潜在抑癌作用。当发生突变、缺失或失活时可引起细胞恶性转化，典型代表有p53和Rb基因。

基因治疗（Gene Therapy）：将外源正常基因或具有治疗功能的基因导入靶细胞，以纠正缺陷基因或提供新的细胞功能，从而达到治疗疾病目的的生物医学技术。

CAR-T细胞疗法（Chimeric Antigen Receptor T-cell Therapy）：通过基因工程技术改造患者T细胞，使其表达嵌合抗原受体，能特异性识别并杀伤肿瘤细胞的免疫治疗方法。

病毒载体（Viral Vector）：经过基因改造的病毒，去除其致病基因后用于携带治疗基因进入靶细胞的基因传递工具。常用载体包括腺相关病毒(AAV)、腺病毒和慢病毒载体。

HIV（Human Immunodeficiency Virus）：人类免疫缺陷病毒，属逆转录病毒，主要感染CD4+ T细胞，导致获得性免疫缺陷综合征(AIDS)。基因组包含gag、pol、env等结构基因和调节基因。

HBV（Hepatitis B Virus）：乙型肝炎病毒，属DNA病毒，具有独特的双链环状DNA基因组和cccDNA(共价闭合环状DNA)复制中间体，可引起急性和慢性肝炎。

细胞因子风暴（Cytokine Storm）：机体免疫系统过度激活时大量释放促炎细胞因子，导致全身炎症反应失控的病理状态，是CAR-T治疗的主要副作用之一。

基因沉默治疗（Gene Silencing Therapy）：通过RNAi、反义寸核苷酸等技术特异性抑制致病基因表达的治疗策略，常用于抑制癌基因或病毒基因的表达。

肿瘤免疫逃逸（Tumor Immune Evasion）：肿瘤细胞通过多种机制逃避机体免疫系统识别和杀伤的现象，包括抗原丢失、免疫抑制因子分泌、免疫检查点激活等。

二、必备核心知识点（5个）

【必备知识点1】原癌基因激活机制
1. 点突变：单个碱基改变导致蛋白质功能异常，如ras基因第12、13、61位密码子突变
2. LTR插入：逆转录病毒长末端重复序列插入原癌基因上游，激活基因转录
3. 基因重排：染色体易位导致原癌基因与强启动子融合，如c-myc基因重排
4. 基因扩增：原癌基因拷贝数增加，导致蛋白质过量表达
5. 缺失：调节序列缺失导致原癌基因失控表达

【必备知识点2】HIV病毒结构与致病机理
1. 结构特征：球形包膜病毒，直径100-120nm，含两条相同的单链RNA基因组
2. 主要基因：gag(结构蛋白)、pol(酶类)、env(包膜蛋白)、调节基因(tat、rev、nef等)
3. 感染机制：gp120结合CD4受体和共受体，膜融合后逆转录整合入宿主基因组
4. 致病机理：破坏CD4+ T细胞，导致免疫功能缺陷，易发生机会性感染和肿瘤
5. 变异特点：逆转录酶缺乏校正功能，突变率高，易产生耐药株

【必备知识点3】基因治疗策略与途径
1. 基因置换：用正常基因替代缺陷基因，治疗单基因遗传病
2. 基因修复：通过同源重组修复突变基因，恢复正常功能
3. 基因增补：导入正常基因补偿缺陷基因功能，不替代原有基因
4. 基因沉默：抑制致病基因表达，如癌基因或病毒基因
5. 自杀基因治疗：导入自杀基因使肿瘤细胞对特定药物敏感而死亡

【必备知识点4】病毒载体与非病毒载体比较
1. 病毒载体优点：转导效率高、靶向性强、可感染非分裂细胞
2. 病毒载体缺点：免疫原性强、载量有限、制备复杂、安全性担忧
3. 非病毒载体优点：安全性高、载量大、制备简单、免疫原性低
4. 非病毒载体缺点：转导效率低、靶向性差、基因表达不稳定
5. 载体选择：根据疾病类型、靶器官、治疗目标选择合适载体系统

【必备知识点5】CAR-T细胞疗法机制与局限性
1. 治疗机制：基因改造T细胞表达嵌合抗原受体，特异性识别肿瘤抗原并杀伤肿瘤细胞
2. 主要优点：靶向性强、杀伤力强、可体外扩增、记忆效应持久
3. 细胞因子风暴：T细胞过度激活释放大量细胞因子，可危及生命
4. 实体瘤限制：肿瘤微环境复杂、抗原异质性、T细胞浸润困难
5. 脱靶效应：可能攻击表达相同抗原的正常组织，引起严重副作用

三、高频必考大题（2道）

【高频考题1】试述原癌基因与抑癌基因在肿瘤发生中的作用机制，并分析两者在细胞生长调控网络中的相互关系。

1. 原癌基因作用机制：正常情况下调节细胞生长分化，激活后促进细胞增殖。激活方式包括点突变、基因扩增、染色体易位、病毒插入等，导致生长因子、受体、信号蛋白过度表达。

2. 抑癌基因作用机制：正常情况下抑制细胞生长，维持细胞周期检查点。失活方式包括基因缺失、点突变、甲基化沉默等，导致细胞周期失控、DNA修复缺陷、凋亡抵抗。

3. 相互关系：两者构成细胞生长调控的正负反馈系统，原癌基因提供"油门"信号，抑癌基因提供"刹车"信号，平衡维持细胞正常生长。

4. 肿瘤发生机制：需要多个原癌基因激活和抑癌基因失活的累积效应，打破生长调控平衡，导致细胞恶性转化。典型例子如p53失活配合ras激活。

5. 临床意义：为肿瘤诊断、预后判断和靶向治疗提供分子标志物，如HER2过表达、p53突变检测等。

【高频考题2】比较分析病毒载体和非病毒载体在基因治疗中的优缺点，并结合具体实例说明如何选择合适的载体系统。

1. 病毒载体优缺点：优点包括转导效率高(可达90%以上)、组织靶向性强、可感染非分裂细胞；缺点包括免疫原性强、载量限制(AAV<5kb)、制备复杂、潜在致癌风险。

2. 非病毒载体优缺点：优点包括安全性高、载量大(可达数百kb)、制备简单、成本低、免疫原性小；缺点包括转导效率低(<10%)、靶向性差、基因表达不稳定、易被核酸酶降解。

3. 载体选择原则：根据目标组织(如AAV适合肌肉、神经组织)、基因大小(大基因选非病毒载体)、治疗要求(急性用病毒载体、慢性用非病毒载体)、安全性要求等因素综合考虑。

4. 具体实例：治疗遗传性肌病选择AAV载体(Zolgensma治疗SMA)；肿瘤基因治疗选择腺病毒载体；大片段基因导入选择脂质体等非病毒载体。

5. 发展趋势：开发新型载体如工程化AAV、靶向性脂质体，结合载体优势，提高安全性和效率，推动基因治疗临床应用。
