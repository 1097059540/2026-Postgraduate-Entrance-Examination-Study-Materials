第二章 染色体与DNA

现代分子生物学 · 考研复习笔记

一、染色体概述

1.1 真核细胞染色体的组成

真核细胞染色体主要由DNA和蛋白质组成，其中蛋白质包括组蛋白和非组蛋白两大类。

1.1.1 组蛋白(Histones)

  • 核心组蛋白：H2A、H2B、H3、H4，形成核小体核心颗粒
  • 连接组蛋白：H1，参与高级结构形成
  • 特点：富含碱性氨基酸(赖氨酸、精氨酸)，与酸性DNA结合
  • 修饰：乙酰化、甲基化、磷酸化等表观遗传修饰

1.1.2 非组蛋白

  • 包括转录因子、DNA聚合酶、拓扑异构酶等
  • 参与DNA复制、转录、修复等生物学过程
  • 种类繁多，功能多样化

1.2 真核基因组DNA序列组成

重要概念：真核基因组DNA按重复程度可分为三类序列

1.2.1 非重复序列(单拷贝序列)

  • 占基因组的40-50%
  • 主要包括蛋白质编码基因
  • 基因密度相对较高

1.2.2 中度重复序列

  • 重复次数为10-10⁵次
  • 包括rRNA基因、tRNA基因、组蛋白基因等
  • 满足细胞对某些产物的大量需求

1.2.3 高度重复序列

  • 卫星DNA：高度重复的短序列
  • 转座子：可移动的DNA序列
  • 假基因：失去功能的基因拷贝

1.3 染色质与核小体结构

染色质是DNA与组蛋白结合形成的复合物，具有多级结构组织。

1.3.1 核小体结构

  • 核心颗粒：147bp DNA缠绕组蛋白八聚体(H2A-H2B-H3-H4)₂
  • 连接DNA：约20-80bp，与H1结合
  • "珠链"结构：10nm纤维，DNA压缩比约6倍

1.3.2 高级结构

  • 30nm纤维：核小体进一步螺旋化，压缩比约40倍
  • 环状结构：30nm纤维形成环状结构域
  • 凝缩染色体：分裂期高度压缩，压缩比达10000倍

二、DNA结构

2.1 DNA的一级结构

DNA的一级结构是指核苷酸的排列顺序，决定了遗传信息的内容。

2.1.1 核苷酸组成

  • 碱基：腺嘌呤(A)、鸟嘌呤(G)、胞嘧啶(C)、胸腺嘧啶(T)
  • 戊糖：2'-脱氧核糖
  • 磷酸基团：连接相邻核苷酸

2.1.2 序列多样性

  • 四种碱基的不同排列组合产生序列多样性
  • n个碱基可形成4ⁿ种不同序列
  • 序列特异性决定生物学功能

2.2 DNA的二级结构

DNA二级结构主要指双螺旋结构，由Watson和Crick于1953年提出。

2.2.1 双螺旋的基本特征

  • 反平行：两条链方向相反(5'→3'和3'→5')
  • 碱基配对：A-T配对(2个氢键)，G-C配对(3个氢键)
  • 右手螺旋：顺时针方向螺旋上升
  • 大沟和小沟：螺旋表面形成两种不同宽度的沟槽

2.2.2 DNA的不同构象

重要知识点：DNA在不同条件下可采用不同的螺旋构象
  • B型DNA：生理条件下的主要形式，右手螺旋，螺距3.4nm
  • A型DNA：脱水条件下形成，右手螺旋，更紧密
  • Z型DNA：高盐条件下形成，左手螺旋，锯齿状主链

2.3 DNA的高级结构

2.3.1 超螺旋结构

超螺旋是DNA双螺旋的进一步扭曲，对DNA的生物学功能具有重要意义。

  • 负超螺旋：DNA双螺旋欠扭曲，有利于DNA解链
  • 正超螺旋：DNA双螺旋过度扭曲，阻碍DNA解链
  • 拓扑异构酶：调节DNA超螺旋度的关键酶类

2.3.2 拓扑异构酶的作用

  • I型拓扑异构酶：产生瞬时单链断裂，改变连环数
  • II型拓扑异构酶：产生瞬时双链断裂，解除拓扑约束
  • 生物学意义：维持DNA拓扑状态，参与复制、转录等过程

三、DNA复制

3.1 DNA复制的基本原理

3.1.1 半保留复制

半保留复制是DNA复制的基本方式，由Meselson-Stahl实验证实。

  • 每条新DNA分子含有一条原有链和一条新合成链
  • 保证遗传信息的准确传递
  • 复制的保真性依赖于碱基配对规则

3.1.2 复制的基本概念

  • 复制子(Replicon)：具有复制起点的DNA单位
  • 复制叉(Replication Fork)：DNA解链形成的Y型结构
  • 复制起点(Origin)：复制开始的特定序列
  • 复制终点：复制结束的区域

3.2 DNA复制的方向和特点

3.2.1 复制方向

  • 单向复制：从起点向一个方向进行
  • 双向复制：从起点向两个方向同时进行(真核细胞主要方式)
  • 汇聚复制：两个复制叉相遇终止

3.2.2 半不连续复制

核心概念：由于DNA聚合酶只能5'→3'方向合成，导致半不连续复制
  • 前导链：连续合成，5'→3'方向
  • 滞后链：不连续合成，形成冈崎片段
  • 冈崎片段：滞后链上的短DNA片段(1000-2000nt)

3.3 DNA聚合酶的结构与功能

3.3.1 DNA聚合酶的结构域

  • 掌心域(Palm)：含有活性中心，催化磷酸二酯键形成
  • 手指域(Fingers)：结合模板DNA和dNTP
  • 拇指域(Thumb)：结合双链DNA，影响酶的移动

3.3.2 DNA聚合酶的功能特性

  • 聚合酶活性：5'→3'方向合成DNA
  • 3'→5'外切酶活性：校对功能，提高保真性
  • 持续合成能力：不同聚合酶的持续性差异很大

四、DNA突变与修复

4.1 DNA损伤的类型

DNA在细胞内外因素作用下可发生多种类型的损伤。

4.1.1 自发性损伤

  • 碱基脱氨：胞嘧啶脱氨形成尿嘧啶
  • 碱基脱落：形成无嘌呤/无嘧啶位点
  • DNA甲基化：5-甲基胞嘧啶脱氨形成胸腺嘧啶

4.1.2 诱发性损伤

  • 紫外线：形成嘧啶二聚体
  • 电离辐射：产生DNA断裂
  • 化学诱变剂：碱基类似物、烷化剂等

4.2 DNA修复机制

4.2.1 错配修复(Mismatch Repair)

  • 识别并修复复制过程中产生的错配
  • 依赖甲基化状态区分新旧链
  • 涉及MutS、MutL、MutH等蛋白

4.2.2 切除修复(Excision Repair)

重要分类：切除修复分为碱基切除修复和核苷酸切除修复
  • 碱基切除修复(BER)：修复单个碱基损伤
  • 核苷酸切除修复(NER)：修复较大的DNA损伤
  • 都涉及损伤识别、切除、合成、连接四个步骤

4.2.3 重组修复

  • 同源重组修复：利用同源序列修复双链断裂
  • 非同源末端连接：直接连接断裂末端
  • 应用：修复复制叉停滞和DNA双链断裂

4.2.4 直接修复

  • 光修复：光解酶修复嘧啶二聚体
  • 甲基转移酶：直接去除烷基化损伤
  • 修复效率高，但底物特异性强

4.3 SOS修复系统

SOS修复是细菌在DNA严重损伤时启动的应急修复系统。

  • 诱导条件：DNA严重损伤，单链DNA积累
  • 调控机制：RecA蛋白激活，LexA阻遏蛋白失活
  • 修复特点：容错性修复，可能引入突变

五、DNA转座

5.1 转座子概述

转座子是能够在基因组内移动位置的DNA序列，广泛存在于各种生物中。

5.1.1 转座子的基本特征

  • 具有移动能力的DNA序列
  • 两端通常含有反向重复序列
  • 编码转座酶等移动所需的蛋白
  • 插入时在靶位点产生短的重复序列

5.2 原核生物转座子

5.2.1 插入序列(IS元件)

  • 最简单的转座子，长度700-1500bp
  • 只编码转座酶
  • 两端为反向重复序列
  • 例如：IS1、IS2、IS5等

5.2.2 复合转座子

  • 由两个IS元件夹持其他基因形成
  • 可携带抗生素抗性基因等
  • 例如：Tn5(卡那霉素抗性)、Tn10(四环素抗性)

5.3 真核生物转座子

5.3.1 玉米转座子

  • Ac/Ds系统：McClintock发现的经典转座子
  • Ac元件：自主转座子，编码转座酶
  • Ds元件：非自主转座子，需要Ac提供转座酶

5.3.2 转座的遗传学效应

  • 基因失活：插入基因内部导致功能丧失
  • 基因激活：插入调控序列附近影响表达
  • 染色体重排：不同位点转座子间重组
  • 基因组进化：提供遗传变异的重要来源

六、单核苷酸多态性(SNP)

6.1 SNP概述

单核苷酸多态性(Single Nucleotide Polymorphism, SNP)是基因组中最常见的遗传变异类型。

6.1.1 SNP的基本特征

  • 单个核苷酸位点的序列变异
  • 人类基因组中约每300bp出现一个SNP
  • 总数超过300万个
  • 大多数SNP为双等位基因变异

6.2 SNP的检测技术

6.2.1 传统检测方法

  • DNA测序：直接测定DNA序列
  • 限制性片段长度多态性(RFLP)：利用酶切位点变化
  • 单链构象多态性(SSCP)：基于DNA构象差异

6.2.2 现代高通量技术

  • SNP芯片：同时检测数十万个SNP
  • 全基因组测序：获得完整的SNP信息
  • 靶向测序：针对特定区域的SNP检测

6.3 SNP的应用

6.3.1 人类单倍型图谱

  • 构建人类基因组SNP分布图
  • 研究人群遗传结构和进化历史
  • 为疾病关联研究提供基础

6.3.2 疾病易感基因研究

  • 全基因组关联研究(GWAS)：寻找疾病相关SNP
  • 药物基因组学：研究SNP与药物反应的关系
  • 个性化医疗：基于SNP信息制定治疗方案

七、重要名词解释

染色质(Chromatin)
真核细胞核内DNA与组蛋白及非组蛋白结合形成的复合物，是遗传物质在细胞核内的存在形式。
核小体(Nucleosome)
染色质的基本重复单位，由147bp DNA缠绕组蛋白八聚体形成，呈现"珠链"状结构。
组蛋白(Histone)
富含碱性氨基酸的蛋白质，与DNA结合形成核小体，包括H1、H2A、H2B、H3、H4五种主要类型。
超螺旋(Supercoiling)
DNA双螺旋的进一步扭曲，分为正超螺旋和负超螺旋，对DNA的生物学功能具有重要调控作用。
拓扑异构酶(Topoisomerase)
调节DNA拓扑状态的酶类，通过产生瞬时断裂改变DNA的超螺旋度，分为I型和II型。
半保留复制(Semi-conservative Replication)
DNA复制的基本方式，每个新DNA分子含有一条原有链和一条新合成链。
冈崎片段(Okazaki Fragment)
DNA复制时滞后链上不连续合成的短DNA片段，长度约1000-2000个核苷酸。
复制叉(Replication Fork)
DNA复制时双链解开形成的Y型结构，是DNA合成的活跃区域。
错配修复(Mismatch Repair, MMR)
修复DNA复制过程中产生的碱基错配的修复机制，提高DNA复制的保真性。
切除修复(Excision Repair)
通过切除损伤的DNA片段并重新合成来修复DNA损伤的机制，包括碱基切除修复和核苷酸切除修复。
转座子(Transposon)
能够在基因组内改变位置的DNA序列，包括插入序列和复合转座子等类型。
单核苷酸多态性(Single Nucleotide Polymorphism, SNP)
基因组中单个核苷酸位点的序列变异，是最常见的遗传变异类型。
SOS修复(SOS Repair)
细菌在DNA严重损伤时启动的应急修复系统，具有容错性但可能引入突变。
同源重组(Homologous Recombination)
利用同源DNA序列作为模板修复DNA双链断裂的精确修复机制。
非同源末端连接(Non-Homologous End Joining, NHEJ)
直接连接DNA双链断裂末端的修复机制，速度快但可能产生小的缺失或插入。

九、深度思考题

1. 分析DNA超螺旋结构在基因表达调控中的作用机制，并说明拓扑异构酶如何参与这一过程。

  1. 超螺旋结构对基因表达的影响：负超螺旋有利于DNA双链的解开，促进转录起始和DNA复制的进行。正超螺旋则会阻碍这些过程。在转录过程中，RNA聚合酶的移动会在其前方产生正超螺旋，在后方产生负超螺旋，这种拓扑变化直接影响转录效率。
  2. 拓扑异构酶的调控作用：I型拓扑异构酶通过产生瞬时单链断裂来释放超螺旋张力，而II型拓扑异构酶则通过双链断裂来解决更复杂的拓扑问题。这些酶的活性受到多种因素调控，包括翻译后修饰、蛋白质相互作用等。
  3. 与染色质结构的关系：超螺旋状态影响核小体的稳定性和染色质的高级结构。负超螺旋有利于核小体的解离，使DNA更容易被转录因子接近，从而激活基因表达。
  4. 药物作用靶点：许多抗癌药物如喜树碱、阿霉素等都是通过抑制拓扑异构酶活性来发挥作用，说明了这一系统在细胞生存中的关键地位。

2. 比较分析原核生物和真核生物DNA修复机制的异同，并讨论其进化意义。

  1. 共同的修复机制：两者都具有错配修复、切除修复、重组修复等基本修复途径。这些机制的基本原理相似，都涉及损伤识别、切除、重新合成和连接等步骤，说明DNA修复的重要性和保守性。
  2. 原核生物特有机制：SOS修复系统是原核生物特有的应急修复机制，在DNA严重损伤时启动容错性修复。光修复系统在某些原核生物中也更为常见和高效。
  3. 真核生物的复杂性：真核生物的修复机制更加复杂精细，涉及更多的蛋白因子和调控网络。细胞周期检验点系统与DNA修复紧密结合，确保基因组稳定性。染色质结构也为修复过程增加了额外的复杂性。
  4. 进化意义：从原核到真核，DNA修复机制的进化体现了生物体对基因组稳定性要求的提高。复杂的修复系统虽然消耗更多能量，但为多细胞生物的发育和长寿命提供了保障。

3. 论述转座子在基因组进化中的双重作用，并分析其对宿主基因组稳定性的影响。

  1. 促进基因组进化的积极作用：转座子为基因组提供了重要的变异来源，通过插入、缺失、重排等方式产生新的基因结构和调控元件。某些转座子携带的序列可能演化为新的功能基因或调控序列，如免疫系统中的V(D)J重组机制就起源于转座子。
  2. 基因组不稳定的消极影响：转座子的随机插入可能破坏重要基因的功能，导致疾病的发生。过度活跃的转座活动会增加基因组的不稳定性，影响细胞的正常功能。某些遗传病如血友病就与转座子插入相关。
  3. 宿主的防御机制：生物体进化出多种机制来控制转座子活动，包括DNA甲基化、异染色质形成、RNA干扰等表观遗传修饰。这些机制在维持基因组稳定性的同时，也保留了转座子作为进化原料的潜力。
  4. 平衡与调控：现代基因组中转座子活动受到严格调控，在特定发育阶段或环境压力下才会激活。这种精细调控体现了宿主与转座子之间长期共进化的结果，实现了基因组稳定性与进化潜力的平衡。

4. 分析SNP在精准医学中的应用前景，并讨论其在疾病预测和个性化治疗中的价值。

  1. 疾病易感性预测：通过大规模GWAS研究，已经发现了数千个与常见疾病相关的SNP位点。这些信息可用于构建多基因风险评分模型，预测个体对糖尿病、心血管疾病、癌症等复杂疾病的易感性，实现疾病的早期预防和干预。
  2. 药物基因组学应用：SNP影响药物代谢酶、转运蛋白和靶点蛋白的功能，导致个体间药物反应的差异。通过检测相关SNP，可以预测药物疗效和不良反应，指导个性化用药方案的制定，提高治疗效果并减少副作用。
  3. 肿瘤精准治疗：肿瘤组织中的体细胞SNP(突变)信息可用于指导靶向治疗药物的选择。液体活检技术的发展使得通过血液检测肿瘤SNP成为可能，为肿瘤的动态监测和治疗调整提供了新的手段。
  4. 技术挑战与发展方向：虽然SNP在精准医学中前景广阔，但仍面临数据解读、种族差异、伦理问题等挑战。未来需要建立更完善的数据库、开发更精确的预测模型，并制定相应的伦理规范，以充分发挥SNP在精准医学中的潜力。
