第10章 基因与发育

特别说明：本章节为考试非重点章节，考生了解即可。主要涉及果蝇发育调控机制和高等植物花发育的基因调控原理。

📚 核心知识内容

一、果蝇的发育与调控

1. 果蝇发育周期

果蝇的完整发育过程包括：胚胎发育 → 幼虫 → 蛹化 → 变态 → 成虫。这一过程体现了昆虫完全变态发育的典型特征。

图1 果蝇完全变态发育过程示意图

2. 卵子发育与卵裂过程

(1) 果蝇卵子形成过程：

  • 卵原细胞经过4次有丝分裂形成16个细胞的并合体
  • 其中1个细胞发育成卵母细胞（双倍体）
  • 其余15个细胞发育成抚育细胞（多倍体）

(2) 果蝇的卵裂过程：

  1. 细胞核以9min/次的高频率复制，直至达到约6000个核并发育成为合胞体
  2. 当出现256个以上的细胞核时，细胞核开始向卵的外周移动，定位到皮层组织

3. 胚胎发育的分子机制

果蝇形体模式的形成是沿前-后轴和背-腹轴进行的。

① 母源效应基因的作用：

  • 前端系统：决定头胸部分节
  • 后端系统：决定腹部分节
  • 末端系统：决定胚胎两端不分节的原头区和尾节
  • 背腹系统：决定胚胎的背-腹轴

② 分节基因的分级调控：

  1. 间隙基因：控制大的身体区域
  2. 成对控制基因：控制相邻体节对
  3. 体节极性基因：控制每个体节内的模式
图2 果蝇胚胎发育的分级基因调控网络

4. 同源域基因(HOM-C)的功能

同源域基因决定躯体体节命运。在果蝇中同源域基因统称为HOM复合体(HOM-C)。典型的同源域基因：

  • 不参与基本躯干的建成
  • 不参与体节和肢芽的完善
  • 只保证体节或肢芽的最终典型特征

二、高等植物花发育的基因调控

根据分生组织的形态特征，将植物从营养生长到生殖生长的转变分为3个阶段：

  1. 营养分生组织阶段
  2. 花序分生组织发生阶段
  3. 花分生组织发生阶段

1. 植物花器官结构与发育过程

花器官发育的层级过程：

顶端分生组织 → 营养分生组织 → 花序分生组织 → 花分生组织 → 花器官原基 → 花器官

2. 调控花器官发育的主要基因

  • 花分生组织决定基因：促进从花序分生组织产生花分生组织并进一步分化产生花器官原基
  • 同源域基因：控制产生花器官的类型

三、控制植物开花时间的分子机制

1. 影响植物开花的因素

  • 外部因素：光照和温度
  • 内部因素：自主途径因子和赤霉素(GA)

2. 拟南芥的开花诱导调控途径

(1) 光周期途径

光周期现象：植物通过感受昼夜长短变化而控制开花的现象。

根据对光周期的反应，植物分为：

  • 长日照植物(LDP)：日照长度必须长于其临界日照长度才能开花，如拟南芥生态型
  • 短日照植物(SDP)：日照长度必须短于其临界日照长度才能开花，如水稻、烟草等
  • 日中性植物(DNP)：对日照长度不敏感，任何日照条件下都可以开花，如印度甘蔗等

光周期的感受及传导机制：

  • 感受光周期刺激的部位是叶片
  • 植物叶片感受到适当的日照长度，会形成开花素/成花素(florigen)
  • 开花素进行长距离运输传导到顶芽分生组织后引起开花
图3 拟南芥开花时间的四个主要调控途径

(2) 春化作用

春化作用是指低温诱导植物开花的过程。春化作用中VIN3识别低温处理，建立对春化关键基因FLC表达的抑制。

(3) 自主途径

自主途径是指基因通过不同机制抑制FLC表达，促进SOC1、FT表达，从而促进开花的开花诱导调节途径。

(4) GA（赤霉素）途径

GA途径是指通过GA合成及信号转导有关组分影响SOC1和LFY表达，调控植物开花时间的开花诱导调节途径。

3. 植物激素对开花过程的表观调控

植物激素包括赤霉素、茉莉酸、脱落酸、乙烯以及生长素都能通过DNA甲基化、组蛋白翻译后修饰等方式，在一定程度上影响染色质的紧密程度，从而影响植物的开花过程。

📖 名词解释库

光周期现象(Photoperiodism)
植物通过感受昼夜长短变化而控制开花的现象。植物能够感知日照长度的变化，并据此调节开花时间，以适应季节变化。
同源域基因(Homeotic Genes, HOM-C)
决定躯体体节命运的基因。在果蝇中统称为HOM复合体，这些基因不参与基本躯干建成，只保证体节或肢芽的最终典型特征。
母源效应基因(Maternal Effect Genes)
由母体基因型决定子代表型的基因。在果蝇胚胎发育中，包括前端系统、后端系统、末端系统和背腹系统四组基因。
分节基因(Segmentation Genes)
参与果蝇分节的基因，包括间隙基因、成对控制基因和体节极性基因，这些基因分级调控同源域基因的表达。
开花素(Florigen, FT protein)
植物叶片感受到适当日照长度后形成的成花信号分子，能够进行长距离运输，传导到顶芽分生组织后引起开花。
春化作用(Vernalization)
低温诱导植物开花的过程。通过VIN3识别低温处理，建立对春化关键基因FLC表达的抑制，从而促进开花。
长日照植物(Long Day Plants, LDP)
日照长度必须长于其临界日照长度才能开花的植物，如拟南芥生态型。这类植物通常在春夏季开花。
短日照植物(Short Day Plants, SDP)
日照长度必须短于其临界日照长度才能开花的植物，如水稻、烟草等。这类植物通常在秋季开花。
花分生组织决定基因(Floral Meristem Identity Genes)
促进从花序分生组织产生花分生组织并进一步分化产生花器官原基的基因，是花发育的关键调控因子。
合胞体(Syncytium)
果蝇早期胚胎发育阶段形成的多核细胞结构，细胞核快速分裂但细胞质不分裂，形成一个含有多个细胞核的大细胞。

🤔 深度思考题

1. 试分析原癌基因激活成为癌基因的分子机制，并结合果蝇发育调控原理，探讨基因表达调控在肿瘤发生中的作用。

参考答案：
  1. 原癌基因激活机制：原癌基因可通过点突变、基因扩增、染色体易位等方式激活。这些机制导致基因产物功能异常或表达水平异常升高，破坏正常的细胞生长调控。
  2. 与果蝇发育调控的相似性：果蝇发育中的分级基因调控网络（母源效应基因→分节基因→同源域基因）体现了基因表达的精确时空调控。肿瘤发生也涉及类似的调控网络失调，如细胞周期调控基因、凋亡调控基因等的异常表达。
  3. 基因表达调控的关键作用：正常发育需要基因在特定时间、特定位置的精确表达。当这种调控机制被破坏时，细胞可能失去正常的分化能力，获得无限增殖的特性，最终导致肿瘤形成。
  4. 临床意义：理解基因表达调控机制有助于开发靶向治疗策略，通过调节关键调控因子的表达来抑制肿瘤生长。

2. 基因治疗作为新兴的医疗技术，面临哪些挑战？请结合植物基因工程的成功经验，分析如何提高基因治疗的效率和安全性。

参考答案：
  1. 基因治疗面临的主要挑战：载体安全性问题、靶向性不足、免疫反应、基因表达调控困难、伦理问题等。载体可能引起免疫反应或插入突变，影响治疗效果和安全性。
  2. 植物基因工程的成功经验：植物基因工程在载体选择（如农杆菌载体）、基因表达调控（组织特异性启动子）、安全性评估等方面积累了丰富经验。
  3. 提高效率的策略：开发更高效的载体系统、优化基因表达调控元件、改进基因递送方法、建立精确的靶向机制等。
  4. 提高安全性的措施：严格的载体安全性评估、建立可控的基因表达系统、完善的临床前安全性试验、长期跟踪监测等。
  5. 未来发展方向：个性化基因治疗、基因编辑技术的应用、多基因协同治疗等将是基因治疗发展的重要方向。

3. HIV作为获得性免疫缺陷病毒，其致病机理与宿主免疫系统的相互作用机制是什么？请分析病毒如何逃避免疫监视。

参考答案：
  1. HIV的基本致病机理：HIV主要感染CD4+ T细胞、巨噬细胞等免疫细胞，通过逆转录将病毒基因组整合到宿主细胞DNA中，导致免疫细胞功能缺陷和数量减少。
  2. 免疫逃避机制：HIV具有高度的遗传变异性，能够快速产生突变逃避抗体和细胞毒性T细胞的识别；病毒蛋白可以干扰MHC分子的表达和抗原呈递过程。
  3. 潜伏感染策略：HIV可在某些细胞中建立潜伏感染，病毒基因组整合到宿主DNA中但不活跃复制，使得免疫系统难以清除这些潜伏的病毒储库。
  4. 免疫系统的对抗反应：机体产生特异性抗体和细胞毒性T细胞反应，但由于病毒的高变异性和免疫逃避机制，难以完全清除病毒。
  5. 治疗策略：高效抗逆转录病毒治疗(HAART)通过多种药物联合使用抑制病毒复制，免疫重建治疗帮助恢复免疫功能。

4. 肿瘤免疫治疗代表了癌症治疗的新方向，请分析CAR-T细胞疗法的作用原理，并讨论其在实体瘤治疗中面临的挑战。

参考答案：
  1. CAR-T细胞疗法的基本原理：通过基因工程技术将嵌合抗原受体(CAR)导入患者T细胞，使T细胞能够特异性识别和攻击表达特定抗原的肿瘤细胞，绕过了传统MHC限制性识别。
  2. CAR结构与功能：CAR包含胞外抗原识别域、跨膜域和胞内信号域，能够同时提供激活信号和共刺激信号，增强T细胞的杀伤活性和持久性。
  3. 在血液肿瘤中的成功：CAR-T疗法在B细胞恶性肿瘤（如急性淋巴细胞白血病）治疗中取得显著成效，主要靶向CD19等表面抗原。
  4. 实体瘤治疗的挑战：肿瘤微环境的免疫抑制、缺乏特异性靶抗原、CAR-T细胞的肿瘤浸润困难、肿瘤异质性等因素限制了疗效。
  5. 改进策略：开发新一代CAR设计、联合免疫检查点抑制剂、改善肿瘤微环境、多靶点CAR-T细胞等策略有望提高实体瘤治疗效果。

5. 病毒在人类疾病中扮演重要角色，请比较DNA病毒与RNA病毒的致病特点，并分析不同类型病毒的预防控制策略。

参考答案：
  1. DNA病毒的致病特点：基因组相对稳定，突变率较低，容易建立持续感染或潜伏感染。如HBV可引起慢性肝炎，HSV可建立神经节潜伏感染。DNA病毒通常具有较大的基因组，编码多种调控蛋白。
  2. RNA病毒的致病特点：基因组变异性高，突变率高，容易产生新的变异株逃避免疫。如流感病毒的抗原漂移和抗原转换，HIV的高度变异性。RNA病毒复制周期通常较短，传播速度快。
  3. DNA病毒的预防控制：疫苗接种效果通常较好且持久（如HBV疫苗），抗病毒药物可针对DNA聚合酶等关键酶，定期筛查有助于早期发现慢性感染。
  4. RNA病毒的预防控制：疫苗需要定期更新以应对变异（如流感疫苗），抗病毒药物需要联合使用防止耐药，加强监测系统追踪病毒变异。
  5. 综合防控策略：建立完善的病毒监测网络、发展广谱抗病毒药物、加强公共卫生措施、推进精准医学在病毒感染诊治中的应用。
