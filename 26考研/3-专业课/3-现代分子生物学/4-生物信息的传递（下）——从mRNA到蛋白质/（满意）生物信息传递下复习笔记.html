生物信息的传递（下）

从mRNA到蛋白质 - 现代分子生物学考研复习笔记

一、遗传密码（Genetic Code）

1.1 遗传密码的基本概念

遗传密码是DNA（或RNA）中的三联体与蛋白质中氨基酸之间的对应关系。1 mRNA上每3个核苷酸翻译成多肽链上的一个氨基酸，这3个核苷酸就称为一个密码子（三联子密码，triplet code）。

重要概念：

  • 密码子（Codon）：代表氨基酸或终止信号的核苷酸三联体
  • 起始密码子：用于开始合成蛋白质的特殊密码子（通常是AUG，也可以是GUG）
  • 终止密码子：导致蛋白质合成终止的三种密码子（UAG、UAA、UGA）
  • 阅读框（Reading frame）：将核苷酸序列作为一系列三联体读取的三种可能方式之一

1.2 遗传密码的性质

  1. 连续性（Commaless）：三联子密码是非重叠和连续的，每三个核苷酸编码一个氨基酸
  2. 简并性（Degeneracy）：64个密码子中，61个编码氨基酸，3个是终止密码子。多个密码子可编码同一氨基酸
  3. 普遍性（Universality）：生物界基本共用同一套遗传密码
  4. 特殊性（Specificity）：在某些生物中存在例外，如线粒体中的特殊密码

考试重点：除色氨酸（UGG）和甲硫氨酸（AUG）只有一个密码子外，其他氨基酸都有一个以上的密码子。

二、转移RNA（tRNA）

2.1 tRNA的结构特点

tRNA是模板与氨基酸之间的接合体，具有独特的三叶草型二级结构和L型三级结构。

2.1.1 tRNA的功能区域

  • 反密码子环：含有与mRNA密码子配对的反密码子
  • 氨基酸臂：3'端的CCA序列，氨基酸结合位点
  • D环和TψC环：参与tRNA的三级结构形成
  • 可变环：不同tRNA分子的可变区域

2.2 氨酰tRNA合成酶

氨酰tRNA合成酶催化氨基酸与对应tRNA的结合，形成氨酰tRNA。这个过程需要消耗ATP，分两步进行：

第一步：氨基酸 + ATP + 酶 → 氨酰-AMP-酶复合物 + PPi

第二步：氨酰-AMP-酶复合物 + tRNA → 氨酰-tRNA + AMP + 酶

2.3 密码子与反密码子的相互作用

tRNA的反密码子通过碱基的反向配对与mRNA上的密码子相互作用。1 这种配对遵循Watson-Crick碱基配对规则，但在第三位存在摆动配对现象。

三、核糖体（Ribosome）

3.1 核糖体的结构组成

核糖体是蛋白质合成的场所，由大小两个亚基组成：

3.1.1 原核生物核糖体（70S）

  • 大亚基（50S）：含有23S rRNA、5S rRNA和约34种蛋白质
  • 小亚基（30S）：含有16S rRNA和约21种蛋白质

3.1.2 真核生物核糖体（80S）

  • 大亚基（60S）：含有28S rRNA、18S rRNA、5.8S rRNA、5S rRNA和约49种蛋白质
  • 小亚基（40S）：含有18S rRNA和约33种蛋白质

3.2 核糖体的功能位点

  • A位点（氨酰位点）：新进入的氨酰tRNA结合位点
  • P位点（肽酰位点）：携带肽链的tRNA结合位点
  • E位点（出口位点）：去酰化tRNA离开前的结合位点

四、蛋白质合成过程（翻译）

4.1 翻译前的准备

4.1.1 氨基酸活化

氨基酸在氨酰tRNA合成酶的催化下与对应的tRNA结合，形成氨酰tRNA，这个过程消耗ATP。

4.1.2 核糖体解离

翻译开始前，核糖体大小亚基处于解离状态，由起始因子维持分离。

4.2 翻译起始

翻译起始是蛋白质合成的第一步，涉及起始密码子的识别和翻译起始复合物的形成。

4.2.1 原核生物翻译起始

  1. 小亚基与mRNA结合，识别Shine-Dalgarno序列
  2. 起始tRNA（fMet-tRNA）在起始因子帮助下结合到P位点
  3. 大亚基结合，形成完整的翻译起始复合物
  4. 起始因子释放，翻译开始

4.2.2 真核生物翻译起始

  1. 小亚基与mRNA的5'帽结构结合
  2. 扫描至起始密码子AUG
  3. Met-tRNA结合到P位点
  4. 大亚基结合，形成翻译起始复合物

4.3 肽链延伸

肽链延伸是一个循环过程，包括以下步骤：

  1. 氨酰tRNA结合：新的氨酰tRNA在延伸因子帮助下结合到A位点
  2. 肽键形成：P位点的肽链转移到A位点的氨基酸上
  3. 核糖体移位：核糖体沿mRNA移动一个密码子的距离
  4. tRNA转位：A位点的肽酰tRNA移至P位点，P位点的去酰化tRNA移至E位点

能量消耗：每添加一个氨基酸需要消耗4个高能磷酸键（2个ATP用于氨基酸活化，2个GTP用于延伸过程）。

4.4 翻译终止

当核糖体遇到终止密码子（UAG、UAA、UGA）时，翻译终止：

  1. 释放因子识别终止密码子
  2. 肽酰tRNA水解，多肽链从tRNA上释放
  3. 核糖体亚基解离
  4. mRNA和tRNA释放

五、蛋白质的加工与修饰

5.1 新生肽链的折叠

新合成的多肽链需要正确折叠才能获得生物活性：

  • 分子伴侣：协助蛋白质正确折叠，如GroEL/GroES系统
  • 二硫键形成：半胱氨酸残基间形成二硫键稳定结构
  • 翻译后修饰：磷酸化、甲基化、乙酰化等

5.2 蛋白质的定向运输

5.2.1 信号肽假说

分泌蛋白和膜蛋白含有信号肽，指导蛋白质的定向运输：

  1. 信号肽从核糖体出现
  2. 信号识别颗粒（SRP）识别并结合信号肽
  3. 翻译暂停，核糖体与内质网结合
  4. 翻译恢复，蛋白质进入内质网腔
  5. 信号肽被切除

5.2.2 蛋白质分选

  • 内质网：分泌蛋白、膜蛋白的加工场所
  • 高尔基体：蛋白质的进一步修饰和分选
  • 线粒体：含有特殊的蛋白质导入机制
  • 细胞核：核蛋白通过核孔复合体进入

六、翻译调控机制

6.1 翻译水平调控

  • 起始因子调控：磷酸化修饰影响翻译起始
  • mRNA结构调控：5'UTR和3'UTR的二级结构影响翻译效率
  • microRNA调控：miRNA与mRNA结合抑制翻译
  • 核糖体开关：小分子代谢物调控翻译

6.2 翻译抑制剂

常见翻译抑制剂：

  • 链霉素：结合30S亚基，干扰翻译起始
  • 氯霉素：抑制肽酰转移酶活性
  • 环己酰亚胺：抑制真核生物翻译延伸
  • 嘌呤霉素：模拟氨酰tRNA，导致肽链提前终止

专业术语库

遗传密码（Genetic Code）
DNA或RNA中的三联体与蛋白质中氨基酸之间的对应关系，是生物信息从核酸传递到蛋白质的桥梁。
密码子（Codon）
mRNA上由三个连续核苷酸组成的编码单位，每个密码子对应一个氨基酸或终止信号。
反密码子（Anticodon）
tRNA分子上与mRNA密码子互补配对的三个核苷酸序列，位于反密码子环中。
转移RNA（Transfer RNA, tRNA）
携带特定氨基酸到核糖体进行蛋白质合成的RNA分子，具有三叶草型二级结构和L型三级结构。
氨酰tRNA合成酶（Aminoacyl-tRNA Synthetase）
催化氨基酸与对应tRNA结合形成氨酰tRNA的酶，保证翻译的准确性。
核糖体（Ribosome）
蛋白质合成的场所，由大小两个亚基组成，含有rRNA和蛋白质。原核生物为70S，真核生物为80S。
翻译（Translation）
以mRNA为模板，在核糖体上合成蛋白质的过程，包括起始、延伸和终止三个阶段。
信号肽（Signal Peptide）
位于新生蛋白质N端的短肽序列，指导蛋白质的定向运输，通常在运输后被切除。
信号识别颗粒（Signal Recognition Particle, SRP）
识别并结合信号肽的核糖核蛋白复合物，介导核糖体与内质网的结合。
分子伴侣（Molecular Chaperone）
协助蛋白质正确折叠、组装和运输的蛋白质，如热休克蛋白（HSP）家族。
多聚核糖体（Polyribosome, Polysome）
多个核糖体同时在一条mRNA上进行翻译形成的复合物，提高翻译效率。
摆动配对（Wobble Pairing）
密码子第三位与反密码子第一位之间的非标准碱基配对，解释了遗传密码的简并性。

选择题测试

深度思考题

1. 分析遗传密码的简并性对生物进化的意义

答案要点：

  1. 减少突变的有害影响：由于多个密码子编码同一氨基酸，DNA序列的点突变可能不会改变蛋白质序列，这种同义突变减少了有害突变的概率，提高了生物的生存能力。
  2. 提供进化的原材料：简并性允许DNA序列在保持蛋白质功能的前提下发生变化，为自然选择提供了更多的遗传变异，促进了物种的适应性进化。
  3. 优化翻译效率：常用氨基酸往往有更多的密码子，这与tRNA的丰度相匹配，提高了翻译效率。同时，摆动配对减少了所需tRNA种类的数量。
  4. 缓冲环境压力：在环境压力下，即使发生突变，简并性也能维持蛋白质的基本功能，增强了生物对环境变化的抗性。
  5. 促进基因重复和分化：简并性使得基因重复后的拷贝可以在保持原有功能的同时积累突变，最终分化出新的功能。

2. 比较原核生物和真核生物翻译过程的差异及其生物学意义

答案要点：

  1. 时空分离差异：原核生物转录和翻译在细质中同时进行，甚至可以边转录边翻译；真核生物转录在核内，翻译在胞质，时空分离允许更复杂的调控机制。
  2. 起始机制差异：原核生物通过Shine-Dalgarno序列识别起始位点，可以多顺反子翻译；真核生物通过5'帽结构和扫描机制，通常单顺反子翻译，调控更精细。
  3. 核糖体结构差异：原核生物70S核糖体，真核生物80S核糖体，结构差异使得某些抗生素可以特异性抑制原核生物翻译而不影响真核生物。
  4. mRNA加工差异：真核生物mRNA经过5'加帽、3'加尾和剪接等加工，增加了调控层次和mRNA稳定性，而原核生物mRNA通常不经过这些加工。
  5. 翻译调控复杂性：真核生物具有更多的翻译调控机制，如miRNA调控、上游开放阅读框（uORF）调控等，适应了多细胞生物复杂的基因表达调控需求。

3. 阐述蛋白质定向运输的分子机制及其在细胞功能中的重要性

答案要点：

  1. 信号肽识别机制：新生蛋白质的信号肽被信号识别颗粒（SRP）识别，SRP结合信号肽后使翻译暂停，并引导核糖体与内质网膜上的SRP受体结合，确保蛋白质正确定位。
  2. 共翻译转位：分泌蛋白和膜蛋白在翻译过程中直接进入内质网，通过Sec61转位子穿越或插入膜，这种机制避免了疏水蛋白在胞质中聚集。
  3. 翻译后转位：某些蛋白质如线粒体蛋白在翻译完成后通过特殊的转位机制进入目标细胞器，需要分子伴侣维持蛋白质的转位能力状态。
  4. 质量控制机制：内质网具有未折叠蛋白反应（UPR）等质量控制机制，确保只有正确折叠的蛋白质才能继续运输，错误折叠的蛋白质被降解。
  5. 细胞功能意义：蛋白质定向运输实现了细胞的区室化，使不同功能的蛋白质在正确的位置发挥作用，维持细胞的正常生理功能和代谢活动。

4. 分析翻译抑制剂的作用机制及其在医学和研究中的应用

答案要点：

  1. 抑制机制分类：翻译抑制剂可分为影响起始（如链霉素）、延伸（如氯霉素、环己酰亚胺）和终止（如嘌呤霉素）等不同阶段的药物，每类都有特定的作用靶点。
  2. 选择性抑制原理：许多抗生素利用原核和真核核糖体结构差异实现选择性抑制，如链霉素特异性结合原核生物30S亚基，而对真核生物毒性较小。
  3. 医学应用价值：翻译抑制剂广泛用作抗菌药物治疗细菌感染，如氯霉素治疗伤寒，红霉素治疗呼吸道感染，为临床提供了重要的治疗手段。
  4. 科研工具应用：在分子生物学研究中，翻译抑制剂用于研究基因表达调控、蛋白质功能分析和细胞周期调控等，是重要的实验工具。
  5. 耐药性挑战：细菌对翻译抑制剂产生耐药性是临床面临的重大挑战，需要不断开发新的抑制剂和合理使用现有药物来应对耐药性问题。
