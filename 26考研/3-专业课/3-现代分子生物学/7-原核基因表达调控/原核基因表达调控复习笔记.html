第7章 原核基因表达调控

考研现代分子生物学复习笔记 | 基于《现代分子生物学》第五版

一、原核基因表达调控概述

（一）基因表达与调控的基本概念

基因表达是指基因经过转录和翻译，产生具有特异性生物学功能的蛋白质产物来执行生命活动，即从DNA到蛋白质的过程。基因表达具有时间特异性（阶段特异性）和空间特异性。

基因表达调控是指发生在基因表达过程中各种各样的调节方式，主要表现在转录水平的调控和转录后水平的调控。

（二）原核基因表达调控分类

调控类型 调节基因产物 作用机制
负转录调控 负控诱导 阻遏蛋白抑制结构基因转录，与诱导物结合后变为非活性，结构基因转录
负控阻遏 阻遏蛋白非活性时不影响转录，与效应物结合后变为活性，抑制转录
正转录调控 正控诱导 诱导物存在使激活蛋白处于活性状态，开启结构基因转录
正控阻遏 激活蛋白促进转录，效应物存在时使激活蛋白非活性，阻遏转录

（三）原核基因表达调控的主要特点

1. 操纵子调控机制

  • 乳糖操纵子：负控诱导系统的典型代表
  • 色氨酸操纵子：负控阻遏系统的典型代表
  • 半乳糖操纵子和阿拉伯糖操纵子：复杂调控系统

2. 转录水平调控

  • 转录起始阶段调控：基因表达调控的关键环节
  • 转录终止阶段调控：包括抗终止作用和弱化作用

3. 转录后调控

  • mRNA自身结构元件对翻译起始的调节
  • 反义RNA对翻译的调控
  • 稀有密码子对翻译的影响
  • 翻译阻遏等现象

二、乳糖操纵子：分解代谢操纵子的典范

🏭 生活化理解

乳糖操纵子就像一个"来活了才开工，而且没好饭吃才拼命干"的智能工厂。它是诱导型的，负责生产消化乳糖的工具（酶）。

（一）乳糖操纵子的基本结构

lacI: 调节基因 | CAP: cAMP结合位点 | P: 启动子 | O: 操纵基因 | lacZ: β-半乳糖苷酶 | lacY: 透过酶 | lacA: 乙酰基转移酶

结构基因功能详解：

  • lacZ基因：编码β-半乳糖苷酶，将乳糖水解为半乳糖和葡萄糖，同时产生少量异构乳糖（真正的诱导物）
  • lacY基因：编码β-半乳糖苷透过酶（permease），负责乳糖跨膜转运，是乳糖进入细胞的"门户"
  • lacA基因：编码β-半乳糖苷乙酰基转移酶，催化有毒代谢产物的乙酰化解毒

（二）双重调控机制：负调控+正调控协同作用

1. 负调控系统（保安机制）

核心元件：由lacI基因编码的阻遏蛋白（Lac repressor, LacI）

无乳糖状态
→
阻遏蛋白以活性构象结合lacO
→
空间位阻阻止RNA聚合酶
→
转录被抑制


有乳糖状态
→
异构乳糖（诱导物）结合阻遏蛋白
→
变构效应使阻遏蛋白失活
→
阻遏蛋白从lacO解离
→
转录解除抑制

关键概念：异构乳糖（allolactose）是真正的诱导物，由β-半乳糖苷酶的转糖基作用产生，形成正反馈调节。

2. 正调控系统（经理机制）

核心元件：分解代谢物激活蛋白（CAP/CRP）与环腺苷酸（cAMP）形成的复合物

详细机制：

  • cAMP的产生：当葡萄糖等优先碳源匮乏时，腺苷酸环化酶活性增强，ATP转化为cAMP
  • cAMP-CAP复合物形成：cAMP与CAP结合，诱导CAP构象变化，获得DNA结合活性
  • 转录激活机制：
    • cAMP-CAP结合到启动子上游的特定位点
    • 通过弯曲DNA分子，增强RNA聚合酶与启动子的亲和力
    • 促进开放复合物形成，大幅提高转录起始效率
  • 葡萄糖效应：葡萄糖存在时抑制腺苷酸环化酶，cAMP水平降低，正调控失效

3. 协同调节的逻辑门控（AND逻辑）

核心原理：乳糖操纵子的高效表达需要同时满足两个条件：

① 诱导物存在（解除阻遏，负调控关闭）AND ② cAMP水平高（CAP激活，正调控开启）

葡萄糖 乳糖 cAMP-CRP 阻遏蛋白状态 转录水平 生物学意义
无 无 有活性 结合lacO 关闭 无底物，不需要酶
有 无 无活性 结合lacO 关闭 优先利用葡萄糖
有 有 无活性 不结合 低水平 葡萄糖优先，乳糖备用
无 有 有活性 不结合 高水平 最佳表达条件

🎯 记忆要点

"等米下锅"策略：只有在缺乏更好的葡萄糖（饿急了）且眼前有乳糖（米来了）时，才不惜成本地大量生产消化工具。这种双重保障确保了能量的高效利用。

三、色氨酸操纵子：合成代谢操纵子的典范

🏭 生活化理解

色氨酸操纵子就像一个"货够了就停产，而且快够了就减速"的精细工厂。它是阻遏型的，负责生产色氨酸（自身需要的氨基酸）。

（一）色氨酸操纵子的基本结构

trpR: 调节基因 | P: 启动子 | O: 操纵基因 | trpL: 前导序列 | a: 弱化子（衰减子） | trpE-A: 结构基因

结构基因功能详解：

  • trpE：编码邻氨基苯甲酸合成酶α亚基，催化色氨酸合成的第一步
  • trpD：编码磷酸核糖基邻氨基苯甲酸转移酶，参与中间代谢
  • trpC：编码吲哚-3-甘油磷酸合成酶，形成吲哚环结构
  • trpB：编码色氨酸合成酶β亚基，与trpA协同完成最后一步
  • trpA：编码色氨酸合成酶α亚基，催化色氨酸的最终合成

（二）层级调控机制：阻遏系统+衰减作用

1. 阻遏系统（总开关机制）

核心元件：由trpR基因编码的阻遏蛋白（aporepressor），本身无活性

详细机制：

  • 辅阻遏物：色氨酸本身作为操纵子终产物，充当辅阻遏物（corepressor）
  • 变构激活：色氨酸与阻遏蛋白结合，诱导变构，形成具有DNA结合活性的复合物
  • 负反馈调节：终产物直接参与调控，实现经典的负反馈抑制
色氨酸充足
→
Trp+阻遏蛋白→活化复合物
→
复合物特异性结合trpO
→
阻遏转录（粗调）


色氨酸缺乏
→
阻遏蛋白无法结合DNA
→
操纵子基础转录
→
合成色氨酸

2. 衰减作用（微调旋钮机制）

发生位置：前导序列（trpL）中的弱化子（attenuator），一个依赖ρ因子的转录终止子

核心机制：转录-翻译偶联调控

前导序列特征：

  • 长度162bp，包含编码前导肽的开放阅读框
  • 前导肽含14个氨基酸，第10、11位为连续的色氨酸密码子（UGG UGG）
  • 含有4个序列互补区域，可形成不同的二级结构（发夹结构）

四个关键区域的配对模式：

  • 区域1：前导肽编码序列（含Trp密码子）
  • 区域2：可与区域1或3配对
  • 区域3：可与区域2或4配对
  • 区域4：终止子序列
色氨酸充足情况
→
核糖体快速翻译前导肽
→
顺利通过Trp密码子，覆盖区域2
→
mRNA形成3:4终止子发夹
→
RNA聚合酶提前终止（细调关闭）


色氨酸缺乏情况
→
核糖体在Trp密码子处停滞
→
无法覆盖区域2
→
mRNA形成2:3抗终止发夹
→
阻止3:4终止子形成，转录通读（细调开启）

3. 层级调控的协同逻辑

双重保障机制：

① 阻遏系统：提供粗调（on/off switch），大幅关闭或开启转录

② 衰减作用：提供细调（fine-tuning），对色氨酸浓度微小变化做出响应

色氨酸浓度 阻遏系统状态 衰减作用 转录水平 调控倍数
极高 完全阻遏 完全终止 几乎为0 ~600倍↓
较高 部分阻遏 大部分终止 极低 ~100倍↓
中等 轻微阻遏 部分终止 中等 ~10倍↓
较低 无阻遏 少量终止 较高 基础水平
极低 无阻遏 无终止 最高 最大表达

🎯 记忆要点

"看菜吃饭"策略：产品够用了就完全停产（阻遏系统），快够用了就降低生产速度（衰减作用），绝不浪费一滴原料。这种层级调控实现了最高可达数百倍的动态调控范围。

四、其他调控机制

（一）SOS应答系统

当细菌DNA遭到破坏时，启动SOS诱导型DNA修复系统：

  • RecA蛋白：SOS反应的发动因子，被激活后表现为蛋白水解酶活性
  • LexA阻遏蛋白：被RecA降解，解除对DNA修复基因的抑制

（二）转录水平的其他调控方式

调控方式 作用机制
σ因子调节 识别启动子共有序列，不同σ因子调控不同基因集
组蛋白类似蛋白 H-NS蛋白等维持DNA高级结构
转录调控因子 与启动子结合，激活或抑制转录
抗终止因子 阻止转录终止，使RNA聚合酶越过终止子

（三）转录后水平调控

1. mRNA结构元件调节

  • 起始密码子：GUG、UUG等非典型起始密码子降低翻译效率
  • 5' UTR：核糖体结合位点（RBS）和SD序列影响翻译起始
  • 核糖开关：感受代谢物浓度变化，改变mRNA二级结构

2. 魔斑核苷酸调节

  • ppGpp和pppGpp：氨基酸缺乏时大量产生
  • 严紧控制：rel⁺基因型，蛋白质和RNA合成速率下降
  • 松散控制：rel⁻基因型，蛋白质合成停止但RNA合成不受影响

3. 其他调控方式

  • 反义RNA：与mRNA互补结合，调节翻译
  • 小RNA：非编码RNA分子调节基因表达
  • 稀有密码子：影响翻译效率
  • 重叠基因：保证翻译偶联

五、重要名词解释

操纵子(Operon)
原核生物基因中，多个功能相关的结构基因串联排列，与上游启动区和操纵区以及下游转录终止区一起构成的基因表达单位。
阻遏蛋白(Repressor Protein)
由调节基因产生的，在转录水平对基因表达产生负控作用的蛋白质，通过结合操纵基因阻碍RNA聚合酶转录。
弱化子(Attenuator)
原核生物操纵子中能显著减弱甚至终止转录作用的核苷酸序列，利用转录与翻译偶联机制对转录进行调节。
反义RNA(Antisense RNA)
与mRNA互补的RNA分子，通过特异性互补结合改变mRNA构象，调节翻译过程。
魔斑核苷酸(Magic Spot Nucleotides)
氨基酸缺乏时细胞内大量产生的ppGpp和pppGpp，是细胞内严紧控制的关键信号分子。
葡萄糖效应(Glucose Effect)
又称降解物阻遏，指葡萄糖和其他糖类同时存在时，葡萄糖优先被利用的现象。
CAP-cAMP复合物(CAP-cAMP Complex)
环腺苷酸受体蛋白与cAMP结合形成的复合物，结合启动子上游CAP位点，促进转录起始。
σ因子(Sigma Factor)
原核生物RNA聚合酶全酶的组分，负责识别启动子共有序列，是转录起始所必需的因子。
SOS应答(SOS Response)
细菌DNA遭到破坏时启动的诱导型DNA修复系统，通过RecA蛋白降解LexA阻遏蛋白实现。
严紧控制(Stringent Control)
细菌在氨基酸饥饿条件下，蛋白质和RNA合成速率下降，大部分活性区域被关闭的调控机制。

六、选择题测试

七、深度思考题

1. 详细分析乳糖操纵子的双重调控机制，并阐述其生物学意义。

答案要点：

  1. 负调控机制（阻遏蛋白调控）：
    • 无乳糖时：LacI阻遏蛋白结合操纵区O，阻止RNA聚合酶转录
    • 有乳糖时：异构乳糖与阻遏蛋白结合，改变其构象，使其脱离O区
    • 机制特点：确保只有在需要时才合成乳糖代谢酶
  2. 正调控机制（CAP-cAMP调控）：
    • 无葡萄糖时：cAMP浓度高，与CRP结合形成复合物
    • cAMP-CRP复合物结合CAP位点，促进RNA聚合酶与启动子结合
    • 有葡萄糖时：cAMP浓度低，无法形成有效的激活复合物
  3. 协同调控的四种状态：
    • 只有在无葡萄糖、有乳糖时，操纵子才完全开启
    • 体现了细菌对碳源利用的精确调控
  4. 生物学意义：
    • 节约能量：避免不必要的酶合成
    • 优先利用：确保优先利用葡萄糖这一更高效的碳源
    • 快速响应：能够迅速适应环境中碳源的变化
    • 精确调控：双重调控确保基因表达的精确性

2. 比较分析色氨酸操纵子的阻遏系统和弱化系统，说明两种机制的协同作用。

答案要点：

  1. 阻遏系统特点：
    • 调控机制：色氨酸作为辅阻遏物与TrpR蛋白结合
    • 作用位点：激活的阻遏蛋白结合操纵区O
    • 调控效果：完全阻断转录起始
    • 响应范围：对色氨酸浓度的大幅变化敏感
  2. 弱化系统特点：
    • 调控机制：通过前导肽翻译感知色氨酰-tRNA浓度
    • 作用位点：前导序列的茎-环结构形成
    • 调控效果：决定转录是否提前终止
    • 响应范围：对色氨酸浓度的细微变化敏感
  3. 两种机制的协同作用：
    • 阻遏系统：粗调，在色氨酸充足时完全关闭操纵子
    • 弱化系统：细调，在色氨酸中等浓度时进行微调
    • 协同效果：实现对色氨酸合成的精确调控
    • 调控范围：覆盖从完全缺乏到完全充足的整个浓度范围
  4. 生物学优势：
    • 节约资源：避免过量合成色氨酸
    • 快速响应：能够迅速调整合成速率
    • 精确控制：双重机制确保调控的精确性
    • 代谢平衡：维持细胞内氨基酸的平衡

3. 阐述原核生物转录后调控的多样性及其调控机制的分子基础。

答案要点：

  1. mRNA结构元件调控：
    • 起始密码子：非AUG起始密码子降低翻译效率
    • SD序列：与16S rRNA互补配对，影响核糖体结合
    • 5' UTR二级结构：影响核糖体接近起始密码子
    • 核糖开关：感受代谢物浓度，改变mRNA构象
  2. 小RNA调控机制：
    • 反义RNA：与靶mRNA互补配对，阻止翻译或促进降解
    • microRNA样分子：调节mRNA稳定性和翻译效率
    • 调控范围：可同时调控多个靶基因
    • 作用特点：快速、可逆、精确
  3. 蛋白质介导的调控：
    • RNA结合蛋白：调节mRNA稳定性和翻译起始
    • 核糖体蛋白：自身调节翻译（翻译阻遏）
    • RNA酶：控制mRNA降解速率
    • 修饰酶：通过RNA修饰调节功能
  4. 代谢调控机制：
    • 魔斑核苷酸：ppGpp和pppGpp调节严紧控制
    • 稀有密码子：通过tRNA可用性调节翻译速率
    • 重叠基因：确保基因产物的化学计量关系
    • 翻译偶联：前一基因翻译影响后一基因表达
  5. 调控的生物学意义：
    • 精细调节：在转录调控基础上进行微调
    • 快速响应：比转录调控响应更迅速
    • 节约能量：避免不必要的蛋白质合成
    • 适应环境：快速适应环境变化

4. 分析σ因子在原核基因表达调控中的作用机制及其调控网络的特点。

答案要点：

  1. σ因子的基本功能：
    • 启动子识别：识别-10和-35区的共有序列
    • 转录起始：引导RNA聚合酶正确起始转录
    • DNA解链：参与启动子区域双链DNA的解开
    • 特异性决定：决定RNA聚合酶的启动子特异性
  2. 不同σ因子的功能分工：
    • σ70：管家基因的转录，维持基本生理功能
    • σ32：热激反应，应对高温胁迫
    • σ54：氮代谢相关基因的转录
    • σS：静止期和胁迫响应基因的转录
  3. σ因子的调控机制：
    • 转录调控：σ因子基因的转录水平调节
    • 蛋白稳定性：通过蛋白酶调节σ因子稳定性
    • 抗σ因子：特异性结合σ因子，阻止其与RNA聚合酶结合
    • 竞争结合：不同σ因子竞争结合RNA聚合酶核心酶
  4. σ因子调控网络特点：
    • 层级调控：形成调控级联，一个σ因子调控另一个σ因子
    • 交叉调控：不同σ因子之间相互影响
    • 时序表达：在不同生长阶段表达不同σ因子
    • 环境响应：根据环境条件选择性激活特定σ因子
  5. 生物学意义：
    • 全局调控：通过改变σ因子实现基因表达的全局重编程
    • 适应性：快速响应环境变化，调整基因表达模式
    • 经济性：用少数调控因子控制大量基因
    • 协调性：确保相关基因的协调表达
