蛋白质降解和氨基酸分解代谢

生物化学考研复习笔记 | 第25章

一、蛋白质降解的两大体系

1. 泛素-蛋白酶体系统（Ubiquitin-Proteasome System, UPS）

泛素-蛋白酶体系统是真核细胞中选择性降解蛋白质的主要途径，主要负责降解胞质和核内蛋白质。

（1）泛素化过程

  • E1酶（泛素激活酶）：以ATP依赖的方式激活泛素
  • E2酶（泛素结合酶）：将激活的泛素转移到E2酶上
  • E3酶（泛素连接酶）：识别特定的靶蛋白，将泛素转移到靶蛋白的赖氨酸残基上
重要提示：蛋白质需要被多个泛素分子标记（形成多泛素链）才能被26S蛋白酶体识别和降解。

（2）26S蛋白酶体结构

  • 20S催化核心：含有蛋白酶活性位点
  • 19S调节亚基：识别多泛素化蛋白质，具有ATP酶和去泛素化酶活性

2. 溶酶体蛋白质降解系统

溶酶体系统主要负责降解细胞外蛋白质、膜蛋白和长寿命蛋白质。

（1）自噬途径（Autophagy）

  • 巨自噬：形成自噬体包裹细胞质成分
  • 微自噬：溶酶体膜直接内陷包裹底物
  • 分子伴侣介导的自噬：特定蛋白质通过LAMP-2A受体进入溶酶体

（2）内吞途径

  • 受体介导的内吞作用
  • 胞饮作用
  • 吞噬作用

二、氨基酸分解代谢的基本过程

1. 氨基酸脱氨基作用

氨基酸分解代谢的第一步是脱除α-氨基，主要通过以下方式进行：

（1）转氨作用（Transamination）

这是氨基酸脱氨基的主要方式，由转氨酶催化。

  • 反应机制：氨基酸 + α-酮戊二酸 ⇌ α-酮酸 + 谷氨酸
  • 辅酶：磷酸吡哆醛（PLP，维生素B6的活性形式）
  • 重要转氨酶：
    • 丙氨酸转氨酶（ALT）：Ala + α-KG ⇌ Pyr + Glu
    • 天冬氨酸转氨酶（AST）：Asp + α-KG ⇌ OAA + Glu

（2）氧化脱氨作用

主要由谷氨酸脱氢酶催化，是唯一能够直接脱除氨基的反应。

  • 反应式：谷氨酸 + NAD(P)⁺ + H₂O → α-酮戊二酸 + NH₃ + NAD(P)H + H⁺
  • 调节：受ADP激活，受ATP和GTP抑制

2. 氨的处理和尿素循环

氨基酸脱氨基产生的氨对细胞有毒，必须及时处理。

（1）氨的毒性

  • 干扰三羧酸循环
  • 影响神经系统功能
  • 破坏细胞的酸碱平衡

（2）尿素循环（Urea Cycle）

尿素循环是哺乳动物处理氨的主要途径，主要在肝脏进行。

尿素循环的五个关键酶：
  1. 氨基甲酰磷酸合成酶I（CPS I）：线粒体内，合成氨基甲酰磷酸
  2. 鸟氨酸氨基甲酰转移酶（OCT）：线粒体内，形成瓜氨酸
  3. 精氨酸琥珀酸合成酶（ASS）：胞质，形成精氨酸琥珀酸
  4. 精氨酸琥珀酸裂解酶（ASL）：胞质，形成精氨酸
  5. 精氨酸酶（ARG）：胞质，形成尿素和鸟氨酸

（3）尿素循环的调节

  • N-乙酰谷氨酸：CPS I的变构激活剂
  • 高蛋白饮食：诱导尿素循环酶的合成
  • 激素调节：胰高血糖素和皮质醇促进酶合成

3. 氨基酸碳骨架的代谢去路

氨基酸脱氨基后的碳骨架可以进入不同的代谢途径。

（1）生糖氨基酸（Glucogenic Amino Acids）

碳骨架可以转化为葡萄糖的氨基酸，包括：

  • 转化为丙酮酸：Ala（丙氨酸）、Cys（半胱氨酸）、Gly（甘氨酸）、Ser（丝氨酸）、Thr（苏氨酸）
  • 转化为α-酮戊二酸：Arg（精氨酸）、Glu（谷氨酸）、Gln（谷氨酰胺）、His（组氨酸）、Pro（脯氨酸）
  • 转化为琥珀酰CoA：Ile（异亮氨酸）、Met（甲硫氨酸）、Thr（苏氨酸）、Val（缬氨酸）
  • 转化为延胡索酸：Asp（天冬氨酸）、Asn（天冬酰胺）
  • 转化为草酰乙酸：Asp（天冬氨酸）、Asn（天冬酰胺）

（2）生酮氨基酸（Ketogenic Amino Acids）

碳骨架转化为酮体的氨基酸：

  • 纯生酮氨基酸：Leu（亮氨酸）、Lys（赖氨酸）
  • 既生糖又生酮：Ile（异亮氨酸）、Phe（苯丙氨酸）、Trp（色氨酸）、Tyr（酪氨酸）

三、重要氨基酸的特殊代谢

1. 苯丙氨酸和酪氨酸代谢

（1）正常代谢途径

  • 苯丙氨酸羟化酶：Phe → Tyr（需要四氢生物蝶呤作辅因子）
  • 酪氨酸羟化酶：Tyr → DOPA（多巴）
  • 最终产物：延胡索酸和乙酰乙酸

（2）代谢异常疾病

  • 苯丙酮尿症（PKU）：苯丙氨酸羟化酶缺陷
  • 酪氨酸血症：酪氨酸代谢酶缺陷
  • 白化病：酪氨酸酶缺陷，影响黑色素合成

2. 含硫氨基酸代谢

（1）甲硫氨酸循环

  • S-腺苷甲硫氨酸（SAM）：重要的甲基供体
  • 同型半胱氨酸：可重新甲基化为甲硫氨酸
  • 转硫途径：同型半胱氨酸 → 胱硫氨酸 → 半胱氨酸

（2）一碳单位代谢

  • 载体：四氢叶酸（THF）
  • 来源：丝氨酸、甘氨酸、组氨酸、色氨酸
  • 用途：嘌呤合成、胸腺嘧啶合成、氨基酸合成

3. 支链氨基酸代谢

缬氨酸（Val）、亮氨酸（Leu）、异亮氨酸（Ile）的代谢特点：

  • 组织特异性：主要在肌肉、脂肪组织代谢
  • 第一步反应：支链氨基酸转氨酶催化
  • 第二步反应：支链α-酮酸脱氢酶复合体催化
  • 代谢异常：枫糖浆尿症（支链α-酮酸脱氢酶缺陷）

四、名词解释库

泛素（Ubiquitin）
一种由76个氨基酸组成的小蛋白质，作为蛋白质降解的标签，标记需要被蛋白酶体降解的蛋白质。
蛋白酶体（Proteasome）
真核细胞中负责降解被泛素标记蛋白质的大型蛋白质复合体，26S蛋白酶体由20S催化核心和两个19S调节亚基组成。
自噬（Autophagy）
细胞通过形成自噬体包裹细胞质成分并与溶酶体融合进行降解的过程，是细胞清除损伤细胞器和蛋白质聚集体的重要机制。
转氨作用（Transamination）
氨基酸与α-酮酸之间氨基转移的反应，由转氨酶催化，需要磷酸吡哆醛作为辅酶，是氨基酸脱氨基的主要方式。
氧化脱氨作用（Oxidative Deamination）
氨基酸直接脱除氨基的反应，主要由谷氨酸脱氢酶催化，产生α-酮酸、氨和NADH。
尿素循环（Urea Cycle）
哺乳动物将有毒的氨转化为无毒尿素的代谢途径，主要在肝脏进行，涉及五个关键酶。
氨基甲酰磷酸合成酶I（CPS I）
尿素循环的限速酶，位于线粒体内，催化氨、CO₂和ATP合成氨基甲酰磷酸，受N-乙酰谷氨酸激活。
生糖氨基酸（Glucogenic Amino Acids）
碳骨架可以转化为葡萄糖的氨基酸，通过糖异生途径产生葡萄糖，包括大部分氨基酸。
生酮氨基酸（Ketogenic Amino Acids）
碳骨架转化为酮体（乙酰乙酸或乙酰CoA）的氨基酸，包括亮氨酸、赖氨酸等。
苯丙酮尿症（Phenylketonuria, PKU）
由苯丙氨酸羟化酶缺陷引起的遗传性疾病，导致苯丙氨酸在体内蓄积，可引起智力发育障碍。
S-腺苷甲硫氨酸（S-Adenosylmethionine, SAM）
重要的甲基供体，由甲硫氨酸和ATP合成，参与DNA甲基化、蛋白质甲基化等反应。
一碳单位（One-Carbon Units）
以四氢叶酸为载体的甲基、甲酰基、亚甲基等基团，参与嘌呤、胸腺嘧啶和某些氨基酸的合成。
支链氨基酸（Branched-Chain Amino Acids, BCAA）
包括缬氨酸、亮氨酸和异亮氨酸，主要在肌肉组织代谢，是重要的能量来源。
枫糖浆尿症（Maple Syrup Urine Disease）
支链α-酮酸脱氢酶复合体缺陷引起的遗传病，导致支链氨基酸及其酮酸在体内蓄积。
磷酸吡哆醛（Pyridoxal Phosphate, PLP）
维生素B6的活性形式，是转氨酶的辅酶，参与氨基酸的转氨、脱羧等反应。
N-乙酰谷氨酸（N-Acetylglutamate）
氨基甲酰磷酸合成酶I的变构激活剂，调节尿素循环的活性，其合成受精氨酸激活。
同型半胱氨酸（Homocysteine）
甲硫氨酸代谢的中间产物，可重新甲基化为甲硫氨酸或通过转硫途径转化为半胱氨酸。
四氢生物蝶呤（Tetrahydrobiopterin, BH4）
苯丙氨酸羟化酶、酪氨酸羟化酶等的重要辅因子，参与芳香族氨基酸的羟化反应。

五、选择题测试

六、深度思考题

1. 比较分析泛素-蛋白酶体系统和溶酶体蛋白质降解系统的异同点，并说明它们在细胞生理功能中的重要意义。

  1. 相同点：
    • 都是细胞内重要的蛋白质降解系统，维持蛋白质稳态
    • 都具有选择性，能够识别特定的底物蛋白质
    • 都参与细胞质量控制，清除异常或损伤的蛋白质
    • 都受到严格的调节控制，确保降解的准确性和时效性
  2. 不同点：
    • 底物特异性：UPS主要降解胞质和核内蛋白质，溶酶体系统主要降解膜蛋白、细胞器和胞外蛋白质
    • 能量依赖性：UPS需要ATP提供能量进行泛素化和蛋白酶体活动，溶酶体降解本身不需要ATP
    • 标记机制：UPS使用泛素作为降解标签，溶酶体通过自噬体包裹或受体介导内吞
    • 降解速度：UPS降解速度快，适合快速调节，溶酶体降解相对较慢但容量大
  3. 生理意义：
    • 维持细胞蛋白质稳态，防止蛋白质聚集和毒性
    • 调节细胞周期、信号转导等重要生理过程
    • 参与细胞应激反应和适应性调节
    • 与多种疾病（如神经退行性疾病、癌症）密切相关

2. 详细阐述尿素循环的分子机制，分析其调节方式，并讨论尿素循环异常对机体的影响。

  1. 分子机制：
    • 第一步（线粒体内）：CPS I催化NH₃ + CO₂ + 2ATP → 氨基甲酰磷酸，需要N-乙酰谷氨酸激活
    • 第二步（线粒体内）：OCT催化氨基甲酰磷酸 + 鸟氨酸 → 瓜氨酸
    • 第三步（胞质）：ASS催化瓜氨酸 + 天冬氨酸 + ATP → 精氨酸琥珀酸
    • 第四步（胞质）：ASL催化精氨酸琥珀酸 → 精氨酸 + 延胡索酸
    • 第五步（胞质）：ARG催化精氨酸 → 尿素 + 鸟氨酸
  2. 调节方式：
    • 变构调节：N-乙酰谷氨酸激活CPS I，精氨酸促进N-乙酰谷氨酸合成
    • 酶诱导：高蛋白饮食、胰高血糖素、皮质醇诱导尿素循环酶合成
    • 底物调节：氨浓度升高促进循环活动
    • 组织特异性：主要在肝脏进行，肠道、肾脏也有一定活性
  3. 异常影响：
    • 高氨血症：可导致神经系统损伤、昏迷甚至死亡
    • 遗传性尿素循环缺陷：如鸟氨酸氨基甲酰转移酶缺陷症
    • 肝功能损伤：影响尿素合成能力，导致氨中毒
    • 营养代谢紊乱：影响氨基酸和蛋白质代谢平衡

3. 分析氨基酸代谢与糖代谢、脂代谢之间的相互关系，说明氨基酸在能量代谢中的作用。

  1. 与糖代谢的关系：
    • 糖异生：生糖氨基酸可通过糖异生途径合成葡萄糖，在饥饿状态下维持血糖稳定
    • 丙氨酸-葡萄糖循环：肌肉产生的丙氨酸运输到肝脏转化为葡萄糖
    • 共同中间产物：丙酮酸、α-酮戊二酸等既是氨基酸代谢产物也是糖代谢中间产物
    • 调节机制：胰岛素促进氨基酸合成蛋白质，胰高血糖素促进氨基酸分解
  2. 与脂代谢的关系：
    • 酮体生成：生酮氨基酸可转化为乙酰CoA，进一步合成酮体
    • 脂肪酸合成：氨基酸代谢产生的乙酰CoA可用于脂肪酸合成
    • 胆固醇合成：某些氨基酸参与胆固醇合成途径
    • 载脂蛋白合成：氨基酸是载脂蛋白合成的原料
  3. 能量代谢作用：
    • 直接供能：氨基酸碳骨架进入TCA循环产生ATP
    • 应急供能：在糖原耗竭时，肌肉蛋白质分解提供氨基酸供能
    • 代谢调节：某些氨基酸（如亮氨酸）具有信号分子功能，调节mTOR通路
    • 组织特异性：不同组织对氨基酸的利用方式不同，如肌肉主要利用支链氨基酸

4. 论述一碳单位代谢的生物学意义，分析其与氨基酸代谢、核酸代谢的关系。

  1. 一碳单位的来源和载体：
    • 主要来源：丝氨酸、甘氨酸、组氨酸、色氨酸等氨基酸的分解
    • 载体系统：四氢叶酸（THF）及其衍生物
    • 一碳单位形式：甲基、甲酰基、亚甲基、甲炔基等
    • 维生素依赖：需要叶酸、维生素B12等维生素参与
  2. 与氨基酸代谢的关系：
    • 氨基酸合成：参与甲硫氨酸、甘氨酸、组氨酸等的合成
    • 甲硫氨酸循环：提供甲基用于同型半胱氨酸的重甲基化
    • 丝氨酸-甘氨酸转换：丝氨酸脱氨产生一碳单位和甘氨酸
    • 代谢调节：一碳单位水平影响氨基酸代谢的方向和速度
  3. 与核酸代谢的关系：
    • 嘌呤合成：提供C2和C8位的碳原子
    • 胸腺嘧啶合成：dUMP甲基化形成dTMP
    • DNA甲基化：通过SAM提供甲基进行表观遗传调节
    • RNA修饰：参与某些RNA的甲基化修饰
  4. 生物学意义：
    • 细胞增殖：DNA合成必需，是细胞分裂的限制因子
    • 表观遗传：DNA和组蛋白甲基化调节基因表达
    • 神经发育：影响神经管闭合，缺乏可导致神经管缺陷
    • 疾病相关：与癌症、心血管疾病、神经退行性疾病相关

5. 结合临床实例，分析氨基酸代谢异常疾病的发病机制、临床表现和治疗策略。

  1. 苯丙酮尿症（PKU）：
    • 发病机制：苯丙氨酸羟化酶缺陷，苯丙氨酸不能转化为酪氨酸
    • 临床表现：智力发育迟缓、癫痫、皮肤色素减少、鼠臭味
    • 治疗策略：低苯丙氨酸饮食、四氢生物蝶呤补充（部分患者）
    • 预防措施：新生儿筛查、遗传咨询
  2. 枫糖浆尿症：
    • 发病机制：支链α-酮酸脱氢酶复合体缺陷
    • 临床表现：尿液有枫糖浆味、喂养困难、神经系统症状
    • 治疗策略：限制支链氨基酸摄入、硫胺素补充（某些类型）
    • 急性处理：血液透析清除毒性代谢产物
  3. 高氨血症：
    • 发病机制：尿素循环酶缺陷或肝功能损伤
    • 临床表现：意识障碍、昏迷、脑水肿
    • 治疗策略：限制蛋白质摄入、苯甲酸钠等氨清除剂
    • 支持治疗：血液净化、营养支持
  4. 同型半胱氨酸尿症：
    • 发病机制：胱硫氨酸β-合成酶缺陷等
    • 临床表现：智力障碍、骨骼异常、血管病变
    • 治疗策略：维生素B6、B12、叶酸补充，低甲硫氨酸饮食
    • 预后改善：早期诊断和治疗可显著改善预后
