生命的分子基础

考研生物化学复习笔记 - 第一章

一、生命属性与生物化学概述

1.1 生命的基本特征

生命的基本特征包括：

  • 新陈代谢：生物体内物质和能量的转换过程
  • 生长发育：从简单到复杂的有序变化
  • 遗传变异：信息的传递和改变
  • 应激性：对环境变化的反应能力
  • 稳态调节：维持内环境相对稳定

1.2 生物化学的研究内容

生物化学是研究生物体内化学成分和化学反应的科学，主要包括：

  • 生物分子的结构与功能
  • 物质代谢及其调节
  • 遗传信息的传递与表达
  • 生物能学和生物热力学

重要概念：生物化学与分子生物学是现代生命科学的核心学科，为理解生命现象的分子机理提供理论基础。

二、生命物质的化学组成

2.1 生物体的元素组成

生物体主要元素：

  • 大量元素：C、H、O、N、P、S（占生物体重量的99%以上）
  • 微量元素：Fe、Zn、Cu、Mn、Mo、I等（含量少但功能重要）

碳元素的特殊性

碳是生命的核心元素，具有以下特点：

  • 能形成稳定的共价键
  • 可形成链状、环状等多种结构
  • 价电子数适中，化学性质活泼适度
  • 能与多种元素结合形成官能团

2.2 生物分子的分类

生物分子分类体系：

小分子化合物 ← → 生物大分子

（分子量 < 1000Da）    （分子量 > 1000Da）

小分子化合物

  • 单糖：葡萄糖、果糖、核糖等
  • 氨基酸：蛋白质的基本单位
  • 核苷酸：核酸的基本单位
  • 脂肪酸：脂质的重要组成
  • 维生素：生物催化剂的辅助因子

生物大分子

  • 蛋白质：由氨基酸聚合而成
  • 核酸：由核苷酸聚合而成（DNA、RNA）
  • 多糖：由单糖聚合而成
  • 脂质：主要包括磷脂、胆固醇等

三、生物分子的三维结构

3.1 构型与构象

构型（Configuration）：分子中原子的空间排列，只能通过化学键的断裂和重新形成来改变。

构象（Conformation）：分子中原子围绕单键自由旋转产生的不同空间排列。

立体异构现象

  • 对映异构体：互为镜像关系的立体异构体
  • 非对映异构体：非镜像关系的立体异构体
  • D/L构型：以甘油醛为标准的构型命名法

3.2 生物分子的手性

生物体中的手性选择性：

  • 天然氨基酸几乎都是L型
  • 天然糖类几乎都是D型
  • 这种选择性是生命进化的结果

四、生物系统中的非共价相互作用

4.1 氢键

氢键形成条件：

X-H···Y（X、Y为电负性强的原子：N、O、F）

氢键的特点：

  • 具有方向性和饱和性
  • 键能：12-30 kJ/mol
  • 在蛋白质和核酸结构中起重要作用

4.2 范德华力

包括三种类型：

  • 取向力：永久偶极间的相互作用
  • 诱导力：永久偶极诱导瞬时偶极产生的作用力
  • 色散力：瞬时偶极间的相互作用

4.3 疏水相互作用

疏水效应：非极性分子在水中聚集，减少与水分子的接触面积，这是一个熵驱动的过程。

4.4 静电相互作用

包括离子键和偶极相互作用，在生物分子的稳定性和功能中发挥重要作用。

五、水和生命

5.1 水的分子结构

水分子的几何结构：

H-O-H键角：104.5°

O-H键长：0.96 Å

5.2 水的特殊性质

水的重要性质：

  • 高沸点和熔点：由于氢键的存在
  • 高比热容：温度缓冲作用
  • 高介电常数：良好的溶剂
  • 密度异常：冰的密度小于液态水

5.3 水的生物学功能

  • 溶剂作用：溶解离子和极性分子
  • 反应介质：许多生化反应的场所
  • 运输载体：营养物质和代谢产物的运输
  • 温度调节：维持体温稳定
  • 润滑作用：关节液、唾液等

5.4 水的电离和pH

水的自电离：

H₂O ⇌ H⁺ + OH⁻

Kw = [H⁺][OH⁻] = 1.0 × 10⁻¹⁴ (25°C)

pH = -log[H⁺]

生物体内pH的重要性：细胞内pH通常维持在7.0-7.4之间，pH的微小变化都会影响蛋白质结构和酶活性。

六、生命的基本单位——细胞

6.1 细胞的基本特征

  • 具有细胞膜包围的结构
  • 含有遗传物质DNA
  • 能进行新陈代谢
  • 具有自我复制能力

6.2 原核细胞与真核细胞

主要区别：

  • 原核细胞：无核膜，DNA直接分布在细胞质中
  • 真核细胞：有核膜，DNA包装在细胞核内

6.3 细胞的分子组成

细胞的化学组成（按重量）：

水：70-85%

蛋白质：10-15%

脂质：2-5%

核酸：1-2%

糖类和其他：1-3%

七、重要名词解释库

生物化学 (Biochemistry)
研究生物体内化学成分的组成、结构及其化学反应的科学，是生命科学的重要分支学科。
分子生物学 (Molecular Biology)
从分子水平研究生命现象的科学，主要研究蛋白质和核酸等生物大分子的结构与功能关系。
构型 (Configuration)
分子中原子的固定空间排列，只能通过化学键的断裂和重新形成来改变。
构象 (Conformation)
分子中原子围绕单键自由旋转产生的不同空间排列，可通过分子内旋转改变。
手性 (Chirality)
分子与其镜像不能重合的性质，具有手性的分子称为手性分子。
对映异构体 (Enantiomer)
互为镜像关系且不能重合的立体异构体，具有相同的物理化学性质但旋光性相反。
氢键 (Hydrogen Bond)
氢原子与电负性强的原子（如N、O、F）之间形成的弱相互作用，具有方向性和饱和性。
疏水相互作用 (Hydrophobic Interaction)
非极性分子在水溶液中相互聚集以减少与水分子接触的现象，是熵驱动的过程。
范德华力 (Van der Waals Force)
分子间的弱相互作用力，包括取向力、诱导力和色散力三种类型。
两性离子 (Zwitterion)
同时带有正电荷和负电荷的离子，如氨基酸在等电点时的存在形式。
等电点 (Isoelectric Point, pI)
分子净电荷为零时溶液的pH值，此时分子在电场中不发生迁移。
缓冲系统 (Buffer System)
能够抵抗外加酸碱对溶液pH值影响的体系，通常由弱酸及其共轭碱组成。

八、选择题测试

九、深度思考题

1. 为什么说水是生命存在的基础？请从分子结构和生物学功能两个角度进行分析。

参考答案：
  1. 分子结构角度：水分子具有极性结构，氧原子带部分负电荷，氢原子带部分正电荷，这使得水分子能够形成氢键。水的这种特殊结构赋予了它许多独特的物理化学性质，如高沸点、高比热容、高介电常数等。
  2. 生物学功能角度：水在生物体内发挥多重关键作用：作为优良溶剂溶解各种生物分子；作为反应介质参与生化反应；作为运输载体运送营养物质和代谢产物；通过高比热容维持体温稳定；参与许多代谢反应如水解反应。
  3. 进化意义：生命起源于水环境，生物体的进化过程中始终依赖水环境，细胞内外的水环境为生命活动提供了必需的条件。
  4. 结构稳定性：水分子通过氢键网络维持生物大分子如蛋白质、核酸的三维结构稳定，这对维持其生物活性至关重要。

2. 生物体为什么选择性地使用L型氨基酸和D型糖？这种手性选择性对生命有什么意义？

参考答案：
  1. 进化选择结果：生物体对L型氨基酸和D型糖的选择是长期进化的结果。在生命起源初期，可能存在多种手性形式，但由于某种偶然因素或选择压力，最终固定为现在的形式。
  2. 结构一致性：使用统一的手性形式确保了生物大分子结构的一致性和规律性。蛋白质中全部使用L型氨基酸使得多肽链能够形成规则的二级结构如α螺旋和β折叠。
  3. 酶的专一性：酶对底物具有严格的立体专一性，统一的手性确保了酶与底物的精确识别和结合，提高了生化反应的效率和准确性。
  4. 遗传信息传递：DNA和RNA中使用D型糖确保了遗传信息传递的准确性，双螺旋结构的形成需要统一的糖环构型。
  5. 代谢协调性：统一的手性使得各种代谢途径能够协调进行，避免了不同手性分子间的相互干扰。

3. 非共价相互作用在生物分子结构和功能中发挥什么作用？请结合具体实例说明。

参考答案：
  1. 维持蛋白质结构：氢键维持蛋白质的二级结构（α螺旋、β折叠）；疏水相互作用驱动蛋白质折叠形成紧密的三级结构；静电相互作用稳定蛋白质的整体构象。例如，血红蛋白的四级结构就是通过多种非共价相互作用维持的。
  2. DNA双螺旋结构：DNA双链间通过氢键配对（A-T形成2个氢键，G-C形成3个氢键），双螺旋的稳定性还依赖于碱基间的π-π堆积作用和疏水相互作用。
  3. 酶-底物识别：酶与底物的结合主要通过氢键、范德华力等非共价相互作用实现，这种结合具有可逆性，有利于产物的释放。
  4. 膜结构形成：磷脂双分子层的形成主要依靠疏水相互作用，磷脂分子的疏水尾部聚集在一起，亲水头部朝向水相。
  5. 分子识别：抗体与抗原的特异性结合、激素与受体的相互作用等都主要依赖非共价相互作用，确保了生物识别的特异性和可逆性。

4. 从化学角度分析，为什么碳元素成为构成生命的核心元素？

参考答案：
  1. 化学键特性：碳原子具有4个价电子，能够形成4个稳定的共价键，既不像氢那样化学性质过于活泼，也不像硅那样过于稳定，化学活性适中。
  2. 结构多样性：碳原子能够形成链状、环状、分支状等多种骨架结构，为构建复杂的生物分子提供了结构基础。碳-碳键的稳定性使得长链分子能够稳定存在。
  3. 官能团多样性：碳能够与氧、氮、硫、磷等多种元素结合形成各种官能团（如羟基、氨基、羧基、磷酸基等），这些官能团赋予了生物分子不同的化学性质和生物活性。
  4. 立体化学：碳原子的四面体构型使得含碳化合物具有丰富的立体化学特性，能够形成对映异构体，为生物分子的精确识别提供了基础。
  5. 能量适宜性：碳-氢键和碳-碳键的键能适中，既能提供足够的稳定性，又能在适宜的条件下发生化学反应，满足了生物体内能量转换的需要。

5. 生物体内的pH缓冲系统是如何工作的？为什么维持pH稳定对生命如此重要？

参考答案：
  1. 缓冲系统组成：生物体内主要的缓冲系统包括碳酸-碳酸氢盐缓冲系统（HCO₃⁻/H₂CO₃）、磷酸缓冲系统（HPO₄²⁻/H₂PO₄⁻）、蛋白质缓冲系统等。这些系统由弱酸及其共轭碱组成。
  2. 缓冲机理：当有酸性物质加入时，缓冲系统中的碱性成分会结合H⁺；当有碱性物质加入时，缓冲系统中的酸性成分会释放H⁺，从而维持pH相对稳定。
  3. 蛋白质结构稳定性：蛋白质的三级结构依赖于氢键、静电相互作用等，pH的变化会影响氨基酸侧链的电离状态，进而影响蛋白质的构象和活性。
  4. 酶活性调节：酶的活性中心通常含有可电离基团，pH的变化会影响这些基团的电离状态，从而影响酶的催化活性。大多数酶只在特定的pH范围内具有最佳活性。
  5. 膜结构稳定性：细胞膜的稳定性也受pH影响，pH的剧烈变化可能导致膜结构破坏，影响细胞的正常功能。
  6. 代谢过程调节：许多代谢反应的进行都需要特定的pH环境，pH的稳定确保了代谢过程的正常进行。
