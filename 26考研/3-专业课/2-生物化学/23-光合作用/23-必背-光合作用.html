国科大生命科学学院 细胞生物学光合作用必背考点

一、核心名词解释（10个）

光合作用（Photosynthesis）：绿色植物利用太阳光能，将二氧化碳和水合成有机物并释放氧气的生物过程，是地球生物圈主要的初级生产模式，包括光反应和暗反应两个阶段。

叶绿体（Chloroplast）：植物细胞中进行光合作用的细胞器，由双分子膜包被、基质和类囊体三部分组成，是光能转化为化学能的主要场所。

光系统Ⅰ（Photosystem I，PSⅠ）：位于类囊体膜上的色素蛋白复合体，吸收峰为700nm，又称P700，参与非循环和循环电子传递，将电子传递给铁氧还蛋白。

光系统Ⅱ（Photosystem II，PSⅡ）：位于类囊体膜上的色素蛋白复合体，吸收峰为680nm，又称P680，催化水的光解产生氧气，是非循环电子传递的起点。

Calvin循环（Calvin Cycle）：光合作用暗反应的主要途径，包括CO₂固定、还原反应和RuBP再生三个阶段，固定3分子CO₂需消耗9分子ATP和6分子NADPH。

RuBisCO酶（Ribulose-1,5-bisphosphate carboxylase/oxygenase）：催化CO₂与RuBP结合的关键酶，是地球上含量最丰富的蛋白质，同时具有羧化酶和加氧酶活性，参与光呼吸过程。

光合磷酸化（Photophosphorylation）：在光反应中由光驱动，利用跨类囊体膜的质子梯度将ADP和Pi合成ATP的过程，包括非循环和循环两种类型。

类囊体（Thylakoid）：叶绿体内由单层膜围成的扁平小囊结构，分为基粒类囊体和基质类囊体，是光反应进行的场所，膜上分布有光系统等蛋白复合体。

叶绿素（Chlorophyll，Chl）：光合作用中的主要光合色素，包括叶绿素a和叶绿素b，主要吸收红光和蓝紫光，反射绿光，是光能吸收和转化的关键分子。

光呼吸（Photorespiration）：RuBisCO酶在高温、强光、低CO₂条件下催化RuBP与O₂结合的过程，消耗ATP和NADPH但不产生糖类，被认为是光合作用的副反应。

二、必备核心知识点（5个）

【必备知识点1】光合作用总反应式及生物学意义
总反应式：6CO₂ + 6H₂O + 光能 → C₆H₁₂O₆ + 6O₂。光合作用是地球生物圈的能量基础，维持大气中碳氧平衡，为异养生物提供有机物和氧气，是生态系统物质循环和能量流动的起点。

【必备知识点2】光反应的分子机理及电子传递链
非循环电子传递：H₂O → PSⅡ → PQ → Cytb₆/f → PC → PSⅠ → Fd → NADP⁺，产生ATP、NADPH和O₂。循环电子传递：PSⅠ → Fd → Cytb₆/f → PC → PSⅠ，仅产生ATP。Z型电子传递模式确保能量逐级传递。

【必备知识点3】Calvin循环的三个阶段及化学计量
1.CO₂固定：RuBisCO催化CO₂与RuBP结合形成3-磷酸甘油酸。2.还原反应：消耗ATP和NADPH将3-磷酸甘油酸还原为3-磷酸甘油醛。3.RuBP再生：消耗ATP使3-磷酸甘油醛重新形成RuBP。固定3分子CO₂需要9ATP + 6NADPH。

【必备知识点4】光合作用的调节机制
光强度：影响光反应速率，存在光补偿点和光饱和点。CO₂浓度：影响暗反应速率，存在CO₂补偿点。温度：主要影响酶活性。C₄途径和CAM途径是适应高温干旱环境的CO₂浓缩机制。

【必备知识点5】叶绿体结构与功能的对应关系
类囊体膜：光反应场所，分布PSⅠ、PSⅡ、Cytb₆/f和ATP合酶。基质：暗反应场所，含RuBisCO等Calvin循环酶系。基粒：类囊体堆叠结构，增大膜面积。双分子膜：控制物质进出，维持光合作用微环境。

三、高频必考大题（2道）

【高频考题1】试述光合作用中光反应和暗反应的联系，并分析两者在时间、空间、条件和产物方面的异同。

标准答案：
1.联系：光反应为暗反应提供ATP和NADPH（同化力），暗反应为光反应提供ADP、Pi和NADP⁺，两者相互依存、协调进行。
2.时间差异：光反应需要光照，暗反应不直接需要光但需要光反应产物。
3.空间差异：光反应在类囊体膜上进行，暗反应在叶绿体基质中进行。
4.条件差异：光反应需要光能、H₂O、ADP、Pi、NADP⁺，暗反应需要CO₂、ATP、NADPH、酶系。
5.产物差异：光反应产生ATP、NADPH、O₂，暗反应产生有机物（糖类）、ADP、Pi、NADP⁺。

【高频考题2】分析影响光合作用速率的主要环境因子，并解释C₄植物和CAM植物对环境的适应机制。

标准答案：
1.光照强度：低光强时为限制因子，存在光补偿点（光合速率=呼吸速率）和光饱和点（光合速率不再增加）。
2.CO₂浓度：影响Calvin循环速率，存在CO₂补偿点，大气CO₂浓度常为限制因子。
3.温度：主要影响酶活性，过高或过低都会降低光合速率。
4.C₄途径：通过PEP羧化酶固定CO₂形成C₄化合物，在维管束鞘细胞释放CO₂供Calvin循环，提高CO₂浓度，减少光呼吸。
5.CAM途径：夜间开放气孔固定CO₂形成苹果酸，白天关闭气孔释放CO₂进行光合作用，适应干旱环境。
