光合作用

生物化学考研复习笔记 | 第23章

一、光合作用概述

1.1 光合作用的定义与意义

光合作用是指绿色植物、藻类和某些细菌利用叶绿素等光合色素，在光照条件下，将二氧化碳和水转化为有机物，并释放氧气的生物化学过程。这是地球上最重要的生物化学反应之一。

图1：光合作用总反应式示意图

光合作用的生物学意义包括：

  • 为地球上几乎所有生物提供有机物和能量来源
  • 维持大气中氧气和二氧化碳的平衡
  • 是生物圈物质循环和能量流动的基础
  • 为人类提供食物、燃料和工业原料

1.2 光合作用的基本过程

光合作用可分为两个相互关联的阶段：

（1）光反应阶段（光依赖反应）

发生在类囊体膜上，需要光照。主要过程包括：

  • 光能吸收：叶绿素等色素吸收光能
  • 水的光解：H₂O → 2H⁺ + ½O₂ + 2e⁻
  • ATP合成：光合磷酸化产生ATP
  • NADPH生成：NADP⁺被还原为NADPH

（2）碳反应阶段（暗反应/Calvin循环）

发生在叶绿体基质中，不直接需要光照。主要过程包括：

  • CO₂固定：CO₂与RuBP结合形成3-磷酸甘油酸
  • 还原反应：利用ATP和NADPH还原3-磷酸甘油酸
  • RuBP再生：部分产物用于再生RuBP

二、光反应的分子机理

2.1 光合色素系统

光合作用中的色素主要包括：

  • 叶绿素a：主要吸收红光和蓝紫光，是光反应中心的主要色素
  • 叶绿素b：辅助色素，扩大光能吸收范围
  • 胡萝卜素：吸收蓝绿光，具有保护作用
  • 叶黄素：吸收蓝紫光，参与光保护

2.2 光系统I和光系统II

光反应通过两个光系统协同完成：

光系统II（PSII）

  • 反应中心：P680（叶绿素a的特殊形式）
  • 主要功能：水的光解，产生O₂、H⁺和电子
  • 电子传递：P680 → 质体醌 → 细胞色素b₆f复合体

光系统I（PSI）

  • 反应中心：P700
  • 主要功能：NADP⁺的还原，产生NADPH
  • 电子传递：P700 → 铁硫蛋白 → NADP⁺还原酶

2.3 电子传递链与ATP合成

光反应中的非循环电子传递过程：

  1. PSII吸收光能，P680被激发失去电子
  2. 失去的电子由水分子提供，同时产生O₂和H⁺
  3. 电子经质体醌、细胞色素b₆f复合体传递到PSI
  4. PSI吸收光能，P700被激发
  5. 电子最终传递给NADP⁺，形成NADPH
  6. 质子梯度驱动ATP合酶合成ATP

三、Calvin循环（碳反应）

3.1 Calvin循环的三个阶段

阶段一：CO₂固定

关键酶：RuBisCO（1,5-二磷酸核酮糖羧化酶/加氧酶）

反应：RuBP + CO₂ → 2分子 3-磷酸甘油酸（3-PGA）

阶段二：还原反应

利用光反应产生的ATP和NADPH：

  • 3-PGA + ATP → 1,3-二磷酸甘油酸
  • 1,3-二磷酸甘油酸 + NADPH → 甘油醛-3-磷酸（G3P）

阶段三：RuBP再生

部分G3P通过复杂的重排反应再生RuBP，消耗ATP。

3.2 Calvin循环的化学计量

固定3分子CO₂，产生1分子G3P（可用于合成葡萄糖）需要：

  • ATP：9分子
  • NADPH：6分子
  • RuBP：3分子（循环再生）

四、光合作用的调节

4.1 光强度的影响

光合作用速率与光强度的关系呈现光饱和现象：

  • 光限制阶段：光强度低时，光反应限制整个过程
  • 光饱和阶段：光强度足够时，暗反应成为限制因子
  • 光抑制阶段：光强度过高时，可能损伤光合器官

4.2 CO₂浓度的影响

CO₂是Calvin循环的底物，其浓度直接影响碳固定效率。

4.3 温度的影响

温度主要影响酶活性，特别是RuBisCO的活性和专一性。

五、C₄途径和CAM途径

5.1 C₄途径

某些植物（如玉米、甘蔗）发展出的CO₂浓缩机制：

  • 在叶肉细胞中，CO₂首先与PEP结合形成C₄化合物
  • C₄化合物转运到维管束鞘细胞
  • 在维管束鞘细胞中释放CO₂，进入Calvin循环
  • 提高了CO₂浓度，减少了光呼吸

5.2 CAM途径

景天酸代谢（CAM）是干旱环境植物的适应机制：

  • 夜间开放气孔，固定CO₂形成苹果酸
  • 白天关闭气孔，分解苹果酸释放CO₂
  • 减少水分损失，适应干旱环境

六、光呼吸

6.1 光呼吸的机理

光呼吸是RuBisCO的加氧酶活性导致的副反应：

  • RuBP + O₂ → 磷酸乙醇酸 + 3-磷酸甘油酸
  • 磷酸乙醇酸经过复杂途径最终产生CO₂
  • 消耗ATP，降低光合效率

6.2 光呼吸的生物学意义

  • 可能具有光保护作用
  • 参与氮代谢
  • 在进化上可能是适应高氧环境的结果

光合作用
(Photosynthesis)
绿色植物、藻类和某些细菌利用光能，将CO₂和H₂O转化为有机物并释放O₂的生物化学过程。
叶绿素
(Chlorophyll)
植物中主要的光合色素，能够吸收光能并参与光反应。主要包括叶绿素a和叶绿素b。
类囊体
(Thylakoid)
叶绿体内的膜结构，光反应发生的场所，含有光系统I和光系统II。
光系统I
(Photosystem I, PSI)
以P700为反应中心的光合单位，主要功能是还原NADP⁺生成NADPH。
光系统II
(Photosystem II, PSII)
以P680为反应中心的光合单位，主要功能是分解水分子，产生O₂、H⁺和电子。
Calvin循环
(Calvin Cycle)
光合作用的碳反应阶段，在叶绿体基质中进行，将CO₂固定并还原为有机物。
RuBisCO
(Ribulose-1,5-bisphosphate carboxylase/oxygenase)
1,5-二磷酸核酮糖羧化酶/加氧酶，Calvin循环中CO₂固定的关键酶，也是光呼吸的关键酶。
光合磷酸化
(Photophosphorylation)
利用光反应产生的质子梯度驱动ATP合酶合成ATP的过程。
Z型电子传递链
(Z-scheme)
光反应中电子从水分子传递到NADP⁺的途径，因其能量变化图形似Z字而得名。
C₄途径
(C₄ Pathway)
某些植物的CO₂浓缩机制，先形成C₄化合物，再释放CO₂进入Calvin循环，提高光合效率。
CAM途径
(Crassulacean Acid Metabolism)
景天酸代谢，干旱植物的适应机制，夜间固定CO₂，白天进行Calvin循环。
光呼吸
(Photorespiration)
RuBisCO的加氧酶活性导致的过程，消耗O₂产生CO₂，降低光合效率。
质体醌
(Plastoquinone, PQ)
类囊体膜中的电子载体，在光系统II和细胞色素b₆f复合体之间传递电子。
质体蓝素
(Plastocyanin, PC)
含铜的电子载体蛋白，在细胞色素b₆f复合体和光系统I之间传递电子。
3-磷酸甘油酸
(3-Phosphoglycerate, 3-PGA)
Calvin循环中CO₂固定的第一个稳定产物，由RuBP和CO₂结合形成。

深度思考题

1. 比较分析光反应和暗反应的异同点，并阐述两者之间的相互关系。

参考答案：
  1. 相同点：都是光合作用的重要组成部分，都需要酶的催化，都涉及能量转换，都对光合作用的总效率有重要影响。两个阶段在时间和空间上紧密联系，共同完成CO₂到有机物的转化过程。
  2. 不同点：光反应发生在类囊体膜上，直接需要光照，主要进行能量转换（光能→化学能），产生ATP和NADPH，同时分解水产生氧气。暗反应发生在叶绿体基质中，不直接需要光照，主要进行物质转换（CO₂→有机物），消耗ATP和NADPH，通过Calvin循环固定和还原CO₂。
  3. 相互关系：光反应为暗反应提供ATP和NADPH这两种高能化合物，暗反应消耗这些高能化合物并产生ADP、Pi和NADP⁺，后者又可被光反应重新利用。这种循环利用保证了光合作用的连续进行。光反应的效率直接影响暗反应的进行速度，而暗反应的速度也会反过来影响光反应产物的消耗。
  4. 调节机制：两个阶段通过多种调节机制保持平衡，包括ATP/ADP比值调节、NADPH/NADP⁺比值调节、以及关键酶活性的调节等。

2. 分析RuBisCO酶的双重功能及其对植物光合效率的影响，并说明C₄植物是如何解决这一问题的。

参考答案：
  1. RuBisCO的双重功能：RuBisCO既具有羧化酶活性（催化CO₂与RuBP结合），又具有加氧酶活性（催化O₂与RuBP结合）。羧化酶活性是光合作用所需的，而加氧酶活性则导致光呼吸，降低光合效率。这种双重功能是由于RuBisCO的活性位点既能结合CO₂也能结合O₂。
  2. 对光合效率的影响：当O₂浓度高、CO₂浓度低或温度高时，RuBisCO的加氧酶活性增强，导致光呼吸增加。光呼吸消耗ATP和还原力，释放CO₂，不产生有用的产物，严重降低光合效率。在正常大气条件下，光呼吸可使光合效率降低20-50%。
  3. C₄植物的解决策略：C₄植物通过空间分离和CO₂浓缩机制解决这一问题。在叶肉细胞中，PEP羧化酶（对CO₂专一性高）催化CO₂与PEP结合形成C₄化合物（如苹果酸）。C₄化合物转运到维管束鞘细胞后分解释放CO₂，在维管束鞘细胞中形成高CO₂浓度环境，使RuBisCO主要发挥羧化酶活性，大大减少光呼吸。
  4. 进化意义：C₄途径是植物适应高温、强光、干旱环境的重要进化策略，特别适合热带和亚热带地区的植物，如玉米、甘蔗、高粱等。

3. 阐述光合作用中能量转换的全过程，并分析其在生物圈能量流动中的重要地位。

参考答案：
  1. 光能捕获与转换：叶绿素等色素分子吸收太阳光能，电子被激发到高能级状态。这是将光能转换为电子激发能的第一步，为后续的化学能转换奠定基础。
  2. 电化学能转换：激发的电子通过Z型电子传递链，将光能转换为电化学势能。在这个过程中，水分子被分解提供电子，同时在类囊体内外形成质子梯度，储存了大量的电化学势能。
  3. 化学能合成：质子梯度驱动ATP合酶合成ATP，电子最终还原NADP⁺形成NADPH。这一步将电化学势能转换为稳定的化学键能，形成了生物体可直接利用的高能化合物。
  4. 有机物合成：ATP和NADPH在Calvin循环中被消耗，将CO₂还原固定为有机物。这是将化学键能转换为稳定有机分子中化学键能的过程，形成了可长期储存的能量形式。
  5. 生物圈地位：光合作用是地球上几乎所有生态系统的能量基础，每年固定约1000亿吨碳，转换的能量约为人类年消耗能量的10倍。它不仅为食物链提供能量来源，还维持了大气中O₂和CO₂的平衡，是生物圈物质循环和能量流动的核心环节。
  6. 环境意义：光合作用调节全球碳循环，影响气候变化，为人类提供食物、燃料、纤维等资源，是维持地球生态平衡的关键过程。

4. 比较C₃、C₄和CAM三种光合途径的特点，并分析它们各自的生态适应意义。

参考答案：
  1. C₃途径特点：最基本的光合途径，CO₂直接进入Calvin循环，第一个稳定产物是3-磷酸甘油酸（C₃化合物）。结构简单，但在高温、强光、低CO₂条件下效率较低，容易发生光呼吸。适应温带、湿润环境，如水稻、小麦、大豆等。
  2. C₄途径特点：具有CO₂浓缩机制，在叶肉细胞中先形成C₄化合物，然后在维管束鞘细胞中释放CO₂进入Calvin循环。具有空间分离特征，能有效抑制光呼吸，在高温、强光条件下效率高。但需要额外的ATP消耗，结构复杂。适应热带、亚热带环境，如玉米、甘蔗、高粱等。
  3. CAM途径特点：具有时间分离特征，夜间开放气孔固定CO₂形成苹果酸储存，白天关闭气孔分解苹果酸释放CO₂进行Calvin循环。能极大减少水分损失，但光合效率相对较低。适应干旱、半干旱环境，如仙人掌、凤梨、景天等。
  4. 生态适应意义：三种途径代表了植物对不同环境条件的适应策略。C₃途径适合温和湿润环境，结构简单能耗低；C₄途径适合高温强光环境，能有效利用强光和高温；CAM途径适合干旱环境，能最大限度节约水分。这种多样性使植物能够占据地球上几乎所有的生态位。
  5. 进化和农业意义：了解这些途径有助于作物改良和农业生产优化，例如在不同气候区选择适宜的作物品种，或通过基因工程改善作物的光合效率。

5. 分析环境因子（光照、温度、CO₂浓度）对光合作用的影响机理，并说明如何在农业生产中应用这些原理。

参考答案：
  1. 光照强度影响：光照强度直接影响光反应的进行。在光限制阶段，光合速率与光强成正比；达到光饱和点后，光合速率不再增加，此时暗反应成为限制因子；光强过高时可能发生光抑制，损伤光合器官。机理是光照影响叶绿素的激发和电子传递链的效率。
  2. 温度影响：温度主要影响酶活性，特别是RuBisCO等关键酶。适宜温度下酶活性高，光合效率高；温度过低时酶活性下降；温度过高时酶变性失活，同时高温促进光呼吸，降低净光合速率。不同植物的最适温度不同，反映了其生态适应性。
  3. CO₂浓度影响：CO₂是Calvin循环的底物，其浓度直接影响碳固定效率。在CO₂限制范围内，光合速率与CO₂浓度成正比；达到CO₂饱和点后，其他因子成为限制因子。高CO₂浓度还能抑制光呼吸，提高光合效率。
  4. 农业应用原理：在温室栽培中，可通过人工补光、温度控制、CO₂施肥等措施优化光合条件。例如，冬季补光延长光照时间，夏季遮阴防止光抑制，通过通风或CO₂发生器调节CO₂浓度，使用加温或降温设备维持适宜温度。
  5. 综合管理策略：根据作物特性和生长阶段，综合调控各环境因子，实现光合作用的最优化。同时考虑成本效益，选择最经济有效的调控措施。现代精准农业中，利用传感器实时监测环境参数，自动调节温室环境，实现智能化管理。
