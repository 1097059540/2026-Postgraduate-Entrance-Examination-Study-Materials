第8章 酶作用机制和酶活性调节

考研生物化学复习笔记 | 基于《生物化学》第四版

一、酶的活性部位

（一）活性部位的概念

酶的活性部位（Active Site）或称活性中心（Active Center），是指酶分子中直接与底物结合并催化底物反应的部位。

活性部位的特点：

  • 只占酶分子很小部分（1%~2%）
  • 组成活性中心的氨基酸残基在空间结构中靠近，但在一级结构中可能相距很远
  • 通常处于或靠近酶分子的表面
  • 底物通过次级键（氢键、范德华力等）结合到酶上
  • 具有柔性或可运动性

活性部位的功能区域：

  1. 结合部位：决定酶的专一性
  2. 催化部位：决定酶的催化能力

（二）研究酶活性部位的方法

  • 酶分子侧链基团的化学修饰法：确定被修饰基团与酶活性的关系
  • 动力学参数测定法：比较底物和抑制剂对酶反应的影响
  • X射线晶体结构分析法：解析酶分子的三维结构
  • 定点诱变法：研究酶的结构和功能关系

二、酶催化反应的独特性质

  1. 反应类型：分为仅涉及电子转移和涉及电子、质子或其他基团转移两类
  2. 催化媒介：由氨基酸侧链上的功能基团和辅酶为媒介
  3. 反应条件：最适pH范围狭小，一般接近中性
  4. 分子大小：活性部位通常稍大于底物
  5. 协同特性：除催化反应所需的活性基团外，还有协同性、多个底物结合部位等特性

三、酶促反应机制

（一）基元催化反应

1. 酸碱催化

通过瞬时的向反应物提供质子或从反应物接受质子以稳定过渡态，加速反应的催化机制。

  • 影响因素：酸或碱的强度（pK值）和质子传递的速率
  • 重要基团：组氨酸咪唑基（pK≈6.0）在生理pH下既可作质子供体又可作质子受体

2. 共价催化

又称亲核催化或亲电子催化，通过形成不稳定的共价中间复合物，降低反应活化能。

  • 常见亲核基团：丝氨酸羟基、半胱氨酸巯基、组氨酸咪唑基
  • 机制：亲核基团释放电子作用于底物的缺电子中心

3. 金属离子催化

金属离子以三种主要途径参与催化：

  • 结合底物为反应定向
  • 可逆地改变金属离子的氧化态，调节氧化还原反应
  • 静电作用稳定或屏蔽负电荷

（二）酶具有高催化能力的原因

1. 邻近效应与定向效应

  • 邻近效应：酶把底物分子富集到活性中心附近，使有效浓度极大升高
  • 定向效应：使反应基团的分子轨道以正确方位相互交叠

2. 扭曲变形和构象变化的催化效应

酶中某些基团使底物分子的敏感键产生"电子张力"，使底物接近过渡态，降低反应活化能。

3. 多元催化和协同效应

几个基元催化反应共同作用，是酶使反应加速的重要因素。

4. 活性中心的微环境

  • 酶活性中心附近往往是疏水的，介电常数低
  • 可加强极性基团间的反应
  • 电荷离子可稳定过渡态的离子

四、酶活性的调节控制

（一）别构调控

1. 别构效应的概念

别构效应（Allosteric Effects）是指效应物（调节物）与别构酶分子中的别构中心（调节中心）结合后，诱导产生或稳定住酶分子的某种构象，使酶活性中心对底物的结合催化作用受到影响。

2. 别构酶的结构特点

  • 多为寡聚蛋白，有一个或多个别构部位
  • 经常位于一个代谢途径的开始或分支点
  • 具有与简单非调节酶不同的理化性质和动力学行为
  • 大都不遵循米氏动力学

3. 别构酶的动力学特征

  • 正协同效应：v-[S]曲线呈S形
  • 负协同效应：v-[S]曲线类似双曲线
  • 协同性参数Ri：Ri=81（无协同），Ri<81（正协同），Ri>81（负协同）

4. 别构模型

  • 齐变模型（WMC模型）：较好地解释了正协同效应
  • 序变模型（KNF模型）：可解释正协同性和负协同性

5. ATCase的别构调节

天冬氨酸转氨甲酰酶（ATCase）是别构酶的典型代表：

  • 结构：由催化亚基（C₆）和调节亚基（R₆）组成
  • 正调节剂：ATP激活酶活性
  • 负调节剂：CTP抑制酶活性（反馈抑制）
  • 生理意义：调节嘧啶核苷酸的合成

（二）酶原的激活

1. 酶原激活的概念

酶原激活是指某些酶先以无活性的酶原形式合成及分泌，然后在到达作用部位时由另外的物质作用，使其失去部分肽段从而形成或暴露活性中心的过程。

2. 酶原激活的机制

  • 酶原分子的内部肽键一处或多处断裂
  • 分子构象发生改变
  • 形成或暴露活性中心
  • 转变为有活性的酶

3. 酶原激活的实例

酶原 激活因子 活性酶 功能
胃蛋白酶原 HCl、胃蛋白酶 胃蛋白酶 蛋白质消化
胰蛋白酶原 肠激酶 胰蛋白酶 蛋白质消化
胰凝乳蛋白酶原 胰蛋白酶 胰凝乳蛋白酶 蛋白质消化
纤维蛋白原 凝血酶 纤维蛋白 血液凝固

4. 生理意义

  • 避免细胞产生的蛋白酶对细胞进行自身消化
  • 使酶在特定的部位和环境中发挥作用
  • 保证体内代谢的正常进行
  • 属于不可逆的调节机制

（三）可逆共价修饰

1. 共价修饰的概念

可逆共价修饰是指酶分子被其他酶催化进行共价修饰，从而在活性形式与非活性形式之间相互转变的调节机制。

2. 主要的共价修饰类型

  • 磷酸化与脱磷酸化：最常见的修饰方式
  • 乙酰化与脱乙酰化
  • 甲基化与脱甲基化
  • 腺苷酰化与脱腺苷酰化
  • ADP-核糖基化

3. 磷酸化修饰的特点

  • 催化酶：蛋白激酶（磷酸化）和蛋白磷酸酶（脱磷酸化）
  • 修饰位点：主要是丝氨酸、苏氨酸和酪氨酸残基
  • 调节特点：快速、可逆、精确
  • 实例：糖原磷酸化酶的调节

（四）同工酶

1. 同工酶的概念

同工酶（Isozyme）是指能催化相同的化学反应，但其蛋白分子结构、理化性质和免疫性能等方面都存在明显差异的一组酶。

2. 同工酶的特点

  • 被不同基因编码
  • 存在于同一个体的不同组织中
  • 甚至存在于同一组织、同一细胞的不同亚细胞结构中
  • 具有组织特异性

3. 乳酸脱氢酶（LDH）同工酶

LDH是研究最清楚的同工酶：

  • 亚基组成：由H型和M型两种亚基组成的四聚体
  • 五种同工酶：
    • LDH₁（H₄）：心肌中占优势
    • LDH₂（H₃M₁）
    • LDH₃（H₂M₂）
    • LDH₄（H₁M₃）
    • LDH₅（M₄）：骨骼肌中占优势
  • 临床意义：不同组织损伤时释放不同的LDH同工酶，可用于疾病诊断

五、重要名词解释

活性部位(Active Site)
酶分子中直接与底物结合并催化底物反应的部位，通常处于或靠近酶分子的表面，只占酶分子很小部分。
共价催化(Covalent Catalysis)
又称亲核催化或亲电子催化，在催化时形成不稳定的共价中间复合物，降低反应活化能，使反应加速。
邻近效应(Proximity Effect)
酶与底物结合形成中间复合物后，使底物和底物之间、酶的催化基团与底物之间结合于同一分子而使有效浓度极大升高的效应。
别构效应(Allosteric Effect)
效应物与别构酶分子中的别构中心结合后，诱导产生或稳定住酶分子的某种构象，使酶活性中心对底物的结合催化作用受到影响的现象。
别构酶(Allosteric Enzyme)
一般都是寡聚酶，除了有和底物结合和催化底物的活性部位，也有和调节物或效应物结合的调节部位的酶。
共价修饰(Covalent Modification)
酶蛋白肽链上的一些基团与某种化学基团发生可逆的共价结合，从而改变酶的活性的过程。
酶原激活(Zymogen Activation)
某些酶先以无活性的酶原形式合成及分泌，然后在到达作用部位时由另外的物质作用，使其失去部分肽段从而形成或暴露活性中心的过程。
蛋白激酶(Protein Kinases, PK)
一类催化蛋白质磷酸化反应的酶，能把ATP上的γ-磷酸转移到蛋白质分子的氨基酸残基上。
同工酶(Isozyme)
能催化相同的化学反应，但其蛋白分子结构、理化性质和免疫性能等方面都存在明显差异的一组酶。
亲和标记(Affinity Labeling)
指对酶的活性部位、受体的结合位点进行特异标记的方法，常用于研究酶的活性部位。

六、选择题测试

七、深度思考题

1. 试分析酶具有高催化效率的分子机制，并解释为什么酶的催化效率远超一般化学催化剂。

答案要点：

  1. 邻近效应与定向效应：
    • 邻近效应：酶将底物分子富集到活性中心附近，使反应物的有效浓度极大升高（可达10⁶-10⁸倍）
    • 定向效应：使反应基团的分子轨道以正确方位相互交叠，大大增加有效碰撞的概率
    • 将分子间反应转化为分子内反应，反应速率显著提高
  2. 过渡态稳定化：
    • 酶对过渡态的结合能力远强于对底物的结合能力
    • 通过稳定过渡态降低反应活化能，使反应更容易进行
    • 诱导契合机制使酶构象变化以最佳方式稳定过渡态
  3. 多元催化机制：
    • 酸碱催化：通过质子转移稳定过渡态
    • 共价催化：形成高反应活性的共价中间物
    • 金属离子催化：提供电子、调节氧化还原、屏蔽电荷
    • 多种催化机制协同作用，产生协同效应
  4. 微环境效应：
    • 活性中心的疏水环境降低介电常数，增强极性基团间相互作用
    • 特定的电荷分布有利于过渡态的形成和稳定
    • 排除水分子的干扰，提高催化效率

2. 详细阐述别构调节的分子机制，并以ATCase为例说明别构调节在代谢调控中的重要意义。

答案要点：

  1. 别构调节的分子机制：
    • 别构酶具有多个亚基，存在活性部位和调节部位
    • 效应物与调节部位结合引起酶构象变化
    • 构象变化通过亚基间相互作用传递到活性部位
    • 导致酶对底物亲和力的改变，表现为正协同或负协同效应
  2. ATCase的别构调节机制：
    • 结构：由6个催化亚基（C₆）和6个调节亚基（R₆）组成
    • CTP负调节：CTP与调节亚基结合，稳定T态（低活性态），抑制酶活性
    • ATP正调节：ATP与调节亚基结合，促进R态（高活性态）形成，激活酶活性
    • 底物天冬氨酸的协同结合：表现为S形动力学曲线
  3. 代谢调控的重要意义：
    • 反馈抑制：CTP作为嘧啶核苷酸合成途径的终产物，抑制ATCase活性，防止过量合成
    • 前馈激活：ATP激活ATCase，当细胞能量充足时促进DNA合成所需的嘧啶核苷酸合成
    • 代谢平衡：协调嘌呤和嘧啶核苷酸的合成比例，维持细胞内核苷酸池的平衡
    • 快速响应：别构调节是瞬时调节，能快速响应细胞代谢状态的变化
  4. 别构调节的普遍意义：
    • 精确调控：通过协同效应实现对代谢流量的精确控制
    • 信号整合：一个酶可接受多种调节信号，整合细胞的代谢信息
    • 代谢网络调节：关键酶的别构调节影响整个代谢网络的活性

3. 比较分析酶原激活、可逆共价修饰和别构调节三种酶活性调节方式的特点、机制和生理意义。

答案要点：

  1. 酶原激活：
    • 机制：蛋白水解酶切除酶原的抑制性肽段，暴露或形成活性中心
    • 特点：不可逆过程，一旦激活无法逆转
    • 调节速度：相对较慢，需要蛋白质合成和分泌
    • 生理意义：防止酶在不适当的时间和地点发挥作用，如消化酶避免自身消化
    • 典型实例：胃蛋白酶原→胃蛋白酶，凝血级联反应
  2. 可逆共价修饰：
    • 机制：通过蛋白激酶和磷酸酶催化的磷酸化/去磷酸化循环
    • 特点：可逆过程，能在活性和非活性状态间快速切换
    • 调节速度：快速（秒-分钟级），响应迅速
    • 生理意义：精确调节代谢酶活性，响应激素信号和细胞能量状态
    • 典型实例：糖原磷酸化酶、丙酮酸脱氢酶的调节
  3. 别构调节：
    • 机制：效应物与别构位点结合引起构象变化，影响活性位点功能
    • 特点：瞬时可逆，无需其他酶参与
    • 调节速度：最快（毫秒级），实时响应
    • 生理意义：实现反馈调节和前馈调节，维持代谢平衡
    • 典型实例：ATCase、磷酸果糖激酶的调节
  4. 三种调节方式的协调作用：
    • 时间尺度：别构调节（毫秒）→共价修饰（秒-分钟）→酶原激活（分钟-小时）
    • 调节精度：从粗调（酶原激活）到精调（别构调节）
    • 生理功能：共同构成完整的酶活性调节网络，适应不同的生理需求
    • 代谢意义：确保细胞能够在不同条件下维持适当的代谢活性

4. 从分子水平分析同工酶的产生机制、结构特点和生物学意义，并以乳酸脱氢酶为例说明同工酶在临床诊断中的应用。

答案要点：

  1. 同工酶的产生机制：
    • 基因重复和分化：同一祖先基因经过重复和突变分化产生不同的基因
    • 组织特异性表达：不同基因在不同组织中选择性表达
    • 发育阶段调控：同一组织在不同发育阶段表达不同的同工酶
    • 亚基组合：不同亚基按不同比例组合形成多种同工酶形式
  2. 结构特点：
    • 催化功能相同：催化相同的化学反应，具有相同的酶活性
    • 结构差异：氨基酸序列、分子量、等电点等理化性质不同
    • 免疫特性：具有不同的抗原性和免疫反应性
    • 动力学参数：Km值、最适pH、热稳定性等可能不同
  3. 乳酸脱氢酶（LDH）同工酶：
    • 亚基组成：由H型（心肌型）和M型（肌肉型）两种亚基组成四聚体
    • 五种形式：LDH₁(H₄)、LDH₂(H₃M)、LDH₃(H₂M₂)、LDH₄(HM₃)、LDH₅(M₄)
    • 组织分布：心肌富含LDH₁和LDH₂，骨骼肌和肝脏富含LDH₄和LDH₅
    • 功能差异：H型亚基适合有氧代谢，M型亚基适合无氧糖酵解
  4. 临床诊断应用：
    • 心肌梗死诊断：血清LDH₁升高，LDH₁/LDH₂比值>1
    • 肝脏疾病诊断：血清LDH₅显著升高
    • 肌肉损伤诊断：血清LDH₄和LDH₅升高
    • 恶性肿瘤监测：某些肿瘤细胞释放特定的LDH同工酶模式
  5. 生物学意义：
    • 代谢适应：不同组织根据代谢特点表达适合的同工酶
    • 发育调控：胚胎发育过程中同工酶模式的转换
    • 进化意义：反映物种进化和基因家族的分化历程
    • 疾病标志：组织损伤时释放特异性同工酶，成为疾病诊断的生化标志物
