维生素和辅酶

生物化学复习笔记 | 考研专业课

一、维生素概论

维生素定义：维生素是维持人和动物机体健康所必需的一类营养素，本质为低分子有机化合物。它们不能在体内合成，或者所合成的量难以满足机体的需要，必须由食物供给。参考：河南大学药理教研室

维生素的特点

  • 外源性：动物体自身不可合成或合成量不足以满足生理所需
  • 微量性：动物体所需量很少，但可以发挥巨大作用
  • 调节性：维生素必须能够调节人体新陈代谢或能量转变
  • 特异性：缺乏某种维生素后，动物将呈现特有的病态

维生素的分类

维生素按溶解性分为两大类：

  • 脂溶性维生素：维生素A、D、E、K
  • 水溶性维生素：B族维生素、维生素C

二、脂溶性维生素

维生素A（视黄醇）

化学结构与性质

维生素A是由β-白芷酮环和两分子2-甲基丁二烯构成的不饱和一元醇。主要形式包括：

  • 维生素A₁：存在于哺乳动物和咸水鱼肝脏中
  • 维生素A₂：存在于淡水鱼肝油中，生理效用仅及A₁的40%
  • β-胡萝卜素：维生素A原，可转化为维生素A

生理功能

  • 维持正常视觉功能，参与视紫红质的合成
  • 促进上皮组织增殖分化
  • 维持黏膜屏障的完整性
  • 促进儿童生长发育

缺乏症

夜盲症、干眼症、角膜软化症、皮肤干燥粗糙、抵抗力下降。

维生素D（钙化醇）

化学结构与代谢

维生素D系固醇类衍生物，主要形式：

  • 维生素D₃（胆钙化醇）：由7-脱氢胆固醇经紫外线照射转变
  • 维生素D₂（钙化醇）：由植物中麦角固醇经紫外线照射产生

生理功能

  • 调节钙、磷代谢
  • 促进肠内钙磷吸收和骨质钙化
  • 维持血钙和血磷的平衡
  • 参与免疫调节

缺乏症

佝偻病、软骨病、骨质疏松症。

维生素E（生育酚）

生理功能

  • 抗氧化作用，保护细胞膜
  • 维持生殖功能
  • 保护红细胞膜的完整性
  • 参与免疫功能调节

维生素K（凝血维生素）

生理功能

  • 参与凝血因子的合成
  • 参与骨代谢
  • 维持正常的凝血功能

缺乏症

凝血时间延长，出血倾向。

三、水溶性维生素

重要特点：水溶性维生素主要作为酶的辅酶或辅基的组成成分，在物质代谢中起重要作用。参考：广东工业大学生物化学课件

B族维生素

维生素B₁（硫胺素）

  • 辅酶形式：硫胺素焦磷酸（TPP）
  • 功能：参与糖代谢，丙酮酸脱羧反应的辅酶
  • 缺乏症：脚气病、神经炎

维生素B₂（核黄素）

  • 辅酶形式：黄素单核苷酸（FMN）、黄素腺嘌呤二核苷酸（FAD）
  • 功能：参与氧化还原反应，电子传递链的重要组成
  • 缺乏症：口角炎、舌炎、脂溢性皮炎

维生素B₃（烟酸/尼克酸）

  • 辅酶形式：烟酰胺腺嘌呤二核苷酸（NAD⁺）、烟酰胺腺嘌呤二核苷酸磷酸（NADP⁺）
  • 功能：参与氧化还原反应，能量代谢的关键辅酶
  • 缺乏症：糙皮病

维生素B₅（泛酸）

  • 辅酶形式：辅酶A（CoA-SH）
  • 功能：参与脂肪酸合成和氧化，柠檬酸循环
  • 缺乏症：疲劳、失眠、消化不良

维生素B₆（吡哆醇/吡哆醛/吡哆胺）

  • 辅酶形式：磷酸吡哆醛（PLP）
  • 功能：参与氨基酸代谢，转氨、脱羧反应
  • 缺乏症：皮炎、贫血、神经症状

维生素B₉（叶酸）

  • 辅酶形式：四氢叶酸（FH₄）
  • 功能：参与一碳单位转移，DNA合成
  • 缺乏症：巨幼红细胞性贫血

维生素B₁₂（钴胺素）

  • 辅酶形式：甲基钴胺素、腺苷钴胺素
  • 功能：参与甲基转移反应，DNA合成
  • 缺乏症：恶性贫血、神经系统损害

生物素（维生素H）

  • 功能：参与羧化反应，脂肪酸合成
  • 缺乏症：皮炎、脱发、神经症状

维生素C（抗坏血酸）

生理功能

  • 促进胶原蛋白合成
  • 抗氧化作用
  • 促进铁的吸收
  • 增强免疫功能

缺乏症

坏血病，表现为牙龈出血、皮下出血、伤口愈合不良。

四、辅酶和金属离子

辅酶的概念

辅酶是与酶蛋白结合疏松，用透析法容易与蛋白部分分开的有机小分子。辅酶在酶催化反应中其化学组分发生变化，可以认为是一种特殊的底物或"第二底物"。参考：百度百科-辅酶

重要辅酶

  • NAD⁺/NADH：参与氧化还原反应，呼吸链电子载体
  • NADP⁺/NADPH：参与合成代谢，提供还原力
  • FAD/FADH₂：参与氧化还原反应，电子传递
  • FMN/FMNH₂：参与电子传递链
  • 辅酶A：参与脂肪酸代谢，柠檬酸循环
  • 四氢叶酸：参与一碳单位转移

金属离子辅因子

  • Fe²⁺/Fe³⁺：血红蛋白、细胞色素、过氧化氢酶
  • Mg²⁺：激酶、磷酸酶、DNA聚合酶
  • Zn²⁺：碳酸酐酶、醇脱氢酶、DNA聚合酶
  • Cu²⁺：细胞色素氧化酶、超氧化物歧化酶
  • Mn²⁺：超氧化物歧化酶、精氨酸酶

七、深度思考题

1. 试述水溶性维生素与脂溶性维生素在化学性质、生理功能和代谢特点方面的主要区别，并分析这些区别对临床营养补充的指导意义。

  1. 化学性质差异：脂溶性维生素（A、D、E、K）具有脂溶性结构，易溶于脂肪和有机溶剂，不溶于水；而水溶性维生素（B族维生素、维生素C）具有亲水性基团，易溶于水。这种化学性质差异决定了它们在体内的吸收、转运、储存和排泄方式完全不同。
  2. 吸收和转运机制：脂溶性维生素需要胆汁酸的帮助才能被吸收，通过淋巴系统转运，与载脂蛋白结合在血液中运输；水溶性维生素可直接被小肠吸收，通过门静脉进入血液循环，大多数以游离形式存在于血液中。
  3. 体内储存能力：脂溶性维生素可在肝脏、脂肪组织等部位大量储存，储存时间较长，因此缺乏症状出现较慢；水溶性维生素在体内储存量很少，多余部分迅速从尿液中排出，因此需要经常补充，缺乏症状出现较快。
  4. 生理功能特点：脂溶性维生素主要参与膜结构维护、基因表达调节、激素样作用等；水溶性维生素主要作为辅酶参与各种代谢反应，是酶活性的必需因子。
  5. 临床指导意义：脂溶性维生素补充需谨慎，过量易中毒，应根据储存状况合理补充；水溶性维生素相对安全，但需要持续补充，特别是在疾病状态、妊娠期、生长发育期等特殊情况下需要增加摄入量。

2. 分析NAD⁺/NADH和NADP⁺/NADPH这两对辅酶在细胞代谢中的不同作用，并说明它们在分解代谢和合成代谢中的具体功能。

  1. 结构差异与识别机制：NAD⁺和NADP⁺在结构上仅差一个磷酸基团，但这个微小差异使得不同的酶能够特异性识别它们。NADP⁺在腺苷的2'位多一个磷酸基团，这个磷酸基团改变了分子的空间构象，使其能够与特定的酶结合位点相互作用。
  2. NAD⁺/NADH在分解代谢中的作用：主要参与糖酵解、柠檬酸循环、脂肪酸β氧化等分解代谢过程。在这些过程中，NAD⁺作为电子受体，接受氢原子形成NADH，然后NADH在电子传递链中被氧化，释放能量用于ATP合成。这个系统主要负责从营养物质中提取能量。
  3. NADP⁺/NADPH在合成代谢中的作用：主要参与脂肪酸合成、胆固醇合成、核苷酸合成等合成代谢过程。NADPH作为还原剂，提供电子和氢原子用于还原反应，是合成代谢中必需的还原力来源。磷酸戊糖途径是NADPH的主要来源。
  4. 细胞内浓度比值的调节意义：细胞内NAD⁺/NADH比值较高，有利于氧化反应的进行；而NADP⁺/NADPH比值较低，NADPH浓度相对较高，有利于还原性合成反应。这种不同的浓度比值确保了分解代谢和合成代谢能够同时高效进行。
  5. 代谢调节的协调机制：这两对辅酶系统通过不同的再生途径和消耗途径，实现了细胞能量状态和合成需求的精确调节。当细胞需要能量时，NAD⁺/NADH系统活跃；当细胞需要合成大分子时，NADP⁺/NADPH系统发挥主导作用。

3. 维生素缺乏症的发生机制是什么？以维生素B₁缺乏导致的脚气病为例，从分子水平解释症状产生的原因。

  1. 维生素缺乏症的一般发生机制：维生素作为辅酶或辅基的前体物质，当摄入不足时，相应的辅酶合成减少，导致依赖这些辅酶的酶活性下降，相关代谢途径受阻，代谢产物积累或缺乏，最终表现为特异性的临床症状。
  2. 维生素B₁的分子作用机制：维生素B₁在体内转化为硫胺素焦磷酸（TPP），TPP是丙酮酸脱氢酶复合体、α-酮戊二酸脱氢酶复合体和转酮醇酶的必需辅酶。这些酶在糖代谢中发挥关键作用，特别是在丙酮酸氧化和磷酸戊糖途径中。
  3. 脚气病的分子病理机制：维生素B₁缺乏时，TPP合成不足，导致丙酮酸脱氢酶活性下降，丙酮酸不能正常氧化为乙酰CoA进入柠檬酸循环。结果是丙酮酸和乳酸在血液和组织中积累，同时ATP产生减少，细胞能量供应不足。
  4. 神经系统症状的产生：神经组织主要依赖葡萄糖代谢提供能量，当糖代谢受阻时，神经细胞能量不足，导致神经传导功能障碍，表现为周围神经炎、感觉异常、肌肉无力等症状。同时，丙酮酸积累可能对神经组织产生毒性作用。
  5. 心血管系统症状的机制：心肌细胞对能量需求极高，糖代谢障碍导致心肌收缩力下降，出现心脏扩大、心力衰竭等症状。此外，代谢产物的积累可能影响心肌细胞的电生理特性，导致心律失常。这说明维生素虽然需要量很少，但其缺乏会在分子水平上引起连锁反应，最终导致严重的临床后果。

4. 比较分析叶酸和维生素B₁₂在DNA合成中的协同作用机制，并解释为什么这两种维生素的缺乏都会导致巨幼红细胞性贫血。

  1. 叶酸在DNA合成中的作用机制：叶酸在体内转化为四氢叶酸（FH₄），作为一碳单位的载体参与嘌呤和嘧啶的合成。特别是在胸腺嘧啶核苷酸（dTMP）的合成中，N⁵,N¹⁰-亚甲基四氢叶酸提供甲基，将脱氧尿苷酸（dUMP）转化为dTMP，这是DNA合成的限速步骤之一。
  2. 维生素B₁₂的甲基循环作用：维生素B₁₂以甲基钴胺素的形式参与甲硫氨酸循环，催化同型半胱氨酸转化为甲硫氨酸。这个反应同时使N⁵-甲基四氢叶酸重新转化为四氢叶酸，维持叶酸的活性形式。如果B₁₂缺乏，叶酸会以N⁵-甲基四氢叶酸的形式被"困住"，无法参与DNA合成。
  3. 两者的协同关系：维生素B₁₂缺乏会导致"甲基陷阱"现象，即使叶酸摄入充足，也无法发挥正常功能。这种协同关系说明了这两种维生素在细胞分裂和DNA复制中的相互依赖性，任何一种缺乏都会影响整个一碳代谢网络。
  4. 巨幼红细胞性贫血的发生机制：DNA合成障碍导致细胞分裂异常，红细胞前体细胞在骨髓中不能正常分裂成熟，形成体积异常增大的巨幼红细胞。这些细胞虽然体积大，但功能异常，携氧能力下降，导致贫血症状。
  5. 临床诊断和治疗意义：理解这种协同机制对临床诊断和治疗具有重要意义。在治疗巨幼红细胞性贫血时，需要明确是叶酸缺乏还是B₁₂缺乏，或者两者同时缺乏。单纯补充叶酸可能掩盖B₁₂缺乏的神经系统症状，延误诊断。因此，需要综合考虑两种维生素的状态，制定合理的治疗方案。
