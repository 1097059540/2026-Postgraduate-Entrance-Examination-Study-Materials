氨基酸、肽和蛋白质

生物化学复习笔记 - 第二章

基于《生物化学》第四版（朱圣庚、徐长法主编）

一、α-氨基酸的结构与性质

1.1 氨基酸的基本结构

α-氨基酸是构成蛋白质的基本单位，具有共同的基本结构：

NH₂-CHR-COOH

其中R基团（侧链）的不同决定了氨基酸的种类和性质。

重要概念：除甘氨酸外，所有α-氨基酸的α-碳原子都是手性碳原子，具有旋光性。天然蛋白质中的氨基酸均为L-型构型。

1.2 氨基酸的分类

根据侧链R基团的性质，20种标准氨基酸可分为：

  • 非极性氨基酸（疏水性）：甘氨酸、丙氨酸、缬氨酸、亮氨酸、异亮氨酸、苯丙氨酸、色氨酸、蛋氨酸、脯氨酸
  • 极性不带电氨基酸：丝氨酸、苏氨酸、酪氨酸、半胱氨酸、天冬酰胺、谷氨酰胺
  • 带正电氨基酸（碱性）：赖氨酸、精氨酸、组氨酸
  • 带负电氨基酸（酸性）：天冬氨酸、谷氨酸

1.3 氨基酸的酸碱性质

氨基酸分子同时含有氨基和羧基，具有两性电解质的性质：

  • 等电点（pI）：氨基酸净电荷为零时的pH值
  • 两性离子：在等电点时，氨基酸以两性离子形式存在
  • 缓冲作用：氨基酸在其pKa附近具有缓冲能力

1.4 氨基酸的化学反应

  • 茚三酮反应：与茚三酮反应产生蓝紫色化合物，用于氨基酸的定性和定量分析
  • 双缩脲反应：含有两个或以上肽键的化合物与Cu²⁺在碱性条件下产生紫色络合物
  • Edman降解：用于蛋白质N端氨基酸序列测定

二、肽的结构与性质

2.1 肽键的形成

肽键是氨基酸之间通过脱水缩合反应形成的共价键：

R₁-CO-NH-R₂

肽键具有部分双键性质，限制了围绕C-N键的自由旋转。

肽键特点：
  • 键长介于单键和双键之间（1.32Å）
  • 具有平面性，ω角接近180°（反式）或0°（顺式）
  • 在脯氨酸残基前，顺式构象较常见

2.2 肽的命名与分类

  • 二肽：由两个氨基酸组成
  • 三肽：由三个氨基酸组成
  • 寡肽：由2-10个氨基酸组成
  • 多肽：由10个以上氨基酸组成
  • 蛋白质：通常由50个以上氨基酸组成的多肽链

2.3 重要的生物活性肽

  • 谷胱甘肽（GSH）：三肽，具有抗氧化作用
  • 胰岛素：由A链（21个氨基酸）和B链（30个氨基酸）组成
  • 血管紧张素II：八肽激素，调节血压

三、蛋白质的结构层次

3.1 蛋白质的一级结构

蛋白质的一级结构是指多肽链中氨基酸的排列顺序，由共价键维持：

  • 肽键：连接氨基酸的主要共价键
  • 二硫键：半胱氨酸残基间形成的共价键
  • N端和C端：多肽链的两端，分别具有自由的氨基和羧基
一级结构的重要性：一级结构决定了蛋白质的高级结构和生物功能。一级结构的改变可能导致蛋白质功能的丧失或改变。

3.2 蛋白质的二级结构

二级结构是指多肽链局部区域的规则构象，主要由氢键维持：

  • α-螺旋：右手螺旋，每圈3.6个氨基酸残基，螺距5.4Å
  • β-折叠片：伸展的多肽链通过氢键形成的片状结构
  • β-转角：连接α-螺旋和β-折叠片的转折区域
  • 无规卷曲：没有规则二级结构的区域

3.3 蛋白质的三级结构

三级结构是指整个多肽链的三维空间构象，由多种非共价相互作用维持：

  • 氢键：极性基团间的相互作用
  • 离子键：带电基团间的静电相互作用
  • 范德华力：原子间的弱相互作用
  • 疏水相互作用：非极性基团聚集，排斥水分子

3.4 蛋白质的四级结构

四级结构是指由两个或多个多肽链（亚基）组成的蛋白质的空间排列：

  • 同源四聚体：由相同亚基组成，如血红蛋白
  • 异源多聚体：由不同亚基组成
  • 亚基间相互作用：主要通过非共价键维持

四、蛋白质的理化性质

4.1 蛋白质的分子大小

  • 分子量范围：从几千到几百万道尔顿
  • 测定方法：SDS-PAGE、质谱法、超离心法
  • 分子量与功能：酶活性中心、结合位点的形成需要一定的分子量

4.2 蛋白质的溶解性

  • 亲水蛋白质：表面多极性基团，易溶于水
  • 疏水蛋白质：膜蛋白等，需要去垢剂溶解
  • 盐析：高盐浓度下蛋白质沉淀

4.3 蛋白质的变性与复性

变性：蛋白质在物理或化学因素作用下，天然构象被破坏，生物活性丧失的过程。
  • 变性因子：高温、极端pH、有机溶剂、重金属离子、尿素等
  • 变性特征：溶解度改变、生物活性丧失、易被蛋白酶水解
  • 复性：在适当条件下，变性蛋白质恢复天然构象和活性

五、蛋白质测序与合成

5.1 蛋白质测序策略

  1. 蛋白质纯化：获得均一的蛋白质样品
  2. 一级结构分析：确定氨基酸组成和序列
  3. 二硫键定位：确定半胱氨酸残基间的连接
  4. 翻译后修饰分析：糖基化、磷酸化等修饰

5.2 测序方法

  • Edman降解法：从N端逐个切除氨基酸进行分析
  • 质谱法：高精度、高通量的现代测序技术
  • 化学裂解：CNBr裂解、胰蛋白酶消化等

5.3 肽与蛋白质的化学合成

  • 固相合成法：Merrifield法，从C端开始合成
  • 保护基团：Fmoc、Boc等保护氨基
  • 偶联试剂：DCC、HBTU等活化羧基
  • 应用：多肽药物、蛋白质工程

六、重要名词解释

Amino Acid

Isoelectric Point, pI
氨基酸或蛋白质分子净电荷为零时的pH值，此时分子不向正极或负极移动。
α-螺旋

β-折叠片

蛋白质变性

两个半胱氨酸残基的巯基（-SH）氧化形成的共价键（-S-S-），对蛋白质结构稳定性起重要作用。
Edman降解

七、选择题测试

进度: 0/15

八、深度思考题

1. 分析蛋白质一级结构与高级结构的关系，并举例说明一级结构改变对蛋白质功能的影响。

参考答案：
  1. 一级结构决定高级结构：蛋白质的一级结构（氨基酸序列）包含了形成特定三维结构所需的全部信息。氨基酸侧链的性质决定了分子内和分子间相互作用的类型和强度，从而决定了蛋白质的折叠方式和最终的空间构象。
  2. 结构与功能的关系：蛋白质的生物功能直接依赖于其特定的三维结构。活性位点的形成、底物结合、催化活性等都需要精确的空间排列。一级结构的改变会影响高级结构，进而影响功能。
  3. 经典实例分析：镰刀型细胞贫血症是由于β-血红蛋白第6位氨基酸从谷氨酸（Glu）突变为缬氨酸（Val）引起的。这个单一氨基酸的改变导致血红蛋白分子聚集，红细胞变形，引起严重的临床症状。
  4. 分子机制：谷氨酸是带负电的亲水性氨基酸，而缬氨酸是疏水性氨基酸。这种改变在血红蛋白表面产生了疏水斑块，导致去氧血红蛋白分子间发生疏水相互作用而聚集成纤维状结构。
  5. 进化意义：蛋白质序列的保守性反映了结构和功能的重要性。关键功能区域（如酶活性中心）的氨基酸序列在进化过程中高度保守，而非关键区域则允许较大的变异。

2. 比较α-螺旋和β-折叠片的结构特点，分析影响二级结构形成的因素。

参考答案：
  1. α-螺旋的结构特点：右手螺旋结构，每圈包含3.6个氨基酸残基，螺距为5.4Å。氢键形成在第n个残基的C=O与第n+4个残基的N-H之间，使结构稳定。φ角约为-60°，ψ角约为-45°。
  2. β-折叠片的结构特点：多肽链呈伸展构象，通过链间或链内氢键形成片状结构。可分为平行和反平行两种类型。φ角约为-120°，ψ角约为+120°。侧链交替分布在片层的两侧。
  3. 氨基酸偏好性：不同氨基酸对二级结构有不同的偏好。丙氨酸、谷氨酸、亮氨酸等倾向于形成α-螺旋；缬氨酸、异亮氨酸、苯丙氨酸等倾向于形成β-折叠片；脯氨酸和甘氨酸是螺旋破坏者。
  4. 环境因素影响：pH值、离子强度、温度等环境因素会影响氨基酸侧链的电离状态和相互作用，从而影响二级结构的稳定性。极端条件可能导致二级结构的破坏。
  5. 生物学意义：α-螺旋常见于膜蛋白的跨膜区域和DNA结合蛋白中；β-折叠片常见于酶的活性中心和免疫球蛋白的结构域中。不同的二级结构适应不同的功能需求。

3. 论述蛋白质变性的机理、影响因素及其在生物技术中的应用。

参考答案：
  1. 变性的分子机理：蛋白质变性是指在外界因素作用下，维持蛋白质天然构象的非共价键（氢键、离子键、疏水相互作用、范德华力）被破坏，导致蛋白质失去天然的三维结构和生物活性。一级结构通常保持不变。
  2. 物理因素：高温破坏氢键和疏水相互作用；高压改变蛋白质体积和构象；紫外线和电离辐射引起化学键断裂；机械搅拌产生剪切力破坏蛋白质结构。
  3. 化学因素：极端pH破坏离子键和氢键；有机溶剂破坏疏水相互作用；变性剂（尿素、胍盐酸）与蛋白质形成氢键；重金属离子与巯基结合；去垢剂破坏疏水相互作用。
  4. 生物技术应用：蛋白质纯化中使用变性条件去除杂蛋白；SDS-PAGE中SDS变性蛋白质用于分子量测定；Western blot中变性抗原暴露抗原决定簇；重组蛋白表达中包涵体的溶解和复性。
  5. 复性与折叠：某些蛋白质在适当条件下可以复性，恢复天然结构和活性。这一过程需要分子伴侣的协助，并且受到蛋白质浓度、温度、pH、离子强度等因素的影响。复性技术在重组蛋白生产中具有重要应用价值。

4. 分析现代蛋白质测序技术的原理、优缺点及发展趋势。

参考答案：
  1. Edman降解法：经典的化学测序方法，使用苯异硫氰酸酯从N端逐个切除氨基酸。优点是准确性高，缺点是速度慢、通量低，只能测定N端50-100个氨基酸。目前主要用于验证和补充质谱数据。
  2. 质谱技术：现代蛋白质组学的核心技术，包括MALDI-TOF、ESI-MS/MS等。通过测定肽段的精确分子量和碎片离子谱图确定序列。优点是高通量、高精度、可检测翻译后修饰，缺点是需要复杂的样品预处理和数据分析。
  3. 串联质谱（MS/MS）：先电离肽段，再进行碰撞诱导解离（CID）产生碎片离子，通过分析碎片离子的质量和强度推断氨基酸序列。结合数据库搜索可实现高通量蛋白质鉴定。
  4. 新兴技术：单分子测序技术正在发展中，如纳米孔测序、单分子荧光技术等。这些技术有望实现实时、长读长的蛋白质测序，但目前仍面临准确性和成本等挑战。
  5. 发展趋势：向着更高通量、更高精度、更低成本的方向发展。结合人工智能和机器学习技术，提高数据分析的准确性和效率。多组学整合分析将成为未来蛋白质研究的重要方向。

5. 探讨氨基酸序列与蛋白质进化的关系，以及比较蛋白质组学在进化研究中的应用。

参考答案：
  1. 分子进化的基础：蛋白质序列的变化反映了基因突变的积累过程。同源蛋白质在不同物种中的序列差异可以用来推断进化关系和分化时间。分子钟假说认为蛋白质序列的变化速率相对恒定。
  2. 保守性与变异性：蛋白质的功能重要区域（如酶活性中心、结构域界面）在进化过程中高度保守，而非关键区域允许较大变异。这种选择压力的差异导致了蛋白质序列进化的不均匀性。
  3. 系统发育分析：通过比较同源蛋白质的序列差异，可以构建系统发育树，揭示物种间的进化关系。多基因分析比单基因分析更可靠，可以减少水平基因转移等因素的干扰。
  4. 功能进化：基因重复、域重排、外显子重组等机制导致新功能的产生。蛋白质家族的扩张和收缩反映了生物适应环境的过程。比较不同物种的蛋白质组可以揭示适应性进化的分子基础。
  5. 蛋白质组学应用：大规模蛋白质序列比较揭示了进化的普遍规律；蛋白质结构比较发现了进化过程中结构的保守性；翻译后修饰的比较研究揭示了调控机制的进化；蛋白质相互作用网络的比较分析揭示了细胞功能的进化。
