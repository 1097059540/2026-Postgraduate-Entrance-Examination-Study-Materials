酶的催化作用

生物化学第六章 - 考研复习笔记

基于《生物化学》第四版（朱圣庚、徐长法主编）

一、酶的基本概念与特性

1.1 酶的定义

酶是由活细胞产生的、对其底物具有高度专一性和高效催化能力的蛋白质或RNA。酶作为生物催化剂，能够显著降低反应的活化能，加速生化反应的进行。

重要提醒：酶只能加速热力学上可能发生的反应，不能改变反应的平衡常数，只能缩短达到平衡的时间。

1.2 酶与一般催化剂的比较

特性 酶 一般催化剂
催化效率 极高（10⁶-10¹⁷倍） 较低
专一性 高度专一 专一性差
反应条件 温和条件（常温、常压、生理pH） 苛刻条件（高温、高压）
调节性 活性可调节 活性不可调节
稳定性 易失活 相对稳定

1.3 酶的化学本质

1.3.1 单纯酶

仅由蛋白质组成的酶，如胃蛋白酶、胰蛋白酶、核糖核酸酶等。

1.3.2 缀合酶（结合酶）

由蛋白质部分（酶蛋白，apoenzyme）和非蛋白质部分（辅助因子，cofactor）组成：

  • 酶蛋白：决定酶的专一性，提供结合位点
  • 辅助因子：
    • 辅酶（coenzyme）：与酶蛋白结合疏松，可透析除去
    • 辅基（prosthetic group）：与酶蛋白结合紧密，不易透析除去
    • 金属离子：如Mg²⁺、Zn²⁺、Fe²⁺等
酶蛋白 + 辅助因子 = 全酶（holoenzyme）
注意：只有全酶才具有催化活性，单独的酶蛋白或辅助因子都没有催化活性。

二、酶的分类与命名

2.1 国际酶学委员会分类法

根据酶催化的反应类型，将酶分为六大类：

编号 酶类 催化反应类型 实例
EC 1 氧化还原酶类 氧化还原反应 乳酸脱氢酶、细胞色素氧化酶
EC 2 转移酶类 基团转移反应 转氨酶、激酶
EC 3 水解酶类 水解反应 胃蛋白酶、淀粉酶、脂肪酶
EC 4 裂合酶类 加成或消除反应 醛缩酶、脱羧酶
EC 5 异构酶类 异构化反应 磷酸葡萄糖异构酶
EC 6 连接酶类 合成反应（需ATP） DNA连接酶、氨酰-tRNA合成酶

2.2 酶的专一性

2.2.1 结构专一性

  • 绝对专一性：只作用于一种底物，如脲酶只能水解尿素
  • 相对专一性：
    • 键专一性：只作用于特定化学键，如肽酶作用于肽键
    • 基团专一性：作用于含有特定基团的化合物

2.2.2 立体异构专一性

  • 旋光异构专一性：只作用于L型或D型氨基酸
  • 几何异构专一性：区分顺式和反式异构体

三、酶的催化机理

3.1 "锁钥学说"（Fischer，1894）

酶与底物的关系如同锁与钥匙，酶的活性中心与底物在空间结构上严格互补，形成酶-底物复合物。

3.2 "诱导契合学说"（Koshland，1958）

酶的活性中心具有一定的柔性，底物结合时诱导酶构象发生变化，使活性中心与底物更好地契合。这一学说更好地解释了酶的催化机理。

诱导契合的意义：
  1. 提高了酶与底物的结合专一性
  2. 有利于催化基团的正确定位
  3. 稳定过渡态，降低活化能
  4. 促进产物的释放

3.3 酶降低活化能的机理

3.3.1 降低活化熵

  • 酶活性部位的邻近和定向效应
  • 使底物反应基团处于最适位置
  • 减少分子运动的自由度

3.3.2 降低活化焓

  • 酶与底物的诱导契合效应
  • 催化基团进入适当位置
  • 削弱反应键，稳定过渡态

四、酶活力测定与分离纯化

4.1 酶活力的概念

酶活力（Enzyme Activity）：酶催化特定化学反应的能力，通常以单位时间内底物的减少量或产物的增加量来表示。
酶活力单位（Unit, U）：在最适条件下，1分钟内转化1μmol底物所需的酶量为1个活力单位。
比活力（Specific Activity）：单位质量蛋白质的酶活力，用U/mg蛋白质表示，是酶纯度的指标。

4.2 酶活力测定的条件

  • 温度：通常采用25℃或37℃
  • pH：最适pH条件
  • 底物浓度：饱和底物浓度（[S] >> Km）
  • 测定时间：反应初期，底物浓度变化<5%
  • 缓冲体系：维持pH稳定
注意：测定酶活力时必须测定反应初速度，此时酶浓度对反应速度呈一级反应关系。

4.3 酶的分离纯化

4.3.1 分离纯化的一般步骤

  1. 细胞破碎：释放胞内酶
  2. 粗分离：盐析、有机溶剂沉淀
  3. 精细分离：离子交换层析、凝胶过滤、等电聚焦
  4. 最终纯化：高效液相色谱、亲和层析

4.3.2 纯化效果的评价指标

  • 回收率：纯化后总活力/初始总活力 × 100%
  • 纯化倍数：纯化后比活力/初始比活力
  • 纯度：目标酶蛋白占总蛋白的百分比

五、重要名词解释

酶（Enzyme）：由活细胞产生的具有催化活性的蛋白质或RNA，能够显著降低反应活化能，加速生化反应。
全酶（Holoenzyme）：由酶蛋白和辅助因子结合形成的完整的、具有催化活性的酶分子。
酶蛋白（Apoenzyme）：缀合酶中的蛋白质部分，决定酶的专一性，但单独存在时无催化活性。
辅酶（Coenzyme）：与酶蛋白结合疏松的有机辅助因子，如NAD⁺、FAD、辅酶A等，可通过透析除去。
辅基（Prosthetic Group）：与酶蛋白结合紧密的非蛋白质辅助因子，如血红蛋白中的血红素。
活性中心（Active Site）：酶分子中直接与底物结合并催化反应的区域，包括结合部位和催化部位。
诱导契合（Induced Fit）：底物与酶结合时，相互诱导构象变化，使活性中心与底物更好契合的机理。
核酶（Ribozyme）：具有催化活性的RNA分子，如核糖体RNA、某些内含子RNA等。
抗体酶（Abzyme）：具有催化功能的抗体分子，通过免疫过渡态类似物获得。
多酶复合体（Multienzyme Complex）：多个酶组成的功能单位，如丙酮酸脱氢酶复合体。

七、深度思考题

1. 试述酶作为生物催化剂的特点，并解释酶为什么具有如此高的催化效率？

参考答案：
  1. 酶的特点：
    • 催化效率极高：比一般催化剂高10⁶-10¹⁷倍
    • 高度专一性：对底物和反应类型都有严格选择
    • 反应条件温和：常温、常压、生理pH下发挥作用
    • 活性可调节：受多种因素调控
    • 易失活：对环境变化敏感
  2. 高催化效率的原因：
    • 显著降低反应活化能：通过稳定过渡态实现
    • 邻近和定向效应：使反应基团处于最适位置
    • 诱导契合机制：优化酶-底物相互作用
    • 多点结合：增强结合特异性和稳定性
    • 构象变化：促进产物形成和释放
  3. 分子机理：
    • 降低活化熵：减少分子运动自由度
    • 降低活化焓：削弱反应键，稳定过渡态
    • 提供替代反应途径：避开高能垒步骤

2. 比较"锁钥学说"和"诱导契合学说"，说明诱导契合学说的优越性及其生物学意义。

参考答案：
  1. 锁钥学说的要点：
    • 酶活性中心与底物结构严格互补
    • 酶-底物结合如锁钥配对
    • 强调结构的刚性匹配
    • 无法解释酶的调节机制
  2. 诱导契合学说的要点：
    • 酶活性中心具有柔性
    • 底物结合诱导酶构象变化
    • 强调动态的相互适应
    • 能解释酶的多种调节现象
  3. 诱导契合学说的优越性：
    • 更好解释酶的专一性机制
    • 阐明了别构调节的分子基础
    • 解释了竞争性抑制现象
    • 符合蛋白质动态结构特点
  4. 生物学意义：
    • 为酶活性调节提供结构基础
    • 有利于代谢途径的精细调控
    • 支持酶工程和药物设计
    • 揭示了蛋白质功能的普遍规律

3. 详述酶活力测定的原理、方法和注意事项，并解释为什么要测定反应初速度？

参考答案：
  1. 测定原理：
    • 基于酶催化反应的速率与酶浓度成正比
    • 通过测定底物消耗或产物生成速率确定酶活力
    • 在饱和底物浓度下，反应速率达到最大值
  2. 测定方法：
    • 连续监测法：实时监测反应进程
    • 终点法：测定固定时间后的产物量
    • 偶联酶法：利用指示反应放大信号
    • 放射性标记法：高灵敏度检测
  3. 注意事项：
    • 控制最适温度和pH
    • 保证底物浓度饱和
    • 选择合适的缓冲体系
    • 设置对照实验
    • 避免产物抑制和底物抑制
  4. 测定初速度的原因：
    • 此时底物浓度基本不变，反应为零级
    • 避免产物抑制和逆反应影响
    • 酶浓度与反应速率呈线性关系
    • 反映酶的真实催化能力
    • 便于不同条件下的比较

4. 论述核酶的发现及其生物学意义，并比较核酶与蛋白质酶的异同点。

参考答案：
  1. 核酶的发现：
    • 1982年Cech发现四膜虫rRNA前体的自我剪接
    • 1983年Altman发现RNase P的RNA亚基具有催化活性
    • 打破了"所有酶都是蛋白质"的传统观念
    • 两人因此获得1989年诺贝尔化学奖
  2. 生物学意义：
    • 支持"RNA世界"假说：RNA既能储存信息又能催化反应
    • 揭示了生命起源的可能途径
    • 为基因治疗提供新工具
    • 拓展了酶学研究领域
  3. 与蛋白质酶的相同点：
    • 都能降低反应活化能
    • 都具有底物专一性
    • 都不改变反应平衡
    • 都可被抑制剂抑制
  4. 与蛋白质酶的不同点：
    • 化学组成：RNA vs 蛋白质
    • 催化效率：一般较低
    • 稳定性：对RNase敏感
    • 结构复杂性：相对简单
    • 进化地位：可能更原始

5. 分析酶工程的基本原理和应用前景，说明其在生物技术中的重要作用。

参考答案：
  1. 酶工程的基本原理：
    • 定向进化：模拟自然选择过程改造酶
    • 理性设计：基于结构-功能关系设计酶
    • 蛋白质折叠工程：优化酶的稳定性
    • 酶的固定化：提高酶的重复使用性
  2. 主要技术手段：
    • 基因突变和筛选
    • DNA重组和基因融合
    • 计算机辅助设计
    • 高通量筛选技术
  3. 应用前景：
    • 工业生物催化：绿色化学工艺
    • 医药工业：药物合成和诊断
    • 食品工业：食品加工和保鲜
    • 环境治理：污染物降解
    • 能源开发：生物燃料生产
  4. 在生物技术中的重要作用：
    • 提供高效专一的生物催化剂
    • 降低生产成本和环境污染
    • 开发新的生物工艺路线
    • 推动生物经济发展
    • 促进可持续发展
