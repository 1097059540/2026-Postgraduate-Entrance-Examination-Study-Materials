RNA的生物合成和加工

考研生物化学专业课复习笔记 | 第32章

一、RNA转录的基本概念与特点

1.1 转录的基本特点

  • 选择性：转录具有时间和空间选择性，只有在特定类型的细胞中或特定发育阶段才能被转录
  • 催化酶：催化转录的酶是RNA聚合酶，属于依赖于DNA的RNA聚合酶
  • DNA解链：在RNA转录过程中，DNA双链要部分解链，释放出模板链DNA，形成DNA-RNA杂合双链
  • 原料：RNA合成的原料是四种核糖核苷三磷酸(ATP、GTP、CTP、UTP)
  • 方向性：RNA合成方向为5'→3'，模板链读取方向为3'→5'
重要提示：转录过程中，RNA聚合酶沿着DNA模板链的3'→5'方向移动，合成的RNA链方向为5'→3'。

1.2 转录与复制的区别

  • 转录只需要一条DNA链作为模板，而复制需要两条链
  • 转录产物是RNA，复制产物是DNA
  • 转录不需要引物，复制需要引物
  • 转录具有选择性，复制是全基因组的

二、RNA聚合酶的结构与功能

2.1 原核生物RNA聚合酶

组成结构：

  • 核心酶：α₂ββ'ω（分子量约400kDa）
  • 全酶：核心酶 + σ因子
  • 各亚基功能：
    • α亚基：酶组装，结合启动子上游元件
    • β和β'亚基：形成催化中心
    • ω亚基：酶组装和稳定
    • σ因子：识别启动子，赋予转录特异性

2.2 真核生物RNA聚合酶

真核生物有三种RNA聚合酶：

  • RNA聚合酶Ⅰ：转录大部分rRNA基因(18S、5.8S、28S rRNA)
  • RNA聚合酶Ⅱ：转录所有mRNA基因和大部分snRNA基因
  • RNA聚合酶Ⅲ：转录tRNA、5S rRNA和U6 snRNA基因
考试重点：真核生物三种RNA聚合酶的功能分工是考试的高频考点，需要准确记忆。

三、启动子与转录起始

3.1 原核生物启动子

保守序列：

  • -10区(Pribnow盒)：TATAAT，σ因子结合位点
  • -35区：TTGACA，σ因子识别序列
  • 转录起始点：通常为嘌呤核苷酸，标记为+1

转录起始过程：

  1. RNA聚合酶全酶与启动子结合形成闭合复合物
  2. DNA双链解开形成开放复合物
  3. 开始RNA合成，σ因子解离
  4. 进入转录延伸阶段

3.2 真核生物启动子

RNA聚合酶Ⅱ启动子元件：

  • TATA盒：位于-25至-30位，TFIID结合位点
  • 起始子元件(Inr)：包含转录起始点的序列
  • CAAT盒：位于-70至-80位，增强转录效率
  • GC盒：含有GGGCGG序列，Sp1转录因子结合位点
注意：真核生物转录起始需要多种转录因子(TFIIA、TFIIB、TFIID、TFIIE、TFIIF、TFIIH)的参与，过程比原核生物复杂。

四、转录终止

4.1 原核生物转录终止

内在终止(ρ-独立终止)：

  • DNA模板含有回文序列
  • 转录产生的RNA形成发夹结构
  • 发夹结构后跟随poly(U)序列
  • RNA聚合酶暂停并解离

ρ-依赖性终止：

  • 需要ρ蛋白参与
  • ρ蛋白具有ATP酶和解旋酶活性
  • 追赶并促使RNA聚合酶解离

4.2 真核生物转录终止

真核生物转录终止机制较为复杂，涉及多聚腺苷化信号序列(AAUAAA)和下游序列元件的协同作用。

五、RNA转录后加工

5.1 真核生物mRNA加工

5'端加帽：

  • 帽子结构：7-甲基鸟苷酸通过5'-5'三磷酸键连接
  • 功能：保护mRNA免受5'外切酶降解，促进翻译起始
  • 时机：转录起始后立即进行

3'端多聚腺苷化：

  • 信号序列：AAUAAA(多聚腺苷化信号)
  • 切割位点：信号序列下游10-30个核苷酸处
  • poly(A)尾：添加约200个腺苷酸残基
  • 功能：增强mRNA稳定性，促进翻译

RNA剪接：

  • 内含子：不编码蛋白质的序列，需要被移除
  • 外显子：编码蛋白质的序列，保留在成熟mRNA中
  • 剪接信号：
    • 5'剪接位点：GU(几乎100%保守)
    • 3'剪接位点：AG(几乎100%保守)
    • 分支点：距3'剪接位点20-50个核苷酸的腺苷酸
    • 多嘧啶序列：分支点与3'剪接位点之间
GU-AG规则：几乎所有的内含子都以GU开始，以AG结束，这是RNA剪接的基本规则。

剪接体机制：

  1. U1 snRNP识别并结合5'剪接位点
  2. U2AF蛋白识别3'剪接位点和多嘧啶序列
  3. U2 snRNP结合分支点腺苷酸
  4. U4/U6·U5 tri-snRNP加入形成剪接体
  5. U1和U4 snRNP解离，U6与5'剪接位点结合
  6. 进行两步转酯反应完成剪接

5.2 选择性剪接

选择性剪接允许一个基因产生多种mRNA异构体，从而编码不同的蛋白质。这是真核生物增加蛋白质多样性的重要机制。

选择性剪接类型：

  • 外显子跳跃：某些外显子被跳过
  • 内含子保留：某些内含子被保留
  • 选择性5'或3'剪接位点：使用不同的剪接位点
  • 互斥外显子：多个外显子中只选择一个

5.3 tRNA和rRNA加工

tRNA加工：

  • 5'端和3'端的切除
  • 3'端添加CCA序列
  • 碱基修饰(如假尿苷化、甲基化)
  • 内含子剪接(某些tRNA)

rRNA加工：

  • 从45S前体rRNA加工为18S、5.8S、28S rRNA
  • 广泛的化学修饰(甲基化、假尿苷化)
  • 与核糖体蛋白组装

六、逆转录

6.1 逆转录酶

  • 来源：逆转录病毒、逆转座子
  • 功能：以RNA为模板合成DNA
  • 特点：具有RNA依赖的DNA聚合酶活性和RNase H活性

6.2 逆转录过程

  1. 逆转录酶结合RNA模板和引物
  2. 合成第一链cDNA
  3. RNase H降解RNA模板
  4. 以第一链cDNA为模板合成第二链DNA
  5. 形成双链DNA

6.3 逆转座子

分类：

  • LTR逆转座子：两端有长末端重复序列
  • 非LTR逆转座子：如LINE和SINE元件
生物学意义：逆转座子是基因组进化的重要驱动力，但也可能导致基因突变和疾病。

名词解释库

转录 (Transcription)
以DNA为模板，在RNA聚合酶催化下合成RNA的过程。是基因表达的第一步，将遗传信息从DNA转移到RNA。
RNA聚合酶 (RNA Polymerase)
催化RNA合成的酶，以DNA为模板，将核糖核苷三磷酸聚合成RNA链。原核生物有一种，真核生物有三种(Pol I、II、III)。
启动子 (Promoter)
DNA上能够与RNA聚合酶结合并启动转录的特定序列。包含转录起始所必需的调控元件。
σ因子 (Sigma Factor, σ)
原核生物RNA聚合酶的亚基，赋予酶识别特定启动子的能力，决定转录的特异性。
TATA盒 (TATA Box)
真核生物启动子中的保守序列，通常位于转录起始点上游25-30个碱基处，是TFIID结合的位点。
5'帽子结构 (5' Cap Structure)
真核生物mRNA 5'端的修饰结构，由7-甲基鸟苷酸通过5'-5'三磷酸键连接形成，保护mRNA并促进翻译。
多聚腺苷化 (Polyadenylation)
在真核生物mRNA 3'端添加多个腺苷酸残基(poly A尾)的过程，增强mRNA稳定性。
RNA剪接 (RNA Splicing)
从前体mRNA中移除内含子并连接外显子的过程，由剪接体催化完成。
内含子 (Intron)
基因中不编码蛋白质的序列，在RNA加工过程中被移除。几乎所有内含子都遵循GU-AG规则。
外显子 (Exon)
基因中编码蛋白质的序列，在RNA剪接后保留在成熟mRNA中。
剪接体 (Spliceosome)
由小核RNA(snRNA)和蛋白质组成的核糖核蛋白复合物，负责催化RNA剪接反应。
逆转录酶 (Reverse Transcriptase, RT)
以RNA为模板合成DNA的酶，具有RNA依赖的DNA聚合酶活性，存在于逆转录病毒和逆转座子中。
逆转座子 (Retrotransposon)
通过RNA中间体进行转座的DNA序列，包含逆转录酶基因，是基因组中的可移动元件。

深度思考题

1. 比较原核生物和真核生物转录调控的主要特点，并分析其生物学意义。

  1. 转录机制差异：原核生物使用单一RNA聚合酶转录所有基因，而真核生物使用三种不同的RNA聚合酶分别转录不同类型的基因。这种分工使真核生物能够对不同类型RNA的合成进行精细调控。
  2. 启动子结构复杂性：真核生物启动子包含多种调控元件(TATA盒、CAAT盒、GC盒等)，需要多种转录因子参与，而原核生物启动子相对简单。这反映了真核生物基因表达调控的复杂性和精确性。
  3. 转录与翻译的时空分离：真核生物转录在细胞核进行，翻译在细胞质进行，为RNA加工提供了时间和空间；而原核生物转录和翻译可以同时进行。这种分离使真核生物能够通过RNA加工增加基因产物的多样性。
  4. 染色质结构的影响：真核生物DNA与组蛋白结合形成染色质，转录调控涉及染色质重塑；原核生物DNA相对裸露，转录调控主要通过蛋白质-DNA相互作用实现。
  5. 生物学意义：这些差异反映了生物复杂性的进化趋势，真核生物更复杂的调控机制使其能够适应更复杂的环境和发育过程。

2. 详细阐述真核生物mRNA转录后加工的三个主要步骤及其生物学功能。

  1. 5'端加帽修饰：在转录起始后立即进行，7-甲基鸟苷酸通过5'-5'三磷酸键连接到mRNA 5'端。功能包括：保护mRNA免受5'外切酶降解；作为核质转运的信号；在翻译起始时被帽结合蛋白识别，促进核糖体结合和翻译起始。这一修饰确保了mRNA的稳定性和翻译效率。
  2. 3'端多聚腺苷化：识别AAUAAA信号序列，在其下游10-30个核苷酸处切割，然后添加约200个腺苷酸残基形成poly(A)尾。功能包括：增强mRNA稳定性，防止3'外切酶降解；促进mRNA从细胞核转运到细胞质；与poly(A)结合蛋白结合，促进翻译起始和延伸。
  3. RNA剪接：通过剪接体识别剪接信号(5'端GU、3'端AG、分支点A、多嘧啶序列)，进行两步转酯反应移除内含子，连接外显子。功能包括：移除非编码序列，形成成熟mRNA；通过选择性剪接增加蛋白质多样性；某些内含子具有调控功能；剪接过程中的质量控制确保mRNA的准确性。
  4. 整体意义：这三个步骤协同作用，不仅确保了mRNA的稳定性和功能性，还为基因表达调控提供了多个控制点，是真核生物基因表达精细调控的重要基础。

3. 分析RNA剪接的分子机制，并讨论选择性剪接在基因表达调控中的作用。

  1. 剪接信号识别：剪接过程依赖于四个关键信号的识别：5'剪接位点的GU序列、3'剪接位点的AG序列、分支点的腺苷酸残基、以及多嘧啶序列。这些信号的保守性确保了剪接的准确性，GU-AG规则几乎适用于所有内含子。
  2. 剪接体组装与催化：剪接体由五种snRNP(U1、U2、U4、U5、U6)和多种蛋白质因子组成。组装过程包括：U1 snRNP识别5'剪接位点，U2 snRNP结合分支点，U4/U6·U5 tri-snRNP加入，随后U1和U4解离，形成催化活性的剪接体。整个过程是动态的，涉及复杂的RNA-RNA和RNA-蛋白相互作用。
  3. 两步转酯反应机制：第一步，分支点腺苷酸的2'-OH攻击5'剪接位点，形成套索中间体；第二步，5'外显子的3'-OH攻击3'剪接位点，释放内含子套索，连接两个外显子。这种机制确保了剪接的精确性和效率。
  4. 选择性剪接的调控机制：通过剪接增强子(ESE)、剪接抑制子(ESS)、内含子剪接增强子(ISE)和内含子剪接抑制子(ISS)等顺式作用元件，以及SR蛋白、hnRNP等反式作用因子的相互作用，实现对剪接位点选择的精细调控。
  5. 选择性剪接的生物学意义：使一个基因能够产生多种mRNA异构体，极大地增加了蛋白质的多样性；在不同组织、发育阶段或环境条件下产生功能不同的蛋白质异构体；为进化提供了原材料，是蛋白质功能多样化的重要机制；在疾病发生中也起重要作用，许多遗传病与剪接异常相关。

4. 论述逆转录在生物学中的意义及其在基因组进化中的作用。

  1. 逆转录的基本机制：逆转录酶具有RNA依赖的DNA聚合酶活性和RNase H活性，能够以RNA为模板合成互补DNA(cDNA)。这一过程违背了中心法则的传统方向，展现了遗传信息流动的多样性。逆转录过程包括第一链cDNA合成、RNA模板降解、第二链DNA合成等步骤。
  2. 逆转录病毒的生活周期：逆转录病毒(如HIV)利用逆转录酶将其RNA基因组转化为DNA，整合到宿主细胞基因组中，实现持久感染。这种机制使病毒能够逃避免疫系统的清除，也是某些病毒致癌的重要原因。逆转录病毒的研究推动了分子生物学技术的发展。
  3. 逆转座子的转座机制：逆转座子通过"复制-粘贴"机制进行转座，首先转录为RNA，然后通过逆转录形成DNA拷贝，最后整合到基因组的新位置。这种机制使逆转座子能够在基因组中大量扩增，成为许多生物基因组的重要组成部分。
  4. 基因组进化的驱动力：逆转座子是基因组大小和结构变化的主要驱动力。它们的插入可以导致基因失活、产生新的调控序列、促进基因重排和染色体重组。在人类基因组中，逆转座子及其残余序列占基因组的45%以上，对基因组的进化产生了深远影响。
  5. 基因功能的多样化：逆转座子的插入可以为基因提供新的启动子、增强子或剪接位点，改变基因的表达模式；某些逆转座子序列被宿主基因组"驯化"，成为功能基因的一部分；逆转座子介导的基因重复为新基因功能的进化提供了原材料。
  6. 疾病与医学意义：逆转座子的异常激活与多种疾病相关，包括癌症、神经退行性疾病等；同时，逆转录技术在基因治疗、疫苗开发等医学应用中具有重要价值；逆转录酶抑制剂是治疗HIV感染的重要药物。
