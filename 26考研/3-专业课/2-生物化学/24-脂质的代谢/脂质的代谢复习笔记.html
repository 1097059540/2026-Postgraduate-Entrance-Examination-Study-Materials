脂质的代谢

基于《生物化学》第四版（朱圣庚、徐长法主编）的考研复习笔记

一、脂质代谢概述

脂质代谢的重要性：脂质是生物体内重要的能量储存物质，也是生物膜的主要组成成分。脂质代谢包括脂肪酸的分解代谢（β氧化）、合成代谢、酮体代谢、磷脂代谢和胆固醇代谢等。

1.1 脂肪的消化、吸收和转运

膳食中脂肪的消化始于胃中的胃脂肪酶，终于小肠中的胰脂肪酶。消化产物脂肪酸和2-单酰甘油被小肠上皮吸收后，重新合成为三酰甘油，与蛋白质形成脂蛋白，经淋巴系统和血液循环运送到肌肉和脂肪组织。

脂肪组织中的脂肪被三酰甘油脂肪酶水解为甘油和脂肪酸，脂肪酸进入血液结合清蛋白后被运送到骨骼肌、心、肝等组织作为燃料。这个过程称为脂肪动员。

二、脂肪酸的β氧化

2.1 脂肪酸的活化和转运

关键步骤：脂肪酸必须先活化为脂酰-CoA，然后通过肉碱转运系统进入线粒体进行β氧化。

2.1.1 脂肪酸的活化

脂肪酸 + CoA + ATP → 脂酰-CoA + AMP + PPi
催化酶：脂酰辅酶A合酶
消耗：2个高能磷酸键

2.1.2 肉碱转运系统

长链脂酰-CoA不能直接穿透线粒体内膜，需要通过肉碱转运系统：

  1. 肉碱脂酰转移酶Ⅰ（CPT-Ⅰ）：脂酰-CoA + 肉碱 → 脂酰肉碱 + CoA
  2. 肉碱/脂酰肉碱转位酶：脂酰肉碱穿过内膜
  3. 肉碱脂酰转移酶Ⅱ（CPT-Ⅱ）：脂酰肉碱 + CoA → 脂酰-CoA + 肉碱
调节要点：CPT-Ⅰ是脂肪酸氧化的限速酶，强烈受丙二酰-CoA抑制，这是脂肪酸合成与分解代谢协调调节的关键机制。

2.2 β氧化的反应过程

β氧化是一个循环过程，每个循环包含4步反应，从脂肪酸的羧基端脱掉一个两碳单元（乙酰-CoA）：

第一步：氧化（脱氢）
脂酰-CoA + FAD → 反式-Δ²-烯酰-CoA + FADH₂
催化酶：脂酰-CoA脱氢酶
第二步：水合
反式-Δ²-烯酰-CoA + H₂O → L-3-羟脂酰-CoA
催化酶：烯酰-CoA水合酶
第三步：氧化（脱氢）
L-3-羟脂酰-CoA + NAD⁺ → 3-酮脂酰-CoA + NADH + H⁺
催化酶：3-羟脂酰-CoA脱氢酶
第四步：硫解
3-酮脂酰-CoA + CoA → 乙酰-CoA + 脂酰-CoA（减少2个C）
催化酶：硫解酶

2.3 β氧化的能量计算

以软脂酸（16C）为例：
  • 经过7轮β氧化，产生8个乙酰-CoA
  • 每轮产生：1个FADH₂ + 1个NADH
  • 总产生：7个FADH₂ + 7个NADH + 8个乙酰-CoA
  • ATP计算：7×1.5 + 7×2.5 + 8×10 - 2 = 106个ATP

2.4 特殊脂肪酸的氧化

2.4.1 不饱和脂肪酸的氧化

不饱和脂肪酸的氧化需要额外的酶：

  • 异构酶：将顺式双键转化为反式双键
  • 2,4-二烯酰-CoA还原酶：处理多不饱和脂肪酸
  • 由于绕过了脂酰-CoA脱氢酶，每个双键少产生1个FADH₂

2.4.2 奇数碳原子脂肪酸的氧化

奇数碳原子脂肪酸在最后一轮β氧化后产生：

  • 1个乙酰-CoA（2个碳）
  • 1个丙酰-CoA（3个碳）
  • 丙酰-CoA可转化为琥珀酰-CoA进入TCA循环，也可用于糖异生

2.5 过氧化物酶体中的β氧化

过氧化物酶体也能进行β氧化，但有以下特点：

  • 第一步氧化直接与O₂反应，产生H₂O₂而非FADH₂
  • 释放的能量以热量形式散失，不产生ATP
  • 主要功能：将超长链脂肪酸缩短为中长链，然后转入线粒体
  • 在植物中，过氧化物酶体是脂肪酸氧化的主要场所

三、酮体代谢

3.1 酮体的概念和组成

酮体包括三种化合物：
  • 乙酰乙酸（Acetoacetate）
  • D-β-羟丁酸（D-β-Hydroxybutyrate）
  • 丙酮（Acetone，少量）

3.2 酮体的生成（酮体生成）

酮体主要在肝脏线粒体中生成，当乙酰-CoA过量时发生：

酮体生成途径：
2 乙酰-CoA → 乙酰乙酰-CoA + CoA
↓ HMG-CoA合酶
乙酰乙酰-CoA + 乙酰-CoA → HMG-CoA
↓ HMG-CoA裂解酶
HMG-CoA → 乙酰乙酸 + 乙酰-CoA
↓ β-羟丁酸脱氢酶
乙酰乙酸 + NADH → D-β-羟丁酸 + NAD⁺

3.3 酮体的利用

酮体在肝外组织（心、脑、肌肉等）被氧化利用：

酮体氧化途径：
D-β-羟丁酸 + NAD⁺ → 乙酰乙酸 + NADH + H⁺
↓ 琥珀酰-CoA转硫酶
乙酰乙酸 + 琥珀酰-CoA → 乙酰乙酰-CoA + 琥珀酸
↓ 硫解酶
乙酰乙酰-CoA + CoA → 2 乙酰-CoA
↓
进入TCA循环

3.4 酮体代谢的生理意义

生理条件下：
  • 饥饿状态：为脑组织提供替代燃料
  • 长期饥饿：脑组织可利用酮体满足60-70%的能量需求
  • 运动状态：为心肌和骨骼肌提供能量
病理条件下：
  • 糖尿病酮症酸中毒：酮体过量产生导致血液pH下降
  • 酮血症：血中酮体浓度异常升高

四、脂肪酸的生物合成

4.1 脂肪酸合成的基本特点

特征 脂肪酸合成 脂肪酸β氧化
场所 细胞溶胶 线粒体基质
方向 甲基端→羧基端 羧基端→甲基端
载体 ACP（酰基载体蛋白） CoA
碳源 丙二酰-CoA（3C→2C） 产生乙酰-CoA（2C）
辅酶 NADPH NAD⁺、FAD
羟基构型 D-型 L-型

4.2 乙酰-CoA的转运

脂肪酸合成的原料乙酰-CoA产生于线粒体，但合成在细胞溶胶进行，需要三羧酸转运系统：

线粒体内：
乙酰-CoA + 草酰乙酸 → 柠檬酸
↓ 穿过线粒体内膜
细胞溶胶中：
柠檬酸 + ATP + CoA → 乙酰-CoA + 草酰乙酸 + ADP + Pi
催化酶：柠檬酸裂解酶

4.3 脂肪酸合成的反应过程

4.3.1 丙二酰-CoA的形成

乙酰-CoA + CO₂ + ATP → 丙二酰-CoA + ADP + Pi
催化酶：乙酰-CoA羧化酶（限速酶）
辅酶：生物素

4.3.2 脂肪酸合酶复合体

在哺乳动物中，脂肪酸合成由一个多酶复合体催化，包含7种酶活性：

  1. 乙酰转移酶：启动反应
  2. 丙二酰转移酶：装载丙二酰基团
  3. 缩合酶：形成C-C键
  4. β-酮脂酰还原酶：第一次还原
  5. 脱水酶：脱水形成双键
  6. 烯脂酰还原酶：第二次还原
  7. 硫酯酶：释放脂肪酸

4.3.3 合成循环

启动：乙酰-CoA → 乙酰-ACP
装载：丙二酰-CoA → 丙二酰-ACP
缩合：乙酰-ACP + 丙二酰-ACP → 乙酰乙酰-ACP + CO₂
还原：乙酰乙酰-ACP + NADPH → D-β-羟丁酰-ACP
脱水：D-β-羟丁酰-ACP → 反式-丁烯酰-ACP
还原：反式-丁烯酰-ACP + NADPH → 丁酰-ACP
释放：软脂酰-ACP + H₂O → 软脂酸 + ACP

4.4 脂肪酸的进一步修饰

4.4.1 碳链延长

软脂酸（C16）可进一步延长：

  • 线粒体系统：逆转β氧化过程
  • 内质网系统：使用丙二酰-CoA作为碳源

4.4.2 去饱和反应

不饱和脂肪酸的形成需要去饱和酶：

软脂酰-CoA + NADH + H⁺ + O₂ → 软油酰-CoA + NAD⁺ + 2H₂O
催化酶：Δ⁹-去饱和酶
必需脂肪酸：哺乳动物缺乏在Δ⁹以外引入双键的酶，因此亚油酸（C18:2）和α-亚麻酸（C18:3）是必需脂肪酸，必须从食物中获得。

4.5 脂肪酸代谢的调节

4.5.1 别构调节

  • 乙酰-CoA羧化酶：
    • 激活剂：柠檬酸
    • 抑制剂：软脂酰-CoA
  • CPT-Ⅰ：被丙二酰-CoA强烈抑制

4.5.2 共价修饰调节

胰高血糖素/肾上腺素激活cAMP-PKA途径：

  • 磷酸化抑制乙酰-CoA羧化酶（合成↓）
  • 磷酸化激活三酰甘油脂肪酶（分解↑）

4.5.3 酶水平调节

长期膳食改变导致相关酶合成水平的调节：

  • 高糖低脂饮食：促进脂肪酸合成酶表达
  • 高脂低糖饮食：促进脂肪酸氧化酶表达

五、复合脂质的代谢

5.1 磷脂的生物合成

5.1.1 磷脂酸的形成

磷脂合成的共同前体是磷脂酸：

甘油-3-磷酸 + 2脂酰-CoA → 磷脂酸 + 2CoA
催化酶：甘油磷酸酰基转移酶

5.1.2 甘油磷脂的合成

有两条主要途径：

途径一（原核生物）：
磷脂酸 + CTP → CDP-二酰甘油 + PPi
CDP-二酰甘油 + 丝氨酸 → 磷脂酰丝氨酸 + CMP
磷脂酰丝氨酸 → 磷脂酰乙醇胺 + CO₂
途径二（真核生物）：
胆碱 + ATP → 磷酸胆碱 + ADP
磷酸胆碱 + CTP → CDP-胆碱 + PPi
CDP-胆碱 + 二酰甘油 → 磷脂酰胆碱 + CMP

5.2 鞘脂的生物合成

5.2.1 神经酰胺的形成

丝氨酸 + 软脂酰-CoA → 3-酮二氢鞘氨醇
↓ 还原、酰化
神经酰胺（鞘脂类的基本骨架）

5.2.2 鞘脂类的形成

  • 鞘磷脂：神经酰胺 + 磷脂酰胆碱 → 鞘磷脂 + 二酰甘油
  • 脑苷脂：神经酰胺 + UDP-葡萄糖 → 葡萄糖脑苷脂 + UDP
  • 神经节苷脂：在脑苷脂基础上进一步糖基化

5.3 鞘脂代谢异常与疾病

鞘脂贮积病：由于鞘脂分解酶缺陷导致的遗传性疾病
  • Tay-Sachs病：β-己糖胺酶A缺陷，GM2神经节苷脂积累
  • Gaucher病：β-葡萄糖脑苷脂酶缺陷，葡萄糖脑苷脂积累
  • Niemann-Pick病：鞘磷脂酶缺陷，鞘磷脂积累

六、胆固醇代谢

  • 胆汁酸的前体
  • 维生素D₃的前体

6.3 胆固醇代谢的调节

6.3.1 HMG-CoA还原酶的调节

  • 转录水平：胆固醇抑制酶基因转录
  • 翻译后修饰：磷酸化抑制酶活性
  • 蛋白质降解：胆固醇促进酶降解

6.3.2 胆固醇的反馈抑制

细胞内胆固醇通过以下机制调节自身合成：

  • 抑制HMG-CoA还原酶活性
  • 激活酰基辅酶A：胆固醇酰基转移酶（ACAT）
  • 抑制LDL受体合成

6.4.1 脂蛋白的分类

6.5.1 胆汁酸的合成

催化酶：胆固醇7α-羟化酶（限速酶）

胆酸、鹅脱氧胆酸

6.5.2 胆汁酸的肠肝循环

  • 肝脏合成胆汁酸，储存在胆囊
  • 进食后释放到小肠，帮助脂质消化
  • 95%的胆汁酸在回肠被重吸收
  • 通过门静脉回到肝脏，形成肠肝循环

七、名词解释库

β氧化
Beta-oxidation
脂肪酸分解代谢的主要途径，通过循环的四步反应（氧化-水合-氧化-硫解）从脂肪酸羧基端逐步脱掉两碳单元，产生乙酰-CoA、FADH₂和NADH。
肉碱转运系统
Carnitine Transport System
长链脂酰-CoA进入线粒体的转运机制，包括CPT-Ⅰ、肉碱/脂酰肉碱转位酶和CPT-Ⅱ三个组分，其中CPT-Ⅰ是脂肪酸氧化的限速步骤。
酮体
Ketone Bodies
包括乙酰乙酸、D-β-羟丁酸和丙酮的总称，主要在肝脏中由乙酰-CoA合成，在肝外组织氧化供能，是饥饿状态下脑组织的重要替代燃料。
脂肪酸合酶
Fatty Acid Synthase, FAS
哺乳动物中催化脂肪酸合成的多酶复合体，包含7种酶活性和1个ACP，能够将乙酰-CoA和丙二酰-CoA合成为软脂酸。
乙酰-CoA羧化酶
Acetyl-CoA Carboxylase, ACC
脂肪酸合成的限速酶，催化乙酰-CoA羧化形成丙二酰-CoA，受柠檬酸激活，受软脂酰-CoA和cAMP依赖的磷酸化抑制。
HMG-CoA还原酶
HMG-CoA Reductase
胆固醇合成的限速酶，催化HMG-CoA还原为甲羟戊酸，是胆固醇合成调节的关键控制点，他汀类药物的作用靶点。
脂肪动员
Fat Mobilization
脂肪组织中储存的三酰甘油被脂肪酶水解为甘油和脂肪酸，脂肪酸释放入血并转运到其他组织进行氧化的过程。
三羧酸转运系统
Tricarboxylate Transport System
将线粒体内的乙酰-CoA以柠檬酸形式转运到细胞溶胶进行脂肪酸合成的机制，涉及柠檬酸载体和柠檬酸裂解酶。
必需脂肪酸
Essential Fatty Acids
人体不能合成必须从食物中获得的脂肪酸，主要包括亚油酸（C18:2）和α-亚麻酸（C18:3），是前列腺素等生物活性物质的前体。
磷脂酸
Phosphatidic Acid, PA
甘油磷脂合成的共同前体，由甘油-3-磷酸经两次酰化反应形成，可进一步合成各种甘油磷脂或三酰甘油。
神经酰胺
Ceramide
鞘脂类的基本骨架，由鞘氨醇与脂肪酸通过酰胺键结合形成，是鞘磷脂和鞘糖脂合成的前体。
鞘脂贮积病
Sphingolipidoses
由于鞘脂分解酶缺陷导致特定鞘脂在细胞内异常积累的遗传性疾病，如Tay-Sachs病、Gaucher病等。
LDL受体
LDL Receptor
细胞表面识别和结合低密度脂蛋白的受体，介导胆固醇的细胞摄取，其缺陷导致家族性高胆固醇血症。
胆固醇逆转运
Reverse Cholesterol Transport
HDL将外周组织的胆固醇转运回肝脏进行代谢或排泄的过程，是机体清除多余胆固醇的重要机制。
肠肝循环
Enterohepatic Circulation
胆汁酸在肝脏合成、胆囊储存、小肠发挥功能、回肠重吸收、门静脉回肝的循环过程，提高胆汁酸的利用效率。
异戊二烯单位
Isoprene Unit
胆固醇等类异戊二烯化合物的基本构建单元，包括异戊烯焦磷酸（IPP）和二甲基烯丙基焦磷酸（DMAPP）。
过氧化物酶体β氧化
Peroxisomal Beta-oxidation
发生在过氧化物酶体中的脂肪酸氧化过程，主要处理超长链脂肪酸，第一步氧化产生H₂O₂而非FADH₂，能量以热量形式散失。

九、深度思考题

1. 比较分析脂肪酸β氧化与脂肪酸合成在细胞定位、反应机制、调节方式等方面的差异，并阐述这些差异的生物学意义。

参考答案：
  1. 细胞定位差异：β氧化发生在线粒体基质，而脂肪酸合成发生在细胞溶胶。这种空间分离避免了合成与分解的同时进行，提高了代谢效率。线粒体是ATP产生的主要场所，β氧化在此进行有利于能量的直接利用。
  2. 反应方向与机制：β氧化从羧基端向甲基端进行，每次脱掉两个碳原子；合成则从甲基端向羧基端进行，每次添加两个碳原子。β氧化产生FADH₂和NADH，合成消耗NADPH。这种相反的机制确保了两个过程的独立调节。
  3. 载体蛋白差异：β氧化使用CoA作为载体，合成使用ACP。不同载体使得两个过程使用不同的酶系统，便于精确调节。
  4. 调节机制：丙二酰-CoA既是合成的原料，又是β氧化限速酶CPT-Ⅰ的抑制剂，实现了合成与分解的协调调节。当合成活跃时，β氧化被抑制，避免了无效循环。
  5. 生物学意义：这些差异确保了机体能够根据能量状态和营养条件灵活调节脂质代谢，在能量充足时促进脂肪合成储存，在能量需求时促进脂肪分解供能。

2. 酮体代谢在人体能量代谢中的作用机制是什么？分析酮体代谢异常与糖尿病酮症酸中毒的关系。

参考答案：
  1. 酮体的生理作用：酮体是脂肪酸β氧化产生的乙酰-CoA的替代利用途径，主要在肝脏产生，在肝外组织利用。正常情况下，酮体为心肌、骨骼肌等组织提供能量，在饥饿状态下成为脑组织的重要替代燃料。
  2. 酮体生成的调节：当草酰乙酸不足（如糖异生增强时）或乙酰-CoA过量时，乙酰-CoA不能充分进入TCA循环，转而生成酮体。这是机体在糖供应不足时的适应性反应。
  3. 酮体的转运与利用：酮体可自由穿过血脑屏障，在肝外组织通过特异性酶系统重新转化为乙酰-CoA进入TCA循环。这种机制使得脂肪酸的能量能够间接供给脑组织。
  4. 糖尿病酮症酸中毒的机制：在未控制的糖尿病中，胰岛素缺乏导致糖利用障碍，脂肪动员和β氧化过度激活，酮体生成大量增加。酮体中的乙酰乙酸和β-羟丁酸是有机酸，大量积累导致代谢性酸中毒。
  5. 临床意义：监测血酮和尿酮是糖尿病管理的重要指标。适度的酮体生成是正常的代谢适应，但过量则提示代谢紊乱，需要及时干预。

3. 胆固醇代谢的调节机制复杂而精密，请分析HMG-CoA还原酶的多层次调节机制及其在胆固醇稳态维持中的作用。

参考答案：
  1. 转录水平调节：胆固醇通过SREBP-SCAP系统调节HMG-CoA还原酶基因转录。当细胞内胆固醇充足时，SREBP被保留在内质网，无法激活基因转录；胆固醇不足时，SREBP转移到核内激活转录。这种机制确保了酶合成与胆固醇需求的匹配。
  2. 翻译后修饰调节：HMG-CoA还原酶受AMP激活的蛋白激酶（AMPK）磷酸化而失活。当细胞能量状态低下时，AMPK激活，抑制耗能的胆固醇合成。胰岛素则通过激活蛋白磷酸酶促进酶去磷酸化激活，协调胆固醇合成与营养状态。
  3. 蛋白质稳定性调节：胆固醇及其代谢产物促进HMG-CoA还原酶的泛素化降解。酶的膜结合域感受胆固醇水平变化，高胆固醇时酶快速降解，低胆固醇时酶稳定存在。这种机制提供了快速的负反馈调节。
  4. 别构调节：虽然HMG-CoA还原酶本身的别构调节相对较弱，但其产物甲羟戊酸的下游代谢物可以反馈调节酶活性，形成精细的调节网络。
  5. 临床意义：他汀类药物通过竞争性抑制HMG-CoA还原酶降低胆固醇合成，是治疗高胆固醇血症的重要药物。理解这些调节机制有助于开发新的治疗策略和理解相关疾病的发病机制。

4. 脂蛋白代谢异常与动脉粥样硬化的关系如何？分析LDL和HDL在胆固醇转运中的不同作用机制。

参考答案：
  1. LDL的致动脉粥样硬化机制：LDL携带胆固醇从肝脏转运到外周组织，当LDL浓度过高或被氧化修饰时，容易在动脉壁沉积。氧化LDL被巨噬细胞吞噬形成泡沫细胞，启动炎症反应，促进动脉粥样硬化斑块形成。
  2. LDL受体途径的调节：细胞通过LDL受体摄取胆固醇，当细胞内胆固醇充足时，LDL受体表达下调，减少胆固醇摄取。但巨噬细胞的清道夫受体不受此调节，持续摄取氧化LDL导致胆固醇过载。
  3. HDL的保护机制：HDL通过胆固醇逆转运系统将外周组织多余的胆固醇转运回肝脏。HDL中的载脂蛋白A-I激活卵磷脂胆固醇酰基转移酶（LCAT），将游离胆固醇酯化便于转运。
  4. HDL的抗动脉粥样硬化作用：除胆固醇逆转运外，HDL还具有抗氧化、抗炎、改善内皮功能等多重保护作用。HDL能够清除氧化LDL，抑制炎症因子释放，维护血管内皮完整性。
  5. 临床应用价值：LDL-C/HDL-C比值是评估心血管疾病风险的重要指标。治疗策略包括降低LDL（他汀类药物）和提高HDL（运动、戒烟、某些药物），综合改善脂蛋白谱以预防动脉粥样硬化。

5. 分析脂质代谢与糖代谢之间的相互关系，特别是在饥饿状态和糖尿病条件下的代谢重编程机制。

参考答案：
  1. 正常状态下的代谢协调：胰岛素促进糖的利用和脂肪的合成，抑制糖异生和脂肪分解；胰高血糖素则相反。柠檬酸-丙酮酸循环协调糖酵解与脂肪酸合成，当糖充足时促进脂肪合成，糖不足时抑制脂肪合成。
  2. 饥饿状态的代谢适应：胰岛素水平下降，胰高血糖素和肾上腺素升高，激活脂肪动员和β氧化。脂肪酸氧化产生的乙酰-CoA抑制糖酵解关键酶，促进糖异生。酮体生成为脑组织提供替代燃料，减少对葡萄糖的依赖。
  3. Randle循环（糖脂竞争）：脂肪酸氧化产生的柠檬酸和ATP抑制磷酸果糖激酶，乙酰-CoA抑制丙酮酸脱氢酶，从而抑制糖酵解。反之，葡萄糖利用产生的丙二酰-CoA抑制CPT-Ⅰ，抑制脂肪酸氧化。
  4. 糖尿病的代谢紊乱：胰岛素缺乏或抵抗导致糖利用障碍，机体误认为处于饥饿状态，过度激活脂肪分解和糖异生。大量脂肪酸氧化产生酮体，可能导致酮症酸中毒。同时，糖异生过度激活导致高血糖。
  5. 治疗策略的代谢基础：糖尿病治疗不仅要控制血糖，还要纠正脂质代谢异常。二甲双胍通过激活AMPK同时改善糖脂代谢；胰岛素治疗恢复正常的代谢调节；生活方式干预通过改善胰岛素敏感性协调糖脂代谢。
