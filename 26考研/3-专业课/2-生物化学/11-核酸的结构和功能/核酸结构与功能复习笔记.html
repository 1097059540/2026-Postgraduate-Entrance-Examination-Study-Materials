核酸的结构和功能

考研生物化学复习笔记 | 基于《生物化学》第四版（朱圣庚、徐长法主编）

一、核酸的历史发现与重要意义

1.1 重要历史节点

  • 1869年：瑞士生物化学家米歇尔（Miescher）首次从白细胞核中分离出核酸
  • 1944年：艾弗里（Avery）等证明DNA是遗传物质
  • 1950年：查戈夫（Chargaff）发现查戈夫定则：[A]=[T]，[G]=[C]
  • 1953年：沃森（Watson）和克里克（Crick）提出DNA双螺旋结构模型
  • 1958年：中心法则的提出：DNA→RNA→蛋白质

1.2 核酸的基本分类

核酸根据所含戊糖的不同分为两大类：

  • 脱氧核糖核酸（DNA）：含脱氧核糖，主要负责遗传信息的储存和传递
  • 核糖核酸（RNA）：含核糖，参与遗传信息的表达和蛋白质合成

二、核酸的化学组成

2.1 基本组成单位

核酸的基本组成单位是核苷酸（nucleotide），每个核苷酸由三部分组成：

（1）含氮碱基

  • 嘌呤碱基：腺嘌呤（A）、鸟嘌呤（G）
  • 嘧啶碱基：胞嘧啶（C）、胸腺嘧啶（T，仅存在于DNA）、尿嘧啶（U，仅存在于RNA）

（2）戊糖

  • 脱氧核糖：DNA中的戊糖，2'位缺少羟基
  • 核糖：RNA中的戊糖，2'位和3'位都有羟基

（3）磷酸基团

连接相邻核苷酸，形成核酸的骨架结构。

2.2 DNA与RNA的组成差异

嘌呤碱基 腺嘌呤（A）、鸟嘌呤（G） 腺嘌呤（A）、鸟嘌呤（G）
嘧啶碱基 胞嘧啶（C）、胸腺嘧啶（T） 胞嘧啶（C）、尿嘧啶（U）

三、核酸的结构层次

3.1 一级结构（Primary Structure）

核酸的一级结构是指核苷酸的排列顺序，即碱基序列。核苷酸之间通过3',5'-磷酸二酯键连接，形成具有方向性的多核苷酸链。

重要特点：

  • 具有方向性：5'端→3'端
  • 磷酸二酯键连接相邻核苷酸的3'碳和5'碳
  • 碱基序列决定遗传信息

3.2 二级结构（Secondary Structure）

3.2.1 DNA的二级结构 - 双螺旋结构

1953年沃森和克里克提出的DNA双螺旋结构模型具有以下特点：

基本特征：
  • 反向平行：两条链方向相反（5'→3'和3'→5'）
  • 右手螺旋：螺旋方向为右手螺旋
  • 碱基配对：A与T配对（2个氢键），G与C配对（3个氢键）
  • 碱基堆积：同一链上相邻碱基间的疏水相互作用
结构参数（B-DNA）：
  • 螺旋直径：2.0 nm
  • 螺距：3.4 nm
  • 每圈碱基对数：10对
  • 相邻碱基对间距：0.34 nm
  • 碱基对转角：36°
DNA的多态性：
DNA构象 螺旋方向 存在条件 特点
A-DNA 右手螺旋 低湿度（75%） 螺旋更紧密，碱基对倾斜
B-DNA 右手螺旋 生理条件（92%湿度） 最常见形式，碱基对垂直于螺旋轴
Z-DNA 左手螺旋 高盐浓度，GC富集区 "之"字形骨架，可能参与基因调控

3.2.2 RNA的二级结构

RNA通常为单链分子，但可通过分子内碱基配对形成复杂的二级结构：

  • 发夹结构（Hairpin）：短的回文序列形成的茎环结构
  • 三叶草结构：tRNA的典型二级结构
  • 假结（Pseudoknot）：复杂的折叠结构

3.3 三级结构（Tertiary Structure）

3.3.1 DNA的三级结构

  • 超螺旋结构：DNA双螺旋的进一步扭曲，多为负超螺旋
  • 核小体：DNA缠绕组蛋白八聚体形成的基本包装单位
  • 染色质纤维：核小体进一步压缩形成的高级结构

3.3.2 RNA的三级结构

以tRNA为例：

  • 倒L型结构：tRNA的三维空间构象
  • 功能区域：氨基酸结合臂、反密码子臂等

四、核酸的生物学功能

4.1 DNA的主要功能

  • 遗传信息储存：以碱基序列形式储存遗传信息
  • 遗传信息传递：通过DNA复制实现遗传信息的垂直传递
  • 基因表达调控：调控序列参与基因表达的时空调控

4.2 RNA的多样化功能

4.2.1 经典功能

  • 信使RNA（mRNA）：携带遗传信息，作为蛋白质合成的模板
  • 转移RNA（tRNA）：转运氨基酸，参与蛋白质合成
  • 核糖体RNA（rRNA）：构成核糖体，催化肽键形成

4.2.2 调控功能

  • 微小RNA（miRNA）：调控基因表达
  • 小干扰RNA（siRNA）：参与RNA干扰
  • 长链非编码RNA（lncRNA）：多种调控功能

4.2.3 催化功能

  • 核酶（Ribozyme）：具有催化活性的RNA分子
  • 例如：肽基转移酶、RNase P、自剪接内含子等

五、重要名词解释库

核苷酸
Nucleotide
核酸的基本组成单位，由含氮碱基、戊糖和磷酸基团组成。是构成DNA和RNA的基本结构单元。
核苷
Nucleoside
由含氮碱基和戊糖通过糖苷键结合形成的化合物，不含磷酸基团。核苷加上磷酸基团即为核苷酸。
查戈夫定则
Chargaff's Rules
在双链DNA中，腺嘌呤的数量等于胸腺嘧啶的数量（[A]=[T]），鸟嘌呤的数量等于胞嘧啶的数量（[G]=[C]）。这一定则为DNA双螺旋结构的提出提供了重要依据。
双螺旋结构
Double Helix Structure
DNA的二级结构，由两条反向平行的多核苷酸链围绕同一中心轴相互缠绕形成的右手螺旋结构。碱基通过氢键配对，维持结构稳定。
碱基配对
Base Pairing
DNA双链中互补碱基之间通过氢键结合的现象。腺嘌呤与胸腺嘧啶配对（A-T，2个氢键），鸟嘌呤与胞嘧啶配对（G-C，3个氢键）。
超螺旋
Supercoiling
DNA双螺旋的进一步扭曲形成的三级结构。生物体内多为负超螺旋，有利于DNA的复制、转录等过程中双链的解开。
核小体
Nucleosome
真核细胞染色质的基本结构单位，由约147个碱基对的DNA缠绕组蛋白八聚体（H2A、H2B、H3、H4各2个）形成，呈现"串珠"状结构。
中心法则
Central Dogma
描述遗传信息流向的基本规律：DNA→RNA→蛋白质。即遗传信息从DNA转录到RNA，再从RNA翻译成蛋白质。
核酶
Ribozyme
具有催化活性的RNA分子，能够催化特定的生化反应。证明了RNA不仅能储存遗传信息，还具有催化功能。
反密码子
Anticodon
tRNA分子上与mRNA密码子互补配对的三个连续碱基序列，决定tRNA携带的氨基酸种类。
变性
Denaturation
在高温、极端pH或化学试剂作用下，DNA双链分离成单链的过程。变性过程是可逆的，条件适宜时可发生复性。
熔解温度
Melting Temperature, Tm
DNA双链50%解离时的温度。Tm值与GC含量正相关，因为G-C碱基对比A-T碱基对多一个氢键，更稳定。

六、选择题自测（随机15题）

七、深度思考题

1. 试述DNA双螺旋结构的发现过程及其科学意义，并分析该模型如何解释遗传信息的储存、复制和传递机制。

答案要点：
  1. 发现过程：
    • 查戈夫定则的发现为碱基配对提供了化学基础
    • 富兰克林的X射线衍射数据（照片51）提供了结构证据
    • 沃森和克里克整合多方面信息，提出双螺旋模型
    • 模型的提出是多学科协作的典型范例
  2. 科学意义：
    • 奠定了分子生物学的基础，开启了分子生物学时代
    • 为理解遗传现象提供了分子基础
    • 推动了基因工程、生物技术等领域的发展
    • 为疾病诊断和治疗提供了新思路
  3. 遗传信息储存机制：
    • 碱基序列作为遗传信息的载体
    • 四种碱基的不同排列组合产生巨大的信息容量
    • 双链结构提供了信息的冗余保护
  4. 复制机制：
    • 半保留复制方式确保遗传信息的准确传递
    • 碱基配对规则保证复制的忠实性
    • 反向平行结构为复制提供了模板
  5. 传递机制：
    • 转录过程将DNA信息转移到RNA
    • 翻译过程将RNA信息转化为蛋白质
    • 中心法则描述了信息流的基本方向

2. 比较分析DNA和RNA在结构、功能和生物学意义方面的异同，并讨论RNA世界假说的科学依据。

答案要点：
  1. 结构差异：
    • 戊糖：DNA含脱氧核糖，RNA含核糖
    • 碱基：DNA含T，RNA含U
    • 链数：DNA通常为双链，RNA通常为单链
    • 稳定性：DNA更稳定，RNA相对不稳定
  2. 功能差异：
    • DNA主要负责遗传信息的长期储存
    • RNA参与遗传信息的表达和调控
    • RNA具有催化功能（核酶）
    • RNA在蛋白质合成中起关键作用
  3. 相同点：
    • 都是核苷酸聚合物
    • 都能储存遗传信息
    • 都遵循碱基配对规则
    • 都具有方向性（5'→3'）
  4. RNA世界假说的科学依据：
    • RNA同时具有信息储存和催化功能
    • 核酶的发现证明RNA的催化能力
    • RNA可以自我复制（某些病毒）
    • 蛋白质合成的核心步骤由RNA催化
    • 许多辅酶含有核苷酸成分
  5. 进化意义：
    • RNA可能是最早的遗传物质
    • DNA的出现提高了遗传信息的稳定性
    • 蛋白质的出现扩展了催化功能
    • 形成了现代的DNA-RNA-蛋白质体系

3. 分析核酸的不同结构层次（一级、二级、三级）及其生物学功能的关系，并举例说明结构与功能的统一性。

答案要点：
  1. 一级结构与功能：
    • 碱基序列决定遗传信息的内容
    • 序列的特异性决定基因的功能
    • 突变改变序列可导致功能异常
    • 例如：血红蛋白基因突变导致镰刀型细胞贫血
  2. 二级结构与功能：
    • DNA双螺旋结构保护遗传信息
    • 碱基配对确保复制的准确性
    • RNA的发夹结构参与调控功能
    • 例如：tRNA的三叶草结构决定其转运功能
  3. 三级结构与功能：
    • DNA超螺旋调节基因表达
    • 染色质结构影响基因可及性
    • RNA的三维结构决定催化活性
    • 例如：核酶的活性位点由三级结构形成
  4. 结构层次间的相互关系：
    • 一级结构是基础，决定高级结构的形成
    • 二级结构是一级结构的空间表现
    • 三级结构是功能实现的关键
    • 各层次结构相互依存，共同决定功能
  5. 结构与功能统一性的实例：
    • tRNA：L型三级结构使氨基酸结合位点和反密码子分离
    • 核糖体RNA：复杂的三级结构形成肽基转移酶活性中心
    • DNA拓扑结构：负超螺旋有利于转录起始
    • miRNA：茎环结构决定其调控特异性

4. 论述核酸在现代生物技术中的应用，并分析其发展前景和潜在挑战。

答案要点：
  1. 基因工程技术：
    • 重组DNA技术：构建表达载体，生产重组蛋白
    • 基因克隆：获得目的基因，研究基因功能
    • 转基因技术：改良作物性状，生产药用蛋白
    • 应用实例：胰岛素、生长激素的工业化生产
  2. 基因诊断与治疗：
    • PCR技术：快速检测病原体，诊断遗传病
    • 基因芯片：大规模基因表达分析
    • 基因治疗：纠正缺陷基因，治疗遗传病
    • 个性化医疗：根据基因型制定治疗方案
  3. 新兴技术：
    • CRISPR-Cas9：精确基因编辑技术
    • RNA干扰：基因沉默技术
    • 合成生物学：设计人工生物系统
    • 纳米生物技术：DNA纳米结构的构建
  4. 发展前景：
    • 精准医疗：基于基因组信息的个体化治疗
    • 基因编辑：治疗更多遗传性疾病
    • 合成生物学：创造新的生物功能
    • 生物计算：DNA作为信息存储介质
  5. 潜在挑战：
    • 伦理问题：基因编辑的道德边界
    • 安全性：基因治疗的副作用
    • 技术限制：递送系统的效率
    • 社会接受度：公众对基因技术的认知
    • 法律法规：相关政策的完善
