糖类和糖生物学

考研生物化学复习笔记 | 基于《生物化学》第四版（朱圣庚、徐长法主编）

一、糖类概述与分类

1.1 糖类的化学本质

糖类（Carbohydrates）是多羟基醛或多羟基酮及其缩聚物和某些衍生物的总称。1 主要由碳、氢、氧三种元素组成，分子通式为(CH₂O)ₙ，但并非所有符合此通式的化合物都是糖类。

糖类具有以下重要特征：

  • 结构多样性：包括单糖、寡糖、多糖等不同层次的结构
  • 立体异构性：存在多种立体异构体，如α-异构体和β-异构体
  • 功能多样性：既是能源物质，又是结构成分和信息分子

1.2 糖类的分类体系

根据分子大小和复杂程度，糖类可分为三大类：

  1. 单糖（Monosaccharides）：不能水解的最简单糖类
    • 醛糖：含醛基的单糖，如葡萄糖、甘露糖、半乳糖
    • 酮糖：含酮基的单糖，如果糖、核酮糖
  2. 寡糖（Oligosaccharides）：由2-10个单糖分子连接而成
    • 双糖：蔗糖、乳糖、麦芽糖
    • 三糖：棉子糖、松三糖
  3. 多糖（Polysaccharides）：由多个单糖分子连接而成
    • 同多糖：淀粉、糖原、纤维素
    • 杂多糖：透明质酸、硫酸软骨素

二、单糖的结构与性质

2.1 单糖的立体化学

单糖分子具有手性碳原子，因此存在立体异构现象。根据分子中距羰基碳最远的手性碳原子上羟基的位置，可将单糖分为D型和L型。

重要概念：

  • 对映异构体：分子式相同但空间构型互为镜像的异构体
  • 差向异构体：仅在一个手性碳原子上构型不同的异构体
  • 异头异构体：环状结构中仅在异头碳原子上构型不同的异构体

2.2 单糖的环状结构

单糖在水溶液中主要以环状半缩醛形式存在。六碳糖形成吡喃环，五碳糖形成呋喃环。环状结构的形成产生了新的手性碳原子——异头碳原子，从而产生α-异构体和β-异构体。

2.3 重要的单糖及其衍生物

生物体内重要的单糖包括：

  • 葡萄糖：生命活动的主要能源物质，血糖的主要成分
  • 果糖：水果中的主要糖类，甜度最高的天然糖
  • 半乳糖：乳糖的组成成分，参与糖脂和糖蛋白的合成
  • 核糖：RNA的组成成分
  • 脱氧核糖：DNA的组成成分

重要的单糖衍生物：

  • 氨基糖：如葡糖胺、半乳糖胺，是糖胺聚糖的组成单位
  • 糖醛酸：如葡糖醛酸，参与解毒反应
  • 糖醇：如山梨醇、甘露醇，具有甜味但不被代谢

三、寡糖与多糖

3.1 重要的双糖

双糖通过糖苷键连接两个单糖分子形成。根据是否保留还原性，可分为还原性双糖和非还原性双糖。

蔗糖 (Sucrose)

由α-D-葡萄糖和β-D-果糖通过α(1→2)糖苷键连接形成的非还原性双糖。是植物中最重要的转运糖类。

乳糖 (Lactose)

由β-D-半乳糖和D-葡萄糖通过β(1→4)糖苷键连接形成的还原性双糖。是哺乳动物乳汁中的主要糖类。

麦芽糖 (Maltose)

由两个D-葡萄糖分子通过α(1→4)糖苷键连接形成的还原性双糖。是淀粉水解的主要产物。

3.2 重要的多糖

3.2.1 储能多糖

  • 淀粉：植物的储能多糖，包括直链淀粉和支链淀粉
    • 直链淀粉：α(1→4)糖苷键连接，呈螺旋结构
    • 支链淀粉：主链为α(1→4)，分支点为α(1→6)糖苷键
  • 糖原：动物的储能多糖，结构类似支链淀粉但分支更多

3.2.2 结构多糖

  • 纤维素：植物细胞壁的主要成分，β(1→4)糖苷键连接
  • 几丁质：节肢动物外骨骼和真菌细胞壁的成分
  • 肽聚糖：细菌细胞壁的主要成分

四、糖缀合物

4.1 糖蛋白

糖蛋白是糖类与蛋白质通过共价键结合形成的复合分子。2 根据连接方式可分为：

  • N-连接糖蛋白：糖链通过N-糖苷键与天冬酰胺残基连接
  • O-连接糖蛋白：糖链通过O-糖苷键与丝氨酸或苏氨酸残基连接

糖蛋白的生物学功能：

  1. 参与细胞识别和细胞间相互作用
  2. 维持蛋白质的正确折叠和稳定性
  3. 调节蛋白质的生物活性
  4. 参与免疫反应和炎症过程

4.2 糖脂

糖脂是糖类与脂质通过糖苷键连接形成的复合分子。3 主要分为：

  • 鞘糖脂：以鞘氨醇为脂质骨架，在神经系统中含量丰富
  • 甘油糖脂：以甘油为脂质骨架，主要存在于植物和细菌中

糖脂主要分布在细胞膜的外侧，参与细胞识别、膜稳定性维持等重要功能。

4.3 糖胺聚糖和蛋白聚糖

糖胺聚糖是由氨基糖和糖醛酸交替连接形成的长链多糖，主要包括：

  • 透明质酸：由N-乙酰葡糖胺和葡糖醛酸组成
  • 硫酸软骨素：含有硫酸基团的糖胺聚糖
  • 肝素：具有抗凝血作用的糖胺聚糖

五、糖类的生物学功能

5.1 能量代谢功能

糖类是生物体内最主要的能源物质：

  • 葡萄糖氧化分解释放能量，1mol葡萄糖完全氧化产生38mol ATP
  • 糖原和淀粉作为储能物质，在需要时快速动员
  • 糖类代谢为其他生物分子的合成提供碳骨架

5.2 结构功能

糖类是重要的结构成分：

  • 植物：纤维素构成细胞壁骨架，提供机械支撑
  • 动物：糖胺聚糖构成细胞外基质，维持组织结构
  • 微生物：肽聚糖构成细菌细胞壁，几丁质构成真菌细胞壁

5.3 信息识别功能

糖类作为生物信息分子参与多种识别过程：

  • 细胞识别：糖蛋白和糖脂参与细胞表面识别
  • 免疫识别：抗原决定簇常含有糖类成分
  • 血型识别：ABO血型由红细胞表面的寡糖链决定
  • 病原体识别：病毒、细菌通过识别宿主细胞表面糖类实现感染

六、重要名词解释库

变旋现象 (Mutarotation)

新配制的单糖溶液旋光度随时间发生变化的现象。由于α-异构体和β-异构体在溶液中可以相互转化，最终达到平衡状态。

糖苷键 (Glycosidic Bond)

单糖分子间通过脱水缩合形成的共价键。根据连接方式分为α-糖苷键和β-糖苷键，根据连接位置标记为(1→4)、(1→6)等。

还原糖 (Reducing Sugar)

具有自由异头碳原子，能够还原费林试剂或本尼迪特试剂的糖类。包括所有单糖和大部分双糖（除蔗糖外）。

糖被 (Glycocalyx)

细胞膜外表面由糖蛋白和糖脂的糖链组成的糖类外壳，参与细胞识别、保护和信号转导等功能。

凝集素 (Lectin)

能够特异性结合糖类的蛋白质，广泛存在于植物、动物和微生物中，参与细胞识别和免疫反应。

糖基化 (Glycosylation)

糖基共价结合到蛋白质或脂质上的翻译后修饰过程，是最复杂的蛋白质修饰方式之一。

异头碳 (Anomeric Carbon)

单糖形成环状结构时，原来的羰基碳原子变成的新的手性碳原子，决定α-异构体和β-异构体的构型。

糖原磷酸化酶 (Glycogen Phosphorylase)

催化糖原分解的关键酶，从糖原的非还原端逐个切除葡萄糖-1-磷酸，受到精密的变构调节。

八、深度思考题

思考题1：糖类结构与功能的关系

题目：分析纤维素和淀粉都是由葡萄糖组成的多糖，但在结构和功能上存在显著差异的原因。从分子水平解释这种差异如何影响它们在生物体内的作用。

  1. 糖苷键类型差异：纤维素由β(1→4)糖苷键连接，形成线性分子链，分子间可形成大量氢键，构成致密的纤维结构，具有很高的机械强度。淀粉由α(1→4)糖苷键连接，形成螺旋结构，分子间作用力较弱，易于水解。
  2. 空间构象差异：β-糖苷键使纤维素分子呈伸展的带状构象，便于平行排列形成纤维；α-糖苷键使淀粉分子呈螺旋构象，内部可容纳碘分子，形成蓝色络合物。
  3. 酶特异性差异：动物体内缺乏β-1,4-葡聚糖酶（纤维素酶），无法消化纤维素；但含有α-淀粉酶，能够水解淀粉。这种酶的特异性决定了两者不同的生物学命运。
  4. 功能分化：纤维素的结构特点使其成为理想的结构材料，为植物提供机械支撑；淀粉的结构特点使其成为优良的储能物质，便于快速动员和利用。
  5. 进化意义：这种结构-功能关系体现了生物分子在长期进化过程中的精确适应，同一种单糖通过不同的连接方式实现了功能的多样化。

思考题2：糖蛋白在细胞识别中的作用机制

题目：详细阐述糖蛋白如何参与细胞识别过程，并分析其在免疫反应、血型识别和病原体感染中的具体作用机制。

  1. 分子识别基础：糖蛋白的糖链部分具有高度的结构多样性，可以形成特异性的三维构象，作为分子识别的"密码"。不同细胞表面的糖蛋白糖链结构不同，形成了细胞的"身份标识"。
  2. 凝集素-糖链相互作用：细胞表面或胞外的凝集素能够特异性识别和结合特定的糖链结构。这种相互作用具有高度的特异性和可逆性，是细胞识别的分子基础。
  3. 免疫反应中的作用：T细胞和B细胞表面的糖蛋白参与抗原识别；主要组织相容性复合体(MHC)分子的糖基化影响其抗原呈递功能；补体系统中的甘露糖结合凝集素识别病原体表面的糖类。
  4. 血型识别机制：ABO血型由红细胞表面糖蛋白和糖脂上的寡糖链决定。A型血含有N-乙酰半乳糖胺末端，B型血含有半乳糖末端，O型血缺乏这些糖基。血清中的相应抗体识别异型糖链引起凝集反应。
  5. 病原体感染机制：许多病毒（如流感病毒）通过识别宿主细胞表面的糖蛋白实现附着和侵入；细菌毒素（如霍乱毒素）特异性结合细胞表面的糖脂受体；这种识别的特异性决定了病原体的组织嗜性和致病性。

思考题3：糖类代谢与其他代谢途径的关系

题目：分析糖类代谢如何与脂质代谢、蛋白质代谢相互关联，并讨论糖类在维持机体代谢平衡中的核心作用。

  1. 糖脂代谢关联：葡萄糖通过糖酵解产生的丙酮酸可转化为乙酰辅酶A，进入脂肪酸合成途径；反之，脂肪酸β氧化产生的乙酰辅酶A可通过糖异生途径转化为葡萄糖。胰岛素和胰高血糖素协调调节这两个过程。
  2. 糖蛋白代谢关联：氨基酸可通过脱氨基作用转化为糖类（糖异生）；糖类代谢产生的α-酮酸可作为氨基酸合成的碳骨架；某些氨基酸（如丙氨酸）是重要的糖异生前体物质。
  3. 能量代谢枢纽：糖类是机体最主要和最快速的能源，ATP的产生主要依赖葡萄糖氧化；在饥饿状态下，糖异生维持血糖稳定，保证大脑等器官的能量需求；糖原作为能量储备，在短期内快速动员。
  4. 代谢调节中心：血糖水平是机体代谢状态的重要指标，通过胰岛素、胰高血糖素等激素的调节，协调糖类、脂质、蛋白质的合成与分解；糖类代谢的关键酶（如磷酸果糖激酶）受到多种代谢物的变构调节。
  5. 病理生理意义：糖尿病等代谢性疾病往往涉及糖类代谢异常，进而影响脂质和蛋白质代谢，说明糖类在维持代谢平衡中的核心地位；糖类代谢异常是许多疾病的共同特征和治疗靶点。

思考题4：糖生物学在现代医学中的应用前景

题目：结合糖类的结构特点和生物学功能，分析糖生物学在疾病诊断、药物开发和治疗方法创新中的应用前景和挑战。

  1. 疾病诊断应用：糖蛋白标志物（如糖化血红蛋白HbA1c）用于糖尿病监测；肿瘤相关糖抗原（如CA125、CEA）用于癌症筛查；糖链结构异常可作为遗传性疾病的诊断指标；糖组学分析有望发现新的疾病生物标志物。
  2. 药物开发前景：糖苷酶抑制剂（如阿卡波糖）用于糖尿病治疗；肝素等糖胺聚糖用作抗凝药物；糖类疫苗（如肺炎球菌多糖疫苗）用于感染性疾病预防；糖类药物载体系统提高药物的靶向性和生物利用度。
  3. 治疗方法创新：糖工程技术改造治疗性蛋白质的糖基化模式，提高药效和稳定性；糖类仿生材料用于组织工程和再生医学；基于糖-蛋白相互作用的靶向治疗策略；糖代谢重编程作为肿瘤治疗的新靶点。
  4. 技术挑战：糖类结构的复杂性和微观不均一性给分析检测带来困难；糖类合成的非模板性使得人工合成复杂糖类困难；糖-蛋白相互作用的低亲和力和高特异性要求精确的检测方法；糖组学数据的标准化和数据库建设仍不完善。
  5. 发展趋势：高通量糖组学技术的发展将加速糖类功能的解析；人工智能在糖类结构预测和功能分析中的应用；个性化糖医学基于个体糖组学特征制定治疗方案；糖类纳米医学在精准治疗中的应用前景广阔。
