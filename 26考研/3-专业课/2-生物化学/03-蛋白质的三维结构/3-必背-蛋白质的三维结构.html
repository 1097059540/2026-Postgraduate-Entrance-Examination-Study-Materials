国科大生命科学学院 细胞生物学蛋白质三维结构必背考点

一、核心名词解释（10个）

蛋白质构象（Protein Conformation）：蛋白质分子在三维空间中的特定几何排列，由氨基酸序列决定，包括主链和侧链原子的空间位置关系，是蛋白质发挥生物学功能的结构基础。

α-螺旋（Alpha Helix，α-helix）：蛋白质二级结构的主要类型，多肽链围绕中心轴呈右手螺旋状盘绕，每圈包含3.6个氨基酸残基，螺距0.54nm，通过主链原子间氢键稳定。

β-折叠（Beta Sheet，β-pleated sheet）：蛋白质二级结构类型，多肽链呈伸展状态形成锯齿状结构，多条链通过链间氢键平行或反平行排列形成片状结构。

肽键平面（Peptide Bond Plane）：肽键中C-N键具有部分双键性质，使得肽键周围的六个原子（C、O、N、H、Cα、Cα）基本处于同一平面，限制了肽键的自由旋转。

结构域（Domain）：蛋白质三级结构中相对独立的功能单元，通常由100-200个氨基酸残基组成，具有特定的空间结构和生物学功能，可独立折叠。

超二级结构（Supersecondary Structure）：由相邻的二级结构元素组合形成的具有特定空间排列的结构单元，如β-发夹、β-桶、螺旋-转角-螺旋等。

蛋白质变性（Protein Denaturation）：在物理或化学因子作用下，蛋白质的天然构象被破坏，导致生物活性丧失，但一级结构保持不变的过程。

分子伴侣（Molecular Chaperone）：协助蛋白质正确折叠、防止错误折叠和聚集的蛋白质，如热休克蛋白（HSP），不参与最终产物的组成。

二硫键（Disulfide Bond）：两个半胱氨酸残基的巯基（-SH）通过氧化反应形成的共价键（-S-S-），是稳定蛋白质三级结构的重要作用力。

Ramachandran图（Ramachandran Plot）：以φ角和ψ角为坐标轴绘制的二面角分布图，用于分析蛋白质主链构象的合理性和预测二级结构类型。

二、必备核心知识点（5个）

【必备知识点1】蛋白质四级结构层次及其稳定作用力
1. 一级结构：氨基酸序列，由肽键连接，决定高级结构
2. 二级结构：主链原子局部构象，主要为α-螺旋和β-折叠，由氢键稳定
3. 三级结构：整条多肽链三维结构，由疏水作用、氢键、离子键、二硫键稳定
4. 四级结构：多个亚基组装，由非共价键维持，具有协同效应

【必备知识点2】α-螺旋和β-折叠的结构特征
1. α-螺旋：右手螺旋，3.6个残基/圈，螺距0.54nm，氢键模式为n→n+4
2. β-折叠：伸展构象，锯齿状，分平行和反平行，链间氢键稳定
3. β-转角：4个残基组成，180°回折，第1和第4残基间氢键
4. 影响因素：氨基酸性质（Pro为螺旋断裂者，Gly增加柔性）

【必备知识点3】蛋白质结构测定的主要方法
1. X射线晶体学：原子级分辨率，需要蛋白质结晶，静态结构
2. 核磁共振（NMR）：溶液状态，动态信息，分子量限制<50kDa
3. 冷冻电镜（Cryo-EM）：大分子复合物，近原子分辨率，天然状态
4. 圆二色谱（CD）：二级结构含量分析，蛋白质折叠研究

【必备知识点4】Anfinsen实验与蛋白质折叠原理
1. 实验设计：核糖核酸酶变性-复性实验，证明序列决定结构
2. 核心结论：蛋白质氨基酸序列包含折叠所需的全部信息
3. 折叠机制：热力学控制，自由能最小化，动力学路径重要
4. 辅助因子：分子伴侣、肽基脯氨酸异构酶、蛋白质二硫键异构酶

【必备知识点5】纤维状蛋白质的结构特点
1. α-角蛋白：α-螺旋为主，二硫键交联，不溶于水，如毛发
2. 胶原蛋白：三股螺旋，Gly-X-Y重复序列，羟脯氨酸丰富
3. 丝心蛋白：β-折叠为主，反平行排列，高强度，如蚕丝
4. 共同特点：结构简单、机械强度高、不溶于水、功能专一

三、高频必考大题（2道）

【高频考题1】试述蛋白质三级结构的稳定化作用力及其相对重要性，并分析环境因素对蛋白质结构稳定性的影响。

1. 疏水相互作用：最主要稳定力，疏水氨基酸聚集在蛋白质内部，减少与水接触，熵增驱动，约占总稳定能的50%
2. 氢键：侧链与主链、侧链与侧链间形成，数量多但单个较弱，对结构特异性重要
3. 离子键（盐桥）：带电氨基酸间静电作用，强度中等，对pH敏感
4. 二硫键：共价键，最强但数量少，主要存在于分泌蛋白中
5. 范德华力：原子间弱相互作用，累积效应显著
6. 环境影响：温度升高破坏弱相互作用；pH改变影响离子键；变性剂破坏疏水作用；金属离子影响配位键

【高频考题2】详述Anfinsen(安芬森)实验的设计思路、主要结果及其对蛋白质折叠理论的重要意义，并说明体内蛋白质折叠的辅助机制。

1. 实验设计：以牛胰核糖核酸酶为材料，用8M尿素+β-巯基乙醇使蛋白质完全变性，然后逐步去除变性剂观察复性
2. 主要结果：变性蛋白质在适当条件下能自发复性至天然构象，恢复100%酶活性，证明折叠信息完全包含在氨基酸序列中
3. 理论意义：确立了"序列决定结构"的中心法则，为蛋白质结构预测提供理论基础，获1972年诺贝尔化学奖
4. 体内折叠辅助机制：分子伴侣(如GroEL/GroES)提供折叠腔室；肽基脯氨酸异构酶催化Pro异构化；蛋白质二硫键异构酶协助二硫键正确形成
5. 生物学意义：解释了蛋白质折叠的热力学基础，为理解蛋白质错误折叠疾病提供依据
