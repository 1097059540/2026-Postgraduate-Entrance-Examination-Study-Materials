激素和信号传导

生物化学第14章 · 考研复习笔记

一、激素概述

1. 激素的定义与特点

激素是由内分泌腺或内分泌细胞分泌的高效生物活性物质，在体内作为信使传递信息，对机体的代谢、生长、发育、繁殖、性别、性欲和性活动等起重要的调节作用。

激素的基本特征：
  • 高效性：激素在体内浓度极低（10⁻⁶～10⁻¹²mol/L），但能产生强烈的生物学效应
  • 特异性：每种激素只对特定的靶器官或靶细胞起作用
  • 信息性：激素是细胞间信息传递的化学信使
  • 调节性：激素不提供能量，也不参与代谢过程，仅起调节作用

2. 激素的化学分类

（1）肽类激素

由氨基酸组成的多肽或蛋白质，如胰岛素、胰高血糖素、生长激素等。这类激素水溶性好，不能通过细胞膜，需要膜受体介导。

（2）氨基酸衍生物

由酪氨酸衍生而来，包括甲状腺激素（T₃、T₄）和儿茶酚胺类（肾上腺素、去甲肾上腺素）。

（3）类固醇激素

以胆固醇为前体合成，包括性激素（雌激素、雄激素、孕激素）、肾上腺皮质激素（糖皮质激素、盐皮质激素）等。脂溶性强，可通过细胞膜。

3. 激素的检测方法

  • 放射免疫分析法（RIA）：利用抗原-抗体反应的特异性和放射性同位素标记技术
  • 酶联免疫吸附试验（ELISA）：基于酶标记的免疫分析方法
  • 化学发光免疫分析法（CLIA）：结合化学发光技术的高灵敏度检测方法

二、信号转导机制

1. 信号转导的基本概念

信号转导是指细胞感受、传递和响应外界信息的过程。包括信号识别、信号转换、信号传递和信号终止四个基本步骤。

信号转导的六种基本机制：
  1. G蛋白偶联受体（GPCR）
  2. 受体酪氨酸激酶（RTK）
  3. 受体鸟苷酸环化酶
  4. 门控离子通道
  5. 整合素
  6. 核受体

2. G蛋白偶联受体系统

（1）cAMP-PKA途径

激素与GPCR结合 → 激活Gₛ蛋白 → 激活腺苷酸环化酶 → ATP转化为cAMP → 激活蛋白激酶A（PKA） → 磷酸化靶蛋白

（2）IP₃-DAG途径

激素与GPCR结合 → 激活Gq蛋白 → 激活磷脂酶C → PIP₂分解为IP₃和DAG → IP₃释放Ca²⁺，DAG激活PKC

3. 受体酪氨酸激酶系统

胰岛素、表皮生长因子等与受体结合后，受体发生二聚化和自磷酸化，激活下游信号通路，如PI3K-Akt通路和MAPK通路。

4. 类固醇激素的作用机制

类固醇激素穿过细胞膜 → 与胞质或核内受体结合 → 激素-受体复合物进入细胞核 → 结合到DNA特定序列 → 调节基因转录 → 合成特异性蛋白质

三、重要激素及其功能

1. 胰岛素

由胰岛β细胞分泌的蛋白质激素，是唯一能降低血糖的激素。主要功能包括促进葡萄糖摄取、糖原合成、脂肪合成，抑制糖异生和脂肪分解。

2. 胰高血糖素

由胰岛α细胞分泌，与胰岛素作用相反，主要促进肝糖原分解、糖异生，升高血糖水平。

3. 肾上腺素

由肾上腺髓质分泌的儿茶酚胺类激素，在应激反应中起重要作用。促进糖原分解、脂肪分解，提高心率和血压。

4. 甲状腺激素

包括T₃和T₄，调节基础代谢率、生长发育和神经系统功能。T₃的生物活性比T₄强3-4倍。

5. 生长激素

由垂体前叶分泌，促进蛋白质合成、骨骼生长和细胞分裂，具有抗胰岛素作用。

四、名词解释库

激素（Hormone）：由内分泌腺或内分泌细胞分泌的高效生物活性物质，在体内作为信使传递信息，调节机体的代谢、生长、发育等生理过程。
第二信使（Second Messenger）：细胞内传递信号的小分子物质，如cAMP、cGMP、IP₃、DAG、Ca²⁺等，将细胞外信号转换为细胞内响应。
G蛋白偶联受体（G Protein-Coupled Receptor, GPCR）：具有七次跨膜结构的受体蛋白，通过G蛋白介导信号转导，是最大的受体家族。
蛋白激酶A（Protein Kinase A, PKA）：cAMP依赖性蛋白激酶，由调节亚基和催化亚基组成，在cAMP作用下激活，磷酸化多种靶蛋白。
腺苷酸环化酶（Adenylyl Cyclase, AC）：位于细胞膜的酶，在Gₛ蛋白激活下催化ATP生成cAMP，是cAMP信号通路的关键酶。
磷脂酶C（Phospholipase C, PLC）：催化PIP₂分解为IP₃和DAG的酶，是IP₃-DAG信号通路的关键酶。
受体酪氨酸激酶（Receptor Tyrosine Kinase, RTK）：具有内在酪氨酸激酶活性的受体，如胰岛素受体、表皮生长因子受体等。
类固醇激素（Steroid Hormone）：以胆固醇为前体合成的激素，包括性激素、肾上腺皮质激素等，脂溶性强，可通过细胞膜。
胰岛素抵抗（Insulin Resistance）：机体对胰岛素的敏感性下降，需要更多胰岛素才能维持正常血糖水平的病理状态。
负反馈调节（Negative Feedback）：激素分泌的自我调节机制，当激素浓度过高时抑制其进一步分泌，维持体内平衡。

六、深度思考题

1. 比较分析水溶性激素和脂溶性激素在作用机制上的异同点，并举例说明。

参考答案：
  1. 相同点：都具有高效性、特异性和调节性；都需要与特异性受体结合才能发挥作用；都参与维持机体内环境稳态。
  2. 不同点：
    • 受体位置：水溶性激素受体位于细胞膜表面（如胰岛素受体），脂溶性激素受体位于细胞质或细胞核内（如雌激素受体）
    • 作用机制：水溶性激素通过第二信使系统发挥作用，脂溶性激素直接调节基因转录
    • 作用速度：水溶性激素作用迅速（秒-分钟级），脂溶性激素作用较慢（小时级）
    • 作用持续时间：水溶性激素作用时间短，脂溶性激素作用时间长
  3. 举例：胰岛素（水溶性）通过RTK-PI3K通路快速调节糖代谢；雌二醇（脂溶性）通过调节基因转录长期调节生殖系统发育。

2. 详细阐述cAMP-PKA信号转导通路的分子机制，并分析其在糖代谢调节中的作用。

参考答案：
  1. 信号转导机制：
    • 激素（如胰高血糖素）与GPCR结合，引起受体构象变化
    • 激活的受体与Gₛ蛋白结合，促使GDP被GTP替换
    • Gₛ-GTP复合物激活腺苷酸环化酶
    • 腺苷酸环化酶催化ATP生成cAMP
    • cAMP与PKA调节亚基结合，释放催化亚基
    • 活化的PKA磷酸化多种靶蛋白
  2. 糖代谢调节作用：
    • 磷酸化激活磷酸化酶激酶，进而激活糖原磷酸化酶，促进糖原分解
    • 磷酸化抑制乙酰辅酶A羧化酶，抑制脂肪酸合成
    • 磷酸化激活激素敏感性脂肪酶，促进脂肪分解
    • 磷酸化CREB转录因子，激活糖异生关键酶基因转录
  3. 调节意义：该通路实现了激素信号的放大和精确调节，是机体应对饥饿和应激状态的重要机制。

3. 分析胰岛素信号转导通路的分子机制，并讨论胰岛素抵抗的发生机制。

参考答案：
  1. 胰岛素信号转导机制：
    • 胰岛素与受体α亚基结合，引起受体二聚化
    • 受体β亚基发生自磷酸化，激活酪氨酸激酶活性
    • 磷酸化的受体招募并磷酸化胰岛素受体底物（IRS）
    • IRS激活PI3K，产生PIP₃
    • PIP₃激活PDK1和Akt/PKB
    • Akt磷酸化多种下游靶点，调节糖脂代谢
  2. 主要生物学效应：
    • 促进葡萄糖转运蛋白GLUT4转位到细胞膜
    • 激活糖原合成酶，促进糖原合成
    • 激活乙酰辅酶A羧化酶，促进脂肪酸合成
    • 抑制糖异生关键酶的表达
  3. 胰岛素抵抗机制：
    • 受体水平：受体数量减少或亲和力下降
    • 受体后水平：IRS蛋白丝氨酸磷酸化增加，酪氨酸磷酸化减少
    • 炎症因子：TNF-α等细胞因子干扰胰岛素信号
    • 脂毒性：游离脂肪酸激活PKC，抑制胰岛素信号

4. 论述类固醇激素受体的结构特点和作用机制，并分析其在基因表达调控中的作用。

参考答案：
  1. 受体结构特点：
    • N端转录激活域：负责转录激活功能
    • DNA结合域：含有锌指结构，识别特定DNA序列
    • 铰链区：连接DNA结合域和配体结合域
    • C端配体结合域：与激素结合，调节受体活性
  2. 作用机制：
    • 激素穿过细胞膜，与胞质或核内受体结合
    • 激素-受体复合物发生构象变化，暴露DNA结合域
    • 复合物以同源二聚体形式结合到靶基因的激素反应元件（HRE）
    • 招募转录共激活因子或共抑制因子
    • 调节RNA聚合酶II的活性，影响基因转录
  3. 基因表达调控：
    • 直接调节：结合到靶基因启动子区域的HRE
    • 间接调节：调节转录因子的表达
    • 表观遗传调节：影响组蛋白修饰和DNA甲基化
    • 转录后调节：影响mRNA稳定性和翻译效率
  4. 生物学意义：类固醇激素通过精确调节基因表达，控制细胞分化、代谢、免疫反应等重要生理过程。

5. 分析激素分泌的调节机制，并讨论反馈调节在维持内分泌平衡中的重要性。

参考答案：
  1. 激素分泌调节机制：
    • 神经调节：下丘脑-垂体轴的神经内分泌调节
    • 体液调节：其他激素或代谢产物的调节作用
    • 自分泌调节：细胞分泌的激素作用于自身
    • 旁分泌调节：激素作用于邻近细胞
  2. 反馈调节类型：
    • 负反馈：最常见，如甲状腺激素对TSH的抑制
    • 正反馈：较少见，如分娩时催产素的作用
    • 短环反馈：垂体激素对下丘脑的反馈
    • 长环反馈：靶腺激素对下丘脑-垂体的反馈
  3. 调节的重要性：
    • 维持激素浓度在生理范围内
    • 防止激素过度分泌造成的病理状态
    • 实现激素分泌的精确时空调控
    • 协调多个内分泌腺的功能
  4. 临床意义：反馈调节异常是多种内分泌疾病的发病机制，如糖尿病、甲亢、库欣综合征等。理解反馈调节机制对疾病诊断和治疗具有重要指导意义。
