第19章 氧化磷酸化作用

考研生物化学复习笔记 | 基于《生物化学》第四版

一、氧化还原电势

（一）氧化还原电势的概念

氧化还原电势（E）是指物质对电子的亲和力大小，反映物质得失电子的倾向。

基本概念：

  • 标准氧化还原电势（E⁰）：在pH 7及标准状态下测得的数值，用伏特(V)表示
  • 电子流向：电子总是从氧化还原电势低的一方流向氧化还原电势高的一方
  • 自由能关系：ΔG⁰' = -nFΔE⁰'

重要规律：

  • 标准还原电势值越大，越倾向于得电子（氧化性越强）
  • 标准还原电势值越小（负值绝对值越大），越倾向于失电子（还原性越强）
  • ΔE⁰'为正值的反应，ΔG⁰'为负值，是放能反应

二、电子传递链（呼吸链）

（一）电子传递链的组成

三、氧化磷酸化

（一）化学渗透假说

目前被广泛接受的氧化磷酸化机制，由Mitchell提出：

基本原理：

  1. 质子泵送：电子传递释放的自由能驱动H⁺从线粒体基质泵出到膜间隙
  2. 质子梯度形成：形成跨线粒体内膜的H⁺浓度梯度和膜电势
  3. 质子动力：由pH梯度和膜电势构成，驱动ATP合成
  4. ATP合成：H⁺通过ATP合酶流回基质，驱动ADP + Pi → ATP

（二）ATP合酶结构与功能

1. F₁单元（球形头部）

  • 位置：朝向线粒体基质
  • 组成：α₃β₃γδε五种亚基
  • 功能：ATP合成的催化部位，只有β亚基具有催化活性

2. F₀单元

  • 位置：嵌于线粒体内膜
  • 组成：abc三种亚基
  • 功能：H⁺跨膜通道

3. 结合变构机制

ATP合酶通过构象变化驱动ATP合成：

  • L构象：松散态，ADP和Pi结合
  • T构象：紧密态，催化ATP合成
  • O构象：开放态，ATP释放

（三）P/O比值

电子供体 进入点 质子泵数 ATP产量
NADH 复合体Ⅰ 3个（Ⅰ、Ⅲ、Ⅳ） 2.5个ATP
FADH₂ 复合体Ⅱ 2个（Ⅲ、Ⅳ） 1.5个ATP
细胞质NADH 穿梭途径 2个（Ⅲ、Ⅳ） 1.5个ATP

重要计算：一个葡萄糖分子完全氧化可产生约30个ATP分子。

四、氧化磷酸化的调节

（一）呼吸控制

电子传递与ATP合成在正常情况下是强制性偶联的：

  • 只有当ATP被合成时，电子才能流经电子传递链
  • 只有当细胞需要ATP时，电子传递过程才会有效进行
  • ADP是氧化磷酸化调节的关键物质

（二）解偶联剂

1. 化学解偶联剂

  • 2,4-二硝基苯酚（DNP）：经典解偶联剂
  • FCCP：三氟甲氧基羰基氰苯腙
  • 作用机制：破坏质子梯度，电子传递继续但不合成ATP
  • 结果：能量以热的形式释放

2. 生理性解偶联

  • 产热蛋白（thermogenin）：存在于褐色脂肪组织
  • 非战栗产热：对新生儿和冬眠动物重要

（三）氧化磷酸化抑制剂

抑制剂类型 代表物质 作用机制
ATP合酶抑制剂 寡霉素 结合F₀亚基，阻断H⁺通道
离子载体抑制剂 缬氨霉素、短杆菌肽 增加膜对一价阳离子的透过性

五、NADH的穿梭机制

线粒体内膜不允许NADH直接通过，细胞通过两种穿梭途径间接转运NADH：

（一）3-磷酸甘油穿梭途径

过程：

  1. 细胞质：NADH + 磷酸二羟丙酮 → NAD⁺ + 3-磷酸甘油
  2. 线粒体内膜：3-磷酸甘油 + FAD → 磷酸二羟丙酮 + FADH₂
  3. FADH₂进入电子传递链（复合体Ⅱ）

特点：

  • 产物：FADH₂（产生1.5个ATP）
  • 分布：广泛存在于各种组织
  • 效率：相对较低

（二）苹果酸-天冬氨酸穿梭途径

过程：

  1. 细胞质：NADH + 草酰乙酸 → NAD⁺ + 苹果酸
  2. 苹果酸通过载体进入线粒体基质
  3. 基质内：苹果酸 + NAD⁺ → 草酰乙酸 + NADH
  4. 草酰乙酸转氨生成天冬氨酸，离开线粒体
  5. 细胞质：天冬氨酸转氨生成草酰乙酸

特点：

  • 产物：NADH（产生2.5个ATP）
  • 分布：主要在心脏和肝脏
  • 效率：相对较高

六、重要名词解释

氧化磷酸化(Oxidative Phosphorylation)
指NADH和FADH₂在电子传递链中被氧化，同时ADP磷酸化生成ATP的过程，是细胞获得ATP的主要途径。
电子传递链(Electron Transport Chain)
又称呼吸链，由一系列电子载体按氧化还原电势递增顺序排列，将电子从NADH或FADH₂传递到氧的载体系统。
化学渗透假说(Chemiosmotic Hypothesis)
Mitchell提出的氧化磷酸化机制，认为电子传递产生的质子梯度驱动ATP合成。
质子动力(Proton-motive Force)
由跨膜质子浓度梯度和膜电势共同构成的驱动力，为ATP合成提供能量。
P/O比(P/O Ratio)
磷氧比，指每消耗一个氧原子所合成的ATP分子数，反映氧化磷酸化的效率。
解偶联剂(Uncoupler)
能破坏电子传递与ATP合成偶联的物质，使电子传递继续进行但不合成ATP，能量以热的形式释放。
呼吸控制(Respiratory Control)
电子传递速率受ATP合成速率控制的现象，体现了氧化磷酸化的调节机制。
ATP合酶(ATP Synthase)
由F₁和F₀两部分组成的跨膜蛋白复合体，利用质子梯度驱动ATP合成。
细胞色素(Cytochrome)
含有血红素辅基的电子传递蛋白，通过铁原子价态变化（Fe³⁺ ⇌ Fe²⁺）传递电子。
辅酶Q(Coenzyme Q, CoQ)
又称泛醌，脂溶性电子载体，能在线粒体内膜中自由扩散，连接复合体Ⅰ、Ⅱ和复合体Ⅲ。

七、选择题测试

八、深度思考题

1. 详细阐述化学渗透假说的分子机制，并分析电子传递链中质子泵的工作原理。

答案要点：

  1. 化学渗透假说的核心机制：
    • 电子传递释放的自由能不直接用于ATP合成，而是用于建立跨膜质子梯度
    • 质子梯度包含化学梯度（浓度差）和电化学梯度（电势差）两个组分
    • 质子动力（proton-motive force）= 化学梯度 + 电化学梯度
    • ATP合酶利用质子回流的驱动力催化ADP + Pi → ATP
  2. 质子泵的工作原理：
    • 复合体Ⅰ：NADH氧化释放的能量驱动4个H⁺从基质泵到膜间隙
    • 复合体Ⅲ：通过Q循环机制，每个QH₂氧化泵出4个H⁺
    • 复合体Ⅳ：细胞色素氧化酶每传递4个电子到O₂，泵出4个H⁺
    • 复合体Ⅱ不是质子泵，FADH₂氧化不直接贡献质子梯度
  3. 能量转换的精确性：
    • 电子传递的自由能变化与质子泵送的化学计量关系
    • 每个复合体的构象变化与质子转运的偶联机制
    • 质子梯度的稳定性和ATP合成的效率
  4. 实验证据支持：
    • 人工建立质子梯度可驱动ATP合成
    • 解偶联剂破坏质子梯度阻止ATP合成但不影响电子传递
    • ATP合酶的结构与功能研究证实了旋转催化机制

2. 比较分析NADH和FADH₂在氧化磷酸化中的不同作用，并解释为什么它们产生的ATP数量不同。

答案要点：

  1. 电子传递路径的差异：
    • NADH：复合体Ⅰ → CoQ → 复合体Ⅲ → Cyt c → 复合体Ⅳ → O₂
    • FADH₂：复合体Ⅱ → CoQ → 复合体Ⅲ → Cyt c → 复合体Ⅳ → O₂
    • FADH₂绕过了复合体Ⅰ，直接将电子传递给CoQ
  2. 质子泵数量的差异：
    • NADH氧化：经过3个质子泵（复合体Ⅰ、Ⅲ、Ⅳ）
    • FADH₂氧化：经过2个质子泵（复合体Ⅲ、Ⅳ）
    • 复合体Ⅱ本身不是质子泵，不贡献质子梯度
  3. ATP产量的计算：
    • NADH：约10个H⁺泵出 ÷ 4个H⁺/ATP = 2.5个ATP
    • FADH₂：约6个H⁺泵出 ÷ 4个H⁺/ATP = 1.5个ATP
    • 每个ATP合成需要3个H⁺通过ATP合酶，转运需要1个H⁺
  4. 生物学意义：
    • 不同代谢途径产生不同类型的还原当量
    • 柠檬酸循环主要产生NADH，脂肪酸氧化产生大量FADH₂
    • 细胞可通过调节不同途径的活性来优化ATP产量
    • 琥珀酸脱氢酶使用FAD而非NAD⁺的热力学必然性

3. 分析氧化磷酸化抑制剂和解偶联剂的作用机制差异，并阐述它们在研究和临床中的应用价值。

答案要点：

  1. 抑制剂的作用机制：
    • 电子传递链抑制剂：阻断特定复合体，停止电子流动
    • ATP合酶抑制剂（如寡霉素）：阻断F₀通道，阻止质子回流
    • 离子载体抑制剂：破坏膜的离子选择性，消散电化学梯度
    • 结果：既阻止ATP合成，也阻止氧消耗
  2. 解偶联剂的作用机制：
    • DNP等脂溶性弱酸：在膜两侧穿梭质子，短路质子梯度
    • 产热蛋白：生理性解偶联，允许质子绕过ATP合酶回流
    • 结果：电子传递继续，氧消耗正常，但不合成ATP
    • 能量以热的形式释放
  3. 研究应用价值：
    • 确定电子传递链组分的顺序和功能
    • 验证化学渗透假说的正确性
    • 研究氧化磷酸化的调节机制
    • 开发新的代谢调节策略
  4. 临床应用价值：
    • 中毒机制研究：氰化物、一氧化碳中毒的分子机制
    • 代谢疾病治疗：线粒体疾病的诊断和治疗
    • 减肥药物开发：DNP曾用作减肥药（现已禁用）
    • 体温调节研究：褐色脂肪组织的产热机制
    • 抗菌药物开发：某些抗生素通过影响细菌呼吸链发挥作用

4. 详细分析细胞质NADH的两种穿梭机制，并从能量效率和组织分布角度比较它们的生物学意义。

答案要点：

  1. 3-磷酸甘油穿梭途径：
    • 机制：细胞质NADH还原磷酸二羟丙酮生成3-磷酸甘油
    • 3-磷酸甘油进入线粒体被FAD氧化生成FADH₂
    • FADH₂进入复合体Ⅱ，产生1.5个ATP
    • 特点：不可逆，能量效率较低
  2. 苹果酸-天冬氨酸穿梭途径：
    • 机制：细胞质NADH还原草酰乙酸生成苹果酸
    • 苹果酸通过载体进入线粒体，被NAD⁺氧化生成NADH
    • 线粒体NADH进入复合体Ⅰ，产生2.5个ATP
    • 特点：可逆，能量效率较高，需要载体蛋白
  3. 能量效率比较：
    • 甘油磷酸穿梭：1个细胞质NADH → 1.5个ATP
    • 苹果酸-天冬氨酸穿梭：1个细胞质NADH → 2.5个ATP
    • 效率差异的原因：进入电子传递链的位点不同
    • 对总ATP产量的影响：影响糖酵解等途径的能量收益
  4. 组织分布和生物学意义：
    • 甘油磷酸穿梭：广泛分布，特别是脑、肌肉等高耗能组织
    • 苹果酸-天冬氨酸穿梭：主要在心脏、肝脏等代谢活跃组织
    • 组织特异性反映了不同组织的代谢需求和调节策略
    • 心脏等高耗能组织优先使用高效率的穿梭途径
    • 穿梭途径的选择影响组织的代谢灵活性和适应性
