蛋白质合成、加工和定位

生物化学第33章 - 考研复习笔记

一、遗传密码与蛋白质合成概述

1.1 遗传密码的基本特征

遗传密码是DNA和RNA中核苷酸序列携带遗传信息的密码规律，具有以下重要特征：

  • 三联体密码：每三个连续的核苷酸组成一个密码子，编码一个氨基酸
  • 通用性：除少数例外，遗传密码在所有生物中基本相同
  • 简并性：多个密码子可编码同一氨基酸，减少突变的有害影响
  • 无重叠性：密码子之间不重叠，按顺序读取
  • 无标点性：密码子连续排列，无间隔符号
重点：64个密码子中，61个编码氨基酸，3个为终止密码子（UAG、UAA、UGA）。起始密码子AUG既编码甲硫氨酸，又作为翻译起始信号。

1.2 RNA对遗传密码的解读机制

  1. 选择性转录：根据需要选择性转录特定基因
  2. 信息组合：通过RNA剪接重新组合外显子
  3. 差错消除：RNA编辑纠正转录错误
  4. 语言转换：tRNA将核酸语言转换为蛋白质语言
  5. 信息调节：通过各种调控机制精确控制表达

二、蛋白质合成相关的RNA和装置

2.1 核糖体结构与功能

核糖体是蛋白质合成的场所，由rRNA和蛋白质组成：

  • 原核生物：70S核糖体（30S小亚基 + 50S大亚基）
  • 真核生物：80S核糖体（40S小亚基 + 60S大亚基）

核糖体具有三个重要位点：

  • A位点（氨酰位点）：结合携带氨基酸的氨酰-tRNA
  • P位点（肽酰位点）：结合携带肽链的肽酰-tRNA
  • E位点（出口位点）：释放去酰化的tRNA

2.2 转移RNA（tRNA）的结构与功能

tRNA是蛋白质合成中的关键分子，具有独特的三叶草结构：

  • 反密码子环：与mRNA密码子配对
  • 氨基酸臂：3'端CCA序列结合氨基酸
  • D环和TψC环：参与tRNA的三级结构形成
  • 可变环：不同tRNA的特征性区域
关键概念：氨酰-tRNA合成酶具有高度特异性，确保正确的氨基酸与对应的tRNA结合，这是翻译准确性的重要保障。

2.3 信使RNA（mRNA）的作用

mRNA携带遗传信息，指导蛋白质合成：

  • 5'非翻译区（5'UTR）：含有调控序列
  • 开放阅读框（ORF）：编码蛋白质的序列
  • 3'非翻译区（3'UTR）：含有稳定性和调控元件
  • SD序列：原核生物中的核糖体结合位点

三、蛋白质合成的详细步骤

3.1 氨酰-tRNA的合成

氨酰-tRNA合成酶催化氨基酸与tRNA的结合，分两步进行：

  1. 活化步骤：氨基酸 + ATP → 氨酰-AMP + PPi
  2. 转移步骤：氨酰-AMP + tRNA → 氨酰-tRNA + AMP
校对机制：氨酰-tRNA合成酶具有3'→5'外切酶活性，能够移除错误结合的氨基酸，确保翻译的准确性。

3.2 翻译起始

原核生物翻译起始：

  1. IF-1、IF-2、IF-3结合30S亚基
  2. fMet-tRNA^fMet与IF-2·GTP复合物结合
  3. mRNA结合，SD序列与16S rRNA配对
  4. 50S亚基结合，形成70S起始复合物
  5. GTP水解，起始因子释放

真核生物翻译起始：

  1. eIF4F识别5'帽子结构
  2. 40S亚基扫描至起始密码子AUG
  3. Met-tRNA^iMet结合P位点
  4. 60S亚基结合，形成80S起始复合物

3.3 翻译延伸

延伸过程包括三个主要步骤：

  1. 氨酰-tRNA结合：EF-Tu·GTP携带氨酰-tRNA进入A位点
  2. 肽键形成：肽酰转移酶催化肽键形成
  3. 移位：EF-G·GTP催化核糖体沿mRNA移动一个密码子
白喉毒素作用机制：白喉毒素通过ADP-核糖基化修饰EF-2，阻断蛋白质合成的延伸步骤，导致细胞死亡。

3.4 翻译终止

当核糖体遇到终止密码子时：

  1. 释放因子（RF1、RF2或eRF1）识别终止密码子
  2. 肽酰转移酶活性转变为水解酶活性
  3. 多肽链从tRNA上释放
  4. 核糖体亚基解离，循环利用

四、蛋白质折叠与翻译后加工

4.1 蛋白质折叠机制

新合成的多肽链必须正确折叠才能获得生物活性：

  • 自发折叠：小蛋白质可自发折叠成正确构象
  • 分子伴侣辅助：大蛋白质需要分子伴侣协助折叠

重要的折叠辅助酶：

  • 蛋白质二硫键异构酶（PDI）：催化二硫键的形成和重排
  • 肽基脯氨酰异构酶（PPI）：催化脯氨酸残基的顺反异构

分子伴侣的分类：

  • Hsp70家族：协助蛋白质折叠和防止聚集
  • Hsp60家族（伴侣素）：提供折叠腔室
  • Hsp90家族：协助信号蛋白的成熟

4.2 翻译后修饰

蛋白质合成后可发生多种修饰：

  • 磷酸化：调节蛋白质活性和定位
  • 糖基化：影响蛋白质稳定性和识别
  • 乙酰化：调节基因表达
  • 甲基化：表观遗传调控
  • 泛素化：标记蛋白质降解
  • 脂质化：膜蛋白的锚定

五、蛋白质定位与转运

5.1 信号序列与蛋白质定位

蛋白质的最终定位由特定的信号序列决定：

信号肽的发现：

Blobel和Dobberstein发现信号假说，解释了蛋白质如何被定向到特定的细胞区室。

主要的定位信号：

  • 信号肽：引导蛋白质进入内质网
  • 核定位信号（NLS）：引导蛋白质进入细胞核
  • 线粒体定位信号：引导蛋白质进入线粒体
  • 叶绿体定位信号：引导蛋白质进入叶绿体
  • 过氧化物酶体定位信号：PTS1和PTS2序列

5.2 内质网转运机制

蛋白质进入内质网有两种方式：

共翻译转运：

  1. 信号识别颗粒（SRP）识别信号肽
  2. 翻译暂停，核糖体-SRP复合物形成
  3. SRP受体引导复合物到内质网膜
  4. Sec61转位子形成蛋白质通道
  5. 信号肽酶切除信号肽

翻译后转运：

某些蛋白质在胞质中合成完成后，通过Sec62/63复合物进入内质网。

5.3 其他细胞器的蛋白质转运

线粒体蛋白质转运：

  • TOM复合物：外膜转位酶
  • TIM复合物：内膜转位酶
  • 前导肽：N端正电荷丰富的序列

核蛋白转运：

  • 核孔复合物：核质交换的通道
  • 输入蛋白（Importin）：识别NLS，介导核输入
  • 输出蛋白（Exportin）：介导核输出
  • Ran-GTP梯度：提供转运的方向性
糖基化与蛋白质定位：N-连接糖基化发生在内质网，是蛋白质正确折叠和质量控制的重要标志。O-连接糖基化主要发生在高尔基体。

六、蛋白质合成的调控与抑制

6.1 翻译水平的调控

  • 起始因子磷酸化：eIF2α磷酸化抑制翻译起始
  • microRNA调控：结合mRNA 3'UTR抑制翻译
  • 铁调节蛋白：调控铁代谢相关蛋白的翻译
  • 核糖体开关：mRNA二级结构变化调控翻译

6.2 蛋白质合成抑制剂

抗生素类抑制剂：

  • 链霉素：结合30S亚基，引起密码子误读
  • 氯霉素：抑制肽酰转移酶活性
  • 红霉素：阻断肽链延伸
  • 嘌呤霉素：模拟氨酰-tRNA，导致肽链提前终止

毒蛋白类抑制剂：

  • 白喉毒素：ADP-核糖基化EF-2
  • 蓖麻毒素：去嘌呤化60S亚基rRNA

6.3 翻译忠实性保障机制

  1. 氨酰-tRNA合成酶校对：确保正确的氨基酸-tRNA配对
  2. 核糖体校对：A位点的构象选择性
  3. EF-Tu的GTP水解：提供校对的时间窗口
  4. 无义介导的mRNA降解：清除含有提前终止密码子的mRNA

名词解释库

遗传密码（Genetic Code）
Genetic Code
DNA和RNA中核苷酸序列携带遗传信息的密码规律，由三个连续核苷酸组成一个密码子，编码一个氨基酸或终止信号。
密码子（Codon）
Codon
mRNA上由三个连续核苷酸组成的编码单位，每个密码子对应一个特定的氨基酸或终止信号。
反密码子（Anticodon）
Anticodon
tRNA分子上与mRNA密码子互补配对的三核苷酸序列，决定tRNA携带的氨基酸种类。
氨酰-tRNA合成酶（Aminoacyl-tRNA Synthetase）
Aminoacyl-tRNA Synthetase, AARS
催化特定氨基酸与对应tRNA结合的酶，具有高度特异性，是翻译准确性的关键保障。
核糖体（Ribosome）
Ribosome
由rRNA和蛋白质组成的核糖核蛋白复合物，是蛋白质合成的场所，具有A、P、E三个功能位点。
信号肽（Signal Peptide）
Signal Peptide
位于蛋白质N端的短肽序列，引导新合成的蛋白质定位到内质网，通常含有疏水性氨基酸。
信号识别颗粒（Signal Recognition Particle）
Signal Recognition Particle, SRP
识别信号肽并引导核糖体到内质网膜的核糖核蛋白复合物，由6个蛋白质和一个7S RNA组成。
分子伴侣（Molecular Chaperone）
Molecular Chaperone
协助蛋白质正确折叠、防止错误折叠和聚集的蛋白质，主要包括Hsp70、Hsp60和Hsp90家族。
翻译后修饰（Post-translational Modification）
Post-translational Modification, PTM
蛋白质合成后发生的化学修饰，包括磷酸化、糖基化、乙酰化等，调节蛋白质的功能、定位和稳定性。
核定位信号（Nuclear Localization Signal）
Nuclear Localization Signal, NLS
引导蛋白质进入细胞核的氨基酸序列，通常富含碱性氨基酸如赖氨酸和精氨酸。
肽酰转移酶（Peptidyl Transferase）
Peptidyl Transferase
催化肽键形成的酶活性，位于核糖体大亚基的23S rRNA（原核）或28S rRNA（真核）中。
延伸因子（Elongation Factor）
Elongation Factor, EF
参与蛋白质合成延伸过程的蛋白质因子，包括EF-Tu、EF-G（原核）和eEF1A、eEF2（真核）。
无义介导的mRNA降解（Nonsense-Mediated mRNA Decay）
Nonsense-Mediated mRNA Decay, NMD
细胞质量控制机制，识别并降解含有提前终止密码子的mRNA，防止产生截短的有害蛋白质。
内质网应激反应（ER Stress Response）
ER Stress Response, UPR
当内质网中未折叠蛋白质积累时启动的细胞保护机制，包括减少蛋白质合成、增强折叠能力和清除错误蛋白质。

深度思考题

1. 试述蛋白质合成过程中翻译忠实性的保障机制，并分析这些机制如何确保遗传信息的准确传递。

标准答案：
  1. 氨酰-tRNA合成酶的双重筛选机制：首先通过活性位点的空间互补性选择正确的氨基酸，然后通过编辑位点的3'→5'外切酶活性移除错误结合的氨基酸。这种双重校对机制将氨基酸选择的错误率降低到10^-4以下。
  2. 核糖体的构象选择性校对：核糖体A位点具有构象选择性，只有正确配对的氨酰-tRNA才能诱导核糖体发生构象变化，促进GTP水解和氨酰-tRNA的稳定结合。错误的氨酰-tRNA会被快速释放。
  3. EF-Tu介导的动力学校对：EF-Tu·GTP的水解为校对提供时间窗口。正确的密码子-反密码子配对稳定EF-Tu·氨酰-tRNA·核糖体复合物，促进GTP水解；错误配对则导致复合物不稳定，氨酰-tRNA在GTP水解前解离。
  4. 翻译后的质量控制：无义介导的mRNA降解（NMD）机制识别含有提前终止密码子的mRNA并将其降解，防止产生截短的蛋白质。内质网的未折叠蛋白反应（UPR）监控蛋白质折叠质量。
  5. 统计学意义：这些机制的协同作用使得蛋白质合成的总错误率控制在10^-4到10^-5之间，确保了遗传信息传递的高保真度，维持了细胞的正常生理功能。

2. 比较分析原核生物和真核生物在蛋白质合成起始阶段的异同点，并解释这些差异的生物学意义。

标准答案：
  1. 相同点：都需要起始氨酰-tRNA（原核为fMet-tRNA^fMet，真核为Met-tRNA^iMet）、起始因子的参与、GTP的水解提供能量、核糖体大小亚基的组装等基本步骤。
  2. 主要差异：
    • mRNA识别机制：原核生物通过SD序列与16S rRNA配对直接定位起始密码子；真核生物通过5'帽子结构识别，40S亚基扫描至第一个AUG
    • 起始因子数量：原核生物有3个起始因子（IF1-3），真核生物有12个以上起始因子（eIF1-6）
    • 起始氨基酸：原核生物使用甲酰甲硫氨酸，真核生物使用甲硫氨酸
    • 调控复杂性：真核生物的起始调控更加复杂，涉及多种调控因子
  3. 生物学意义：真核生物更复杂的起始机制反映了其基因表达调控的精细化需求，5'帽子结构和扫描机制确保了翻译的准确起始，多样化的起始因子为不同生理条件下的差异化调控提供了分子基础。
  4. 进化意义：这些差异体现了从原核到真核生物进化过程中基因表达调控系统的复杂化，适应了真核细胞更加复杂的生理需求和环境变化。

3. 详细阐述蛋白质信号序列介导的定位机制，并分析不同细胞器蛋白质转运的特点和调控机制。

标准答案：
  1. 信号序列的基本特征：
    • 内质网信号肽：N端疏水性序列，被信号肽酶切除
    • 线粒体前导肽：N端富含正电荷，形成两亲性α螺旋
    • 核定位信号：富含碱性氨基酸的短序列，通常不被切除
    • 叶绿体转运肽：N端序列，含有羟基氨基酸
  2. 内质网转运机制：SRP识别信号肽→翻译暂停→SRP受体引导→Sec61转位子形成通道→共翻译转运→信号肽酶切除。该机制确保分泌蛋白和膜蛋白的正确定位。
  3. 线粒体转运特点：翻译后转运，需要ATP和膜电位驱动，通过TOM和TIM复合物，前导肽被基质加工肽酶切除。转运过程需要分子伴侣协助维持蛋白质的转运能力构象。
  4. 核转运的独特性：通过核孔复合物进行，依赖Ran-GTP梯度提供方向性，输入蛋白和输出蛋白介导双向转运，NLS和NES序列决定转运方向。
  5. 调控机制：信号序列的可变性、转运机器的调控、分子伴侣的参与、翻译后修饰的影响等多层次调控确保蛋白质的精确定位，维持细胞区室化的完整性。

4. 分析蛋白质翻译后修饰的主要类型及其生物学功能，并举例说明这些修饰如何调控蛋白质的功能和细胞过程。

标准答案：
  1. 磷酸化修饰：主要发生在丝氨酸、苏氨酸和酪氨酸残基上，由蛋白激酶催化，磷酸酶逆转。功能包括酶活性调控、蛋白质相互作用、细胞定位等。例如：糖原磷酸化酶的磷酸化激活糖原分解；p53的磷酸化调控细胞周期检查点。
  2. 糖基化修饰：包括N-连接糖基化（天冬酰胺）和O-连接糖基化（丝氨酸/苏氨酸）。功能包括蛋白质折叠、稳定性、细胞识别、免疫反应等。例如：抗体的糖基化影响其效应功能；病毒表面糖蛋白的糖基化影响宿主识别。
  3. 泛素化修饰：泛素通过E1-E2-E3酶级联反应结合到目标蛋白的赖氨酸残基。功能包括蛋白质降解、细胞定位、信号转导等。例如：多聚泛素链标记蛋白质进入蛋白酶体降解；单泛素化调控膜蛋白的内吞。
  4. 乙酰化修饰：主要发生在组蛋白和转录因子上，由乙酰转移酶催化，去乙酰化酶逆转。功能包括基因表达调控、蛋白质相互作用等。例如：组蛋白乙酰化促进转录激活；p53的乙酰化增强其转录活性。
  5. 甲基化修饰：主要发生在组蛋白的赖氨酸和精氨酸残基，参与表观遗传调控。例如：H3K4me3标记转录起始位点；H3K27me3参与基因沉默。
  6. 生物学意义：这些修饰构成了复杂的调控网络，实现了蛋白质功能的精细调控，是细胞信号转导、基因表达调控、细胞周期控制等重要生物过程的分子基础。
