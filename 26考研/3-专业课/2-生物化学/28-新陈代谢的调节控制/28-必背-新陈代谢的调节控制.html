国科大生命科学学院 细胞生物学【新陈代谢的调节控制】必背考点

一、核心名词解释（10个）

别构调节（Allosteric Regulation）：调节分子与酶的别构位点结合，引起酶构象变化，从而改变酶活性的调节方式，是代谢调节的重要机制。

共价修饰调节（Covalent Modification Regulation）：通过可逆的共价键修饰（如磷酸化/去磷酸化）改变酶活性的调节方式，响应速度快，调节幅度大。

能荷（Energy Charge）：细胞中高能磷酸状态的量化指标，计算公式为([ATP]+1/2[ADP])/([ATP]+[ADP]+[AMP])，正常值为0.8-0.95。

反馈抑制（Feedback Inhibition）：代谢产物对催化其生成的酶或代谢途径起抑制作用，防止产物过量积累的负调节机制。

前馈激活（Feedforward Activation）：代谢底物对下游酶活性的正调节作用，确保代谢途径的协调进行和底物的有效利用。

级联反应（Cascade Reaction）：第一级反应激活的蛋白质催化下一级反应，逐级放大信号的酶促反应系列，如凝血反应和激素信号转导。

cAMP-PKA信号通路（cAMP-PKA Signaling Pathway）：胰高血糖素和肾上腺素激活的经典信号转导途径，通过激活蛋白激酶A调节糖脂代谢。

代谢网络（Metabolic Network）：细胞内各代谢途径通过关键中间代谢物（6-磷酸葡萄糖、丙酮酸、乙酰辅酶A）相互连接形成的整体系统。

同工酶（Isozyme）：催化相同反应但分子结构、理化性质和调节特性不同的酶，适应不同组织和生理条件的代谢需求。

底物循环（Substrate Cycle）：一对催化相反方向、代谢上不可逆反应的酶同时活跃，导致ATP无效消耗的现象，需要严格调节防止。

二、必备核心知识点（5个）

【必备知识点1】酶活性调节的四种主要方式
1. 别构调节：效应物与别构位点结合改变酶构象，分为正效应（激活）和负效应（抑制）
2. 共价修饰调节：磷酸化/去磷酸化最常见，响应快速，可逆性强
3. 酶原激活：蛋白酶切除抑制肽段激活酶活性，如凝血因子激活
4. 酶含量调节：通过调节酶的合成和降解速率改变酶浓度

【必备知识点2】能荷调节机制及其生物学意义
1. 能荷计算：([ATP]+1/2[ADP])/([ATP]+[ADP]+[AMP])，正常值0.8-0.95
2. 高能荷时：抑制分解代谢关键酶，激活合成代谢关键酶
3. 低能荷时：激活分解代谢关键酶，抑制合成代谢关键酶
4. 生物学意义：维持细胞能量稳态，协调产能和耗能过程

【必备知识点3】胰岛素和胰高血糖素的拮抗调节机制
1. 胰岛素作用：促进糖原合成、脂肪合成，抑制糖异生和脂肪分解
2. 胰高血糖素作用：激活cAMP-PKA途径，促进糖原分解和糖异生
3. 分子机制：通过磷酸化/去磷酸化调节关键酶活性
4. 生理意义：维持血糖稳态，适应饥饿和饱食状态

【必备知识点4】糖酵解和糖异生的协同调节
1. 关键调节酶：己糖激酶/葡萄糖激酶、6-磷酸果糖激酶-1、丙酮酸激酶
2. 别构调节：ATP抑制糖酵解关键酶，AMP激活糖酵解关键酶
3. 共价修饰：PKA磷酸化抑制糖酵解，促进糖异生
4. 底物循环防止：酶的细胞内分隔化分布，避免无效循环

【必备知识点5】信号转导途径的级联放大机制
1. cAMP-PKA途径：G蛋白→腺苷酸环化酶→cAMP→PKA激活→底物磷酸化
2. IP3-DAG途径：磷脂酶C→IP3+DAG→Ca2+释放+PKC激活
3. 级联特点：信号逐级放大，快速响应，精确调节
4. 生物学意义：微弱信号产生强烈生物学效应，实现精细调控

三、高频必考大题（2道）

【高频考题1】试述酶活性调节的主要方式，并以糖原磷酸化酶为例说明共价修饰调节的分子机制及生物学意义。

参考答案：
1. 酶活性调节方式：别构调节、共价修饰调节、酶原激活、酶含量调节四种主要方式，其中共价修饰调节响应最快速。
2. 糖原磷酸化酶的调节机制：磷酸化酶激酶催化其磷酸化生成活性a型，蛋白磷酸酶-1催化去磷酸化生成无活性b型。
3. 信号转导过程：胰高血糖素→cAMP-PKA途径→磷酸化酶激酶激活→糖原磷酸化酶磷酸化→糖原分解。
4. 生物学意义：快速响应激素信号，调节糖原分解速率，维持血糖稳态，适应机体能量需求变化。

【高频考题2】分析能荷对细胞代谢调节的作用机制，并说明其在维持细胞能量稳态中的重要意义。

参考答案：
1. 能荷定义及计算：能荷=([ATP]+1/2[ADP])/([ATP]+[ADP]+[AMP])，反映细胞高能磷酸状态，正常值0.8-0.95。
2. 高能荷调节：ATP作为负效应物抑制糖酵解关键酶（6-磷酸果糖激酶-1），激活合成代谢酶，减少ATP产生。
3. 低能荷调节：AMP作为正效应物激活糖酵解关键酶，抑制合成代谢酶，促进ATP生成和能量释放。
4. 生物学意义：实现产能和耗能过程的精确平衡，维持细胞能量稳态，确保生命活动正常进行，体现代谢调节的精密性。
