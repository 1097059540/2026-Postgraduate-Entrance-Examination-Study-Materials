新陈代谢的调节控制

考研生物化学复习笔记 | 基于朱圣庚《生物化学》第四版

一、代谢调节控制概述

1. 细胞代谢的三个方面

  • 物质代谢：糖类、脂类、蛋白质、核酸等生物大分子的合成与分解
  • 能量代谢：ATP的产生、储存和利用，维持细胞的各种生命活动
  • 信息代谢：遗传信息的复制、转录、翻译和调控

2. 代谢调节控制的三个水平

  1. 酶水平调节：通过调节酶的活性和含量来控制代谢流
  2. 细胞水平调节：通过细胞结构和膜系统的调控作用
  3. 整体水平调节：通过神经系统和内分泌系统的协调作用

3. 关键中间代谢物

  • ATP/ADP比值：反映细胞的能荷状态
  • NAD+/NADH比值：反映细胞的氧化还原状态
  • 乙酰辅酶A：连接糖、脂、蛋白质代谢的关键分子
  • 柠檬酸：柠檬酸循环的关键调节点

二、酶水平的代谢调节

1. 酶活性调节

(1) 别构调节

  • 正别构效应：别构激活剂结合，增强酶活性
  • 负别构效应：别构抑制剂结合，降低酶活性
  • 协同效应：多个亚基间的相互作用

(2) 共价修饰调节

  • 磷酸化/去磷酸化：最重要的共价修饰方式
  • 甲基化：主要调节基因表达
  • 乙酰化：调节蛋白质功能和定位
  • 泛素化：标记蛋白质降解

(3) 反馈调节

  • 负反馈抑制：终产物抑制代谢途径的关键酶
  • 正反馈激活：产物激活代谢途径（较少见）
  • 前馈调节：底物激活下游酶活性

2. 酶含量调节

  • 转录水平调节：控制mRNA的合成
  • 转录后调节：mRNA的稳定性和翻译效率
  • 翻译后修饰：蛋白质的成熟和激活
  • 蛋白质降解：通过蛋白酶体系统降解

3. 能荷调节

能荷(Energy Charge) = [ATP + 0.5ADP] / [ATP + ADP + AMP]

  • 能荷高(0.8-0.95)：抑制分解代谢，促进合成代谢
  • 能荷低(0.2-0.5)：促进分解代谢，抑制合成代谢
  • 关键调节酶：磷酸果糖激酶、柠檬酸合酶、异柠檬酸脱氢酶

三、细胞结构对代谢的调控

1. 酶的区域化分布

  • 胞质：糖酵解、戊糖磷酸途径、脂肪酸合成
  • 线粒体基质：柠檬酸循环、脂肪酸β氧化、氨基酸脱氨
  • 线粒体内膜：电子传递链、ATP合酶
  • 内质网：蛋白质合成、脂质合成
  • 过氧化物酶体：脂肪酸氧化、过氧化氢分解

2. 膜的基本功能

  1. 屏障功能：维持细胞内外环境的差异
  2. 载体功能：选择性物质转运
  3. 识别功能：受体介导的信号转导

3. 膜转运对代谢的调节

  • 载体蛋白：调节底物和产物的跨膜转运
  • 通道蛋白：控制离子梯度和膜电位
  • ATP-ADP载体：调节线粒体能量输出
  • 柠檬酸-丙酮酸载体：连接糖代谢和脂代谢

四、整体水平的神经和激素调节

1. 激素调节机制

(1) 胰岛素(Insulin)

  • 促进作用：糖原合成、脂肪合成、蛋白质合成
  • 抑制作用：糖异生、脂肪分解、蛋白质分解
  • 信号转导：受体酪氨酸激酶途径

(2) 胰高血糖素(Glucagon)

  • 促进作用：糖原分解、糖异生、脂肪分解
  • 抑制作用：糖原合成、脂肪合成
  • 信号转导：cAMP-PKA途径

(3) 肾上腺素(Epinephrine)

  • 促进作用：糖原分解、脂肪分解、糖异生
  • 信号转导：cAMP-PKA途径(β受体)，IP3-DAG途径(α受体)

2. 神经调节

  • 交感神经：促进分解代谢，释放肾上腺素
  • 副交感神经：促进合成代谢，刺激胰岛素分泌
  • 下丘脑-垂体轴：调节多种内分泌腺的功能

3. 信号转导途径

(1) cAMP-PKA途径

  • 激素结合G蛋白偶联受体 → 激活腺苷酸环化酶 → cAMP升高 → 激活PKA
  • PKA磷酸化关键酶：ACC(抑制)、HMG-CoA还原酶(抑制)、磷酸化酶激酶(激活)

(2) IP3-DAG途径

  • 激素结合受体 → 激活磷脂酶C → 产生IP3和DAG → Ca2+释放和PKC激活

(3) 受体酪氨酸激酶途径

  • 胰岛素结合受体 → 受体自磷酸化 → 激活下游信号分子 → 调节代谢酶活性

五、各器官代谢的相互协调

1. 肝脏的代谢功能

  • 糖代谢：糖原合成与分解、糖异生
  • 脂代谢：脂肪酸合成、胆固醇合成、酮体生成
  • 蛋白质代谢：血浆蛋白合成、氨基酸脱氨、尿素合成
  • 解毒功能：药物代谢、有毒物质转化

2. 肌肉组织的代谢特点

  • 骨骼肌：糖原储存、乳酸产生、蛋白质合成
  • 心肌：主要利用脂肪酸，需要持续的ATP供应
  • 平滑肌：代谢较慢，主要进行有氧代谢

3. 脂肪组织的代谢调节

  • 脂肪合成：胰岛素促进，胰高血糖素抑制
  • 脂肪分解：肾上腺素、胰高血糖素促进
  • 瘦素分泌：调节食欲和能量平衡

4. 脑组织的代谢需求

  • 葡萄糖依赖：正常情况下主要利用葡萄糖
  • 酮体利用：饥饿状态下可利用酮体
  • 血脑屏障：选择性允许某些物质通过

六、代谢紊乱相关疾病

1. 糖尿病

(1) 1型糖尿病

  • 病因：胰岛β细胞破坏，胰岛素绝对缺乏
  • 特点：发病年龄轻，需要胰岛素治疗

(2) 2型糖尿病

  • 病因：胰岛素抵抗和相对缺乏
  • 特点：多见于成年人，与肥胖相关

2. 肥胖症

  • 能量失衡：摄入大于消耗
  • 瘦素抵抗：瘦素信号传导异常
  • 代谢综合征：伴随胰岛素抵抗、高血压等

3. 饥饿和营养不良

  • 短期饥饿：糖原分解、糖异生增加
  • 长期饥饿：蛋白质分解、酮体生成
  • 严重营养不良：免疫功能下降、器官功能衰竭

名词解释库

能荷 (Energy Charge)
反映细胞能量状态的指标，计算公式为[ATP + 0.5ADP] / [ATP + ADP + AMP]，正常细胞的能荷维持在0.8-0.95之间，是调节代谢途径的重要参数。
别构调节 (Allosteric Regulation)
调节分子结合到酶的别构位点，通过改变酶的构象来调节酶活性的机制。包括正别构效应（激活）和负别构效应（抑制）。
共价修饰调节 (Covalent Modification Regulation)
通过在酶分子上添加或移除化学基团来调节酶活性的机制，最常见的是磷酸化/去磷酸化修饰。
反馈抑制 (Feedback Inhibition)
代谢途径的终产物抑制该途径中关键酶活性的调节机制，防止产物过量积累，维持代谢平衡。
前馈调节 (Feed-forward Regulation)
代谢途径的底物或早期产物激活下游酶活性的调节机制，确保代谢流的顺畅进行。
酶的区域化 (Enzyme Compartmentalization)
不同的酶分布在细胞的不同区域或细胞器中，通过空间分离来调节代谢途径，提高代谢效率。
信号转导 (Signal Transduction)
细胞接收、处理和响应外界信号的过程，包括信号识别、信号传递和细胞响应三个步骤。
血脑屏障 (Blood-Brain Barrier, BBB)
由脑毛细血管内皮细胞形成的选择性屏障，限制血液中某些物质进入脑组织，保护脑组织免受有害物质侵害。
瘦素 (Leptin)
由脂肪细胞分泌的激素，调节食欲和能量平衡，抑制食欲，增加能量消耗，被称为"饱腹激素"。
胰岛素抵抗 (Insulin Resistance)
机体对胰岛素的敏感性降低，需要更多胰岛素才能维持正常血糖水平的病理状态，是2型糖尿病的主要特征。

深度思考题

1. 试述细胞能荷调节在代谢控制中的作用机制，并分析其在不同生理状态下的调节意义。

  1. 能荷的定义和计算：能荷(Energy Charge)是反映细胞能量状态的重要指标，计算公式为[ATP + 0.5ADP] / [ATP + ADP + AMP]。正常细胞的能荷维持在0.8-0.95之间，这个范围确保了细胞既有足够的能量进行各种生命活动，又不会因ATP过量而浪费资源。
  2. 高能荷状态的调节机制：当能荷较高时(>0.8)，细胞会抑制分解代谢途径，如糖酵解和脂肪酸氧化，同时促进合成代谢途径，如糖原合成、脂肪酸合成和蛋白质合成。关键调节酶如磷酸果糖激酶被ATP抑制，而乙酰辅酶A羧化酶被激活。
  3. 低能荷状态的调节机制：当能荷较低时(<0.5)，细胞会激活分解代谢途径以快速产生ATP，同时抑制耗能的合成代谢。AMP激活磷酸果糖激酶和异柠檬酸脱氢酶，促进糖酵解和柠檬酸循环，同时激活AMP激活的蛋白激酶(AMPK)，抑制合成代谢。
  4. 生理意义和适应性：能荷调节使细胞能够根据能量需求自动调整代谢流向，在运动时快速提供能量，在休息时储存能量。这种调节机制具有高度的敏感性和快速响应能力，是细胞代谢稳态维持的基础。

2. 分析胰岛素和胰高血糖素在代谢调节中的拮抗作用，并阐述其信号转导机制的差异。

  1. 胰岛素的代谢调节作用：胰岛素是主要的合成代谢激素，促进糖原合成、脂肪合成和蛋白质合成，同时抑制糖异生、脂肪分解和蛋白质分解。它通过激活关键的合成酶如糖原合酶、乙酰辅酶A羧化酶，并抑制分解酶如激素敏感性脂酶来发挥作用。
  2. 胰高血糖素的代谢调节作用：胰高血糖素是主要的分解代谢激素，促进糖原分解、糖异生和脂肪分解，抑制糖原合成和脂肪合成。它主要在肝脏发挥作用，通过激活糖原磷酸化酶和磷酸烯醇式丙酮酸羧激酶等关键酶来调节代谢。
  3. 信号转导机制的差异：胰岛素通过受体酪氨酸激酶途径发挥作用，激活PI3K-Akt信号通路，直接磷酸化并调节代谢酶的活性。而胰高血糖素通过G蛋白偶联受体激活腺苷酸环化酶，提高cAMP水平，激活蛋白激酶A(PKA)，通过磷酸化调节酶活性。
  4. 拮抗作用的生理意义：这种拮抗调节确保了机体能够根据营养状态和能量需求精确调节代谢。进食后胰岛素占主导，促进营养物质储存；饥饿时胰高血糖素占主导，动员储存的能量。这种精密的调节系统维持了血糖稳态和整体代谢平衡。

3. 论述酶的区域化分布在代谢调节中的重要性，并举例说明其在提高代谢效率方面的作用。

  1. 区域化分布的基本概念：酶的区域化是指不同的代谢酶分布在细胞的特定区域或细胞器中，形成功能性的代谢区室。这种空间组织使得相关的代谢途径能够在特定的微环境中高效进行，避免了不同途径间的相互干扰。
  2. 线粒体的代谢区域化：线粒体基质含有柠檬酸循环的所有酶、脂肪酸β氧化酶系和部分氨基酸代谢酶，而内膜含有电子传递链和ATP合酶。这种分布使得脂肪酸氧化产生的乙酰辅酶A能够直接进入柠檬酸循环，NADH能够高效地被电子传递链氧化产生ATP。
  3. 胞质的代谢区域化：糖酵解酶系、戊糖磷酸途径酶系和脂肪酸合成酶系主要分布在胞质中。这种分布使得葡萄糖能够通过糖酵解快速产生ATP和丙酮酸，戊糖磷酸途径能够提供NADPH用于脂肪酸合成，形成了高效的代谢网络。
  4. 提高代谢效率的机制：区域化分布通过以下方式提高效率：(1)减少底物和产物的扩散距离；(2)维持局部的最适pH和离子浓度；(3)防止相互竞争的途径间的干扰；(4)便于协调调节相关酶的活性。例如，脂肪酸合成需要大量NADPH，而戊糖磷酸途径正好在同一区室提供NADPH，实现了高效的底物供应。

4. 分析糖尿病患者代谢紊乱的分子机制，并探讨现代治疗策略的生化基础。

  1. 1型糖尿病的分子机制：1型糖尿病是由于自身免疫反应破坏胰岛β细胞，导致胰岛素绝对缺乏。缺乏胰岛素使得葡萄糖不能有效进入细胞，血糖升高，同时糖原合成受阻，糖异生增强。脂肪分解增加导致酮体生成过多，可能引起酮症酸中毒。蛋白质分解增加，导致肌肉萎缩和负氮平衡。
  2. 2型糖尿病的分子机制：2型糖尿病主要特征是胰岛素抵抗和相对性胰岛素缺乏。胰岛素受体或其下游信号通路异常，导致胰岛素敏感性降低。初期胰岛β细胞代偿性增加胰岛素分泌，但随着病程进展，β细胞功能逐渐衰竭。肥胖、炎症因子和游离脂肪酸等都参与胰岛素抵抗的发生。
  3. 代谢紊乱的连锁反应：高血糖导致蛋白质糖基化，形成糖化终末产物(AGEs)，损伤血管和神经。糖代谢异常影响脂代谢，导致血脂异常和动脉粥样硬化。肾脏长期高糖负荷导致糖尿病肾病。这些并发症的发生都有明确的生化机制基础。
  4. 现代治疗策略的生化基础：胰岛素治疗直接补充缺乏的激素；二甲双胍通过激活AMPK改善胰岛素敏感性，抑制肝糖输出；磺脲类药物刺激胰岛β细胞分泌胰岛素；α-糖苷酶抑制剂延缓碳水化合物吸收；GLP-1受体激动剂促进胰岛素分泌并抑制胰高血糖素。这些治疗方法都基于对糖尿病分子机制的深入理解。
