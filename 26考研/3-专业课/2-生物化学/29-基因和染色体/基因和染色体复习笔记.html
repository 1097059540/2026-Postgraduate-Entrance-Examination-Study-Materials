第29章 基因和染色体

生物化学复习笔记 | 考研专业课

一、生物大分子与遗传信息

1.1 生物大分子的功能分工

生物功能由生物不同层次结构所决定，而不同层次结构则由生物大分子装配而成。各类生物大分子具有明确的功能分工：

  • 蛋白质：主要的生物功能分子，承担催化、结构、调节等多种功能
  • 核酸（DNA和RNA）：主要的遗传信息携带分子
  • 多糖和脂质化合物：协助蛋白质完成其功能
  • 生物膜：由脂质化合物与蛋白质构成，形成细胞的基本结构
重要概念：一切生命现象均产生于生物分子的相互作用之中。先存结构在子代细胞形成中起指导作用，因此也携带一定的遗传信息。

1.2 表观遗传学

表观遗传学研究不涉及DNA序列变化的、可遗传的、表观改变的规律。这是对传统遗传学的重要补充，研究基因之外的遗传机理。

二、基因概念的演变

2.1 历史发展脉络

  1. 孟德尔的遗传因子假说（1865年）
    • 否定了"融合遗传"，确立了"颗粒遗传"思想
    • 发现遗传因子的显性和隐性之分
    • 发现遗传的分离和自由组合定律
  2. 摩尔根的基因学说和染色体理论（20世纪初）
    • 研究果蝇杂交，建立基因学说
    • 发现性连锁现象
    • 通过重组绘制基因连锁图
  3. 分子生物学时期的基因概念
    • 1945年：Beadle和Tatum提出"一个基因一个酶"假设
    • 1953年：Watson和Crick提出DNA双螺旋结构模型
    • 1958年：Crick提出中心法则（DNA→RNA→蛋白质）
现代基因定义：基因是产生一条多肽链或功能RNA所必需的全部核苷酸序列

三、基因组结构特点

3.1 原核生物基因组

原核生物基因组特点：
  • 基因组较小，大部分为编码序列
  • 基因无内含子
  • 相关基因组成操纵子
  • 重复序列少、较短

病毒基因组

病毒基因组可看作游离的基因，变化较大：

  • 核酸类型：DNA或RNA
  • 链型：单链或双链
  • 形状：线状或环状
  • 特点：有些病毒基因相互重叠

3.2 真核生物基因组

真核生物基因组特点：
  • 基因组较大
  • 基因有内含子
  • 不组成操纵子
  • 重复序列多

人类基因组数据

  • 大小：3.2×10⁹bp
  • 基因数量：约3万个基因
  • 基因占比：占基因组长度25%
  • 编码序列：仅占1.2%

调节序列类型

  • 增强子（Enhancer）
  • 沉默子（Silencer）
  • 绝缘子（Insulator）
  • 启动子（Promoter）
  • 各类调节元件

四、染色体结构与组装

4.1 染色体基本概念

染色体是真核细胞有丝分裂期由基因组DNA致密组装而成的易染色实体。组成染色体的物质称为染色质，具有不同层次结构，使DNA具有不同的压缩比。

4.2 细菌拟核结构

拟核（nucleoid）是原核生物中没有核膜包被的细胞核，含有一个环形DNA分子，与蛋白质组成，含有多个突环（超螺旋结构域），相当于转录单位，DNA呈负超螺旋。

4.3 真核生物染色质

4.3.1 染色质分类

  • 常染色质：结构较疏松，是转录活性区
  • 异染色质：结构致密
    • 组成型异染色质：永久性致密
    • 功能型异染色质：可变性致密

4.3.2 核小体结构

核小体——染色质的基本结构单位：
  • 组成：DNA + 组蛋白八聚体（H2A、H2B、H3、H4）₂
  • DNA缠绕：以左手螺旋（负超螺旋）绕组蛋白八聚体1.65圈
  • DNA长度：核心DNA 146~147 bp，连接DNA 20~60 bp
  • 平均长度：核小体DNA平均为200bp
  • 压缩比：长度压缩7倍

4.3.3 组蛋白特性

  • 富含碱性氨基酸，结构保守
  • 中心区为α螺旋构成的折叠结构域
  • N端：碱性氨基酸较多，伸出核小体外
  • C端：疏水氨基酸较多
  • H1组蛋白：位于连接DNA靠近核小体的入口处

4.3.4 核小体组装过程

  1. （H3-H4）₂四聚体与DNA结合
  2. 加入2个H2A-H2B二聚体
  3. DNA盘绕八聚体1⅔圈
  4. H1结合到连接DNA处

4.4 染色质高级结构

染色质组装层次：
第一级组装：念珠状核小体链（11 nm）
↓
第二级组装：30 nm纤丝（螺线管或锯齿状）
↓
第三级组装：玫瑰花结螺旋（300 nm）
↓
第四级组装：螺旋圈（700 nm）
↓
第五级组装：染色单体

支架蛋白结合的DNA区为支架联结区（SAR），捆绑染色质丝的蛋白质为染色体结构维持蛋白（SMC）和连接蛋白。

五、染色质重塑

5.1 基本概念

染色质重塑是指染色质结构的改变，核小体重塑是染色质重塑的基础。

5.2 组蛋白修饰

5.2.1 组蛋白变异体功能

  • H3.3：转录激活
  • CENP-A：组装着丝粒
  • H2AX：DNA修复和重组
  • H2AZ：基因表达，染色体分离
  • macroH2A：X染色体失活，转录阻遏

5.2.2 组蛋白修饰类型

在细胞周期特定时间可发生：

  • 甲基化
  • 乙酰化
  • 磷酸化
  • ADP核糖基化
修饰意义：所有修饰都降低组蛋白的正电荷，改变染色体结构，影响转录活性，使调控蛋白易于与染色质接触。

5.2.3 组蛋白修饰酶

  • HAT（组蛋白乙酰转移酶）：催化乙酰基加到组蛋白上
  • HDAC（组蛋白脱乙酰酶）：催化组蛋白脱去乙酰基
  • HMT（组蛋白甲基转移酶）：加甲基到组蛋白上
  • HDM（组蛋白脱甲基酶）：脱去组蛋白的甲基

5.3 DNA修饰

5.3.1 DNA甲基化

  • 主要方式：甲基位于CpG岛胞嘧啶的第五位碳上
  • 效果：DNA甲基化后失去转录活性
  • 区别：与组蛋白甲基化不同，DNA甲基化只能失活

5.3.2 生物学意义

  • 基因沉默：DNA甲基化使基因沉默或异染色质化
  • 性别决定和剂量补偿：调节X染色体基因表达
  • 基因组印记：失活基因带有甲基化印记
剂量补偿机制：
  • 雌性哺乳动物：随机关闭一个X染色体，形成巴氏小体
  • 雄性果蝇：X染色体组蛋白乙酰化和磷酸化，转录活性提高一倍
  • 结果：雌雄个体X连锁基因表达产物达到相同水平

六、基因进化

6.1 生物进化的分子基础

生物进化是一个受动力学束缚的开放系统热力学过程。在前生命期的化学进化中：

  1. 无机分子 → 有机分子
  2. 小分子 → 大分子
  3. 随机合成的大分子 → 自我复制的大分子

6.2 "RNA世界"假说

认为最早出现的具有自我复制能力的生物大分子是RNA。遗传密码的出现使"RNA世界"转为"三驾并驱的世界"（DNA、RNA、蛋白质）。

6.3 分子进化理论

  • 中性漂变理论：分子水平变异多数是中性的，主要作用是中性漂变而非自然选择
  • 分子钟：生物大分子进化速度恒定
  • 获得性遗传：基因表达方式受环境影响，可通过生殖细胞传给后代

6.4 新基因产生方式

  1. 基因倍增和歧化：基因复制后功能分化
  2. 基因模块改组：不同基因片段重新组合
  3. RNA逆转录：RNA逆转录成DNA整合到基因组
进化的关键：生物进化不仅是基因数的增加，更重要的是基因表达时空调节的进化。

七、重要名词解释

基因（Gene）
Gene
产生一条多肽链或功能RNA所必需的全部核苷酸序列。现代分子生物学认为，基因是编码多肽链和RNA的一段DNA。
表观遗传学（Epigenetics）
Epigenetics
研究不涉及DNA序列变化的、可遗传的、表观改变的规律。是对传统遗传学的重要补充，研究基因之外的遗传机理。
核小体（Nucleosome）
Nucleosome
染色质的基本结构单位，由DNA和组蛋白八聚体（H2A、H2B、H3、H4）₂组成。DNA以左手螺旋绕组蛋白八聚体1.65圈，长146~147 bp。
染色质重塑（Chromatin Remodeling）
Chromatin Remodeling
染色质结构的改变，基础是核小体重塑。包括组蛋白被变异体置换、组蛋白修饰和DNA甲基化等过程。
操纵子（Operon）
Operon
原核生物中相关基因组成的转录调节单位，包括结构基因、启动子、操作子等元件，可协调调节相关基因的表达。
拟核（Nucleoid）
Nucleoid
原核生物中没有核膜包被的细胞核，含有一个环形DNA分子，与蛋白质组成，含有多个超螺旋结构域。
异染色质（Heterochromatin）
Heterochromatin
结构致密的染色质，分为组成型异染色质（永久性致密）和功能型异染色质（可变性致密）。
常染色质（Euchromatin）
Euchromatin
结构较疏松的染色质，是转录活性区，基因表达活跃。
剂量补偿（Dosage Compensation）
Dosage Compensation
补偿雌雄性细胞X染色体连锁基因数量不同的机制，使雌性与雄性细胞基因表达产物达到相同水平。
巴氏小体（Barr Body）
Barr Body
雌性哺乳动物胚胎发育早期随机关闭的X染色体凝缩成的致密结构，实现剂量补偿。
基因组印记（Genomic Imprinting）
Genomic Imprinting
失活基因带有的印记，其DNA被甲基化，组蛋白被修饰，显示表观遗传学特征。
中心法则（Central Dogma）
Central Dogma
由Crick提出的遗传信息流动规律：DNA→RNA→蛋白质，概括了遗传信息的传递和表达过程。

九、深度思考题

1. 试述染色质重塑在基因表达调节中的作用机制，并分析组蛋白修饰与DNA甲基化的异同点。

参考答案：
  1. 染色质重塑的作用机制：
    • 通过改变核小体结构，调节DNA的可接触性
    • 依赖ATP的染色质重塑复合物水解ATP提供能量
    • 解开组蛋白与DNA间的作用力，使转录因子能够结合
    • 招致特殊功能蛋白质结合，完成各类染色质功能
  2. 组蛋白修饰的特点：
    • 可逆性：既可以激活也可以抑制基因表达
    • 多样性：包括甲基化、乙酰化、磷酸化等多种修饰
    • 位点特异性：不同位点的修饰产生不同效果
    • 动态性：可根据细胞需要快速改变
  3. DNA甲基化的特点：
    • 单向性：主要起抑制作用，导致基因沉默
    • 稳定性：具有长期效应，可维持较长时间
    • 遗传性：可通过细胞分裂传递给子代细胞
    • 位点特异性：主要发生在CpG岛的胞嘧啶上
  4. 两者的协同作用：
    • 共同参与基因表达的精细调节
    • 在发育、分化和疾病过程中发挥重要作用
    • 形成表观遗传学调控网络

2. 比较原核生物与真核生物基因组的结构特点，并分析这些差异对基因表达调节的影响。

参考答案：
  1. 基因组大小差异：
    • 原核生物基因组较小，编码密度高，大部分序列为编码区
    • 真核生物基因组较大，含有大量非编码序列和调节元件
    • 影响：真核生物具有更复杂的调节机制和更精细的表达控制
  2. 基因结构差异：
    • 原核生物基因无内含子，转录产物直接翻译
    • 真核生物基因有内含子，需要剪接加工
    • 影响：真核生物可通过选择性剪接产生多种蛋白质异构体
  3. 基因组织方式差异：
    • 原核生物相关基因组成操纵子，协调表达
    • 真核生物基因分散分布，独立调节
    • 影响：原核生物适合快速响应环境变化，真核生物适合复杂的发育调节
  4. 重复序列差异：
    • 原核生物重复序列少且短
    • 真核生物含有大量重复序列，包括转座子、串联重复等
    • 影响：真核生物具有更多的调节元件和进化潜力

3. 阐述核小体结构与功能的关系，并分析染色质高级结构形成的生物学意义。

参考答案：
  1. 核小体的结构特点：
    • DNA以负超螺旋形式缠绕组蛋白八聚体1.65圈
    • 组蛋白富含碱性氨基酸，与DNA的负电荷形成稳定结合
    • H1组蛋白位于连接DNA处，稳定核小体间连接
    • 核小体间距约200bp，形成念珠状结构
  2. 核小体的功能作用：
    • DNA包装：将长链DNA压缩约7倍，节省核内空间
    • 基因调节：通过核小体定位调节基因的可接触性
    • DNA保护：保护DNA免受核酸酶降解
    • 表观调节：通过组蛋白修饰调节基因表达
  3. 染色质高级结构的形成：
    • 第一级：念珠状核小体链（11nm）
    • 第二级：30nm纤丝（螺线管或锯齿状）
    • 第三级：300nm玫瑰花结螺旋
    • 第四级：700nm螺旋圈
    • 最终形成染色单体，压缩比达10,000倍
  4. 生物学意义：
    • 空间节约：使长达2米的DNA装入直径约10μm的细胞核
    • 有序分离：确保染色体在细胞分裂时准确分离
    • 基因调节：不同压缩程度对应不同的转录活性
    • 遗传稳定：保护遗传物质免受损伤

4. 分析"RNA世界"假说的科学依据，并讨论其对理解生命起源和进化的意义。

参考答案：
  1. "RNA世界"假说的科学依据：
    • RNA的双重功能：既能储存遗传信息，又具有催化活性（核酶）
    • RNA的自我复制能力：某些RNA分子能够催化自身的复制
    • RNA在现代生物中的关键作用：rRNA、tRNA、mRNA等在蛋白质合成中不可缺少
    • RNA的化学稳定性：在原始地球条件下比DNA更容易形成
  2. 从RNA世界到现代生物系统：
    • RNA首先出现，具有遗传和催化双重功能
    • DNA的出现提供了更稳定的遗传信息储存
    • 蛋白质的出现提供了更高效的催化功能
    • 形成现代的"三驾并驱"系统：DNA-RNA-蛋白质
  3. 对生命起源的意义：
    • 解决了"鸡蛋问题"：RNA既是遗传物质又是催化剂
    • 提供了从无机到有机、从简单到复杂的进化路径
    • 解释了现代生物系统中RNA的核心地位
    • 为实验室合成生命提供了理论指导
  4. 对进化理论的贡献：
    • 支持了分子进化的连续性观点
    • 解释了遗传密码的起源和进化
    • 为理解早期生命的代谢网络提供了框架
    • 促进了对表观遗传学机制的理解

5. 论述表观遗传学在现代生物学中的重要性，并分析其在疾病发生和治疗中的应用前景。

参考答案：
  1. 表观遗传学的重要性：
    • 补充了传统遗传学：解释了基因型相同但表型不同的现象
    • 环境与基因的桥梁：阐明了环境因素如何影响基因表达
    • 发育调控机制：解释了细胞分化和器官发育的分子基础
    • 遗传多样性来源：提供了除DNA序列变异外的另一种遗传变异形式
  2. 在疾病发生中的作用：
    • 肿瘤发生：DNA甲基化异常导致抑癌基因沉默
    • 神经系统疾病：组蛋白修饰异常影响神经发育和功能
    • 代谢性疾病：表观遗传修饰影响代谢基因的表达
    • 免疫系统疾病：表观调节影响免疫细胞的分化和功能
  3. 在疾病治疗中的应用：
    • 表观遗传药物：HDAC抑制剂、DNA甲基转移酶抑制剂等
    • 精准医疗：基于表观遗传标记的个体化治疗
    • 基因治疗：通过调节表观修饰恢复基因正常表达
    • 预防医学：通过生活方式干预调节表观遗传状态
  4. 未来发展前景：
    • 诊断标志物：开发基于表观遗传的疾病诊断方法
    • 药物开发：设计更特异性的表观遗传调节剂
    • 再生医学：利用表观调节促进组织再生
    • 衰老研究：通过表观遗传干预延缓衰老过程
