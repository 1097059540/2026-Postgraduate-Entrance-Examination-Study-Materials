氨基酸生物合成和生物固氮

考研生物化学复习笔记 | 基于权威教材整理

一、生物固氮

1.1 生物固氮的意义

生物固氮是指某些微生物将大气中的分子氮（N₂）还原为氨（NH₃）的过程，这是自然界中氮循环的重要环节。生物固氮为生物圈提供了可利用的氮源，是维持生态系统氮平衡的关键过程。

重要概念：生物固氮是唯一能够将大气中惰性的N₂转化为生物可利用氮化合物的生物学过程。

1.2 固氮酶系统

1.2.1 固氮酶的组成

  • 固氮酶蛋白（Component I）：含有钼-铁辅基，分子量约240 kDa，是催化N₂还原的主要酶
  • 固氮酶还原酶（Component II）：含有铁-硫簇，分子量约60 kDa，负责电子传递

1.2.2 固氮反应机理

固氮反应的总反应式为：

N₂ + 8H⁺ + 8e⁻ + 16ATP → 2NH₃ + H₂ + 16ADP + 16Pi

  1. ATP结合到固氮酶还原酶上，引起构象变化
  2. 电子从固氮酶还原酶传递到固氮酶蛋白
  3. N₂在钼-铁辅基上被逐步还原
  4. 产生NH₃和H₂作为副产物

1.3 固氮的调节

  • 氧气敏感性：固氮酶对氧气极其敏感，需要厌氧或微氧环境
  • 氨的反馈抑制：产物NH₃对固氮酶活性有反馈抑制作用
  • 能量需求：固氮过程消耗大量ATP，受细胞能量状态调节

二、氨基酸生物合成

2.1 氨基酸分类

2.1.1 营养学分类

  • 必需氨基酸（Essential amino acids）：人体不能合成或合成量不足，必须从食物中获得
    • 组氨酸（His）、异亮氨酸（Ile）、亮氨酸（Leu）、赖氨酸（Lys）
    • 甲硫氨酸（Met）、苯丙氨酸（Phe）、苏氨酸（Thr）
    • 色氨酸（Trp）、缬氨酸（Val）
  • 非必需氨基酸（Non-essential amino acids）：人体可以合成
    • 丙氨酸（Ala）、天冬氨酸（Asp）、天冬酰胺（Asn）
    • 谷氨酸（Glu）、谷氨酰胺（Gln）、甘氨酸（Gly）
    • 脯氨酸（Pro）、丝氨酸（Ser）
  • 半必需氨基酸：特殊生理状态下需求量增加
    • 精氨酸（Arg）、半胱氨酸（Cys）、酪氨酸（Tyr）

2.2 氨基酸合成的前体物质

氨基酸的合成以糖酵解、柠檬酸循环和磷酸戊糖途径的中间产物为碳骨架前体：

2.2.1 来自糖酵解途径

  • 3-磷酸甘油酸 → 丝氨酸 → 甘氨酸、半胱氨酸
  • 磷酸烯醇式丙酮酸 → 芳香族氨基酸（苯丙氨酸、酪氨酸、色氨酸）
  • 丙酮酸 → 丙氨酸、缬氨酸、亮氨酸、异亮氨酸

2.2.2 来自柠檬酸循环

  • α-酮戊二酸 → 谷氨酸 → 谷氨酰胺、脯氨酸、精氨酸
  • 草酰乙酸 → 天冬氨酸 → 天冬酰胺、苏氨酸、甲硫氨酸、赖氨酸

2.3 重要氨基酸的合成途径

2.3.1 谷氨酸和谷氨酰胺的合成

谷氨酸脱氢酶反应：

α-酮戊二酸 + NH₃ + NADPH + H⁺ → 谷氨酸 + NADP⁺ + H₂O

谷氨酰胺合成酶反应：

谷氨酸 + NH₃ + ATP → 谷氨酰胺 + ADP + Pi

2.3.2 芳香族氨基酸的合成

苯丙氨酸、酪氨酸和色氨酸的合成都经过莽草酸途径：

  1. 磷酸烯醇式丙酮酸 + 赤藓糖-4-磷酸 → 莽草酸
  2. 莽草酸 → 分支酸
  3. 分支酸 → 苯丙氨酸、酪氨酸或色氨酸

2.3.3 含硫氨基酸的合成

甲硫氨酸循环：

  • 甲硫氨酸 → S-腺苷甲硫氨酸（SAM）
  • SAM → S-腺苷同型半胱氨酸
  • 同型半胱氨酸 → 甲硫氨酸（需要维生素B₁₂和叶酸）

2.4 氨基酸合成的调节

2.4.1 反馈抑制

  • 终产物对合成途径中关键酶的抑制
  • 例：苯丙氨酸抑制3-脱氧-D-阿拉伯庚酮糖酸-7-磷酸合成酶

2.4.2 酶的诱导和阻遏

  • 氨基酸缺乏时诱导合成酶的产生
  • 氨基酸充足时阻遏合成酶的产生

三、一碳单位代谢

3.1 一碳单位的概念

一碳单位是指含有一个碳原子的基团，包括甲基（-CH₃）、甲酰基（-CHO）、亚甲基（-CH₂-）、甲炔基（-CH≡）等。这些基团在氨基酸、核苷酸和其他生物分子的合成中起重要作用。

3.2 叶酸辅酶系统

3.2.1 四氢叶酸（THF）

四氢叶酸是一碳单位的主要载体，由叶酸经二氢叶酸还原酶还原而成。

3.2.2 一碳单位的载体形式

  • N⁵-甲基-THF：甲基供体
  • N¹⁰-甲酰基-THF：甲酰基供体
  • N⁵,N¹⁰-亚甲基-THF：亚甲基供体
  • N⁵,N¹⁰-甲炔基-THF：甲炔基供体

3.3 一碳单位在氨基酸合成中的作用

  • 甘氨酸合成：丝氨酸 + THF → 甘氨酸 + N⁵,N¹⁰-亚甲基-THF
  • 甲硫氨酸合成：同型半胱氨酸 + N⁵-甲基-THF → 甲硫氨酸 + THF
  • 组氨酸合成：需要N¹⁰-甲酰基-THF参与

四、氨基酸合成异常与疾病

4.1 苯丙酮尿症（PKU）

病因：苯丙氨酸羟化酶缺陷，导致苯丙氨酸不能转化为酪氨酸

后果：苯丙氨酸及其代谢产物（苯丙酮酸、苯乳酸）在体内蓄积

症状：智力发育迟缓、皮肤和毛发色素减少、特殊体味

治疗：低苯丙氨酸饮食，补充酪氨酸

4.2 枫糖尿症

病因：支链氨基酸（缬氨酸、亮氨酸、异亮氨酸）脱氢酶复合体缺陷

症状：尿液有枫糖味、神经系统损害

4.3 同型半胱氨酸尿症

病因：胱硫醚β-合成酶缺陷或甲硫氨酸合成酶缺陷

后果：同型半胱氨酸蓄积，导致血管病变和智力障碍

五、名词解释库

固氮酶 (Nitrogenase)
催化大气中分子氮还原为氨的酶复合体，由固氮酶蛋白和固氮酶还原酶组成，含有钼-铁和铁-硫辅基。
必需氨基酸 (Essential Amino Acids)
人体不能合成或合成量不足以满足生理需要，必须从食物中获得的氨基酸，共9种。
非必需氨基酸 (Non-essential Amino Acids)
人体能够合成，不需要从食物中获得的氨基酸，但在某些生理状态下需求量可能增加。
谷氨酸脱氢酶 (Glutamate Dehydrogenase, GDH)
催化α-酮戊二酸与氨结合生成谷氨酸的关键酶，是氨基酸合成中氮素固定的重要途径。
谷氨酰胺合成酶 (Glutamine Synthetase, GS)
催化谷氨酸与氨结合生成谷氨酰胺的酶，是细胞内氨解毒的主要途径。
莽草酸途径 (Shikimate Pathway)
芳香族氨基酸（苯丙氨酸、酪氨酸、色氨酸）的共同合成途径，以莽草酸为关键中间产物。
四氢叶酸 (Tetrahydrofolate, THF)
叶酸的活性形式，是一碳单位代谢的主要载体，参与氨基酸和核苷酸的合成。
一碳单位 (One-Carbon Units)
含有一个碳原子的基团，如甲基、甲酰基、亚甲基等，在生物合成中起重要作用。
S-腺苷甲硫氨酸 (S-Adenosylmethionine, SAM)
甲硫氨酸的活化形式，是细胞内主要的甲基供体，参与多种甲基化反应。
苯丙酮尿症 (Phenylketonuria, PKU)
由苯丙氨酸羟化酶缺陷引起的遗传性疾病，导致苯丙氨酸代谢异常和智力发育障碍。
转氨酶 (Transaminase, Aminotransferase)
催化氨基酸与α-酮酸之间氨基转移的酶类，是氨基酸合成和分解的关键酶。
PRPP (5-Phosphoribosyl-1-pyrophosphate)
5-磷酸核糖-1-焦磷酸，是嘌呤、嘧啶核苷酸和组氨酸、色氨酸合成的重要前体。
分支酸 (Chorismate)
莽草酸途径的关键中间产物，是苯丙氨酸、酪氨酸、色氨酸合成的共同前体。
同型半胱氨酸 (Homocysteine)
甲硫氨酸代谢的中间产物，可重新甲基化生成甲硫氨酸或转硫生成半胱氨酸。
支链氨基酸 (Branched-Chain Amino Acids, BCAA)
包括缬氨酸、亮氨酸、异亮氨酸，具有支链结构的必需氨基酸。
反馈抑制 (Feedback Inhibition)
代谢途径的终产物对该途径中关键酶活性的抑制作用，是代谢调节的重要机制。
酶的诱导 (Enzyme Induction)
在特定条件下，细胞增加某种酶的合成量，以适应代谢需要的调节机制。
酶的阻遏 (Enzyme Repression)
在特定条件下，细胞减少某种酶的合成量，以节约能量和资源的调节机制。
氮平衡 (Nitrogen Balance)
机体氮的摄入量与排出量之间的平衡状态，反映蛋白质代谢的整体状况。
氨基酸库 (Amino Acid Pool)
机体内游离氨基酸的总和，是蛋白质合成和分解的动态平衡体系。

七、深度思考题

1. 试述生物固氮的生物学意义及其在生态系统中的作用，并分析固氮酶系统的结构特点和催化机理。

  1. 生物学意义：生物固氮是自然界中唯一能将大气中惰性的N₂转化为生物可利用氮化合物的过程，为生物圈提供了约80%的固定氮源。这一过程打破了氮的化学惰性，使氮元素能够进入生物循环，是维持生态系统氮平衡和生物多样性的基础。
  2. 生态系统作用：固氮微生物通过固氮作用为植物提供氮源，支持初级生产力；在农业生态系统中，豆科植物与根瘤菌的共生固氮可减少化肥使用，维护土壤健康；在海洋生态系统中，蓝藻等固氮生物是海洋食物链的重要基础。
  3. 固氮酶结构特点：固氮酶系统由两个组分组成：Component I（固氮酶蛋白）含有钼-铁辅基（FeMo-co），是催化活性中心；Component II（固氮酶还原酶）含有铁-硫簇，负责电子传递。两个组分必须结合才能发挥催化活性。
  4. 催化机理：固氮过程需要大量ATP提供能量，每固定一分子N₂需要消耗16个ATP。反应过程中，电子从还原酶传递到固氮酶蛋白，N₂在钼-铁辅基上被逐步还原，同时不可避免地产生H₂作为副产物。整个过程对氧气极其敏感，需要严格的厌氧环境。
  5. 调节机制：固氮过程受到多重调节，包括氧气的抑制、氨的反馈抑制、能量状态的调节等，确保固氮活性与环境条件和细胞需求相适应。

2. 分析必需氨基酸和非必需氨基酸的营养学意义，并阐述氨基酸合成途径与中心代谢途径的关系。

  1. 必需氨基酸的营养学意义：必需氨基酸（9种）是人体无法合成或合成量不足的氨基酸，必须通过饮食获得。它们在蛋白质合成中起限制性作用，任何一种必需氨基酸的缺乏都会影响蛋白质的合成效率。在营养评价中，必需氨基酸的组成和比例决定了蛋白质的生物学价值。
  2. 非必需氨基酸的意义：非必需氨基酸虽然可以体内合成，但在特殊生理状态（如生长发育、妊娠、疾病恢复期）下，需求量可能超过合成能力，此时也需要从饮食中补充。它们参与多种生理功能，如神经递质合成、免疫功能调节等。
  3. 与糖酵解途径的关系：多种氨基酸以糖酵解中间产物为前体：3-磷酸甘油酸是丝氨酸、甘氨酸、半胱氨酸的前体；磷酸烯醇式丙酮酸参与芳香族氨基酸合成；丙酮酸是丙氨酸和支链氨基酸的前体。
  4. 与柠檬酸循环的关系：α-酮戊二酸是谷氨酸家族氨基酸的前体，通过谷氨酸脱氢酶反应形成谷氨酸，进而合成谷氨酰胺、脯氨酸、精氨酸；草酰乙酸是天冬氨酸家族氨基酸的前体，包括天冬氨酸、天冬酰胺、苏氨酸、甲硫氨酸、赖氨酸。
  5. 代谢整合意义：氨基酸合成与中心代谢途径的紧密联系体现了细胞代谢的整体性和协调性，使得氨基酸合成能够根据细胞的能量状态和代谢需求进行调节，实现代谢的经济性和高效性。

3. 详述一碳单位代谢的生物学意义，分析叶酸辅酶系统在氨基酸合成中的作用机制。

  1. 一碳单位代谢的生物学意义：一碳单位代谢是细胞内重要的代谢网络，涉及氨基酸、核苷酸、维生素等多种生物分子的合成。它连接了蛋白质代谢、核酸代谢和脂质代谢，是细胞生长、分裂和分化的基础。一碳单位的转移反应是表观遗传修饰（如DNA甲基化）的重要基础。
  2. 叶酸辅酶系统的结构：四氢叶酸（THF）是叶酸的活性形式，含有蝶啶环、对氨基苯甲酸和谷氨酸残基。THF分子上的N⁵和N¹⁰位点可以结合不同的一碳单位，形成多种载体形式：N⁵-甲基-THF、N¹⁰-甲酰基-THF、N⁵,N¹⁰-亚甲基-THF等。
  3. 在氨基酸合成中的作用：叶酸辅酶参与多种氨基酸的合成：在甘氨酸合成中，丝氨酸羟甲基转移酶催化丝氨酸分解为甘氨酸，同时形成N⁵,N¹⁰-亚甲基-THF；在甲硫氨酸合成中，N⁵-甲基-THF提供甲基，使同型半胱氨酸甲基化生成甲硫氨酸；在组氨酸合成中，N¹⁰-甲酰基-THF参与嘌呤环的形成。
  4. 调节机制：一碳单位代谢受到多种因素调节，包括叶酸营养状态、维生素B₁₂水平、甲硫氨酸循环的反馈调节等。叶酸缺乏会导致一碳单位代谢障碍，影响DNA合成和甲基化，导致巨幼红细胞性贫血和神经管缺陷。
  5. 临床意义：叶酸补充在预防神经管缺陷、治疗巨幼红细胞性贫血中具有重要作用。抗叶酸药物（如甲氨蝶呤）通过抑制二氢叶酸还原酶，阻断THF的生成，用于癌症化疗。

4. 分析苯丙酮尿症的发病机理、临床表现和治疗原理，并讨论其对氨基酸代谢调节的启示。

  1. 发病机理：苯丙酮尿症是由苯丙氨酸羟化酶基因突变引起的常染色体隐性遗传病。苯丙氨酸羟化酶缺陷导致苯丙氨酸无法正常转化为酪氨酸，使苯丙氨酸在体内大量蓄积。蓄积的苯丙氨酸通过替代代谢途径转化为苯丙酮酸、苯乳酸、苯乙酸等异常代谢产物。
  2. 临床表现：患儿出生时正常，但随着苯丙氨酸及其代谢产物的蓄积，出现智力发育迟缓、行为异常、癫痫发作等神经系统症状；由于酪氨酸合成受阻，影响黑色素生成，患儿皮肤和毛发色素减少，呈现白化表现；异常代谢产物的排出使尿液和汗液具有特殊的鼠尿味。
  3. 治疗原理：采用低苯丙氨酸饮食疗法，严格控制苯丙氨酸的摄入量，维持血中苯丙氨酸浓度在安全范围内（2-6 mg/dL）。同时补充酪氨酸，因为酪氨酸在PKU患者中成为必需氨基酸。治疗需要终生坚持，特别是在大脑发育的关键期（0-12岁）。
  4. 对代谢调节的启示：PKU揭示了氨基酸代谢途径的相互依赖性，一个酶的缺陷可以影响整个代谢网络；说明了代谢途径的分支和替代途径的重要性，当主要途径受阻时，替代途径可能产生有害产物；体现了早期诊断和干预的重要性，新生儿筛查可以及时发现并治疗PKU。
  5. 分子医学意义：PKU是基因治疗和酶替代治疗的重要研究模型，为理解遗传性代谢病的发病机制和治疗策略提供了重要参考。同时也说明了营养基因组学的重要性，即基因型决定了个体对特定营养素的需求。

5. 论述氨基酸合成的调节机制，分析反馈抑制和酶诱导/阻遏在代谢调节中的作用及其生物学意义。

  1. 反馈抑制机制：反馈抑制是氨基酸合成调节的重要机制，终产物氨基酸与合成途径中关键酶（通常是第一个酶）的变构位点结合，抑制酶活性。例如，苯丙氨酸抑制3-脱氧-D-阿拉伯庚酮糖酸-7-磷酸合成酶，组氨酸抑制ATP磷酸核糖基转移酶。这种调节是快速、可逆的，能够及时响应细胞内氨基酸浓度的变化。
  2. 酶诱导机制：当细胞内某种氨基酸缺乏时，相应的合成酶基因表达增加，酶蛋白合成增多，提高合成能力。这种调节涉及转录水平的控制，通过调节蛋白（如阻遏蛋白、激活蛋白）与操纵子的相互作用实现。诱导机制响应较慢但持续时间长，适合应对长期的营养缺乏。
  3. 酶阻遏机制：当氨基酸充足时，合成酶的基因表达受到抑制，减少不必要的酶合成，节约细胞资源。阻遏机制通过阻遏蛋白与氨基酸结合后与操纵子结合，阻止RNA聚合酶转录实现。这种机制确保细胞在营养充足时不会过度合成代谢酶。
  4. 多重调节的整合：氨基酸合成途径通常受到多种调节机制的共同控制，包括变构调节、共价修饰调节、转录调节等。这种多层次的调节网络确保了代谢的精确控制和快速响应。例如，谷氨酰胺合成酶同时受到反馈抑制、共价修饰和转录调节的控制。
  5. 生物学意义：精确的代谢调节机制具有重要的生物学意义：保证代谢的经济性，避免不必要的能量和资源浪费；维持细胞内氨基酸的动态平衡，满足蛋白质合成和其他生理功能的需要；提供代谢的灵活性，使细胞能够适应不同的营养条件和生理状态；为代谢工程和生物技术应用提供理论基础，通过调节关键酶的表达可以改善氨基酸的生产效率。
