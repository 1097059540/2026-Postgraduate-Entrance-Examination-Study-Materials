国科大生命科学学院 生物化学柠檬酸循环必背考点

一、核心名词解释（10个）

三羧酸循环（Tricarboxylic Acid Cycle，TCA）：又称柠檬酸循环或克雷布斯循环(Krebs cycle)，是需氧生物体内普遍存在的代谢途径，乙酰CoA与草酰乙酸缩合生成柠檬酸，经过一系列脱氢脱羧反应，最终重新生成草酰乙酸的循环过程。

柠檬酸合酶（Citrate Synthase，CS）：催化乙酰CoA与草酰乙酸缩合生成柠檬酸的关键酶，是TCA循环的第一步反应酶，受ATP、NADH、琥珀酰CoA的抑制调节。

异柠檬酸脱氢酶（Isocitrate Dehydrogenase，IDH）：催化异柠檬酸氧化脱羧生成α-酮戊二酸的NAD+依赖性酶，是TCA循环的限速酶之一，受ATP抑制、ADP激活。

α-酮戊二酸脱氢酶复合体（α-Ketoglutarate Dehydrogenase Complex，α-KGDHC）：催化α-酮戊二酸氧化脱羧生成琥珀酰CoA的多酶复合体，由三种酶组成，受ATP和琥珀酰CoA抑制。

琥珀酰CoA（Succinyl-CoA）：α-酮戊二酸氧化脱羧的产物，含有高能硫酯键的四碳化合物，在琥珀酰CoA合成酶催化下水解生成琥珀酸并产生GTP。

琥珀酸脱氢酶（Succinate Dehydrogenase，SDH）：催化琥珀酸氧化生成富马酸的酶，同时是电子传递链复合体II，以FAD为辅酶，产生FADH2。

草酰乙酸（Oxaloacetate，OAA）：TCA循环的起始和终产物，四碳二羧酸化合物，与乙酰CoA缩合启动循环，由苹果酸脱氢酶催化苹果酸氧化生成。

底物水平磷酸化（Substrate-Level Phosphorylation，SLP）：琥珀酰CoA直接转移高能磷酸键给GDP生成GTP的过程，是TCA循环中唯一的直接ATP生成步骤。

丙酮酸脱氢酶复合体（Pyruvate Dehydrogenase Complex，PDC）：催化丙酮酸氧化脱羧生成乙酰CoA的多酶复合体，连接糖酵解与TCA循环，受产物反馈抑制调节。

能量电荷（Energy Charge，EC）：Atkinson(阿特金森)提出的细胞能量状态指标，计算公式为EC=([ATP]+0.5[ADP])/([ATP]+[ADP]+[AMP])，正常细胞维持在0.7-0.95范围内。

二、必备核心知识点（5个）

【必备知识点1】柠檬酸循环的基本特征与总反应式
发生部位：真核细胞线粒体基质，原核细胞胞质。总反应式：乙酰-CoA + 3NAD+ + FAD + GDP + Pi + 2H2O → 2CO2 + 3NADH + FADH2 + GTP + 3H+ + CoA-SH。能量产出：每轮循环产生1个GTP、3个NADH、1个FADH2，经氧化磷酸化可产生约10个ATP。循环特点：草酰乙酸既是起始物又是终产物，实现循环代谢。

【必备知识点2】八步反应的关键酶与不可逆步骤
第1步：柠檬酸合酶催化，乙酰CoA + 草酰乙酸 → 柠檬酸（不可逆，ΔG°′ = -31.4 kJ/mol）。第3步：异柠檬酸脱氢酶催化，异柠檬酸 → α-酮戊二酸 + CO2 + NADH（不可逆）。第4步：α-酮戊二酸脱氢酶复合体催化，α-酮戊二酸 → 琥珀酰CoA + CO2 + NADH（不可逆）。第5步：琥珀酰CoA合成酶催化，琥珀酰CoA → 琥珀酸 + GTP（底物水平磷酸化）。第6步：琥珀酸脱氢酶催化，琥珀酸 → 富马酸 + FADH2（与电子传递链偶联）。

【必备知识点3】三大关键调节酶的调控机制
柠檬酸合酶：受ATP、NADH、琥珀酰CoA抑制，受ADP激活。异柠檬酸脱氢酶：受ATP、NADH抑制，受ADP、Ca2+激活，需Mg2+或Mn2+作为辅助因子。α-酮戊二酸脱氢酶复合体：受ATP、NADH、琥珀酰CoA、GTP抑制，受Ca2+激活。调节原理：ATP/ADP和NADH/NAD+比值是主要调节因子，能量充足时抑制循环，能量不足时激活循环。

【必备知识点4】柠檬酸循环的代谢枢纽功能
糖类代谢：丙酮酸经丙酮酸脱氢酶复合体生成乙酰CoA进入循环。脂类代谢：脂肪酸β氧化产生乙酰CoA，甘油转化为磷酸二羟丙酮进入糖酵解。蛋白质代谢：氨基酸脱氨基后进入循环，如丙氨酸→丙酮酸，谷氨酸→α-酮戊二酸，天冬氨酸→草酰乙酸。合成代谢：循环中间产物用于合成氨基酸、血红蛋白、胆固醇等生物大分子。

【必备知识点5】回补反应与乙醛酸途径
回补反应：当循环中间产物被消耗用于合成时，需要回补反应维持循环正常进行。主要回补反应：丙酮酸羧化酶催化丙酮酸 + CO2 + ATP → 草酰乙酸 + ADP + Pi。乙醛酸途径：植物和微生物特有，绕过两个脱羧步骤，实现脂肪向糖的净转化。关键酶：异柠檬酸裂解酶（异柠檬酸→琥珀酸+乙醛酸）、苹果酸合成酶（乙醛酸+乙酰CoA→苹果酸）。

三、高频必考大题（2道）

【高频考题1】试述柠檬酸循环的调节机制，包括关键调节酶、调节因子及其生物学意义。

1. 三大关键调节酶及其调节机制：
- 柠檬酸合酶：受ATP、NADH、琥珀酰CoA负反馈抑制，受ADP正向激活
- 异柠檬酸脱氢酶：受ATP、NADH抑制，受ADP、Ca2+激活，需Mg2+/Mn2+作辅助因子
- α-酮戊二酸脱氢酶复合体：受ATP、NADH、琥珀酰CoA、GTP抑制，受Ca2+激活

2. 主要调节因子：
- ATP/ADP比值：能量电荷高时抑制循环，能量电荷低时激活循环
- NADH/NAD+比值：还原当量充足时抑制循环，缺乏时激活循环
- Ca2+离子：激活异柠檬酸脱氢酶和α-酮戊二酸脱氢酶，增强循环活性

3. 生物学意义：
- 根据细胞能量状态调节ATP产生速率，维持能量代谢平衡
- 协调分解代谢与合成代谢，确保代谢物合理分配
- 响应生理需求变化，如运动时Ca2+升高激活循环产生更多ATP

【高频考题2】比较柠檬酸循环与糖酵解在能量产生、调节机制和代谢意义方面的异同，并说明两者的协调关系。

1. 能量产生比较：
- 糖酵解：每分子葡萄糖净产生2个ATP（底物水平磷酸化），2个NADH
- 柠檬酸循环：每分子乙酰CoA产生1个GTP、3个NADH、1个FADH2，经氧化磷酸化约产生10个ATP
- 效率差异：柠檬酸循环能量产出效率远高于糖酵解，是细胞主要产能途径

2. 调节机制比较：
- 糖酵解：主要受磷酸果糖激酶调节，ATP、柠檬酸抑制，AMP、ADP激活
- 柠檬酸循环：三个关键酶协同调节，ATP/ADP和NADH/NAD+比值为主要调节因子
- 共同特点：均受能量电荷调节，遵循负反馈抑制原理

3. 代谢意义与协调关系：
- 糖酵解：快速产能，无氧条件下仍可进行，为柠檬酸循环提供丙酮酸
- 柠檬酸循环：完全氧化有机物，是糖、脂、蛋白质代谢的共同通路和代谢枢纽
- 协调机制：柠檬酸抑制磷酸果糖激酶，防止糖酵解过度进行；丙酮酸脱氢酶复合体连接两个途径，受ATP、NADH调节
