核酸的物理化学性质和研究方法

生物化学考研复习笔记 | 基于朱圣庚、徐长法《生物化学》第四版

一、核心知识体系

1. 核酸的水解

1.1 酸性水解

核酸在酸性条件下水解时，嘌呤碱基（腺嘌呤A和鸟嘌呤G）的糖苷键比嘧啶碱基（胞嘧啶C、胸腺嘧啶T、尿嘧啶U）的糖苷键更容易断裂。这是因为嘌呤环系统的电子云密度较高，质子化后更容易发生糖苷键的断裂。

1.2 碱性水解

RNA在碱性条件下可以被完全水解，产生2'和3'核苷酸的混合物。这是因为RNA分子中2'羟基的存在使得磷酸二酯键在碱性条件下不稳定。而DNA由于缺乏2'羟基，在碱性条件下相对稳定，不易被水解。

重要提示：利用RNA和DNA对碱性水解敏感性的差异，可以选择性地去除RNA而保留DNA，这在核酸分离纯化中具有重要应用价值。

1.3 酶解

核酸酶可以特异性地水解核酸分子：

  • DNA酶（DNase）：专一性水解DNA分子
  • RNA酶（RNase）：专一性水解RNA分子
  • 限制性内切酶：识别特定的DNA序列并在特定位点切割

2. 核酸的酸碱性质

核酸分子具有两性电解质的性质，主要由以下基团贡献：

  • 酸性基团：磷酸基团（pKa约为1-2）
  • 碱性基团：嘌呤和嘧啶碱基上的氨基和亚氨基

在生理pH条件下（pH 7.4），核酸分子带负电荷，这是由于磷酸基团完全电离而碱基基团大部分未质子化所致。

3. 核酸的紫外吸收

3.1 紫外吸收的分子基础

核酸的紫外吸收主要来源于嘌呤和嘧啶碱基环上的共轭双键系统。这些芳香环化合物在240-290nm波长范围内有强烈的紫外吸收，最大吸收峰位于260nm附近。

核酸最大吸收波长：λmax = 260nm
蛋白质最大吸收波长：λmax = 280nm

3.2 增色效应（Hyperchromic Effect）

当DNA双螺旋结构被破坏（变性）时，碱基从内侧暴露出来，碱基间的相互作用减弱，导致紫外吸收显著增加，这种现象称为增色效应。增色效应的幅度通常为30-40%。

3.3 核酸纯度的测定

利用核酸和蛋白质在不同波长下的吸收差异，可以通过测定A₂₆₀/A₂₈₀比值来评估核酸样品的纯度：

  • 纯DNA：A₂₆₀/A₂₈₀ ≈ 1.8
  • 纯RNA：A₂₆₀/A₂₈₀ ≈ 2.0
  • 比值偏低：表明样品中含有蛋白质污染
  • 比值偏高：表明样品中可能含有RNA或其他杂质

4. 核酸的变性、复性和杂交

4.1 DNA变性（Denaturation）

DNA变性是指双螺旋结构中的氢键断裂，两条互补链分离，形成无规则线团状构象的过程。变性只涉及次级键的断裂，不影响一级结构。

引起变性的因素：

  • 高温（最常用的变性方法）
  • 极端pH值（强酸或强碱）
  • 有机溶剂（甲醇、乙醇、甲酰胺等）
  • 变性剂（尿素、盐酸胍等）

变性后的理化性质变化：

  • 溶液黏度降低
  • 增色效应（紫外吸收增加30-40%）
  • 旋光性改变
  • 浮力密度增加
  • 流动双折射消失

4.2 熔解温度（Tm值）

Tm值是指DNA热变性过程中，紫外吸收增加量达到最大增量50%时的温度，此时50%的双螺旋结构被破坏。

Tm = 69.3 + 0.41(%G+C) + 16.6(lg[Na⁺])

影响Tm值的因素：

  • G+C含量：G-C碱基对有3个氢键，A-T碱基对只有2个氢键，因此G+C含量越高，Tm值越高
  • 离子强度：高离子强度可以屏蔽DNA链间的静电斥力，提高Tm值
  • DNA长度：较长的DNA分子具有更高的Tm值
  • pH值：极端pH会影响碱基的质子化状态，从而影响Tm值

4.3 DNA复性（Renaturation）

复性是变性DNA在适当条件下重新形成双螺旋结构的过程，也称为退火（Annealing）。

复性的最适条件：

  • 温度：比Tm值低25°C左右
  • 冷却速度：缓慢冷却（急速冷却会阻止复性）
  • 离子强度：0.18-0.2 mol/L Na⁺
  • DNA浓度：适当的浓度有利于互补链的碰撞结合

4.4 Cot值和复性动力学

Cot值用于描述DNA复性的动力学特征，其中C₀为单链DNA的起始浓度，t为复性时间。

Cot₁/₂ ∝ 基因组复杂性

基因组越复杂（非重复序列越多），达到50%复性所需的Cot₁/₂值越大。

4.5 分子杂交（Hybridization）

分子杂交是指来自不同来源的核酸单链在适当条件下形成杂化双链的过程。这是现代分子生物学技术的重要基础。

杂交的类型：

  • DNA-DNA杂交：用于基因定位、基因组比较等
  • DNA-RNA杂交：用于基因表达分析
  • RNA-RNA杂交：用于RNA结构和功能研究

5. 核酸的分离和纯化

5.1 细胞裂解和核酸释放

核酸的分离首先需要破坏细胞壁和细胞膜，释放细胞内的核酸。常用方法包括：

  • 物理方法：超声波破碎、液氮研磨、高压均质等
  • 化学方法：去垢剂（SDS）、蛋白酶K处理等
  • 酶解方法：溶菌酶处理细菌细胞壁

5.2 蛋白质的去除

核酸提取过程中需要去除结合的蛋白质：

  • 苯酚-氯仿抽提：利用蛋白质在有机相中的溶解性
  • 蛋白酶处理：蛋白酶K可以消化大部分蛋白质
  • 盐析法：高盐浓度下沉淀蛋白质

5.3 DNA和RNA的分离

利用DNA和RNA的理化性质差异进行分离：

  • 碱性水解：NaOH处理可选择性水解RNA
  • RNase处理：特异性消化RNA
  • 氯化锂沉淀：高浓度LiCl可选择性沉淀RNA

5.4 核酸的纯化方法

色谱法：

  • 离子交换色谱：利用核酸的负电荷特性
  • 分子筛色谱：根据分子大小分离
  • 反相色谱：利用疏水相互作用
  • 亲和色谱：利用特异性结合

电泳法：

  • 琼脂糖凝胶电泳：分离较大的DNA片段（>100bp）
  • 聚丙烯酰胺凝胶电泳：分离较小的DNA片段（<1000bp）
  • 脉冲场凝胶电泳：分离超大DNA分子（>50kb）

5.5 密度梯度离心

  • 蔗糖密度梯度：根据沉降系数分离
  • 氯化铯密度梯度：根据浮力密度分离，可区分不同类型的核酸

6. 核酸序列测定

6.1 第一代测序技术

Sanger双脱氧链终止法（1977年）：

这是目前应用最广泛的DNA测序方法，其基本原理是利用DNA聚合酶合成DNA时，如果加入缺乏3'-OH的双脱氧核苷三磷酸（ddNTP），就会导致链合成终止。

测序步骤：

  1. 模板准备：制备单链DNA模板
  2. 引物退火：特异性引物与模板结合
  3. 链延伸反应：在含有dNTP和少量ddNTP的体系中进行DNA合成
  4. 电泳分离：变性聚丙烯酰胺凝胶电泳分离不同长度的片段
  5. 序列读取：通过荧光检测或放射自显影读取序列

Maxam-Gilbert化学法（1977年）：

利用特异的化学试剂作用于不同碱基，然后用哌啶切断反应碱基的多核苷酸链。虽然准确性高，但操作复杂，现已较少使用。

6.2 第二代测序技术（高通量测序）

454测序法：

  • 核心技术：焦磷酸测序法，边合成边测序
  • 优点：读长较长（400-800bp），速度快
  • 缺点：无法准确识别连续相同核苷酸

Solexa测序法（Illumina）：

  • 核心技术：桥式PCR扩增和可逆终止测序
  • 优点：通量高，成本低，准确性高
  • 缺点：读长较短（50-300bp）

SOLiD测序法：

  • 核心技术：连接酶测序和双碱基编码
  • 优点：准确性极高，通量大
  • 缺点：读长短，数据分析复杂

6.3 第三代测序技术（单分子测序）

SMRT测序（PacBio）：

  • 原理：单分子实时测序，利用零模波导孔技术
  • 优点：读长极长（平均10-15kb），可检测碱基修饰
  • 缺点：错误率相对较高，成本高

Nanopore测序：

  • 原理：DNA通过纳米孔时产生特征性电流信号
  • 优点：读长无理论上限，便携性好
  • 缺点：错误率较高，通量相对较低

7. DNA微阵技术（DNA芯片）

7.1 基本原理

DNA微阵技术是将大量已知序列的单链DNA片段按设计顺序固定在固体表面，形成微观阵列，用作探针与标记的靶核酸分子进行杂交检测。

7.2 制作方法

  • 接触打印法：使用机械臂将DNA溶液点样到芯片表面
  • 照相平版印刷法：利用光刻技术原位合成寡核苷酸
  • 喷墨法：类似喷墨打印机的原理制作芯片

7.3 主要应用

  • 基因表达谱分析：检测不同条件下基因表达水平的变化
  • 基因型检测：检测单核苷酸多态性（SNP）
  • 基因突变检测：筛查疾病相关的基因突变
  • DNA重测序：验证已知序列的准确性

8. 核酸的化学合成

8.1 固相亚磷酰胺法

这是目前DNA自动合成仪普遍采用的方法，具有高效、快速的特点。

合成循环（四步反应）：

  1. 去保护：用三氯醋酸去除5'端的DMTr保护基
  2. 偶联反应：活化的亚磷酰胺单体与5'-OH发生偶联
  3. 加帽反应：用醋酐封闭未反应的5'-OH
  4. 氧化反应：用碘将三价磷氧化为五价磷

合成特点：

  • 合成方向：3'→5'
  • 偶联效率：>99%
  • 合成长度：通常50-100个核苷酸
  • 纯化：需要去除保护基并进行HPLC纯化

二、重要名词解释库

指DNA双螺旋结构中的氢键断裂，两条互补链分离，形成无规则线团状构象的过程。变性只涉及次级键的断裂，不影响一级结构。
复性（Renaturation）
DNA Renaturation / Annealing
变性DNA在适当条件下重新形成双螺旋结构的过程。需要缓慢冷却和适当的离子强度。
熔解温度（Tm值）
Melting Temperature, Tm
DNA热变性过程中，紫外吸收增加量达到最大增量50%时的温度。Tm值与G+C含量成正比。
增色效应
Hyperchromic Effect
DNA变性时碱基暴露导致紫外吸收显著增加的现象，增幅通常为30-40%。
Cot值
Cot Value
用于描述DNA复性动力学的参数，C₀为单链DNA起始浓度，t为复性时间。Cot₁/₂与基因组复杂性成正比。
分子杂交
Molecular Hybridization
来自不同来源的核酸单链在适当条件下通过碱基配对形成杂化双链的过程。
Sanger测序法
Sanger Sequencing / Chain Termination Method
利用双脱氧核苷三磷酸（ddNTP）作为链终止剂的DNA测序方法，是第一代测序技术的代表。
高通量测序
High-throughput Sequencing / Next-generation Sequencing, NGS
第二代测序技术的总称，特点是通量高、成本低，包括454、Solexa、SOLiD等平台。
单分子测序
Single Molecule Sequencing
第三代测序技术，可以对单个DNA分子进行实时测序，读长长，包括SMRT和Nanopore技术。
DNA微阵技术
DNA Microarray / DNA Chip
将大量已知序列的DNA片段固定在固体表面形成阵列，用于基因表达分析和基因型检测的技术。
固相合成法
Solid-phase Synthesis
DNA化学合成的主要方法，将第一个核苷酸固定在不溶性载体上，逐步延伸合成目标序列。
亚磷酰胺法
Phosphoramidite Method
目前DNA自动合成仪普遍采用的化学合成方法，包括去保护、偶联、加帽、氧化四个步骤的循环。
Southern印迹杂交
Southern Blotting
将DNA转移到硝酸纤维素膜或尼龙膜上进行分子杂交的技术，用于DNA检测和分析。
Northern印迹杂交
Northern Blotting
将RNA转移到膜上进行分子杂交的技术，用于RNA检测和基因表达分析。
脉冲场凝胶电泳
Pulsed Field Gel Electrophoresis, PFGE
用于分离超大DNA分子（>50kb）的电泳技术，通过改变电场方向使大分子DNA能够在凝胶中迁移。
密度梯度离心
Density Gradient Centrifugation
利用不同密度的介质形成密度梯度，根据分子的浮力密度差异进行分离的技术。

三、选择题测试

四、深度思考题

1. 试分析DNA变性和复性的分子机制，并阐述其在分子生物学技术中的应用价值。

  1. DNA变性的分子机制：DNA双螺旋结构的稳定性主要依赖于碱基间的氢键和碱基堆积力。当外界条件（如高温、极端pH、变性剂等）破坏这些相互作用时，氢键断裂，双链分离。变性过程是协同性的，在很窄的温度范围内完成，表现为典型的S型熔解曲线。变性后DNA分子从紧密的双螺旋结构变为松散的无规线团，导致黏度降低、紫外吸收增加（增色效应）等理化性质的改变。
  2. DNA复性的分子机制：复性是变性的逆过程，需要适当的温度（通常比Tm低25°C）、离子强度和时间。复性包括成核和拉链化两个步骤：首先是互补序列的随机碰撞和初始配对（成核），然后是沿着分子逐步配对形成完整双螺旋（拉链化）。成核是限速步骤，因此DNA浓度对复性速度有重要影响。
  3. 在PCR技术中的应用：PCR的每个循环都包括变性、退火和延伸三个步骤。高温变性使模板DNA链分离，降温退火使引物与模板结合，适温延伸合成新链。这一过程的成功依赖于对DNA变性复性规律的精确控制。
  4. 在分子杂交技术中的应用：Southern、Northern杂交等技术都基于核酸的变性复性原理。通过控制杂交条件（温度、盐浓度等），可以实现特异性检测，广泛应用于基因定位、表达分析、突变检测等领域。
  5. 在基因克隆和测序中的应用：DNA文库的构建、载体与插入片段的连接、测序反应中的模板制备等都涉及DNA的变性复性过程。理解这些原理有助于优化实验条件，提高实验成功率。

2. 比较第一代、第二代和第三代DNA测序技术的原理、特点及应用前景。

  1. 第一代测序技术（Sanger法）：基于链终止原理，使用ddNTP作为终止剂，通过毛细管电泳分离不同长度的片段。特点是准确性高（>99.9%），读长适中（500-1000bp），但通量低，成本高。目前仍是验证测序和小规模项目的金标准，在基因克隆、突变检测、引物设计验证等方面有重要应用。
  2. 第二代测序技术（高通量测序）：包括454（焦磷酸测序）、Illumina（可逆终止测序）、SOLiD（连接酶测序）等平台。共同特点是通量高、成本低、并行处理能力强，但读长相对较短（50-800bp）。革命性地推动了基因组学研究，使得全基因组测序、转录组测序、表观基因组学等大规模研究成为可能。在疾病基因发现、个性化医疗、农业育种等领域有广泛应用。
  3. 第三代测序技术（单分子测序）：以PacBio SMRT和Oxford Nanopore为代表，可以对单个DNA分子进行实时测序。最大优势是读长极长（可达数十kb甚至更长），能够跨越重复序列和结构变异区域，并可检测碱基修饰。虽然错误率相对较高，但在基因组组装、结构变异检测、全长转录本测序等方面具有独特优势。
  4. 技术发展趋势：未来测序技术将朝着更长读长、更高准确性、更低成本、更快速度的方向发展。第三代技术的错误率正在快速降低，第四代技术（如蛋白质纳米孔测序）也在研发中。不同代技术将在不同应用场景中发挥各自优势，形成互补格局。
  5. 应用前景展望：随着技术成熟和成本降低，测序将更广泛地应用于临床诊断、药物开发、精准医疗、生物技术等领域。长读长测序技术将解决复杂基因组区域的组装问题，实时测序技术将实现即时诊断，多组学整合分析将提供更全面的生物学信息。

3. 阐述核酸紫外吸收的分子基础，并分析其在核酸定量和纯度检测中的应用原理。

  1. 紫外吸收的分子基础：核酸的紫外吸收主要来源于嘌呤（腺嘌呤、鸟嘌呤）和嘧啶（胞嘧啶、胸腺嘧啶、尿嘧啶）碱基环上的共轭双键系统。这些芳香环化合物具有π电子共轭体系，能够吸收特定波长的紫外光。当紫外光照射时，π电子从基态跃迁到激发态，产生特征性吸收峰。核酸的最大吸收波长在260nm附近，这是由于碱基的电子跃迁能级决定的。
  2. 增色效应的机制：在DNA双螺旋结构中，碱基位于分子内侧，相邻碱基间存在π-π堆积相互作用，这种相互作用会抑制紫外吸收。当DNA变性时，双螺旋结构解开，碱基暴露，堆积相互作用减弱，导致紫外吸收显著增加，这就是增色效应。增色效应的幅度通常为30-40%，是DNA变性的重要指标。
  3. 核酸定量的原理：根据Beer-Lambert定律，在一定浓度范围内，溶液的吸光度与溶质浓度成正比。通过测定260nm处的吸光度，可以计算核酸浓度。对于双链DNA，A₂₆₀=1.0相当于约50μg/mL；对于RNA，A₂₆₀=1.0相当于约40μg/mL。这种方法简便快速，是实验室常用的核酸定量方法。
  4. 纯度检测的原理：不同生物分子在紫外区有不同的吸收特征。蛋白质由于含有芳香族氨基酸（苯丙氨酸、酪氨酸、色氨酸），在280nm处有最大吸收。通过测定A₂₆₀/A₂₈₀比值，可以评估核酸样品的纯度。纯DNA的比值约为1.8，纯RNA约为2.0。比值偏低表明有蛋白质污染，比值偏高可能含有RNA或其他杂质。
  5. 实际应用中的注意事项：pH值、离子强度、温度等因素都会影响紫外吸收，因此需要在标准条件下测定。样品中的污染物（如苯酚、EDTA等）也会干扰测定结果。对于高精度要求的实验，还需要结合其他方法（如荧光定量、凝胶电泳等）进行验证。现代分光光度计还可以测定A₂₆₀/A₂₃₀比值来检测有机污染物的存在。

4. 分析影响DNA熔解温度（Tm值）的主要因素，并探讨其在分子生物学实验中的指导意义。

  1. 碱基组成的影响：G-C碱基对之间有三个氢键，而A-T碱基对只有两个氢键，因此G+C含量越高，DNA双螺旋越稳定，Tm值越高。经验公式Tm=69.3+0.41(%G+C)很好地反映了这种关系。这一规律在引物设计、PCR条件优化、杂交实验等方面具有重要指导意义。例如，设计PCR引物时需要考虑引物的Tm值匹配，通常要求正向和反向引物的Tm值相差不超过5°C。
  2. 离子强度的影响：DNA分子带负电荷，链间存在静电斥力。高离子强度可以屏蔽这种斥力，稳定双螺旋结构，提高Tm值。Na⁺浓度的影响可用公式Tm=69.3+0.41(%G+C)+16.6(lg[Na⁺])表示。在实际实验中，通过调节缓冲液的离子强度可以控制杂交的严紧性，实现特异性检测。
  3. DNA分子长度的影响：较长的DNA分子具有更多的碱基对，协同效应使得整体结构更稳定，Tm值更高。但这种影响主要体现在相对较短的分子中，对于长DNA分子，长度对Tm值的影响趋于平缓。这一特性在寡核苷酸探针设计中需要特别考虑，通常要求探针长度在18-25个核苷酸之间以获得适当的Tm值。
  4. pH值的影响：极端pH会影响碱基的质子化状态，破坏氢键形成，降低Tm值。在碱性条件下，碱基去质子化会影响Watson-Crick配对；在酸性条件下，碱基质子化也会干扰正常配对。因此，核酸实验通常在中性或弱碱性条件下进行。
  5. 在分子生物学实验中的指导意义：①PCR实验：退火温度通常设定为引物Tm值减去5-10°C，以确保引物特异性结合；②杂交实验：杂交温度和洗涤温度的设定需要根据探针和靶序列的Tm值来确定，高严紧性条件可以提高特异性；③DNA变性实验：了解样品的Tm值有助于选择合适的变性条件；④引物和探针设计：需要计算Tm值以确保实验条件的合理性；⑤基因组分析：不同物种或基因组区域的G+C含量差异可以通过Tm值分析来研究。

5. 论述现代DNA测序技术的发展对生命科学研究和医学诊断带来的革命性影响。

  1. 基因组学研究的革命：从人类基因组计划耗时13年、花费30亿美元，到现在个人基因组测序只需几天、成本不到1000美元，测序技术的飞速发展彻底改变了基因组学研究。大规模基因组测序项目（如千人基因组计划、万人基因组计划）揭示了人类遗传变异的全貌，为理解疾病机制、药物反应、进化历史等提供了前所未有的数据基础。全基因组关联研究（GWAS）发现了数千个与疾病相关的遗传位点，推动了精准医学的发展。
  2. 转录组学和表观基因组学的兴起：RNA测序技术使得全转录组分析成为可能，不仅可以检测已知基因的表达水平，还能发现新的转录本、剪接变体、长非编码RNA等。单细胞RNA测序技术进一步揭示了细胞异质性和发育轨迹。表观基因组测序（如ChIP-seq、ATAC-seq、甲基化测序等）阐明了基因调控的分子机制，为理解疾病发生发展提供了新的视角。
  3. 精准医学的实现：测序技术使得个体化医疗成为现实。通过分析患者的基因组信息，可以预测疾病风险、选择最适合的治疗方案、避免药物不良反应。肿瘤基因组测序已成为癌症诊断和治疗的重要工具，指导靶向治疗和免疫治疗的选择。药物基因组学研究为个体化用药提供了科学依据。
  4. 感染性疾病诊断的变革：宏基因组测序技术可以在不培养病原体的情况下直接检测感染性疾病，大大缩短了诊断时间。在COVID-19疫情中，快速基因组测序为病毒溯源、变异监测、疫苗开发等提供了关键信息。抗生素耐药基因的检测有助于合理使用抗生素，减少耐药性的产生。
  5. 农业和环境科学的应用：作物基因组测序加速了分子育种进程，培育出抗病、抗逆、高产的新品种。环境宏基因组学揭示了微生物群落的结构和功能，为生态保护、污染治理、生物能源开发等提供了新思路。古DNA测序技术重建了物种进化历史，为保护生物学提供了重要信息。
  6. 未来发展前景：随着测序成本的进一步降低和技术的不断完善，测序将更广泛地应用于临床常规检测、健康管理、药物开发等领域。长读长测序技术将解决复杂基因组区域的分析难题，实时测序技术将实现床旁即时诊断，多组学整合分析将提供更全面的生物学信息。人工智能和大数据技术的结合将进一步提升数据分析和解释的能力，推动生命科学研究和医学诊断进入新的时代。

🤔 深度思考题

1. 比较分析DNA变性和复性过程的分子机制，并讨论影响这两个过程的主要因素。

参考答案：

  1. DNA变性的分子机制： DNA变性是指双链DNA分子在物理或化学因素作用下，氢键断裂，双螺旋结构解开，形成单链DNA的过程。主要涉及碱基间氢键的断裂和碱基堆积力的破坏。变性过程中，DNA分子的紫外吸收增加（增色效应），黏度下降，浮力密度增加，但一级结构（核苷酸序列）保持不变。
  2. DNA复性的分子机制： DNA复性是变性DNA在适当条件下重新形成双螺旋结构的过程。包括成核阶段（互补链的初始结合）和拉链阶段（氢键的逐步形成）。复性过程遵循二级反应动力学，其速率与DNA浓度的平方成正比。
  3. 影响变性的主要因素： 温度（主要因素）、pH值（极端pH破坏氢键）、离子强度（低盐浓度促进变性）、变性剂（如尿素、甲酰胺）、DNA的G+C含量（G+C含量高的DNA更稳定）。
  4. 影响复性的主要因素： 温度（最适温度为Tm-25°C）、离子强度（适中的盐浓度有利于复性）、DNA浓度（浓度越高复性越快）、DNA片段长度（短片段复性更快）、序列复杂性（重复序列复性更快）。
  5. 生物学意义： DNA的变性和复性特性是许多分子生物学技术的基础，如PCR、分子杂交、DNA测序等。理解这些过程对于优化实验条件和开发新技术具有重要意义。

2. 详细阐述第一代、第二代和第三代DNA测序技术的原理、特点及应用，并分析各自的优缺点。

参考答案：

  1. 第一代测序技术（Sanger测序）： 原理基于链终止法，使用ddNTP作为链终止剂。特点是准确性高（99.9%），读长较长（800-1000bp），但通量低，成本高。主要应用于小规模测序项目、验证性测序和高精度要求的项目。优点是准确性极高，技术成熟；缺点是通量低，不适合大规模项目。
  2. 第二代测序技术（高通量测序）： 包括454、Illumina、SOLiD等平台。原理基于边合成边测序或连接测序。特点是通量极高，成本大幅降低，但读长相对较短（50-300bp）。广泛应用于全基因组测序、转录组分析、表观遗传学研究等。优点是通量高、成本低；缺点是读长短，难以跨越重复序列。
  3. 第三代测序技术（单分子测序）： 包括PacBio SMRT和Oxford Nanopore等。原理基于单分子实时测序。特点是读长极长（可达数万bp），可直接检测DNA修饰，但错误率相对较高。主要应用于基因组组装、结构变异检测、全长转录本分析等。优点是读长长，可检测修饰；缺点是错误率高，成本仍然较高。
  4. 技术发展趋势： 各代技术互补发展，第二代技术用于大规模筛查，第三代技术用于精细结构分析，第一代技术用于验证。未来发展方向包括提高准确性、降低成本、实现便携化等。
  5. 选择策略： 根据研究目的选择合适的测序技术：高通量筛查选择第二代，结构分析选择第三代，精确验证选择第一代。

3. 分析核酸分离纯化的各种方法，并讨论如何根据不同的研究目的选择合适的分离纯化策略。

参考答案：

  1. 基于分子大小的分离方法： 凝胶电泳是最常用的方法，琼脂糖凝胶适合分离大分子DNA，聚丙烯酰胺凝胶适合分离小分子DNA和RNA。脉冲场凝胶电泳专门用于分离超大DNA分子（>50kb）。超速离心可根据沉降系数分离不同大小的核酸分子。
  2. 基于密度差异的分离方法： 氯化铯密度梯度离心可分离不同类型的核酸（DNA、RNA）和不同组成的DNA（如重同位素标记的DNA）。蔗糖密度梯度离心主要用于分离不同大小的核酸分子，条件温和，适合分离完整的大分子。
  3. 基于化学性质的分离方法： 苯酚-氯仿抽提用于去除蛋白质，利用蛋白质在有机相中的溶解性。碱性条件下可选择性水解RNA，保留DNA。RNase处理可特异性降解RNA。氯化锂沉淀可选择性沉淀RNA。
  4. 色谱分离方法： 离子交换色谱基于电荷差异分离，适合分离不同类型的核酸。分子筛色谱基于分子大小分离。反相色谱可分离修饰的核酸。亲和色谱可特异性纯化特定序列的核酸。
  5. 选择策略： 根据研究目的选择：基因克隆需要高纯度完整DNA，选择温和的分离方法；PCR扩增对纯度要求不高，简单的裂解和沉淀即可；RNA研究需要防止RNase污染，选择快速有效的方法；结构研究需要保持天然构象，避免变性条件。

4. 论述分子杂交技术的原理及其在现代分子生物学研究中的应用，并分析影响杂交特异性的因素。

参考答案：

  1. 分子杂交的基本原理： 基于Watson-Crick碱基配对规则，互补的单链核酸分子在适当条件下可以形成稳定的双链结构。杂交过程包括变性、退火和检测三个步骤。杂交的稳定性主要由氢键和碱基堆积力维持。
  2. 主要应用技术： Southern印迹杂交用于DNA检测和基因定位；Northern印迹杂交用于RNA检测和基因表达分析；原位杂交用于染色体定位和组织表达分析；DNA微阵列技术用于大规模基因表达谱分析；FISH技术用于染色体和基因的可视化定位。
  3. 影响杂交特异性的因素： 温度是最重要因素，高温提高特异性但降低敏感性；盐浓度影响DNA稳定性，低盐提高特异性；探针长度和序列同源性直接影响杂交效率；甲酰胺等添加剂可降低杂交温度；pH值影响碱基质子化状态。
  4. 严紧性控制： 高严紧性条件（高温、低盐）提高特异性，适合检测完全匹配的序列；低严紧性条件（低温、高盐）提高敏感性，可检测部分匹配的序列。根据研究目的调整严紧性条件。
  5. 现代发展： 从传统的放射性标记发展到荧光标记，从单一检测发展到高通量检测，从定性分析发展到定量分析。新技术如数字PCR、单细胞测序等进一步扩展了应用范围。

5. 结合核酸的物理化学性质，分析DNA在细胞内的存在状态及其生物学意义。

参考答案：

  1. DNA的超螺旋结构： 细胞内DNA以超螺旋形式存在，包括正超螺旋和负超螺旋。负超螺旋占主导，有利于DNA的解链和转录。拓扑异构酶调节超螺旋度，维持DNA的功能状态。超螺旋结构影响基因表达和DNA复制。
  2. DNA与蛋白质的相互作用： 组蛋白与DNA结合形成核小体结构，是染色质的基本单位。非组蛋白如转录因子、DNA结合蛋白等调节基因表达。蛋白质结合改变DNA的局部结构和可及性，影响生物学功能。
  3. DNA的甲基化修饰： 胞嘧啶的甲基化是重要的表观遗传修饰，影响基因表达而不改变DNA序列。甲基化模式在细胞分化和发育过程中起关键作用。异常甲基化与多种疾病相关。
  4. 染色质的层次结构： 从DNA双螺旋到核小体，再到30nm纤维，最终形成染色体。不同层次的结构调节基因的可及性和表达。异染色质和常染色质的转换调节基因活性。
  5. 生物学意义： DNA的物理化学性质决定了其在细胞内的存在形式和功能。理解这些性质有助于阐明基因调节机制，为疾病治疗和生物技术应用提供理论基础。DNA结构的动态变化是生命活动的重要基础。
