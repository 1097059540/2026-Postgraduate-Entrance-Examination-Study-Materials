蛋白质的性质、分离纯化和鉴定

考研生物化学复习笔记 | 基于朱圣庚《生物化学》第四版

一、蛋白质的理化性质

1. 酸碱性质

蛋白质分子含有氨基和羧基，具有两性电解质的性质，在不同pH环境中表现出不同的电荷状态。

等电点（pI）

蛋白质分子净电荷为零时溶液的pH值。在等电点时，蛋白质在电场中不发生迁移，溶解度最小，最易沉淀。

  • pH < pI：蛋白质带正电荷，向负极迁移
  • pH = pI：蛋白质净电荷为零，不迁移
  • pH > pI：蛋白质带负电荷，向正极迁移

2. 胶体性质

蛋白质分子量大，在水中形成胶体溶液，具有以下特点：

  • 丁达尔效应：光束通过蛋白质溶液时产生散射现象
  • 布朗运动：蛋白质分子在溶液中不规则运动
  • 电泳现象：在电场作用下发生定向迁移
  • 渗透压低：相同浓度下渗透压比小分子物质低

3. 沉淀性质

蛋白质可通过多种方法从溶液中沉淀出来：

盐析法

常用硫酸铵进行盐析，原理是高浓度盐离子破坏蛋白质的水化层，使蛋白质聚集沉淀。盐析是可逆过程，不破坏蛋白质结构。

有机溶剂沉淀法

加入乙醇、丙酮等有机溶剂，降低蛋白质溶解度，使其沉淀。需在低温下进行以防变性。

等电点沉淀法

调节pH至蛋白质等电点，此时溶解度最小，易于沉淀。

二、蛋白质分离纯化方法

蛋白质分离纯化一般分为两个阶段：粗分离阶段和精细纯化阶段。

1. 根据溶解度差异的分离方法

盐析法

利用不同蛋白质在不同盐浓度下的溶解度差异进行分离。硫酸铵是最常用的盐析剂，具有溶解度大、对蛋白质无害等优点。

有机溶剂分级沉淀

逐步增加有机溶剂浓度，使不同蛋白质依次沉淀。常用乙醇，需在低温（-5°C）下操作。

2. 根据分子大小的分离方法

凝胶过滤层析（分子筛）

原理：利用多孔凝胶珠的分子筛效应，大分子蛋白质无法进入凝胶孔隙，先被洗脱；小分子蛋白质进入孔隙，后被洗脱。

  • 优点：条件温和，不使用变性剂，分离效果好
  • 缺点：分离时间长，样品稀释严重
  • 常用介质：Sephadex、Sepharose、Bio-Gel等

超滤法

利用半透膜的选择透过性，根据分子量大小进行分离浓缩。

3. 根据电荷性质的分离方法

离子交换层析

原理：利用蛋白质表面电荷的差异，使其与离子交换树脂结合，然后通过改变pH或离子强度进行洗脱分离。

  • 阳离子交换树脂：结合带正电荷的蛋白质（如CM-纤维素）
  • 阴离子交换树脂：结合带负电荷的蛋白质（如DEAE-纤维素）

电泳技术

聚丙烯酰胺凝胶电泳（PAGE）

根据蛋白质的电荷和分子量进行分离：

  • 天然PAGE：保持蛋白质天然结构
  • SDS-PAGE：SDS破坏蛋白质二级和三级结构，主要根据分子量分离
  • 尿素-PAGE：尿素作为变性剂
等电点聚焦电泳（IEF）

在pH梯度中进行电泳，蛋白质迁移至其等电点位置停止，分辨率极高。

4. 根据生物学特异性的分离方法

亲和层析

利用蛋白质与特定配体的特异性结合进行分离，是最高效的蛋白质纯化方法之一。

  • 酶亲和层析：利用酶与底物、抑制剂的特异性结合
  • 免疫亲和层析：利用抗原-抗体反应
  • 金属螯合亲和层析：利用组氨酸标签与金属离子结合

5. 根据疏水性质的分离方法

疏水作用层析（HIC）

利用蛋白质表面疏水区域的差异进行分离。在高盐浓度下，疏水性强的蛋白质与疏水介质结合，然后用低盐缓冲液洗脱。

反相高效液相色谱（RP-HPLC）

利用蛋白质的疏水性差异，在有机溶剂梯度中进行分离，分辨率高但条件较苛刻。

三、蛋白质相对分子质量的测定

1. 凝胶过滤法

通过测定蛋白质在凝胶过滤柱上的洗脱体积，与标准蛋白质比较确定分子量。适用于天然状态下的蛋白质。

2. SDS-聚丙烯酰胺凝胶电泳法

log(分子量) = a × 迁移率 + b

通过蛋白质在SDS-PAGE中的迁移距离，与标准分子量蛋白质比较确定分子量。

3. 质谱法

最准确的分子量测定方法：

  • MALDI-TOF MS：基质辅助激光解吸电离飞行时间质谱
  • ESI-MS：电喷雾电离质谱

4. 超速离心法

通过测定蛋白质的沉降系数计算分子量，适用于大分子蛋白质复合物。

四、蛋白质含量测定与纯度鉴定

1. 蛋白质含量测定方法

紫外吸收法（A280法）

蛋白质浓度 = A280 × 稀释倍数 / 消光系数

基于蛋白质中酪氨酸和色氨酸在280nm处的紫外吸收。快速简便，但受核酸干扰。

双缩脲法

蛋白质肽键与Cu²⁺在碱性条件下形成紫色络合物，在540nm处测定吸光度。

Lowry法

结合双缩脲反应和Folin试剂反应，灵敏度高，但操作复杂，易受干扰。

Bradford法

考马斯亮蓝G-250与蛋白质结合后最大吸收峰从465nm移至595nm。操作简便，干扰少，是最常用的方法之一。

BCA法

蛋白质在碱性条件下将Cu²⁺还原为Cu⁺，Cu⁺与BCA形成紫色络合物。兼容性好，重现性佳。

2. 蛋白质纯度鉴定

电泳分析

  • SDS-PAGE：检测蛋白质的均一性和分子量
  • Native-PAGE：检测天然状态下的蛋白质
  • 等电点聚焦：检测蛋白质的等电点均一性

色谱分析

  • HPLC：高分辨率分析蛋白质纯度
  • 毛细管电泳：高效分离分析

质谱分析

最准确的纯度鉴定方法，可检测微量杂质和蛋白质修饰。

3. 蛋白质活性检测

通过测定蛋白质的生物学活性评估其功能完整性：

  • 酶活性测定：对于酶蛋白
  • 结合活性测定：对于结合蛋白
  • 免疫活性测定：对于抗体蛋白

五、重要名词解释

等电点
(Isoelectric Point, pI)
蛋白质分子净电荷为零时溶液的pH值。在等电点时，蛋白质在电场中不发生迁移，溶解度最小，最容易沉淀析出。
盐析
(Salting Out)
在蛋白质溶液中加入大量中性盐（如硫酸铵），使蛋白质沉淀的方法。盐析是可逆过程，不破坏蛋白质的天然结构。
凝胶过滤层析
(Gel Filtration Chromatography)
又称分子筛层析，利用多孔凝胶珠的分子筛效应，根据分子大小分离蛋白质的方法。大分子先洗脱，小分子后洗脱。
离子交换层析
(Ion Exchange Chromatography)
利用蛋白质表面电荷的差异，使其与离子交换树脂发生可逆结合，然后通过改变pH或离子强度进行洗脱分离的方法。
SDS-PAGE
(SDS-Polyacrylamide Gel Electrophoresis)
十二烷基硫酸钠-聚丙烯酰胺凝胶电泳。SDS破坏蛋白质的二级和三级结构，使蛋白质主要根据分子量大小进行分离。
等电点聚焦
(Isoelectric Focusing, IEF)
在pH梯度凝胶中进行的电泳技术，蛋白质迁移至其等电点位置后停止移动，具有极高的分辨率。
亲和层析
(Affinity Chromatography)
利用蛋白质与特定配体（如底物、抑制剂、抗体等）的特异性结合进行分离纯化的高效方法。
疏水作用层析
(Hydrophobic Interaction Chromatography, HIC)
利用蛋白质表面疏水区域的差异进行分离的方法。在高盐浓度下结合，用低盐缓冲液洗脱。
Bradford法
(Bradford Assay)
利用考马斯亮蓝G-250与蛋白质结合后吸收峰移动的原理测定蛋白质浓度的方法，操作简便，干扰少。
质谱法
(Mass Spectrometry, MS)
通过测定蛋白质的质荷比精确确定分子量的方法，包括MALDI-TOF MS和ESI-MS等技术。

选择题测试

从30道题库中随机抽取15道题进行测试

深度思考题

1. 设计一个完整的蛋白质分离纯化方案，从大肠杆菌细胞中纯化一个重组蛋白质。请详细说明每个步骤的原理、操作要点和预期效果。

参考答案：
  1. 细胞破碎与粗提取：采用超声波破碎或高压匀浆破碎细胞壁，释放胞内蛋白质。离心去除细胞碎片，获得粗蛋白提取液。此步骤的目的是从复杂的细胞环境中初步分离目标蛋白质。
  2. 盐析初步分离：使用硫酸铵分级沉淀，通常在30-70%饱和度范围内收集目标蛋白质。盐析能够去除大部分杂蛋白，同时浓缩目标蛋白质，且不破坏蛋白质的天然结构。
  3. 离子交换层析：根据目标蛋白质的等电点选择合适的离子交换树脂。如果蛋白质pI>7，选择阳离子交换树脂；如果pI<7，选择阴离子交换树脂。通过盐梯度或pH梯度洗脱，可以有效分离电荷相近的蛋白质。
  4. 凝胶过滤层析：根据目标蛋白质的分子量选择合适孔径的凝胶介质，进一步去除分子量差异较大的杂质蛋白，同时实现缓冲液置换。
  5. 亲和层析精细纯化：如果目标蛋白质带有His标签，可使用镍柱亲和层析；如果是酶蛋白，可设计底物类似物亲和柱。亲和层析具有高度特异性，能够实现一步高纯度纯化。
  6. 纯度检测与活性验证：使用SDS-PAGE检测蛋白质纯度和分子量，通过活性测定验证蛋白质功能完整性，确保纯化过程未导致蛋白质失活。

2. 比较SDS-PAGE和Native-PAGE的原理、应用场景和各自的优缺点。在什么情况下应该选择哪种电泳方法？

参考答案：
  1. SDS-PAGE原理与特点：SDS（十二烷基硫酸钠）是一种强阴离子去垢剂，能够破坏蛋白质的二级和三级结构，使蛋白质完全变性并带上均匀的负电荷。在还原条件下（加入β-巯基乙醇或DTT），还能断裂二硫键。因此蛋白质主要根据分子量大小进行分离，分离效果好，重现性高。
  2. Native-PAGE原理与特点：在非变性条件下进行电泳，保持蛋白质的天然结构和电荷状态。蛋白质的迁移率取决于其天然电荷、分子量和形状的综合效应。能够保持蛋白质的生物活性，适用于蛋白质复合物的分析。
  3. 应用场景选择：当需要确定蛋白质亚基组成、分子量或检测蛋白质纯度时，选择SDS-PAGE；当需要保持蛋白质活性、分析蛋白质复合物或进行活性染色时，选择Native-PAGE。
  4. 优缺点比较：SDS-PAGE的优点是分离效果好、重现性高、操作标准化，缺点是破坏蛋白质结构和活性；Native-PAGE的优点是保持蛋白质天然状态和活性，缺点是分离效果相对较差、条件优化复杂。
  5. 实际应用考虑：在蛋白质纯化过程中，通常先用Native-PAGE监测活性蛋白质的纯化效果，再用SDS-PAGE确认分子量和纯度。对于多亚基蛋白质复合物的研究，两种方法常常结合使用以获得完整信息。

3. 分析影响蛋白质分离纯化效果的主要因素，并提出相应的优化策略。

参考答案：
  1. pH值的影响：pH影响蛋白质的电荷状态和稳定性。优化策略包括：选择蛋白质稳定的pH范围，在离子交换层析中合理设计pH梯度，避免在等电点附近操作以防沉淀。对于酸敏感蛋白质，应在中性或弱碱性条件下操作。
  2. 离子强度的控制：离子强度影响蛋白质的溶解度和层析分离效果。在离子交换层析中，起始缓冲液离子强度要低，洗脱时逐渐增加；在疏水层析中则相反。需要根据具体蛋白质特性优化盐浓度梯度。
  3. 温度条件：低温有利于保持蛋白质稳定性和活性，但会降低分离效率。通常在4°C下操作，对于热稳定蛋白质可适当提高温度以改善分离效果。需要在稳定性和效率之间找到平衡点。
  4. 蛋白酶抑制剂的使用：添加蛋白酶抑制剂（如PMSF、EDTA等）防止蛋白质降解。选择合适的抑制剂组合，注意抑制剂的稳定性和与后续纯化步骤的兼容性。
  5. 样品预处理：去除核酸、脂质等杂质，调节样品的pH和离子强度，必要时进行浓缩或稀释。预处理质量直接影响后续分离效果。
  6. 层析条件优化：选择合适的层析介质、柱床高度、流速和梯度条件。流速过快会降低分离效果，过慢会导致蛋白质扩散和活性损失。需要通过实验优化找到最佳条件。

4. 论述现代蛋白质组学研究中蛋白质鉴定技术的发展趋势和应用前景。

参考答案：
  1. 质谱技术的革命性发展：从传统的MALDI-TOF发展到高分辨率的Orbitrap和Q-TOF质谱仪，质量精度和分辨率大幅提升。串联质谱（MS/MS）技术使蛋白质序列鉴定更加准确，能够实现大规模蛋白质组的定量分析。
  2. 高通量分离技术：超高效液相色谱（UHPLC）与质谱联用，大大提高了分离效率和检测灵敏度。毛细管电泳-质谱联用技术在单细胞蛋白质组学中显示出巨大潜力。
  3. 生物信息学的深度整合：蛋白质数据库的不断完善和搜索算法的优化，使蛋白质鉴定的准确性和覆盖度显著提高。机器学习和人工智能在蛋白质结构预测和功能注释中发挥越来越重要的作用。
  4. 单细胞蛋白质组学：技术发展使得在单细胞水平进行蛋白质分析成为可能，为理解细胞异质性和疾病机制提供了新的研究手段。这将推动精准医学和个性化治疗的发展。
  5. 临床应用前景：蛋白质生物标志物的发现和验证在疾病诊断、预后评估和药物开发中具有重要价值。蛋白质组学技术在肿瘤、神经退行性疾病等重大疾病研究中展现出巨大潜力。
  6. 技术挑战与发展方向：需要进一步提高检测灵敏度和动态范围，发展更加温和高效的蛋白质分离技术，建立标准化的实验流程和数据分析方法。未来将朝着更高通量、更高精度、更加自动化的方向发展。
