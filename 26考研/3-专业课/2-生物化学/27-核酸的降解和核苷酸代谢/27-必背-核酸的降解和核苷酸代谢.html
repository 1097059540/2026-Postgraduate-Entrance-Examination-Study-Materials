国科大生命科学学院 细胞生物学核酸的降解和核苷酸代谢必背考点

一、核心名词解释（10个）

从头合成途径（De novo synthesis pathway）：利用磷酸核糖、氨基酸、一碳单位及CO₂等简单物质为原料合成嘌呤或嘧啶核苷酸的过程，是体内核苷酸合成的主要途径。

补救途径（Salvage pathway）：利用体内游离嘌呤、嘧啶碱基或核苷，经简单反应过程重新生成核苷酸的过程，又称重新利用途径，在脑、骨髓等组织中尤为重要。

PRPP（5-磷酸核糖-α-焦磷酸，5-Phosphoribosyl-α-pyrophosphate）：由5-磷酸核糖与ATP在磷酸核糖焦磷酸激酶催化下生成的活化形式，是嘌呤和嘧啶核苷酸从头合成的共同起始物质。

IMP（次黄嘌呤核苷酸，Inosine monophosphate）：嘌呤核苷酸从头合成的第一个产物，是AMP和GMP合成的共同前体，由PRPP经10步反应生成。

核糖核苷酸还原酶（Ribonucleotide reductase）：催化核糖核苷二磷酸（NDP）还原生成脱氧核糖核苷二磷酸（dNDP）的关键酶，是DNA合成的限速酶，受严格的变构调节。

胸苷酸合酶（Thymidylate synthase）：催化dUMP转化为dTMP的酶，需要N⁵,N¹⁰-甲烯四氢叶酸作为甲基供体，是DNA合成中胸腺嘧啶核苷酸生成的唯一途径。

黄嘌呤氧化酶（Xanthine oxidase）：催化次黄嘌呤氧化为黄嘌呤，进一步氧化为尿酸的酶，是嘌呤核苷酸分解代谢的关键酶，其活性异常与痛风症密切相关。

Lesch-Nyhan综合征（Lesch-Nyhan syndrome）：由HGPRT（次黄嘌呤-鸟嘌呤磷酸核糖转移酶）缺陷引起的X连锁隐性遗传病，导致严重痛风和神经系统异常，患者常有自残行为。

抗代谢物（Antimetabolite）：结构类似于天然代谢物但功能相反的化合物，如5-氟尿嘧啶、6-巯基嘌呤等，通过竞争性抑制相关酶活性发挥抗肿瘤作用。

一碳单位（One-carbon unit）：以四氢叶酸为载体的含一个碳原子的基团，包括甲基、甲酰基、甲烯基等，在嘌呤和胸腺嘧啶核苷酸合成中起重要作用。

二、必备核心知识点（5个）

【必备知识点1】嘌呤核苷酸从头合成的基本过程
1. 起始物质：5-磷酸核糖经ATP活化生成PRPP
2. 合成顺序：先合成IMP（10步反应），再分别转化为AMP和GMP
3. 原料来源：甘氨酸、天冬氨酸、谷氨酰胺、CO₂、一碳单位
4. 关键调节酶：PRPP酰胺转移酶（限速酶，受反馈抑制）
5. 能量平衡：AMP合成需要GTP，GMP合成需要ATP，实现浓度平衡

【必备知识点2】嘧啶核苷酸从头合成的特点
1. 合成策略：先合成嘧啶环，再与PRPP结合形成核苷酸
2. 关键步骤：氨基甲酰磷酸+天冬氨酸→氨基甲酰天冬氨酸→二氢乳清酸→乳清酸
3. 限速酶：天冬氨酸氨基甲酰转移酶（受CTP反馈抑制，ATP激活）
4. 产物转化：UMP→CMP（需CTP合成酶和谷氨酰胺）
5. 胞浆定位：主要在胞浆中进行（与嘌呤合成相同）

【必备知识点3】脱氧核糖核苷酸的生成机制
1. 还原底物：核糖核苷二磷酸（NDP）直接还原为dNDP
2. 关键酶：核糖核苷酸还原酶（含自由基，需硫氧还蛋白还原）
3. 变构调节：活性位点受ATP/dATP调节，特异性位点决定底物选择性
4. dTMP合成：dUMP经胸苷酸合酶催化，N⁵,N¹⁰-甲烯四氢叶酸提供甲基
5. 抗癌靶点：该酶是阿糖胞苷、羟基脲等抗癌药物的作用靶点

【必备知识点4】嘌呤核苷酸分解代谢与痛风症
1. 分解途径：AMP/GMP→次黄嘌呤→黄嘌呤→尿酸（终产物）
2. 关键酶：黄嘌呤氧化酶（催化后两步氧化反应）
3. 痛风成因：尿酸生成过多或排泄不足，导致尿酸盐结晶沉积
4. 治疗药物：别嘌呤醇（黄嘌呤氧化酶抑制剂）
5. 补救途径缺陷：HGPRT缺乏→从头合成亢进→Lesch-Nyhan综合征

【必备知识点5】核苷酸代谢的调节机制与抗代谢物
1. 反馈调节：终产物对限速酶的变构抑制（如AMP、GMP抑制PRPP酰胺转移酶）
2. 前馈激活：PRPP对嘌呤合成的正调节作用
3. 抗代谢物机制：5-氟尿嘧啶抑制胸苷酸合酶，6-巯基嘌呤抑制IMP转化
4. 叶酸拮抗剂：甲氨蝶呤抑制二氢叶酸还原酶，阻断一碳单位循环
5. 临床应用：抗代谢物广泛用于抗肿瘤治疗，利用肿瘤细胞DNA合成旺盛的特点

三、高频必考大题（2道）

【高频考题1】试述嘌呤核苷酸从头合成与补救途径的区别，并分析Lesch-Nyhan综合征的发病机制及临床特征。

参考答案：
1. 从头合成途径：以PRPP为起始物，利用氨基酸、CO₂、一碳单位等简单物质，经10步反应合成IMP，再转化为AMP和GMP；消耗大量ATP，是主要合成途径。
2. 补救途径：利用游离嘌呤碱基或核苷，通过HGPRT、APRT等酶催化，直接生成核苷酸；节约能量和原料，在脑、骨髓等组织中重要。
3. Lesch-Nyhan综合征发病机制：HGPRT基因缺陷（X连锁隐性遗传）→补救途径受阻→从头合成代偿性亢进→尿酸生成增加→严重痛风。
4. 临床特征：严重痛风、智力发育迟缓、肌张力异常、自残行为（咬手指、嘴唇）；别嘌呤醇只能缓解痛风，无法改善神经症状。

【高频考题2】分析抗代谢物在抗肿瘤治疗中的作用机制，举例说明5-氟尿嘧啶和甲氨蝶呤的具体作用靶点及临床意义。

参考答案：
1. 抗代谢物作用原理：结构类似天然代谢物，通过竞争性抑制关键酶活性，阻断DNA/RNA合成，选择性杀伤增殖旺盛的肿瘤细胞。
2. 5-氟尿嘧啶（5-FU）机制：转化为5-FdUMP后与胸苷酸合酶形成稳定三元复合物，阻断dUMP→dTMP转化，导致胸腺嘧啶核苷酸缺乏，DNA合成受阻。
3. 甲氨蝶呤（MTX）机制：竞争性抑制二氢叶酸还原酶，阻断四氢叶酸再生，使一碳单位循环中断，影响嘌呤合成和dTMP生成。
4. 临床意义：广泛用于白血病、结直肠癌等治疗；可联合亚叶酸钙（四氢叶酸）进行解救治疗，降低正常细胞毒性。
