糖异生和糖的其他代谢途径

考研生物化学复习笔记 | 专业课重点整理

一、糖异生作用概述

1.1 糖异生的定义与意义

糖异生作用（Gluconeogenesis）是指机体利用非糖物质（如乳酸、甘油、生糖氨基酸等）合成葡萄糖的代谢过程。2 这一过程对维持血糖恒定具有重要意义，特别是在禁食状态下，当肝糖原储备耗尽后，糖异生成为维持血糖水平的主要途径。

糖异生主要发生在肝脏和肾脏，这两个器官含有完整的糖异生酶系。在长时间饥饿状态下，肾脏产生的葡萄糖可占总糖异生量的20%。2

1.2 糖异生的关键酶

糖异生过程中有四个关键限速酶，它们催化糖酵解中三个不可逆反应的逆转：

  1. 丙酮酸羧化酶（Pyruvate Carboxylase, PC）：催化丙酮酸生成草酰乙酸，需要生物素和ATP，受乙酰辅酶A变构激活
  2. 磷酸烯醇式丙酮酸羧激酶（PEPCK）：催化草酰乙酸生成磷酸烯醇式丙酮酸，消耗GTP 2
  3. 果糖-1,6-二磷酸酶（FBPase）：催化果糖-1,6-二磷酸水解为果糖-6-磷酸，是糖异生的第二步限速酶 1
  4. 葡萄糖-6-磷酸酶（G6Pase）：催化葡萄糖-6-磷酸水解为葡萄糖，仅存在于肝脏和肾脏 2

1.3 糖异生的调节机制

胰岛素是糖异生最主要的抑制剂，而胰高血糖素是重要的促进剂。胰高血糖素通过以下机制促进糖异生：

  • 激活腺苷酸环化酶，产生cAMP
  • 通过PKA磷酸化并灭活丙酮酸激酶
  • 上调PEPCK的表达
  • 增加果糖-1,6-二磷酸酶活性，同时抑制PFK-1 2

二、乙醛酸循环

2.1 乙醛酸循环的定义与发现

乙醛酸循环（Glyoxylate Cycle）是柠檬酸循环的一种变体，由Hans Kornberg爵士和他的导师Hans Krebs（克雷布斯）于1957年在牛津大学发现。1 这是一种发生在植物、细菌、原生生物和真菌中的合成代谢途径。

2.2 乙醛酸循环的关键酶

乙醛酸循环使用与柠檬酸循环相关的8种酶中的5种，但有两个关键的特异性酶：

  1. 异柠檬酸裂解酶（Isocitrate Lyase, ICL）：将异柠檬酸分解为琥珀酸和乙醛酸 2
  2. 苹果酸合酶（Malate Synthase）：催化乙醛酸与乙酰CoA结合生成苹果酸 2

2.3 乙醛酸循环的生物学意义

乙醛酸循环的总反应式为：
2乙酰CoA + NAD+ + 2H₂O → 琥珀酸 + 2CoA + NADH + H+

该循环的重要意义包括：

  • 脂肪到糖的转化：在油料植物种子萌发期间，储藏的脂肪通过乙醛酸循环转化为糖类，为幼苗生长提供能量 2
  • 碳源利用：允许微生物在缺乏葡萄糖时利用乙酸等二碳化合物作为碳源 1
  • 代谢灵活性：提高生物体利用乙酰CoA的能力，为TCA循环提供四碳单位

三、寡糖和糖蛋白的生物合成

3.1 寡糖的生物合成

寡糖的生物合成主要通过糖基转移酶催化，将活化的糖基供体（如UDP-葡萄糖、GDP-甘露糖等）的糖基转移到受体分子上。这一过程在细胞膜系统中进行，特别是在内质网和高尔基体中。

3.2 糖蛋白的生物合成

糖蛋白的生物合成包括两种主要类型的糖基化：

  1. N-连接糖基化：糖链连接到Asn（天冬酰胺）残基上，在内质网中开始
  2. O-连接糖基化：糖链连接到Ser（丝氨酸）或Thr（苏氨酸）残基上，主要在高尔基体中进行

四、葡萄糖转运

4.1 葡萄糖转运蛋白（GLUT）

葡萄糖转运蛋白是一类跨膜蛋白，负责葡萄糖在细胞膜上的转运。不同类型的GLUT具有不同的组织分布和动力学特性：

  • GLUT1：广泛分布，Km值低，保证基础葡萄糖摄取
  • GLUT2：主要在肝脏和胰岛β细胞，Km值高，摄取速率与血糖浓度成正比 5
  • GLUT4：主要在肌肉和脂肪细胞，受胰岛素调节 5

名词解释库

糖异生作用(Gluconeogenesis)
机体利用非糖物质（乳酸、甘油、生糖氨基酸等）合成葡萄糖的代谢过程，主要发生在肝脏和肾脏，是维持血糖恒定的重要途径。
磷酸烯醇式丙酮酸羧激酶(Phosphoenolpyruvate Carboxykinase, PEPCK)
糖异生过程中的关键限速酶，催化草酰乙酸生成磷酸烯醇式丙酮酸，消耗GTP，受胰高血糖素正调节，受胰岛素负调节。
果糖-1,6-二磷酸酶(Fructose-1,6-bisphosphatase, FBPase)
糖异生途径中第二步的限速酶，催化果糖-1,6-二磷酸水解为果糖-6-磷酸，受AMP和果糖-2,6-二磷酸抑制，被ATP、柠檬酸激活。
乙醛酸循环(Glyoxylate Cycle)
柠檬酸循环的变体，存在于植物、细菌和真菌中，通过异柠檬酸裂解酶和苹果酸合酶绕过TCA循环的脱羧步骤，实现脂肪向糖的转化。
异柠檬酸裂解酶(Isocitrate Lyase, ICL)
乙醛酸循环的关键酶，将异柠檬酸分解为琥珀酸和乙醛酸，是植物和微生物特有的酶，动物细胞中不存在。
苹果酸合酶(Malate Synthase)
乙醛酸循环的另一关键酶，催化乙醛酸与乙酰CoA结合生成苹果酸，与异柠檬酸裂解酶共同实现二碳单位向四碳单位的转化。
乙醛酸循环体(Glyoxysome)
植物细胞中的特殊细胞器，含有乙醛酸循环的关键酶，主要存在于油料种子萌发期间，负责脂肪向糖的转化过程。
葡萄糖转运蛋白(Glucose Transporter, GLUT)
负责葡萄糖跨膜转运的载体蛋白家族，不同亚型具有不同的组织分布、动力学特性和调节机制。
N-连接糖基化(N-linked Glycosylation)
糖链通过N-糖苷键连接到蛋白质天冬酰胺残基上的翻译后修饰过程，在内质网中开始，对蛋白质折叠和功能具有重要意义。
O-连接糖基化(O-linked Glycosylation)
糖链通过O-糖苷键连接到蛋白质丝氨酸或苏氨酸残基上的翻译后修饰过程，主要在高尔基体中进行。

选择题测试

以下是从题库中随机抽取的15道选择题，请认真作答：

深度思考题

1. 分析糖异生和糖酵解的相互调节机制，说明机体如何在不同生理状态下精确调控这两个相反的代谢途径？

  1. 酶水平的相互调节：糖异生和糖酵解在三个关键步骤上使用不同的酶。糖异生使用PEPCK、FBPase和G6Pase，而糖酵解使用丙酮酸激酶、PFK-1和己糖激酶。这些酶受到相反的变构调节，如FBPase被ATP激活、被AMP抑制，而PFK-1则相反，确保两个途径不会同时高度活跃。
  2. 激素调节的精确性：胰岛素和胰高血糖素通过不同的信号转导途径实现精确调控。胰岛素激活糖酵解相关酶并抑制糖异生酶的表达，而胰高血糖素通过cAMP-PKA途径激活糖异生并抑制糖酵解。这种激素调节确保了血糖水平的精确维持。
  3. 代谢物的反馈调节：关键代谢物如果糖-2,6-二磷酸作为重要的调节分子，在饥饿状态下其浓度降低，从而激活FBPase并抑制PFK-1，促进糖异生。ATP/AMP比值的变化也直接影响两个途径的酶活性。
  4. 组织特异性调节：不同组织对糖异生和糖酵解的调节存在差异。肝脏主要负责糖异生以维持血糖，而肌肉主要进行糖酵解以产生ATP。这种组织特异性确保了整体代谢的协调性。

2. 乙醛酸循环在植物生命周期中的作用机制是什么？为什么动物细胞不具备这一代谢途径？

  1. 植物种子萌发的能量需求：油料植物种子在萌发初期缺乏光合作用能力，必须依靠储藏的脂肪提供能量和碳骨架。乙醛酸循环通过异柠檬酸裂解酶和苹果酸合酶绕过TCA循环的脱羧步骤，将脂肪酸β氧化产生的乙酰CoA转化为四碳化合物，进而合成糖类。
  2. 碳骨架的有效利用：乙醛酸循环的关键在于避免了CO₂的丢失，使得二碳单位能够有效转化为四碳单位。这对于需要大量碳水化合物合成细胞壁成分（如纤维素）的植物细胞至关重要。
  3. 动物细胞的代谢特点：动物细胞不具备乙醛酸循环主要因为：①动物可以通过摄食获得现成的糖类，不需要从脂肪大量合成糖；②动物的糖异生主要利用氨基酸、乳酸等三碳或四碳化合物，而非二碳单位；③动物细胞缺乏乙醛酸循环体这一特殊的细胞器结构。
  4. 进化适应性意义：乙醛酸循环的存在体现了植物对环境的适应性。种子萌发期间无法进行光合作用，必须依靠内源性储藏物质，乙醛酸循环提供了一种高效的脂肪-糖转化机制，确保幼苗在获得光照前能够正常生长。

3. 糖蛋白的N-连接和O-连接糖基化在细胞生物学功能上有何差异？这些差异如何影响蛋白质的功能？

  1. 结构和位置差异：N-连接糖基化发生在天冬酰胺残基上，糖链结构复杂且相对保守，主要在内质网开始并在高尔基体中修饰完成。O-连接糖基化发生在丝氨酸或苏氨酸残基上，糖链结构相对简单但多样性更大，主要在高尔基体中进行。
  2. 蛋白质折叠的影响：N-连接糖基化对蛋白质正确折叠至关重要，糖链可以作为分子伴侣帮助蛋白质折叠，并参与质量控制机制。错误折叠的糖蛋白会被内质网相关降解途径清除。O-连接糖基化主要影响蛋白质的稳定性和功能域的暴露。
  3. 细胞识别和信号转导：N-连接糖基化形成的复杂糖链结构是细胞表面受体识别的重要标志，参与细胞间相互作用、免疫识别等过程。O-连接糖基化则更多参与蛋白质-蛋白质相互作用的调节和信号转导的精细调控。
  4. 疾病相关性：N-连接糖基化缺陷常导致严重的遗传性疾病，如先天性糖基化缺陷综合征。O-连接糖基化异常则与肿瘤转移、炎症反应等病理过程密切相关。这些差异反映了两种糖基化修饰在维持细胞正常功能中的不同作用。

4. 在糖尿病患者中，糖异生途径的异常激活如何加剧高血糖症状？现有的治疗药物如何针对这一机制发挥作用？

  1. 胰岛素抵抗导致的糖异生失调：在2型糖尿病中，胰岛素抵抗使得胰岛素无法有效抑制肝脏糖异生，导致即使在血糖已经升高的情况下，肝脏仍然持续产生葡萄糖。同时，胰高血糖素的作用相对增强，进一步激活糖异生途径。
  2. 关键酶表达和活性的异常：糖尿病患者肝脏中PEPCK、FBPase和G6Pase的表达显著上调，这些酶的过度激活使得糖异生能力大大增强。特别是PEPCK作为限速酶，其表达增加直接导致糖异生通量的显著提升。
  3. 二甲双胍的多重作用机制：二甲双胍作为一线抗糖尿病药物，通过多种机制抑制肝脏糖异生：①激活AMPK，改善胰岛素敏感性；②抑制线粒体电子传递链复合物I，影响ATP生成；③直接抑制线粒体甘油磷酸脱氢酶，阻断甘油磷酸穿梭，提高NADH水平，从而抑制糖异生前体的氧化。
  4. 新型治疗靶点的开发：针对糖异生关键酶的特异性抑制剂正在开发中，如FBPase抑制剂已进入临床试验阶段。这些药物通过直接抑制糖异生限速酶，可以更精确地控制肝脏葡萄糖产生，为糖尿病治疗提供新的策略。
